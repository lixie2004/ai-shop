## 一、用户画像（菲律宾跨境电商核心消费者）

### 1.1 人口统计

| 维度 | 描述 |
|------|------|
| **年龄** | 18-35 岁（核心），35-45 岁（次核心） |
| **性别** | 女性占比 ~65%（美妆/时尚/母婴），男性 ~35%（电子/运动） |
| **收入** | 月收入 PHP 15,000-40,000（~$270-720），中层消费者 |
| **地域** | 马尼拉大都会 ~40%，省级城市 ~40%，偏远 ~20% |
| **语言** | Tagalog（日常）、English（电商/品牌消费） |

### 1.2 消费心理画像

| 特征 | 描述 |
|------|------|
| **社交驱动** | 社交媒体种草是首要购买触发点 |
| **价格敏感** | "value for money" 是最高频评价词 |
| **品牌意识觉醒** | 从白牌转向品牌，偏好 Korean/日本品牌 |
| **冲动购买** | TikTok Shop 驱动大量冲动消费 |
| **信任门槛** | 假货问题导致对新卖家信任低 |
| **颜值导向** | 产品包装/主图美学是转化关键 |

## 二、好评关键词分析

### 2.1 各品类高频好评词

| 品类 | Top 好评关键词 | 频次 |
|------|---------------|------|
| **美妆** | "有效的"(mabisa)、"美白"(pampaganda)、"便宜"(mura)、"正品"(original) | 极高 |
| **宠物用品** | "可爱"(cute)、"质量好"、"狗/猫喜欢"、"超值" | 高 |
| **时尚** | "合身"、"和图片一样"、"便宜"、"质量好" | 高 |
| **运动** | "舒服"、"适合"、"值得买"、"便宜" | 中-高 |
| **家居** | "实用"、"省空间"、"便宜"、"质感好" | 中-高 |

### 2.2 好评驱动因素排序

1. **价格合适** (35%) — 性价比是第一好评因子
2. **产品描述准确** (20%) — "和图片一致"很重要
3. **质量好** (18%) — 耐用性/效果
4. **物流快** (15%) — 超出预期的物流速度带来好评
5. **客服好** (12%) — 响应及时解决问题

## 三、差评痛点分析

### 3.1 各品类 Top 差评原因

| 品类 | Top 差评关键词 | 差评率 | 核心问题 |
|------|---------------|--------|---------|
| **美妆** | "假货"、"过敏"、"和描述不符" | 15-20% | 真伪 + 效果 |
| **时尚** | "尺码不准"、"颜色不同"、"面料差" | 15-18% | 品质 + 描述准确性 |
| **宠物** | "质量一般"、"物流坏了" | 10-15% | 物流 + 耐用性 |
| **电子** | "不工作"、"不耐用"、"虚标参数" | 12-18% | 质量 + 真伪 |
| **运动** | "质量一般"、"和图片不同" | 10-14% | 一致性 |
| **家居** | "太小"、"材质薄"、"安装困难" | 10-13% | 规格说明 |

### 3.2 差评驱动因素排序

1. **假货/非正品** (28%) — 最大痛点
2. **产品描述不符** (25%) — 图片/描述过度美化
3. **质量差** (22%) — 耐用性/用料
4. **物流问题** (15%) — 慢/丢/损
5. **客服差** (10%) — 不回应/不解决

## 五、用户购买旅程

```
社交媒体种草 (TikTok/Facebook)
    ↓
搜索验证 (Shopee/Google)
    ↓
比价 (跨平台)
    ↓
评价检查 (Review + 评分)
    ↓
下单 (大促期间冲动购买概率高)
    ↓
物流跟踪 (焦虑期)
    ↓
开箱体验 (第一印象决定 Review)
    ↓
使用和 Review (1-7天内)
```

### 关键转化节点

| 节点 | 影响因素 | 卖家优化建议 |
|------|---------|-------------|
| 种草→搜索 | 视频内容质量 | KOL 合作 + 直播 |
| 搜索→比价 | 主图 + 标题 + 评分 | 优化 listing SEO |
| 比价→下单 | 价格 + 评价 + 认证 | 正品标识 + 合理定价 |
| 物流→开箱 | 包装 + 惊喜感 | 精美包装 + 小样 |
| 开箱→Review | 产品质量 + 描述一致 | QC + 准确描述 |
