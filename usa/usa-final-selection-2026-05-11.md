## 二、TOP 3 风险项与缓解方案

### 🔴 风险 #1：LED美容面罩 — FDA Class II 510(k) 认证

| 维度 | 详情 |
|------|------|
| **风险描述** | FDA Class II 510(k) 认证周期 6-9 个月，费用 $35K-$100K，是全部选品中最长的先导时间 |
| **影响** | 无法在认证完成前上架销售，延迟上市时间 |
| **缓解方案** | ① 立即启动 FDA 510(k) 预提交（Pre-Sub）流程 ② 聘请有 FDA 经验的合规顾问 ③ 先以"美容工具"（非医疗器械）定位测试市场反应 ④ 认证期间完成供应链搭建和内容储备 |
| **责任人** | 合规经理 + 外部 FDA 顾问 |
| **启动时间** | 立即 |

### 🔴 风险 #2：适应原补充剂 — FDA 膳食补充剂注册 + cGMP

| 维度 | 详情 |
|------|------|
| **风险描述** | FDA 膳食补充剂注册 + cGMP 合规需 3-6 个月，费用 $15K-$50K |
| **影响** | 无法在合规完成前上架，且标签声明需严格审核 |
| **缓解方案** | ① 选择有 cGMP 认证的代工厂 ② 提前准备标签文案（避免治疗/预防疾病声明） ③ 进行第三方成分检测 ④ 注册 FDA 膳食补充剂设施 |
| **责任人** | 合规经理 + 供应链经理 |
| **启动时间** | 立即 |

### 🔴 风险 #3：关税政策不确定性

| 维度 | 详情 |
|------|------|
| **风险描述** | 美国对华关税政策持续变化，可能影响 7 个低风险 SKU 的利润率 |
| **影响** | 关税从 25% 升至更高将直接压缩 5-15% 的利润空间 |
| **缓解方案** | ① 预留 5% 关税缓冲金 ② 考虑越南/墨西哥中转仓方案 ③ 高价值品类（#2 #9）优先走海外仓模式 ④ 持续关注政策动态，建立快速响应机制 |
| **责任人** | 财务分析师 + 供应链经理 |
| **启动时间** | 持续监控 |
