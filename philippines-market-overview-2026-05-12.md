# 菲律宾跨境电商市场宏观分析简报

**报告日期**：2026年5月12日  
**分析师**：趋势研究员  
**报告类型**：市场概览

---

## 执行摘要

菲律宾是**东南亚增长最快的数字经济体**，2025年电商GMV预计达$28-36亿（全口径数字 economy GMV达$36B），年增速15-18%。市场呈现**三极格局**：Shopee以55%的GMV份额主导，TikTok Shop以29%快速崛起（+53% YoY），Lazada份额萎缩至16%。消费者高度移动化（78%交易来自手机）、社交化（日均社交媒体使用3.5小时+）和直播购物驱动。跨境卖家面临物流碎片化（7,641岛屿）、COD占比仍高（45%）等挑战，但数字支付渗透率快速提升（64%）、政府政策积极、RCEP框架下关税优惠，为跨境电商提供了结构性机会窗口。

### 趋势分析

菲律宾电商市场处于**高速增长期**，增速显著高于东南亚平均水平（约10-12%）。关键驱动力：

1. **人口红利**：1.1亿人口，中位年龄25岁，数字化原住民占比高
2. **互联网渗透率提升**：8600万+互联网用户，智能手机渗透率超70%
3. **数字支付爆发**：GCash等电子钱包用户超7300万
4. **疫情后行为固化**：线上购物习惯已不可逆

**与东南亚其他市场对比**：
- 规模：东南亚第四（仅次于印尼、泰国、越南），已超越越南重回第三
- 增速：东南亚最快之一（15-18% vs 区域平均10-12%）
- 渗透率：仍低于印尼和越南，意味着更大增长空间

### 消费行为特征

#### 1. 移动优先（Mobile-First）
- **78%**的电商交易来自移动设备
- Shopee在菲律宾的会话绝大多数来自手机端
- **启示**：产品图片、详情页、结账流程必须为移动端优化，桌面优先设计会直接损失转化率

#### 2. 社交购物（Social Commerce）
- 菲律宾人日均社交媒体使用时间**3小时32分钟**，全球最高之一
- **44%**的购物者依赖网红推荐做购买决策
- 社交电商收入预计2026年达**$2.3-2.8B**，2030年达$4.17B
- Facebook仍是最主要社交平台，但TikTok增长最快
- **74%**的本地企业将直播带货视为关键获客策略

#### 3. 直播购物（Live Selling）
- 菲律宾是**东南亚直播购物渗透率最高**的市场之一
- TikTok Shop美妆品类28%的GMV占比直接证明了直播带货的转化效率
- 直播购物的核心驱动力：**真实性 + 紧迫感（FOMO）+ 互动性**
- **启示**：美妆、时尚、食品类卖家必须建立直播能力，这不是"加分项"而是"必选项"

#### 4. 促销驱动型购物
- 菲律宾消费者极度热衷促销：**9.9、11.11、12.12**等大促是全年GMV的关键节点
- Shopee Q4（旺季）占全年GMV的**31%**
- 消费者习惯跨平台比价寻找最优deal
- **启示**：定价策略需预留大促空间，日常价格锚定要合理

#### 5. 信任与风险意识
- **45%**的消费者在大额购买（PHP 2,000+）时仍偏好货到付款（COD）
- 产品评价和社交证明对购买决策影响巨大
- 对陌生品牌的信任建立需要时间

### 平台深度分析

#### 1. Shopee — 不可撼动的领导者

**核心优势**：
- **购买频率最高**：菲律宾用户年均**11次**下单，东南亚最高
- **习惯性消费**：菲律宾消费者默认在Shopee购物，大促期间尤为明显
- **品类广度**：时尚(27%) + 电子(24%) + 美妆(14%) + 健康(8%)
- **物流整合**：2026年初起强制使用Shopee自有物流网络（J&T Express被移除）
- **即时配送**：2026年2月推出Instant Mart，利用ShopeeFood基础设施实现**4小时内送达**

**对跨境卖家的意义**：
- 仍是进入菲律宾市场的**第一优先级平台**
- 但竞争激烈，需要差异化定位
- 物流必须适配Shopee自有网络

#### 2. TikTok Shop — 颠覆性挑战者

**核心优势**：
- **增速最快**：FY2025 GMV $6B（+53% YoY），且增速在加快
- **Q4超越Lazada**：2025年Q4 TikTok Shop GMV占三大平台合计的35%，成为第二大平台
- **美妆品类王者**：美妆占其GMV的28%，是所有平台中美妆占比最高的
- **内容驱动**：算法推荐+直播带货=高转化率
- **低AOV但高频率**：AOV仅PHP 198，但冲动购买频次高

**对跨境卖家的意义**：
- **必须布局**的增量渠道
- 适合美妆、时尚、食品等**可演示性强**的品类
- 需要投入内容创作和直播运营能力
- 早期红利仍在，但窗口期可能有限

#### 3. Lazada — 战略性收缩

**现状**：
- GMV从FY2024的~$4.5B降至FY2025的$3B（-34%）
- Q4旺季仅占三大平台合计GMV的10%
- 但AOV最高（PHP 416），聚焦高客单价品牌商品

**战略定位**：
- 退守**品牌正品**和**高客单价电子**品类
- LazMall正品保证是核心差异化
- 电子和家电占其GMV的26%，高于Shopee的24%和TikTok Shop的12%

**对跨境卖家的意义**：
- 适合**品牌型卖家**和**高客单价电子产品**
- 不应作为主要增长渠道
- 可作为品牌正品形象的补充展示窗口

## 四、跨境电商基础设施

### 1. 物流

| 指标 | 数值 | 来源 |
|------|------|------|
| **同日达覆盖城市** | 15个（2024年为8个） | CloudEcommerce |
| **城市快递覆盖率** | 85%城区24小时内 | CloudEcommerce |
| **农村配送时效** | 3-4天（2024年为5-7天） | CloudEcommerce |
| **LBC Express网点** | 2,800+ | CloudEcommerce |
| **2GO Express覆盖率** | 95%的市镇 | CloudEcommerce |
| **微型履约中心** | Ninja Van/J&T在45个城市 | CloudEcommerce |
| **自动化履约容量增长** | +40% | CloudEcommerce |
| **国际清关时效** | 2天（从5天改善） | CloudEcommerce |

#### 主要物流服务商

| 服务商 | 类型 | 特点 |
|--------|------|------|
| **Shopee自有物流** | 平台物流 | 2026年起强制使用，覆盖最广 |
| **LBC Express** | 传统物流 | 2,800+网点，最广泛的线下网络 |
| **J&T Express** | 电商快递 | 性价比高，但在Shopee上被限制 |
| **Ninja Van** | 电商快递 | 45个微型履约中心 |
| **2GO Express** | 综合物流 | 覆盖95%市镇，海运+空运 |
| **GoGoX / Lalamove** | 即时配送 | 马尼拉/宿务/达沃同日达 |

#### 跨境物流建议

1. **头程**：中国→菲律宾，建议走深圳/广州直飞马尼拉的海运或空运
2. **清关**：利用NAIA和Clark机场的跨境清关设施（已缩短至2天）
3. **海外仓**：在Laguna/Cavite/Bulacan的自动化履约中心设仓，覆盖马尼拉大都会
4. **区域仓**：在宿务和达沃设区域仓，覆盖维萨亚斯和棉兰老岛
5. **COD处理**：必须支持COD的物流合作伙伴（LBC最佳）

### 2. 支付

#### 支付方式分布

| 支付方式 | 使用率 | 趋势 | 来源 |
|----------|--------|------|------|
| **数字钱包（GCash/Maya等）** | **64%** | ↑ 快速增长 | CloudEcommerce |
| **货到付款（COD）** | **45%**（大额购买） | ↓ 缓慢下降 | CloudEcommerce |
| **信用卡/借记卡** | **25%** | ↑ 稳步增长 | CloudEcommerce |
| **银行转账** | **15%** | ↔ 稳定 | CloudEcommerce |
| **ShopeePay** | ~10% | ↑ 增长中 | Cube.asia |
| **SPayLater（先买后付）** | 达到"teen"占比 | ↑ 快速增长 | Cube.asia |

> 注：因消费者可使用多种支付方式，各方式使用率之和超过100%。

#### 主要数字钱包

| 钱包 | 用户数 | 特点 |
|------|--------|------|
| **GCash** | 6500万+ | 市场领导者，生态最完整（储蓄+贷款+保险+投资） |
| **Maya（原PayMaya）** | 4500万+ | 增长最快，数字银行牌照，信用卡功能 |
| **ShopeePay** | — | Shopee生态内，SPayLater先买后付 |
| **GrabPay** | — | Grab生态内 |
| **Coins.ph** | — | 加密货币友好 |

#### 跨境卖家支付建议

1. **必须接入**：GCash + Maya（覆盖80%+数字支付用户）
2. **COD不可忽略**：45%的大额购买仍用COD，不提供COD等于放弃近半市场
3. **BNPL/SPayLater**：接入先买后付可提升转化率，尤其对$50+商品
4. **信用卡**：虽然仅25%，但高客单价用户偏好信用卡，必须支持
5. **支付安全**：BSP（菲律宾央行）对数字钱包有严格监管，选择合规支付服务商

## 五、政策环境与贸易协定

### 关键政策法规

| 政策/法规 | 内容 | 对跨境电商的影响 |
|-----------|------|-----------------|
| **Konektadong Pinoy法案** | 促进数字基础设施建设，降低互联网成本 | 正面：扩大互联网覆盖，增加潜在消费者 |
| **电子治理法案（E-Governance Act）** | 数字化公共服务 | 正面：推动全民数字化，间接促进电商 |
| **BSP数字支付监管框架** | 规范GCash/Maya等数字钱包 | 中性：提高消费者信任，但增加合规要求 |
| **消费者保护法（修订）** | 加强在线消费者权益保护 | 中性：需关注退货政策和消费者争议处理 |
| **数据隐私法（DPA）** | 个人数据保护 | 需合规：跨境数据传输需注意 |
| **跨境电商税收政策** | 进口关税+增值税 | 需关注：不同品类税率不同 |

### 贸易协定

| 协定 | 内容 | 对跨境电商的影响 |
|------|------|-----------------|
| **RCEP（区域全面经济伙伴关系）** | 中国与菲律宾同属RCEP成员 | **重大利好**：关税减免、贸易便利化 |
| **中国-东盟FTA升级版** | 进一步降低关税 | 利好：多数商品关税已降至0-5% |
| **ASEAN电商协议** | 区域内电商便利化 | 利好：简化跨境电商流程 |
| **WTO信息技术协定** | 信息技术产品零关税 | 利好：电子产品出口关税优惠 |

### 关税与税收

| 品类 | 进口关税 | 增值税 | 备注 |
|------|---------|--------|------|
| 电子产品 | 0-5% | 12% | 多数电子产品ITA零关税 |
| 服装/纺织品 | 5-15% | 12% | 取决于具体品类 |
| 美妆护肤品 | 3-10% | 12% | 需FDA认证 |
| 食品/保健品 | 5-20% | 12% | 需FDA/BFAD认证 |
| 家居用品 | 5-10% | 12% | — |

### 合规要求

1. **FDA认证**：美妆、护肤、食品、保健品需菲律宾FDA注册
2. **BOC清关**：通过菲律宾海关局（Bureau of Customs）正规清关
3. **BIR税务登记**：在菲律宾税务局（BIR）登记，缴纳增值税
4. **DPA合规**：消费者数据收集和处理需遵守数据隐私法
5. **产品标签**：需有英文或菲律宾语标签，含成分、原产地等信息

## 六、2025-2026年市场趋势预测

### 趋势一：社交电商成为主流渠道

**预测**：社交电商GMV将从2025年的~$2.3B增长至2026年的~$2.8B（+17-20%），2030年达$4.17B。

**驱动因素**：
- 菲律宾人日均社交媒体使用3.5小时+，全球最高
- TikTok Shop在菲律宾的GMV增速（+53% YoY）远超传统电商
- 直播带货转化率远高于静态商品页
- 44%消费者依赖网红推荐

**跨境卖家行动**：建立TikTok Shop运营能力，培养或合作菲律宾本地KOL，投资直播基础设施。

### 趋势二：AI驱动的个性化购物

**预测**：AI推荐引擎将驱动35%以上的产品发现，AI交互偏好率达92%。

**驱动因素**：
- Lazada等平台AI交互在12.12大促中达92%偏好率
- 78%消费者会放弃无法提供个性化体验的品牌
- AI视觉搜索、智能客服、个性化推荐快速普及

**跨境卖家行动**：优化产品数据（标签、描述、图片）以适应AI推荐算法，接入AI客服工具。

### 趋势三：即时配送与快速商务

**预测**：同日达/即时配送将成为马尼拉标配，覆盖城市从15个扩展至25+。

**驱动因素**：
- Shopee Instant Mart（4小时达）2026年2月上线
- 自动化履约容量+40%
- 消费者期望持续提升

**跨境卖家行动**：在Laguna/Cavite/Bulacan设海外仓，接入Shopee即时配送网络。

### 趋势四：数字支付全面普及

**预测**：数字支付占比将从64%提升至70%+，COD降至35%以下。

**驱动因素**：
- GCash 6500万+用户，Maya 4500万+用户
- SPayLater等BNPL快速增长
- BSP推动金融普惠

**跨境卖家行动**：全面接入GCash+Maya+SPayLater，优化数字支付结账体验。

### 趋势五：AR/VR购物体验升级

**预测**：AR试穿/试用将成为美妆和时尚品类标配，AR交互可提升转化率94%。

**驱动因素**：
- 5G用户占比9%+，持续增长
- AR/VR市场收入预计2024年达$1.82亿
- Shopee/Lazada/TikTok均已上线AR功能

**跨境卖家行动**：为美妆/时尚产品创建3D模型和AR试用功能。

### 趋势六：品类结构变化

**预测品类增长排名**：

| 品类 | 预测增速 | 驱动因素 |
|------|---------|---------|
| 健康保健 | **+45%** | 健康意识提升、远程医疗整合 |
| 生鲜/杂货 | **+30%+** | 快速消费习惯养成、即时配送 |
| 美妆护肤 | **+20%** | 直播带货、KOL推荐 |
| 时尚服装 | **+15%** | 社交电商驱动 |
| 电子产品 | **+12%** | 5G换机、智能设备 |
| 家居生活 | **+18%** | 居家趋势延续、改善需求 |

### 趋势七：跨境竞争加剧

**预测**：Temu等超低价平台将继续施压，但品牌化卖家将获得溢价空间。

**驱动因素**：
- Temu 2023年底进入菲律宾，快速崛起
- 菲律宾消费者对价格敏感度极高
- 但Dentsu报告显示消费者越来越"价值驱动"而非纯"价格驱动"

**跨境卖家行动**：避免纯低价竞争，通过品牌故事、产品质量、服务体验建立差异化。

## 七、跨境卖家战略建议

### 市场进入策略

| 阶段 | 时间 | 行动 | 目标 |
|------|------|------|------|
| **验证期** | 0-3月 | Shopee开店+TikTok Shop测试 | 月销$5K-10K验证PMF |
| **扩张期** | 3-6月 | 加大优胜品类投入+海外仓 | 月销$30K-50K |
| **品牌期** | 6-12月 | 独立站+品牌内容+多渠道 | 月销$100K+ |

### 平台优先级

1. **Shopee**（必选）：最大流量和GMV，第一优先级
2. **TikTok Shop**（必选）：增速最快，美妆/时尚品类必布局
3. **Lazada**（可选）：品牌正品定位，高客单价品类补充
4. **独立站**（中期）：Shopify建站，长期品牌建设

### 品类推荐

| 优先级 | 品类 | 理由 |
|--------|------|------|
| **S级** | 美妆护肤 | TikTok Shop核心品类，直播带货高转化，中国供应链优势 |
| **S级** | 时尚服装 | 最大品类，社交电商驱动，供应链优势 |
| **A级** | 健康保健 | +45%增速，高复购，蓝海 |
| **A级** | 家居生活 | 增长稳定，差异化空间大 |
| **A级** | 消费电子 | 高客单价，Lazada品牌正品定位 |
| **B级** | 母婴用品 | 增长稳定但合规要求高 |
| **B级** | 食品/饮料 | 需FDA认证，物流要求高 |

### 关键成功因素

1. **本地化**：产品描述、客服、营销内容需有菲律宾英语/他加禄语版本
2. **COD管理**：必须支持COD，同时通过数字支付引导降低COD比例
3. **物流时效**：海外仓是竞争力的关键，纯直邮模式难以满足时效期望
4. **内容能力**：TikTok/Instagram内容创作能力是差异化核心
5. **合规先行**：FDA认证、BOC清关、BIR税务登记必须提前规划
