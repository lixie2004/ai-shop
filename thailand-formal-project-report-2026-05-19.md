## 1. 执行摘要

### 核心结论

| 维度 | 结论 |
|------|------|
| **市场吸引力** | ⭐⭐⭐⭐ 强烈推荐进入 |
| **首选品类** | 宠物用品（蓝海）> 美妆个护（利润）> 家居收纳（低竞争） |
| **首选平台** | Shopee（最大用户基）> TikTok Shop（最高增速）> Lazada（品牌友好） |
| **预计月销售额** | THB 500,000-1,000,000（US$14,000-28,000） |
| **预计综合毛利率** | 35-40% |
| **最大风险** | 2026年免税门槛取消 + FDA认证周期 |

### 关键数据

- 泰国2024年电商GMV：**US$21.83bn**，增速11.5%
- 含社交电商2025年GMV：**US$35.5bn**，增速51.8%
- 网购渗透率：**85%**，约4350万网购消费者
- 跨境购物占比：**30%**
- 移动端购物：**>80%**
