## 一、方法论与成本模型框架

### 1.1 成本计算模型

```
总成本率 = 采购成本率 + 头程物流率 + 关税率 + 平台佣金率 + 尾程配送率 + 退货损耗率 + 汇率波动风险率

毛利率 = 1 - 总成本率

盈亏平衡销量（月）= 固定成本 / (零售价 × 毛利率)
```

### 1.2 费率基准（2026年6月）

| 成本项 | 依据说明 | 基准参数 |
|--------|---------|---------|
| **头程物流（空运）** | 中国→马来西亚 空运（5-8天） | **¥18-28/kg**（约 MYR 11-17/kg） |
| **头程物流（海运）** | 中国→马来西亚 海运拼箱（15-20天） | **¥6-12/kg**（约 MYR 3.7-7.3/kg） |
| **头程物流（快递）** | 国际快递DHL/FedEx（3-5天） | **¥35-55/kg**（约 MYR 21-34/kg） |
| **进口关税** | 马来西亚 Customs Duties Order 2025 | 0-30% 视品类（SST另加） |
| **SST销售税** | 马来西亚 Sales & Service Tax | 5% 或 10%（依品类） |
| **Shopee佣金** | Shopee MY Marketplace 2026年2月更新 | **3-12% + 5% PSF = 8-17%** |
| **Lazada佣金** | Lazada MY 2025年7月更新 | **4-22.5%**（含All-In） |
| **TikTok Shop佣金** | TikTok Shop MY 2025年9月更新 | **2-8% + 1.8% 交易费 + 10-20% 达人佣金** |
| **尾程配送（SLS）** | Shopee Logistics Service 西马 | **MYR 4-12/首kg** |
| **尾程配送（本地仓）** | 第三方仓储代发 | **MYR 3-8/单** |

### 1.3 重要说明

- 本报告基于**跨境直邮模式 + 本土仓中转模式**两种场景建模
- 平台佣金取**Shopee Marketplace标准费率**为基准，因Shopee占马来西亚电商份额~55%
- 汇率波动风险率取历史年度波动区间 3-5%
- 退货损耗率基于竞品报告中各品类差评率和行业平均退货率估算
- 所有金额以**马来西亚令吉（MYR）**为单位，除非特别标注

| **分批结汇** | 分批次结汇，降低单点风险 | 中小卖家 |
| **本地化比例提升** | 逐步将部分采购/生产转到马来西亚本地 | 规模化后（月销>1,000件） |
| **定价动态调整** | 每季度根据汇率调整零售价（±5%范围内）| 所有卖家 |
