# Malaysia 跨境电商品类机会矩阵（2026‑05‑25）

**数据来源**

- GMV 2026 预测 $2.18 bn，CAGR 13.67%（Mordor Intelligence）【1】
- 平台格局：Shopee 55‑60%，Lazada 25‑30%，TikTok Shop 8‑12%（增长最快）【2】
- 热门品类占比及增长：时尚服装 26.67%，电子/数码 31%，食品饮料 CAGR 16.46%，健康美妆 CAGR 12‑14%（Mordor Intelligence）【3】
- 消费者特征：移动端 72.67%，数字钱包 34.92%，清真需求强、促销敏感（IPSOS 2025）【4】
- 跨境来源：中国占 70%（行业调研）【5】

## 1. 评分模型（1‑10）

| 维度 | 说明 |
|------|------|
| **市场潜力** | 结合品类规模、增长率与平台渗透度。 |
| **竞争度**（逆向） | 竞争越低得分越高。 |
| **跨境适配度** | 中国供应链匹配度、易于跨境采购。 |
| **利润率预期** | 毛利空间、定价弹性。 |
| **合规风险**（逆向） | 合规难度越低得分越高（如清真、药监、商品安全等）。 |
