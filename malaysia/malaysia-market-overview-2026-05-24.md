## 4. 跨境进口政策 & 趋势

| 政策 | 内容 | 影响 |
|------|------|------|
| **进口关税** | 大宗商品平均 **6.5 %** | 相对低，鼓励进口 |
| **SST (2025 改革)** | 7 月生效，新税率 5‑10 % (按品类) | 增加跨境成本，尤其服饰、化妆品 |
| **免值门槛** | 无低值免税 | 所有进口需清关 |
| **海关预清** | 关税 + 清关周期 缩短 30 % | 缓解物流痛点 |
| **外资持股** | 100 % 允许 | 便利外资企业设立 |
| **数据本地化** | 必须在本土存储 | 合规成本略升 |
| **主要采购来源** | 中国 (最大)、新加坡、日韩、美洲 | 中国商品占比 40‑50 %，占跨境总 GMV 的 70 %+ |
| **政策趋势** | 增加税收、完善海关数字化、支持跨境电商平台合规 | 需要关注 SST、物流成本和清真、药品监管 |

**总结**：马来西亚跨境电商监管环境总体友好，但 SST 改革将提升电商额外税负，尤其是化妆品、服饰类。海关数字化提升清关效率，对时效敏感型卖家有利。中国商品依旧占据主导地位。

## 5. 热门品类趋势

| 主要品类 | 线上份额 | CAGR | 关键趋势 |
|----------|----------|------|-----------|
| **电子/数码** | **~31 %** | 4‑6 % | 热门配件、5G 设备、可穿戴 |
| **时尚服装** | **~30 %** | 8‑10 % | 穆斯林时尚、可持续面料、正品企划 |
| **健康美妆** | 15‑18 % | 12‑14 % | 清真认证、美妆有机、护肤功能固化 |
| **食品饮品** | 10‑12 % | **16.5 %** | 快速消费品进口、健康食品、零食跨境 | 
| **家居生活** | 8‑10 % | 5‑7 % | 智能家居、环保家具 |
| **母婴/玩具** | 12 % | 7‑9 % | 安全认证、品牌化、亲子产品 |

**机会**：
- *电子配件* 受 5G 需求推动，制造链完善。
- *穆斯林时尚* 结合清真认证可形成差异化。
- *清真美妆* 与本土需求高度契合。
- *食品饮品* 近年增速最快，适合快速试水。

**风险**：高端化妆品与医药类仍受监管严格；物流成本在东马不平衡，可能影响成本管控。
