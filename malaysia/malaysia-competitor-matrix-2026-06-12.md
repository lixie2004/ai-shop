## 二、马来西亚市场背景

### 2.1 电商宏观环境

| 指标 | 数值 |
|------|------|
| 人口 | 3,580万（2025） |
| 互联网渗透率 | 97.7% |
| 社交媒体用户 | 2,510万（70.2%） |
| 社交电商购买率 | 68% |
| 电商市场规模 | ~USD 120亿（2025） |
| 平台份额 | Shopee ~55% / Lazada ~25% / TikTok Shop ~20% |
| 穆斯林人口占比 | ~63%（JAKIM认证是信任核心） |
| 人均GDP | ~USD 12,500 |
| COD比例 | ~15% |

### 2.2 市场关键特征

- **Halal 认证 = 信任基石**：JAKIM认证已从宗教要求演变为全民消费信任符号，无Halal认证产品搜索转化率下降40%+
- **三语并行**：马来语+英语+中文是标配，缺一不可
- **热带气候适配**：从美妆到食品到家具，不适合热带气候是被吐槽最多的问题
- **社交电商领导者**：68%用户通过社交内容触达产品，TikTok（1,850万用户）是增长最快渠道
- **认证双保险**：JAKIM（Halal）+ NPRA（保健品）+ SFA（食品）是消费者"信任三角"
