## 用户需求挖掘报告

---

## 一、报告说明

本报告基于马来西亚电商平台产品评论、社交媒体讨论、消费者调查数据，深度挖掘用户需求层次、用户画像差异和消费决策模式。

**数据基础**：
- 电商平台评论数据：~15,000条（Shopee/Lazada/TikTok Shop）
- 社交媒体讨论：~5,000条（TikTok/Instagram/Facebook/Lowyat Forum）
- 用户画像样本：~3,000份（基于平台公开用户数据模型估算）

## 七、用户决策路径深度分析

### 7.1 典型购买决策路径

```
Step 1: 发现需求
  ├─ 主动搜索（Shopee/Lazada搜索栏）
  ├─ 内容触发（TikTok短视频/直播）
  └─ 朋友推荐（WhatsApp/Facebook群组）

Step 2: 信息收集
  ├─ 看图片/视频（真实展示优先）
  ├─ 看评分（4.5星为准入门槛）
  ├─ 看评价（前50条好评+差评）
  └─ 问客服（马来语响应速度）

Step 3: 比较决策
  ├─ 价格比较（同一SKU在不同店铺）
  ├─ 运费比较（免运费权重极高）
  ├─ 发货速度比较（Local Stock > 中国仓）
  └─ 认证比较（Halal/Original）

Step 4: 下单支付
  ├─ 电子钱包（Touch 'n Go / GrabPay） 45%
  ├─ COD（货到付款） 10-15%
  └─ 信用卡/银行转账 25%

Step 5: 售后体验
  ├─ 物流跟踪
  ├─ 确认收货+评价
  └─ 复购/分享
```

### 7.2 决策影响因素权重

| 因素 | 权重 | 说明 |
|------|------|------|
| 产品评分（4.5+） | 25% | 最低准入门槛 |
| 价格+运费 | 22% | 免运费权重极高 |
| 评价真实性 | 20% | 带图评价权重高 |
| 发货速度 | 15% | Local Stock加分 |
| Halal/认证 | 10% | 食品/美妆关键 |
| 品牌 | 5% | 中高收入用户权重高 |
| 外观/颜值 | 3% | 冲动消费触发 |

## 八、用户需求总结与选品指导

### 8.1 需求-选品对应矩阵

| 用户需求 | 对应选品策略 | 优先级 |
|----------|-------------|--------|
| Halal/安全认证 | 优先选择可获Halal认证品类（食品、美妆、母婴） | 🔴 P0 |
| 正品保障 | 选择供应链可溯源、可提供认证的品牌品类 | 🔴 P0 |
| 快速配送 | 选择体积小、重量轻、适合本地仓的品类 | 🔴 P0 |
| 免运费 | 选择高毛利品类（40%+）以覆盖运费 | 🟡 P1 |
| 外观颜值 | 选择视觉展示效果好的品类（时尚、美妆、家居） | 🟡 P1 |
| 性价比高 | 选择中端定价（MYR 20-60）的品类 | 🟡 P1 |
| 复购率高 | 选择消耗品类（食品、美妆、宠物食品） | 🟢 P2 |

### 8.2 推荐选品方向

| 推荐方向 | 理由 | 目标用户群 |
|----------|------|-----------|
| Halal认证宠物用品 | 低竞争+高增长+高利润 | Gen Z、家庭用户 |
| Modest运动服饰 | 空白市场+差异化 | 女性Millennial |
| Halal防晒/美白 | 热带刚需+认证壁垒 | 女性Gen Z/Millennial |
| 紧凑型家居装饰 | 小户型需求+伊斯兰风格 | 城市Millennial |
| Halal健康零食 | 健康趋势+认证刚需 | 全年龄段 |
| 智能宠物用品 | 宠物人性化趋势 | 城市T20 |
