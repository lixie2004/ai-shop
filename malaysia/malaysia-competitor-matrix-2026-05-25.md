## 📊 马来西亚市场总览

| 指标 | 数值 |
|------|------|
| 人口 | 3,410万 |
| 电商GMV（2025年） | ~$80亿+，CAGR ~18% |
| 穆斯林人口占比 | ~63%（约2,150万） |
| 平台格局 | Shopee ~55-60% / Lazada ~20-25% / TikTok Shop增长最快 |

| 消费者信心指数 | ASEAN中等偏上 |
| 中位数年龄 | 30.8岁（年轻化市场） |

**核心消费特征**:
- **清真认证是第一信任门槛**——JAKIM认证对穆斯林消费者是购买决策必要条件
- **多族群市场**（马来~63%/华裔~23%/印裔~7%）——不同族群消费偏好差异大
- **"研究型购买"模式**——马来西亚消费者购买前会看评测、对比价格、看评价
- **高信用卡渗透率**——电子支付占比远高于越南/菲律宾
- **Shopee + Lazada双平台**——品牌通常双平台运营
