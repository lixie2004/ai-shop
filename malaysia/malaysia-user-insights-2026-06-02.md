### 1.4 需求优先级排序

| 优先级 | 需求 | 说明 | 解决收益 |
|--------|------|------|---------|
| **Must-have** | **Halal 认证** | 60%+ 穆斯林消费者的第一筛选条件，无 Halal 转化率下降 40%+ | 获得 60%+ 人口的市场准入 |
| **Must-have** | **NPRA 注册（NOT 编号）** | 合法销售的前提，也是信任信号 | 合规保障，避免下架风险 |
| **Must-have** | **小颗粒/软糖剂型** | "难吞咽"是 #1 差评，软糖/小胶囊是差异化利器 | 降低服用门槛，提升复购 |
| **Must-have** | **三语产品描述（MS/EN/ZH）** | 马来语覆盖主流，英语城市白领，中文华人社区 | 覆盖 100% 目标用户 |
| **Should-have** | **明确效果周期提示** | "效果不明显"来自预期管理失败，需标注见效周期 | 降低差评率 2-3pp |
| **Should-have** | **防伪验证系统** | "假货担忧"是差评 Top 4，扫码验真可建立信任 | 提升信任转化率 |
| **Should-have** | **便携包装/即饮形态** | 胶原蛋白饮品已验证，液体/条包形式更受欢迎 | 差异化竞争 |
| **Should-have** | **男士专属功能配方** | 护肝/抗疲劳/睾酮支持等细分功能几乎空白 | 开辟男性增量市场 |
| **Nice-to-have** | **本地化口味（无腥味）** | 鱼油/Omega-3 的鱼腥味是常见痛点 | 提升体验满意度 |
| **Nice-to-have** | **订阅制/自动补货** | 保健品是高频消耗品，订阅制可锁定复购 | 提升 LTV 30-50% |
| **Nice-to-have** | **TikTok 科普内容** | 成分解读/效果展示类内容可降低决策门槛 | 降低获客成本 |
