## SKU 1：清真防晒霜 SPF50+（化妆品）

| 项目 | 详情 |
|------|------|
| **认证名称** | NPRA 化妆品通知 + JAKIM 清真认证 |
| **认证编号格式** | NPRA: NOT-XXXX-XXXXX / JAKIM: JAKIM-HC-XXXX |
| **认证费用** | NPRA: MYR 2,000-3,000 + JAKIM: MYR 3,000-5,000 |
| **认证周期** | NPRA: 2-3 个月 / JAKIM: 3-6 个月 |
| **风险评级** | 🟡 **中** |
| **具体要求** | ① 成分不含酒精/动物衍生物 ② 需在 NPRA 官网通知 ③ 包装标注清真标志 ④ 防晒指数需符合 MS 2634:2019 |
| **不合规替代方案** | ① 改为普通防晒霜（放弃清真认证，降低售价） ② 与已有清真认证工厂合作（OEM） ③ 先以普通化妆品上架，同步申请清真认证 |
| **违规后果** | 产品下架 + 罚款 MYR 5,000-50,000 + 刑事责任 |
