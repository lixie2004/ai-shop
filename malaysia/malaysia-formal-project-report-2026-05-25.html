3. 搜索热点分析

3.1 搜索行为特征

  • 移动端搜索占比：>85%
  • 语言分布：马来语 45%、英语 35%、中文 15%
  • 意图分布：交易型 58%、信息型 30%、导航型 12%

3.2 关键发现

  1. "Halal" 是核心搜索修饰词：在美妆、食品、保健品中是最关键的信任信号
  2. 价格敏感度高：cheap, affordable, best price 频繁出现
  3. 社交证明驱动：review, best, top rated 占比高
  4. 季节性强：Raya/斋月、11.11、农历新年是三大搜索高峰

4. 舆情趋势分析

4.1 消费者决策路径

社交媒体发现(48%) → 搜索验证 → 查看评价(80%) → 比价 → 促销时下单

4.2 核心痛点（跨品类）

  1. 清真认证真实性：需要 JAKIM 官方认证
  2. 成分/配料安全：要求透明标注
  3. 价格与效果不匹配：高性价比需求
  4. 假货问题：10% 美妆收入被假冒侵蚀
  5. 尺码/色差：跨境电商通病

4.3 舆情风险预警

风险 等级 应对

文化冒犯 🔴 高 本地化审核
