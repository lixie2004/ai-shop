# Malaysia（马来西亚）跨境电商宏观环境分析简报

**报告日期**：2026年5月12日  
**编制**：趋势研究员  
**报告周期**：2025-2026 年度综合评估

---

## 执行摘要

Malaysia 是东南亚第三大电商市场，也是**增速第二快**的市场（仅次于泰国）。2025年电商市场规模达 **US$10.62B**，同比增长 **47.6%**（东南亚第二），预计2031年达 **US$23.11B**（CAGR 13.67%）。三巨头 Shopee、TikTok Shop、Lazada 合计控制99%市场。马来西亚拥有 **3,300万人口**、人均GNI约 **US$12,000**、智能手机渗透率超 **90%**，5G覆盖达 **82.4%**。DuitNow QR 实时支付覆盖 260万个商户点，数字钱包占线上支付 **34.92%**。对中国卖家而言，Malaysia 的优势在于**人均消费力强、增速快、清真市场独特、物流基础设施完善**，且与 Singapore 地理相邻，可作为进入东南亚的双枢纽策略之一。

### 1.2 增速分析

  - **直播电商**：TikTok Shop GMV爆发，直播带货在马来西亚极受欢迎
  - **清真电商**：马来西亚是全球清真经济中心，清真认证线上货架解锁农村需求

## 三、平台格局与市场份额

### 3.1 马来西亚平台市场份额（2025年）

| 平台 | 全区域市场份额 | 马来西亚市场地位 | GMV趋势 |
|------|-------------|----------------|---------|
| **Shopee**（Sea Ltd） | **~50%+** | **#1**，绝对领先 | 稳健增长 |
| **TikTok Shop**（ByteDance） | **~29%**（全区域） | **#2**，快速崛起 | **翻倍增长** |
| **Lazada**（Alibaba） | **~11%**（全区域） | **#3**，品牌导向 | 基本持平 |
| **其他**（PGMall等） | ~10% | 边缘化 | 持续被挤压 |

**来源**：Momentum Works 2026 / KR Asia / Nikkei Asia

### 3.2 三巨头GMV对比（东南亚全区域）

| 平台 | 2024年GMV | 2025年GMV | 增速 |
|------|----------|----------|------|
| Shopee | US$66.8B | US$83.2B | +24.6% |
| TikTok Shop（含Tokopedia） | US$22.6B | US$45.6B | **+101.8%** |
| Lazada | ~US$18B | US$18B | ~0% |

### 3.3 各平台马来西亚市场特征

#### Shopee — 流量王者
- **优势**：最大用户基础，ShopeePay生态，马来西亚物流网络完善
- **马来西亚特色**：Shopee Mall（清真品牌旗舰店）增长快。注意：2024年TikTok Shop曾因监管被暂停2个月（损失约US$2亿/月GMV），Shopee在此期间获益
- **卖家视角**：流量最大但竞争激烈

#### TikTok Shop — 增长引擎
- **优势**：内容电商天然流量，在马来西亚年轻用户中极受欢迎
- **监管风险**：2024年曾被暂停运营2个月，需本地注册合规
- **卖家视角**：直播GMV占平台总GMV达22%，转化率是静态广告的3-5倍

#### Lazada — 品牌阵地
- **优势**：Alibaba供应链，AI购物助手（近1亿用户），超50%产品页面使用AI技术
- **马来西亚特色**：超8,000个天猫品牌入驻Lazada，Startup期间GMV超US$1亿
- **卖家视角**：品牌溢价空间大，适合中高端定位

#### PGMall — 本土玩家
- **定位**：马来西亚本土平台，本土化优势
- **市场份额**：<3%，持续边缘化

## 四、跨境电商政策与监管环境

### 4.1 进口关税框架

| 政策维度 | 详情 | 来源 |
|----------|------|------|
| **关税制度** | ASEAN Harmonized Tariff Nomenclature (AHTN) | Malaysian Customs |
| **进口关税** | **0-40%** 不等，多数消费品 0-10% | Malaysian Customs |
| **SST（销售与服务税）** | **6-10%**（因品类而异，多数商品6%） | Malaysian Customs |
| **免税额度** | **MYR 500（~US$115）** 以下免关税 | Malaysian Customs |
| **跨境电商综合税** | 关税 + SST（多数品类实际税负 6-16%） | 行业估计 |
| **清关效率** | 电子清关（MyTRADE），多数商品 **1-3个工作日** | Malaysian Customs |

### 4.2 跨境电商监管政策

| 政策领域 | 详情 |
|----------|------|
| **电商注册** | 外国电商企业需在马来西亚公司法下注册，年销售额超MYR 500,000需注册SST |
| **清真认证** | **核心差异化**——食品/化妆品/药品需JAKIM清真认证才能在穆斯林消费者中建立信任 |
| **消费者保护** | 《消费者保护法》适用于电商，平台需审核卖家资质 |
| **数据隐私** | PDPA（个人数据保护法）2013年生效，2020年修订加强 |
| **平台责任** | 电商平台需对卖家资质审核负责，假货/侵权产品平台连带责任 |
| **预清关** | 马来西亚海关与中国/新加坡的预清关系统将过境时间缩短30% |

### 4.3 对中国卖家的政策环境

| 维度 | 评估 | 说明 |
|------|------|------|
| **关税优惠** | ⭐⭐⭐⭐ | 中国-东盟FTA（ACFTA）提供关税减免 |
| **清真机会** | ⭐⭐⭐⭐⭐ | **独特优势**——马来西亚是全球清真经济中心，获清真认证可辐射全球穆斯林市场 |
| **跨境电商政策** | ⭐⭐⭐ | 政府积极推动，但合规要求趋严 |
| **外汇管制** | ⭐⭐⭐ | 利润汇出需通过正规渠道，相对规范 |
| **政治关系** | ⭐⭐⭐⭐ | 中马关系稳定，RCEP框架下贸易便利化 |

### 4.4 关键合规要求

1. **清真认证（JAKIM）**：食品、化妆品、药品、保健品——这是进入马来西亚市场的最大差异化门槛
2. **NPRA认证**：药品/保健品需国家药品监管局批准
3. **SIRIM认证**：电子产品需马来西亚标准认证
4. **标签要求**：产品标签需包含马来语信息
5. **SST注册**：年销售额超MYR 500,000必须注册SST

## 五、物流基础设施与支付生态

### 5.1 物流基础设施

| 维度 | 来源 |
|------|------|
| **物流市场** | 马来西亚是东南亚物流最发达的市场之一 |
| **主要物流商** | Pos Malaysia、J&T Express、Ninja Van、DHL、GD Express、City-Link、Flash Express |
| **中国跨境物流** | 4PX、云途、燕文、递四方等均有马来西亚专线 |
| **海外仓** | Shopee/Lazada官方仓、第三方海外仓 |

#### 配送时效

| 服务类型 | 时效 | 覆盖范围 |
|----------|------|---------|
| **西马（半岛）同城** | 当日达/次日达 | 吉隆坡/槟城/柔佛核心区域 |
| **西马全境** | 1-3工作日 | 主要城市 |
| **东马（沙巴/砂拉越）** | 3-7工作日 | 偏远地区更慢 |
| **中国→马来西亚跨境** | 5-10工作日（直邮）/ 1-3日（海外仓） | — |

#### 物流成本

| 物流方式 | 首重费用 | 备注 |
|----------|---------|------|
| Pos Malaysia（平邮） | ~US$1.5-3 | 最便宜但最慢 |
| J&T Express | ~US$2-3.5 | 性价比高，快速增长 |
| Ninja Van | ~US.5-3.5 | 覆盖广，API完善 |
| DHL Express | ~US$8-15 | 最快但最贵 |
| 海外仓发货 | ~US$1.5-3 | 最快，但需备货 |

#### 物流挑战

| 挑战 | 详情 |
|------|------|
| **东马配送成本高** | 沙巴/砂拉越物流成本是西马的2-3倍，抑制电商渗透 |
| **最后一英里** | 城市拥堵，农村地区覆盖有限 |
| **退货处理** | 逆向物流基础设施仍在完善中 |

### 5.2 支付生态

| 支付方式 | 市场份额 | 特征 |
|----------|---------|------|
| **DuitNow QR** | **~20-25%** | 马来西亚央行实时支付系统，覆盖260万商户 |
| **Grab Pay** | ~15-18% | 最大电子钱包之一 |
| **Touch'n Go eWallet** | ~12-15% | 马来西亚本土最大电子钱包 |
| **ShopeePay** | ~10-12% | Shopee生态内 |
| **Boost** | ~8-10% | Axiate旗下电子钱包 |
| **信用卡/借记卡** | ~15-20% | 普及率持续提升 |
| **银行转账（FPX）** | ~10-15% | 网银直接转账 |
| **COD（货到付款）** | **~15-20%** | 低于泰国，呈下降趋势 |
| **BNPL** | ~5-8%（快速增长）CAGR 17.18% | Atome/Grab PayLater |

#### 支付生态关键洞察

1. **DuitNow 是基础设施**：马来西亚央行推出的实时QR支付系统，类似新加坡的PayNow，预计2026年所有授权钱包必须接入
2. **数字钱包碎片化**：无单一主导钱包，3.2个活跃账户/用户，需接入多个支付渠道
3. **COD 下降趋势**：从2023年的~25%降至2025年的~15-20%
4. **跨境QR互联**：DuitNow已与新加坡PayNow、泰国PromptPay、印尼QRIS互联，支持跨境支付

## 七、综合评估

### 7.1 SWOT 分析（中国卖家视角）

| | 正面 | 负面 |

### 7.2 市场吸引力评分

| 维度 | 评分（1-10） | 说明 |
