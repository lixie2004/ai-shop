## 1. 市场概览

### 1.1 宏观数据

| 指标 | 数据 |
|------|------|
| 电商GMV（2025） | $106.2亿（平台口径）/$346亿（全市场） |
| CAGR（2024-2027） | 13.67% |
| 人均GDP（PPP） | ~$33,000（东南亚最高之一） |
| 互联网普及率 | >95%（3,300万+用户） |
| 移动端购物占比 | 72.67% |
| 数字支付渗透率 | >80% |

### 1.2 平台格局

| 平台 | 市场份额 | 核心优势 |
|------|---------|---------|
| Shopee | ~60% | 流量最大，活动丰富 |
| Lazada | ~30% | 品牌旗舰店，跨境物流成熟 |
| TikTok Shop | 5-8% | 内容电商增速最快 |
| 其他 | 2-5% | 独立站、Facebook Shop等 |

### 1.3 消费者特征

- **多元文化**：马来族（60%）、华裔（23%）、印度裔（7%）
- **年轻化**：25-34岁占网购主力52%
- **价格敏感**：38%搜索含"murah/cheap"修饰词
- **Halal刚需**：马来族消费者首要考虑Halal认证
- **移动优先**：65%+电商交易通过手机完成

### 10.3 核心建议

1. **立即启动**：弹力带健身套装 + 磁吸手机支架（无认证门槛，快速验证市场）
