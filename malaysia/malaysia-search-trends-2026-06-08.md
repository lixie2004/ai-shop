## 4. 平台搜索算法洞察

### Shopee SEO 关键因子
1. **标题关键词** → 直接影响搜索排名
2. **销量/转化率** → 加权排序
3. **店铺评分** → 4.5+ 获流量倾斜
4. **促销活动** → Flash Sale/Voucher 期间权重提升
5. **物流时效** → 快速发货店铺获排名提升
6. **Chat响应** → 1小时内回复率影响搜索权重

### TikTok Shop 搜索特点
- **内容驱动**: 短视频播放量直接影响商品曝光
- **直播加权**: 正在直播的商品获额外搜索权重
- **达人关联**: KOL搜索词与商品关联曝光

## 6. 搜索热点总结

### 核心发现
1. **时尚品类搜索占主导** → 女装(baju kurung/blouse)是最大搜索品类
2. **3C新品驱动搜索** → iPhone 17效应仍在持续
3. **家居新增长点** → Tumbler/保温杯成现象级搜索词
4. **文化特定词不可忽视** → 马来语搜索词(baju kurung/kebaya)搜索量大
5. **中国品牌崛起** → Xiaomi在搜索榜排名靠前
