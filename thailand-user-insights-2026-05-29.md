## 用户画像

### 核心用户群体：Z世代 + 千禧一代（17-36岁）

| 维度 | 描述 |
|------|------|
| **年龄段** | 17-36岁，占网购人群 **62%** |
| **性别** | 女性占比略高（55%+），尤其时尚/美妆 |
| **地域** | 曼谷都会区（最大）、清迈、春武里、呵叻 |

## 消费者旅程

```
发现 → 研究 → 比较 → 决策 → 购买 → 收货 → 评价 → 复购
  │       │       │       │       │       │       │
  │       │       │       │       │       │       └── 好评+推荐 = 复购
  │       │       │       │       │       └── 期望匹配=好评, 不匹配=差评
  │       │       │       │       └── 支付方式影响转化率
  │       │       │       └── 价格+评价+时效=最终决策
  │       │       └── Shopee/Lazada比价
  │       └── 看Read reviews + 视频测评
  └── TikTok / LINE / 朋友推荐
```

### 关键转化节点

| 节点 | 优化策略 | 预期提升 |
|------|---------|---------|
| **发现** | KOL+短视频 | +40% 流量 |
| **研究** | 视频测评+详细描述 | +25% 停留 |
| **比较** | 价格锚定+捆绑销售 | +15% 转化 |
| **决策** | 限时优惠+库存提示 | +20% 转化 |
| **购买** | 多支付选项 | +10% 完成 |
| **收货** | 快速+精美包装 | +30% 好评 |
| **复购** | LINE+订阅+会员 | +25% LTV |

## Buyer Persona

### Persona 1: "时尚 Ting"（25岁女性，曼谷）
- 月收入: ~25,000 泰铢
- 购物动机: 追赶潮流、社交分享
- 平台: Shopee + TikTok Shop
- 购买品类: 时尚、美妆、家居装饰
- 核心痛点: 怕买到假货，希望图片与实物一致
- 营销触达: TikTok KOL、闪购通知

### Persona 2: "宠物妈妈 Ploy"（30岁女性，春武里）
- 月收入: ~35,000 泰铢
- 购物动机: 给宠物最好的
- 平台: Shopee + Lazada
- 购买品类: 宠物食品、玩具、美容
- 核心痛点: 食品安全、效果不确定
- 营销触达: Facebook 群组、宠物KOL

### Persona 3: "健身男 Bank"（28岁男性，曼谷）
- 月收入: ~30,000 泰铢
- 购物动机: 增肌塑形
- 平台: Shopee + LINE Shopping
- 购买品类: 运动装备、蛋白粉、电子配件
- 核心痛点: 产品质量、效果
- 营销触达: YouTube 测评、Instagram

### Persona 4: "居家男 Gun"（35岁男性，呵叻）
- 月收入: ~40,000 泰铢
- 购物动机: 提升生活品质
- 平台: Lazada + Shopee
- 购买品类: 家居、电子、工具
- 核心痛点: 品质保证、售后
- 营销触达: Facebook、搜索广告
