## 七、搜索趋势对选品的指导意义

### 7.1 选品关键词策略

| 策略 | 具体行动 |
|------|----------|
| **标题优化** | 必含 `murah`、`free shipping`、`original` |
| **本地化关键词** | 马来语标题权重最高（覆盖70%用户） |
| **TikTok关联词** | 加入 `viral`、`trending` 提升内容曝光 |
| **Halal标识** | Halal相关产品标题和描述突出标注 |
| **季节性备货** | 4月/11月/12月前2个月开始准备 |

### 7.2 高潜力搜索机会

| 机会方向 | 理由 |
|----------|------|
| Halal宠物用品 | `pet food halal` 搜索↑50%，竞争低 |
| Modest运动用品 | `baju senaman`、`yoga pants` 增长18% |
| 防晒/防晒相关 | `sunscreen`搜索↑30%，热带气候刚需 |
| 智能家居小件 | `LED strip light` 增长40% |
| 智能宠物用品 | `pet bed` 增长35%，蓝海 |
