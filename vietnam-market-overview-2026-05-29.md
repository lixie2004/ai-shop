## 执行摘要

越南是**东南亚第三大电商市场**（仅次于印尼和泰国），2025年电商GMV预计达 **163.5亿-280亿美元**（不同口径），正处于高速增长向成熟竞争过渡的关键阶段。Shopee 与 TikTok Shop 双头垄断格局已定，合计占据 **97%** 的市场份额。政策端，2025年新电商法（Law No. 122/2025）将于 **2026年7月1日** 生效，对外国卖家提出了明确的注册与合规要求，跨境电商的监管红利期正在结束。对跨境卖家而言，越南市场机会巨大但合规门槛显著提高。

### 3.1 市场份额（2025年）

> **数据来源**：Metric VN Online Retail Platform Market Report 2025 / KR-Asia / Vietnam Briefing / Fintech News SG  
> **关键结论**：Shopee + TikTok Shop 合计占据 **97%** 的市场份额，已形成实质性的双头垄断。

### 3.2 平台竞争态势

**Shopee（Sea Group旗下）**
- 仍保持领先地位，但增长放缓（Q3 2025 YoY仅+4%）
- 应对策略：ShopeeVIP忠诚度计划、Shopee Live直播升级、强化物流基础设施
- 品牌旗舰店战略：仅2.64%店铺贡献35.7%销售额
- 利润率：FY2026 调整后EBITDA/GMV预计改善至0.9%（FY2025为0.7%）

**TikTok Shop（字节跳动旗下）**
- H1 2025 GMV增长 **69%**（全平台最快），市场份额从29%升至39-42%
- 核心武器：**"Shoppertainment"（购物娱乐化）**——直播购物 + 短视频发现 + 原生闭环交易
- 平台特色：消费娱乐化、冲动消费、低客单价但高交易量
- 本地市场已被亚马逊视为东南亚电商出口枢纽

**Lazada（阿里巴巴旗下）**
- 份额萎缩至约3%，2025年销售额下降约48%
- 仍保有忠实用户群体，但在内容和价格战中处于劣势

**Tiki**
- 2025年收入暴跌 **80%**，份额不足1%
- 面临被边缘化的风险

**其他竞争者**
- Amazon 在越南被视为跨境购物信任平台，越南消费者通过 Weshop.com.vn、fado.vn 等中间服务购买
- Sendo、The Gioi Di Dong（Mobile World）等本地化平台在细分领域仍有存在

## 五、跨境电商政策环境

### 5.1 现行法规框架

| 法规 | 状态 | 关键内容 |
|------|------|----------|
| **第52/2013/ND-CP号法令** | 现行（将废止） | 电商活动基本法规，工贸部（MoIT）主管 |
| **第117/2025/ND-CP号法令** | 2025年6月生效 | 修订版电商法令，扩大监管范围至直播、网红、跨境平台 |
| **电商法 No. 122/2025/QH15** | **2026年7月1日生效** | 取代法令层级的旧规，上升为法律层级 |
| **第01/2025/QĐ-TTg号决定** | 2025年2月18日生效 | 取消低值进口免税 |

### 5.2 2025年新电商法核心要点

**（1）外国平台/卖家义务** ⚠️
- 必须向 **工贸部（MoIT）申请许可证**
- 必须在越南设立 **代表处** 或指定 **法律授权代表**
- 承担消费者权益保护、卖家信息核验、纠纷先行赔付等责任
- 必须向越南监管机构共享卖家信息

**（2）卖家身份认证**
- 强制通过 **VNeID**（国家身份认证系统）核实卖家身份
- 目的：遏制假冒伪劣、知识产权侵权、逃税

**（3）直播与网红监管**
- 直播卖货、社交电商卖家必须进行工商登记和税务登记
- 年收入超过VND 1亿须缴纳个人所得税和增值税
- 广告内容须符合越南广告法，明确标注付费推广

**（4）平台税务代扣义务** 
- 月销售额超过VND 1亿的个人卖家，平台须代扣代缴 **VAT + 个人所得税**
- 平台须向税务机关月度申报
- 适用于境内和境外卖家

**（5）假冒伪劣与知识产权**
- 各平台须联合承担平台上侵权行为的连带责任
- 加强打击走私假冒伪劣商品

### 5.3 关税与进口税收政策

| 税种 | 标准税率 | 备注 |
|------|----------|------|
| **进口关税** | 10-30%（因品类而异） | CPTPP/VFTA等自贸协定可享优惠税率 |
| **增值税（VAT）** | 10%（标准）/ 8%（优惠期至2026年底） | 2025年2月18日起取消低值免税 |
| **特别消费税** | 因品类（含酒精、烟草、高端消费品等） | — |
| **企业所得税** | 20%（标准） | — |

### 5.4 低值进口免税取消（重大变化）

| 政策 | 详情 |
|------|------|
| **旧政策**（2010-2025.2.17） | ≤VND 100万（~$40）进口快件免征进口关税和VAT |
| **新政策**（2025.2.18起） | **取消免税门槛**，所有进口商品无论金额大小一律征收进口关税（如适用）和VAT |
| **自动征税** | 自2025年8月1日起，海关对VND 100万以下小件进口商品自动征收VAT |
| **政府目的** | 公平竞争（本地商品VAT 10% vs 进口小额免税）、扩大税基、保护国内制造业 |

### 5.5 进口商品正面清单

- 政府将维护一份 **通过电商渠道允许进口的商品清单**
- 外国商品和服务在越南销售须符合本地产品标准、技术要求和安全规定
- **电子烟**自2025年1月起禁止进口
- 危险品、违禁品等按既有法规执行

### 5.6 CPTPP / 自贸协定优惠

| 协定 | 相关内容 |
|------|----------|
| **CPTPP** | 第13/2025/ND-CP号法令（2025.1.21生效），修订2022-2027年特殊优惠进口关税 |
| **RCEP** | 区域全面经济伙伴关系，原产地累积规则 |
| **VFTA** | 越南-欧盟自贸协定（2020年生效） |
| **美国贸易** | 美越达成互惠贸易框架协议，美国对越维持20%对等关税率，部分产品可降至0% |

### 5.6 合规路线图（对跨境卖家）

```
现在 → 2025年底:
├── 评估是否需要注册越南实体或指定本地代理
├── 审查产品是否在允许进口的正面清单内
├── 适应新VAT/进口税成本结构（低值免税取消）
└── 确保产品符合越南技术标准和标签要求

2026年7月1日前 (电商法生效):
├── 完成 MoIT 许可证申请
├── 通过 VNeID 完成卖家身份认证
├── 建立税务申报和代扣代缴系统
├── 确保消费者保护机制到位
└── 若为平台：任命越南本地法律代表
```

## 六、关键趋势与战略启示

### 6.1 五大关键趋势

1. **双头垄断加剧**：Shopee + TikTok Shop 占 97% 份额，新进入者门槛极高
2. **内容电商为王**：TikTok Shop 的"Shoppertainment"模型重塑消费路径
3. **监管收紧**：电商法2026年生效，跨境卖家合规成本上升
4. **低值免税终结**：2025年2月起所有进口商品均征税，跨境小包裹成本上升
5. **卖家结构升级**：品牌旗舰店贡献近30%收入但仅占3.4%店铺，消费者偏好官方正品

### 6.2 对跨境卖家的建议

| 优先级 | 行动 | 时间窗口 |
|--------|------|----------|
| 🔴 紧急 | 评估产品关税+VAT成本重算，调整定价策略 | 立即 |
| 🔴 紧急 | 确认产品是否在越南允许进口正面清单 | 立即 |
| 🟠 重要 | 部署MoIT注册和合规准备 | 2026年7月1日前 |
| 🟠 重要 | 建立越南本地代理或法律实体 | 2026年Q1前 |
| 🟡 建议 | 入驻 Shopee 和/或 TikTok Shop 越南站 | 3-6个月内 |
| 🟡 建议 | 布局直播电商和KOL合作（越南市场最有效获客方式） | 持续 |
| 🟡 建议 | 考虑布局本地仓储/物流（跨境直邮成本上升） | 6-12个月内 |
