## 五、消费者购买决策路径

```
发现产品 → 查看Reviews → 对比价格 → 搜索"original/正品" → 检查BPOM/清真标识 → 下单
  ↑                    ↑                ↑                    ↑
 TikTok/IG           Shopee评分      比价工具/平台比价     评论区询问
```

**关键转化节点**：
1. **Reviews数量和质量**：>100条好评是购买信心底线
2. **BPOM/清真标识**：美妆和食品必须可见
3. **价格锚点**：折扣vs原价对比是转化加速器
4. **KOL信任度**：医生/成分党KOL > 颜值博主
