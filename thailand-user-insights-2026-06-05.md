## 二、泰国 Gen Z 消费者洞察

| 维度 | 数据 |
|------|------|
| 出生年份 | 1997-2012 |
| 电商用户占比 | ~55%（线上购物主力） |
| 在线购物频率 | 每周2-3次 |
| 主要消费品类 | 美妆、时尚、食品、电子产品 |
| 价格敏感度 | 中高，折扣驱动 |
| 平台偏好 | TikTok Shop > Shopee > Instagram购物 |
| 驱动因素 | 社交趋势、KOL推荐、折扣代码 |

### Gen Z 消费趋势

1. **战略购物**：泰国Gen Z引领亚洲战略购物行为（Bangkok Post）
2. **比较购物**：Price comparison是最重要的购物动机
3. **社交购物**：51%受社交媒体趋势影响
4. **发现式购物**：TikTok Shop的发现式购物体验深受欢迎

## 三、泰国 Millennials 消费者洞察

| 维度 | 数据 |
|------|------|
| 出生年份 | 1981-1996 |
| 电商用户占比 | ~45% |
| 在线购物频率 | 每周1-2次 |
| 客单价 | 高于Gen Z约40% |
| 主要消费品类 | 家电、母婴、家居、3C |
| 价格敏感度 | 中，品质驱动 |
| 平台偏好 | Shopee > Lazada > LINE Shopping |

### Millennials 消费趋势

1. **品牌化消费**：愿意为品牌和品质支付溢价
2. **Loyalty驱动**：服务体验好则复购率高
3. **家庭需求**：母婴、健康补充剂增长
4. **智能家居**：智能家电需求增长
