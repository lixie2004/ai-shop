## 泰国法规框架概览

| 法规 | 管理机构 | 影响 |
|------|---------|------|
| **Thai FDA Act** | 泰国食品药品监督管理局 | 食品、药品、化妆品、医疗器械 |
| **Consumer Protection Act** | 消费者保护委员会 | 标签、广告、退货 |
| **Digital Platform Decree** | ETDA | 数字平台运营 |
| **PDPA** | 个人数据保护委员会 | 用户数据 |
| **Import Tariff Act** | 海关厅 | 进口关税 |
| **IP Law (Trademark Act)** | 知识产权厅 | 商标/专利 |
| **Electronic Transactions Act** | ETDA | 电子合同/签名 |

## 通用合规要求

### 1. 产品标签要求
| 要素 | 要求 |
|------|------|
| 语言 | **必须泰语** |
| 内容 | 品名、成分/材质、产地、经销商信息、使用说明 |
| 标注方式 | 永久标签或包装印刷 |
| 违规处罚 | 罚款 10,000-50,000 泰铢 |

### 2. 进口关税

| 品类 | 关税率 | VAT | 综合税率 |
|------|--------|-----|---------|
| 电子产品 | 0-10% | 7% | 7-17% |
| 塑料制品 | 5-15% | 7% | 12-22% |
| 宠物用品 | 5-10% | 7% | 12-17% |
| 纺织品 | 10-20% | 7% | 17-27% |
| **< 1500 ฿ 包裹** | **0%** | **0%** | **0%** ✅ |

### 3. 数字平台法令合规

| 要求 | 说明 |
|------|------|
| 境外卖家 | 需在泰国指定本地联系人 |
| 平台报告 | 需向 ETDA 提交年交易数据 |
| 违规后果 | 最高 90 天内暂停运营 |
| 建议 | 通过平台官方店入驻可满足大部分要求 |

### 4. PDPA 合规（个人数据保护）

| 要求 | 说明 |
|------|------|
| 用户同意 | 收集数据需明确同意 |
| 目的限制 | 仅用于声明目的 |
| 数据出境 | 需通过 PDPA 标准合同 |
| 违规处罚 | 最高 500 万泰铢罚款 |

### 5. IP 保护

| 步骤 | 说明 |
|------|------|
| 1 | 在泰国 DBD 注册品牌商标（先到先得） |
| 2 | 注册费用 ~3,000-5,000 泰铢/类 |
| 3 | 审核周期 12-18 个月 |
| 4 | 建议在开售 3 个月前提交 |
