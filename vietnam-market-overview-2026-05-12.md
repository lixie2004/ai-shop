# 越南跨境电商市场宏观分析简报

**报告日期**：2026年5月12日  
**分析师**：趋势研究员  
**报告类型**：市场概览

---

## 执行摘要

越南是**东南亚增长最快的电商市场之一**，2025年GMV达**$26-28亿**（四大平台），同比增速**34.75%**，整体电商市场规模达**$31-33亿**。消费者网购渗透率快速提升，**96%**的消费者使用智能手机网购，日均在线消费约**4800万美元**。市场由Shopee（~65%）和TikTok Shop（~28%）主导，Lazada和Tiki为补充。**社交电商**是越南最显著的特征——50%消费者通过社交平台购物，远超全球平均。跨境卖家面临COD占比高（67%）、物流基础设施不完善等挑战，但RCEP关税优惠、数字支付快速增长、政府数字化政策支持力度大，为跨境电商提供了结构性机会窗口。

## 一、电商市场规模（GMV）及增速

### 核心数据

| 指标 | 数值 | 来源 |
|------|------|------|
| **2025年GMV（四大平台）** | **$16.35B**（VND 429.7万亿） | Metric / VCCI |
| **2025年整体电商市场规模** | **$26-33B** | IMARC / Mordor Intelligence |
| **同比增长率（四大平台）** | **+34.75%** YoY | Metric |
| **2024年GMV** | ~$25B（整体）/ $16B（四大平台） | VECOM / Opollo |
| **CAGR（2024-2027）** | **~23%** | PCMI |
| **CAGR（2025-2034）** | **25.94%** | IMARC Group |
| **2034年预测规模** | **$250B** | IMARC Group |
| **东南亚排名** | **第三**（仅次于印尼、泰国） | 行业分析 |
| **日均在线消费** | **$4800万**（VND 1.233万亿） | VietnamPlus |
| **电商渗透率** | 占零售总额约**9%** | Opollo |

### 趋势分析

越南电商市场处于**高速增长期**，增速显著高于东南亚平均水平。关键驱动力：

1. **人口红利**：1亿人口，中位年龄31岁，数字化原住民占比高
2. **智能手机渗透率极高**：96%消费者使用智能手机网购，亚太最高
3. **社交电商爆发**：社交电商2025年预计达$50亿，增速25.4%
4. **平台竞争加剧**：TikTok Shop快速崛起，2024年GMV增长69%
5. **政策支持**：政府数字化政策持续推动

**与东南亚其他市场对比**：
- 印尼：规模最大但增速放缓
- 泰国：增速最快（+51.8%）
- 越南：增速第二（+34.75%），规模第三，但增长质量高

| **日均手机上网时长** | 亚太最高之一 | 行业分析 |

### 消费行为特征

#### 1. 社交电商大国
- **50%**消费者通过社交平台购物，**全球平均仅23%**（Havas Vietnam）
- 社交电商2025年预计达**$50亿**，增速25.4%
- 视频电商占GMV约**20%**（东南亚趋势）
- 直播购物快速普及，TikTok Shop是主要驱动力

#### 2. 移动优先（Mobile-First）
- **96%**消费者使用智能手机网购（亚太最高）
- 移动端占GMV超**80%**
- 消费者从月购转向**周购**，购物频次持续提升

#### 3. 价格敏感但品牌意识提升
- 价格/促销是网购首要驱动因素
- 消费者广泛研究后购买，依赖评价和社交证明
- 对正品和品质的要求持续提升
- 大促期间（9.9、11.11、12.12）消费集中爆发

#### 4. O2O行为普遍
- **45%**的农村消费者"线上搜索→线下购买"
- 消费者日益融合线上线下购物体验

### 区域分布

| 区域 | 特征 |
|------|------|
| **胡志明市** | 最大市场，购买力最强 |
| **河内** | 第二大市场，增长快速 |
| **岘港/芹苴等二线城市** | 增长潜力大 |
| **农村地区** | 45%人口，COD占比高，增长空间大 |

## 三、平台格局

### 市场份额数据

| 平台 | GMV份额（2024-2025） | 趋势 | 来源 |
|------|---------------------|------|------|
| **Shopee** | **~65%** | 领导者，但面临TikTok挑战 | Opollo / YouNet ECI |
| **TikTok Shop** | **~28%** | **爆发式增长**（+69% YoY） | Opollo |
| **Lazada** | 次要 | 稳定但份额被挤压 | 行业分析 |
| **Tiki** | 次要 | 本土平台，正品定位 | 行业分析 |
| **Shopee+TikTok合计** | **~93%** | 高度集中 | — |

### 平台深度分析

#### 1. Shopee — 领导者但面临挑战

**核心优势**：
- **~65%** GMV份额，最大流量和用户基础
- 频繁的促销活动和优惠券策略
- 自建物流网络Shopee Express覆盖广泛
- 品类广度最大

**挑战**：
- TikTok Shop快速抢占市场份额
- 佣金率上升压缩卖家利润
- 7000+卖家在2025年上半年退出市场（VietData）

**对跨境卖家的意义**：
- 仍是进入越南市场的**第一优先级平台**
- 但需关注TikTok Shop的增量机会

#### 2. TikTok Shop — 颠覆性挑战者

**核心优势**：
- **~28%** GMV份额，2024年增长**+69% YoY**
- 短视频+直播带货=高转化率
- 在时尚、美妆、食品等品类表现最强
- 社交电商与越南消费者行为完美匹配

**战略方向**：
- 快速扩张品类覆盖
- 提升平台变现能力
- 投资直播基础设施

**对跨境卖家的意义**：
- **必须布局**的增量渠道
- 适合时尚、美妆、食品等**可演示性强**的品类
- 需要投入内容创作和直播运营能力

#### 3. Lazada — 稳定但份额被挤压

**现状**：
- 份额被Shopee和TikTok Shop挤压
- 阿里巴巴资源支持，技术基础设施先进
- 正品认证体系是差异化

**对跨境卖家的意义**：
- 作为补充渠道，不应作为主要增长来源
- 适合品牌型卖家

#### 4. Tiki — 本土正品平台

**现状**：
- 越南本土平台，正品保证是核心差异化
- 物流自建，配送体验好
- 份额较小但忠诚用户多

## 四、跨境电商政策与监管环境

### 关键政策法规

| 政策/法规 | 内容 | 对跨境电商的影响 |
|-----------|------|-----------------|
| **RCEP** | 中越同属成员 | **重大利好**：关税减免、贸易便利化 |
| **越南数字化转型政策** | 政府大力推动数字经济 | 正面：扩大互联网覆盖，增加潜在消费者 |
| **跨境电商税收** | 进口关税+VAT | 需关注：不同品类税率不同 |
| **数据保护法（PDPD）** | 个人数据保护 | 需合规：跨境数据传输需注意 |
| **消费者权益保护法** | 7天无理由退货等 | 需合规：退货政策需明确 |
| **平台监管** | 电商平台需注册和合规 | 需关注：平台责任和卖家资质要求 |

### 关税与税收

| 税种 | 税率 | 说明 |
|------|------|------|
| **进口关税** | 0-40%（取决于品类） | RCEP框架下多数商品0-10% |
| **增值税（VAT）** | **10%** | 标准税率 |
| **特别消费税** | 特定品类 | 烟草、酒精、化妆品等 |
| **企业所得税** | 20% | 适用于在越南注册的企业 |

### 合规要求

1. **进口许可证**：部分品类需进口许可证
2. **产品认证**：食品、化妆品、电子产品需相关认证
3. **产品标签**：需有越南语标签
4. **数据合规**：遵守越南《个人数据保护法》
5. **平台注册**：在越南运营的电商平台需向ETDA注册

## 五、物流基础设施现状

### 物流服务商

| 服务商 | 类型 | 特点 |
|--------|------|------|
| **Vietnam Post** | 国有邮政 | 全国覆盖最广 |
| **Shopee Express** | 平台物流 | Shopee生态内，覆盖广 |
| **J&T Express** | 民营快递 | 快速扩张，性价比高 |
| **Ninja Van** | 区域电商物流 | 东南亚网络 |
| **Giao Hàng Nhanh (GHN)** | 本土快递 | 越南最大本土快递之一 |
| **Giao Hàng Tiết Kiệm (GHTK)** | 本土快递 | 覆盖面广，价格优 |
| **DHL/FedEx** | 国际快递 | 高端跨境物流 |

### 物流时效与覆盖

| 指标 | 数值 |
|------|------|
| **胡志明/河内** | 1-2天 |
| **主要城市** | 2-3天 |
| **省级城市** | 3-4天 |
| **偏远地区** | 4-7天 |
| **COD覆盖率** | 全国（67%交易使用COD） |

### 物流挑战

1. **最后一公里成本高**：农村地区配送成本是城市的2-3倍
2. **仓储设施不足**：现代化仓储设施仍缺乏
3. **冷链物流不完善**：食品/生鲜跨境受限
4. **COD效率低**：67%的COD占比导致高退货率和现金流压力

### 跨境物流建议

1. **头程**：中国→越南，建议走深圳/广州直飞胡志明/河内的空运
2. **海外仓**：在胡志明设仓是覆盖越南市场的最优解
3. **COD管理**：必须支持COD，同时通过数字支付优惠引导预付
4. **物流合作伙伴**：与GHTK或GHN合作，覆盖最广且成本最优

## 六、支付生态

### 支付方式分布

| 支付方式 | 占比 | 趋势 |
|----------|------|------|
| **COD（货到付款）** | **67%** | ↓ 缓慢下降 |
| **电子钱包** | **~18%** | ↑ 快速增长 |
| **信用卡/借记卡** | **~8%** | ↑ 稳步增长 |
| **BNPL（先买后付）** | **~2%** | ↑ 快速增长 |
| **银行转账** | ~5% | ↔ 稳定 |

### 主要支付工具

| 支付工具 | 用户数/份额 | 特点 |
|----------|------------|------|
| **MoMo** | **3500万+** / **68%**电子钱包份额 | 最大电子钱包，已发展为超级App（支付+投资+保险+信贷） |
| **ZaloPay** | **53%**电子钱包份额 | 集成于Zalo消息平台，用户基础大 |
| **VNPay** | — | 二维码支付领导者 |
| **VietQR** | — | 国家二维码支付标准 |
| **ShopeePay** | — | Shopee生态内 |

### 支付市场关键数据

- 越南数字支付市场2025年规模超**$400亿**
- QR码支付是增长最快的细分市场
- **49%**消费者对BNPL感兴趣
- MoMo已从简单P2P转账演变为AI驱动的超级App

### 跨境卖家支付建议

1. **必须支持COD**：67%交易使用COD，不提供COD等于放弃大部分市场
2. **接入电子钱包**：MoMo + ZaloPay（覆盖80%+数字支付用户）
3. **信用卡**：Visa/Mastercard必须支持
4. **BNPL**：接入先买后付可提升转化率
5. **数字支付优惠**：通过折扣引导消费者从COD转向数字支付

## 七、市场进入的关键机会与挑战

### 🟢 关键机会

#### 1. 社交电商红利
- 越南50%消费者通过社交平台购物，全球最高之一
- TikTok Shop增速+69%，早期红利仍在
- 直播带货在时尚、美妆、食品品类转化率极高

#### 2. 人口红利+数字化加速
- 1亿人口，中位年龄31岁
- 96%智能手机网购渗透率（亚太最高）
- 政府数字化政策大力支持

#### 3. RCEP关税优惠
- 中越同属RCEP成员，多数商品关税0-10%
- 贸易便利化措施持续推出

#### 4. 数字支付快速增长
- 电子钱包用户快速增长，MoMo 3500万+用户
- BNPL渗透率仅2%，增长空间巨大
- 数字支付市场超$400亿

#### 5. 品类升级机会
- 消费者对品牌和品质要求提升
- 跨境商品在美妆、时尚、电子品类有显著优势

### 🔴 关键挑战

#### 1. COD占比高（67%）
- 导致高退货率（时尚品类可达20-30%）
- 现金流压力大
- 需要建立高效的COD管理系统

#### 2. 物流基础设施不完善
- 最后一公里成本高
- 农村地区配送时效长
- 冷链物流缺乏

#### 3. 平台竞争激烈
- Shopee和TikTok Shop双寡头格局
- 7000+卖家在2025年上半年退出市场
- 价格战激烈，利润空间被压缩

#### 4. 合规要求提高
- 数据保护法（PDPD）合规
- 产品认证和标签要求
- 跨境电商税收政策变化

#### 5. 本地化要求高
- 越南语产品详情和客服是基本要求
- 文化差异需要深度理解
- 本地KOL合作是获客关键
