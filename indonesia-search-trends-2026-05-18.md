## 6. 营销建议

### 6.1 关键词策略

| 策略 | 建议 |
|------|------|
| **核心词** | 品类词 + 功能修饰词（如 "hijab olahraga anti keringat"） |
| **长尾词** | 品类词 + 功能 + 价格（如 "bros hijab magnet murah 3 pcs"） |
| **信任词** | halal/SNI/BPOM 必须加入标题和描述 |
| **否定词** | 排除不相关词，降低无效点击 |

### 6.2 内容策略

| 渠道 | 内容类型 | 关键词重点 |
|------|---------|-----------|
| **TikTok Shop** | 短视频+直播 | 功能展示+痛点解决 |
| **Shopee** | 搜索广告+展示 | 品类词+价格词 |
| **Google Ads** | 搜索广告 | 长尾词+意图词 |
| **Instagram** | KOL合作 | 品牌词+生活方式 |

### 6.3 本地化建议

| 维度 | 建议 |
|------|------|
| **语言** | 印尼语优先，英语为辅 |
| **价格** | 标注IDR，支持COD |
| **信任** | 展示Halal/SNI/BPOM认证 |
| **物流** | 强调"gratis ongkir"（免运费） |
| **尺码** | 提供详细尺码指南 |
