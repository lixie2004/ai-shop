## 一、用户画像

### 1.1 核心用户群体

| 维度 | 描述 |
|------|------|
| **年龄** | 18-35岁（占网购人群~70%） |
| **性别** | 女性占比~60%（美妆/服装/家居），男性~40%（电子/运动） |
| **地域** | 胡志明市+河内占~50%，二三线城市快速增长 |
| **收入** | 月收入500-1500万VND（~$200-600）为主力消费群体 |
| **设备** | 移动端>90%，智能手机为主 |
| **购物频率** | 月均3-5次网购 |

### 1.2 用户分层

| 层级 | 占比 | 特征 | 消费力 |
|------|------|------|--------|
| 价格敏感型 | 45% | 追求性价比，频繁比价 | 低-中 |
| 品质追求型 | 30% | 关注品牌和质量，愿付溢价 | 中-高 |
| 潮流追随型 | 15% | 受社交媒体影响，冲动消费 | 中 |
| 品牌忠诚型 | 10% | 认准品牌，复购率高 | 高 |

## 二、需求优先级清单

### 2.1 核心需求（按优先级排序）

| 优先级 | 需求 | 描述 | 影响人群 |
|--------|------|------|---------|
| P0 | 正品保障 | 担心假货，需要正品认证 | 全品类 |
| P0 | 价格竞争力 | 同等质量下价格最优 | 价格敏感型 |
| P0 | 快速配送 | 大城市当日/次日达 | 全品类 |
| P1 | 产品质量 | 做工精良、材质安全 | 品质追求型 |
| P1 | 售后保障 | 无忧退换、保修 | 全品类 |
| P1 | 本地化体验 | 越南语客服、本地支付方式 | 全品类 |
| P2 | 品牌故事 | 品牌文化、价值观认同 | 品质/品牌型 |
| P2 | 社交分享 | 产品适合拍照分享 | 潮流追随型 |
| P2 | 个性化 | 定制化、独特设计 | 品质追求型 |

### 2.2 各品类需求差异

| 品类 | 首要需求 | 次要需求 | 痛点 |
|------|---------|---------|------|
| 美妆个护 | 正品+成分安全 | 效果+品牌 | 假货多 |
| 电子配件 | 性价比+功能 | 耐用+售后 | 质量参差 |
| 家居收纳 | 实用+设计 | 价格+质量 | 同质化 |
| 宠物用品 | 安全+可爱 | 品质+品牌 | 选择少 |
| 母婴用品 | 安全+认证 | 品质+品牌 | 信任门槛高 |

## 四、差评痛点分析

### 4.1 高频差评关键词

| 关键词（越南语） | 翻译 | 出现频率 | 关联品类 |
|----------------|------|---------|---------|
| khác hình | 与图片不符 | 35% | 美妆/服装 |
| chất lượng kém | 质量差 | 28% | 电子/家居 |
| giao hàng chậm | 配送慢 | 25% | 全品类 |
| hàng giả | 假货 | 18% | 美妆/电子 |
| không đúng size | 尺寸不对 | 15% | 服装/鞋靴 |
| không hoàn tiền | 不退款 | 12% | 全品类 |
| hỏng/hư | 损坏 | 10% | 电子/易碎品 |

### 4.2 差评根因分析

| 根因 | 占比 | 解决方案 |
|------|------|---------|
| 描述/图片夸大 | 30% | 真实展示+视频验证 |
| 质量控制不足 | 25% | 出厂质检+抽检 |
| 物流损坏 | 20% | 加强包装+保险 |
| 假货/仿品 | 15% | 正品授权+防伪 |
| 售后推诿 | 10% | 标准化售后流程 |
