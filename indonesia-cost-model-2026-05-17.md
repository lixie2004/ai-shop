## 核心假设与参数

### 平台费率（2026年Q2 有效）

| 费用项 | Shopee Indonesia (Non-Mall) | TikTok Shop by Tokopedia (Marketplace) |
|--------|---------------------------|----------------------------------------|

> 注：Shopee Indonesia 2026年5月21日起上调本地卖家佣金，电子品类保持稳定，时尚/FMCG/生活方式上调1–3%。本模型采用调整后费率。TikTok Shop by Tokopedia 自2026年2月11日起采用新佣金结构。

| 税项 | 税率 | 说明 |
|------|------|------|

| 增值税（PPN） | 11%（实际） | 名义12%，有效税率11%（DPP Nilai Lain 11/12） |

| 企业所得税（PPh Badan） | 22% | 按净利润征收 |

## 参数来源

| 参数 | 数据来源 |

| PPN/VAT 11%（有效） | PwC Indonesia Pocket Tax Book 2026 + pajak.go.id |

| 尾程配送费 | Shopee Logistik标准费率 + JNE/J&T标准 |

| 汇率基准 | 2026年5月IDR/USD中间价16,500 |

## 计算公式

```
总成本 = 采购成本 + 头程物流 + 进口关税 + PPN(VAT) + PPh 22
       + 平台佣金 + 交易费 + 广告费 + 尾程配送 
       + 退货损耗 + 仓储费 + 汇率缓冲 + 包装耗材 
       + 认证分摊 + App分摊 + 促销分摊

综合毛利率 = (零售价 - 总成本) / 零售价 × 100%

净利率 = (零售价 - 总成本 - 企业所得税22%) / 零售价 × 100%

盈亏平衡月销量 = 固定成本 / (单件毛利 × 6个月)
```

## 风险声明

1. **汇率风险**：IDR对USD/CNY年波动5–8%，所有进口SKU均暴露。建议使用远期锁汇或CNY结算。
2. **关税政策风险**：2025年PMK 4/2025将免税门槛降至$3，低价值包裹全面征税。东盟-中国FTA（FORM E）可大幅降低关税。
3. **平台费率上升风险**：Shopee Indonesia 2026年5月21日再次上调佣金。TikTok Shop by Tokopedia 2026年2月调整佣金结构。预计费率仍有上升空间。
4. **认证风险**：BPOM（保健品）周期6–12周，SNI（电子）周期8–12周，Halal认证4–8周。建议提前启动。
5. **合规风险**：印尼对纺织品（#3、#9）、化妆品/食品（#1、#8）进口监管趋严。确保HS编码准确、认证齐全。
6. **本模型为理论测算**，实际毛利率受库存周转、季节波动、平台促销力度、退货率波动等多因素影响。建议每季度更新一次成本模型。
