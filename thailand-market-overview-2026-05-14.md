# Thailand（泰国）跨境电商宏观环境分析简报

**报告日期**：2026年5月12日  
**编制**：趋势研究员  
**报告周期**：2025-2026 年度综合评估

---

## 执行摘要

Thailand 是东南亚第二大电商市场，也是**增长最快的市场之一**。2025年平台电商GMV增速达 **51.8%**，领跑东南亚（马来西亚47.6%、越南/菲律宾/新加坡均>20%）。Shopee（~50%+）、TikTok Shop（快速崛起至第二）、Lazada（第三）三巨头控制99%市场。泰国拥有 **7,160万人口**、人均GDP约 **US$7,080**、互联网渗透率超 **88%**，消费者数字化程度极高。跨境进口关税以 **0-30%** 为主，VAT 7%，清关效率在东盟中属中上水平。PromptPay 实时支付系统覆盖全民，COD（货到付款）仍占重要份额。对中国卖家而言，Thailand 是东南亚人口最多、增速最快、社交电商渗透最深的市场，但竞争烈度也在快速上升，**2025-2026年是入场窗口期的最后阶段**。

## 一、电商市场规模（GMV）及增速

### 1.1 市场规模

| 指标 | 数值 | 来源 |
|------|------|------|
| 2025年东南亚平台电商总GMV | **US$157.6B** | Momentum Works 2026 |
| 2025年泰国平台电商GMV增速 | **+51.8% YoY** | Momentum Works 2026 / Yicai |
| 泰国在东南亚增速排名 | **#1**（51.8%），马来西亚#2（47.6%） | Momentum Works 2026 |
| Shopee 2025年GMV（全区域） | US$83.2B（2024年为US$66.8B） | KR Asia / Nikkei |
| TikTok Shop 2025年GMV（全区域） | US$45.6B（2024年为US$22.6B，翻倍） | KR Asia / Nikkei |
| Lazada 2025年GMV（全区域） | US$18B（与2024年基本持平） | KR Asia / Nikkei |
| 内容电商（直播+短视频）GMV | US$49.7B（占东南亚总GMV 31.5%） | KR Asia / Nikkei |

### 1.2 增速分析

- **泰国是东南亚增长极**：51.8% 的增速远超区域平均（22.8%），是新加坡（~20%）的2.5倍
- **TikTok Shop 是最大变量**：全区域GMV从US$22.6B翻倍至US$45.6B，泰国是TikTok Shop增长最快的市场之一
- **内容电商爆发**：直播+短视频带货已占东南亚总GMV的31.5%，泰国是核心战场
- **增长驱动因素**：
  - 智能手机渗透率超90%，移动端购物占主导
  - TikTok/Line/Instagram 社交电商渗透率东南亚最高
  - PromptPay 实时支付普及降低交易摩擦
  - 政府"泰国4.0"数字经济政策持续推动

### 1.3 市场预测

| 时间线 | 预测 |
|--------|------|
| 2025-2030 | 东南亚电商GMV预计翻倍至US$410B（DBS预测） |
| 泰国角色 | 增速持续领跑，预计占东南亚总GMV 15-18% |
| 关键趋势 | 从"补贴驱动"转向"盈利优先"，内容电商占比持续提升 |

## 三、平台格局与市场份额

### 3.1 泰国平台市场份额（2025年）

| 平台 | 全区域市场份额 | 泰国市场地位 | GMV趋势 |
|------|-------------|-------------|---------|
| **Shopee**（Sea Ltd） | **~50%+** | **#1**，绝对领先 | 稳健增长 |
| **TikTok Shop**（ByteDance） | **~29%**（全区域） | **#2**，快速崛起 | **翻倍增长** |
| **Lazada**（Alibaba） | **~11%**（全区域） | **#3**，品牌导向 | 基本持平 |
| **其他**（JD Central/TikTok Shop泰国本土等） | ~10% | 边缘化 | 持续被挤压 |

**来源**：Momentum Works 2026 / KR Asia / Nikkei Asia

### 3.2 三巨头GMV对比（东南亚全区域）

| 平台 | 2024年GMV | 2025年GMV | 增速 |
|------|----------|----------|------|
| Shopee | US$66.8B | US$83.2B | +24.6% |
| TikTok Shop（含Tokopedia） | US$22.6B | US$45.6B | **+101.8%** |
| Lazada | ~US$18B | US$18B | ~0% |

### 3.3 各平台泰国市场特征

#### Shopee — 流量王者
- **优势**：最大用户基础，ShopeePay 生态，物流网络最广
- **泰国特色**：Shopee Mall（品牌旗舰店）增长快，Shopee Live 直播功能在泰国极受欢迎
- **卖家视角**：流量最大但竞争最激烈，佣金费率2-6%（因品类而异）
- **适合**：全品类，尤其是价格敏感型标品

#### TikTok Shop — 增长引擎
- **优势**：内容电商天然流量，泰国是TikTok全球渗透率最高的国家之一
- **泰国特色**：直播带货文化极浓，美妆/时尚/食品是Top品类
- **卖家视角**：内容制作能力是关键门槛，佣金费率约3-5%
- **适合**：美妆、时尚、食品、家居装饰等视觉驱动型品类
- **关键数据**：TikTok Shop在泰国日均直播场次东南亚第一

#### Lazada — 品牌阵地
- **优势**：Alibaba供应链支持，品牌旗舰店模式，Lazada Logistics物流
- **泰国特色**：AI购物助手用户近1亿，超50%产品页面使用AI技术
- **卖家视角**：品牌溢价空间更大，适合中高端定位
- **适合**：品牌型卖家、3C电子、家居、时尚品牌

#### JD Central（京东）— 小众玩家
- **定位**：正品保障、快速配送
- **市场份额**：<3%，持续边缘化

## 四、跨境电商政策与监管环境

### 4.1 进口关税框架

| 政策维度 | 详情 | 来源 |
|----------|------|------|
| **关税制度** | ASEAN Harmonized Tariff Nomenclature (AHTN)，8位编码 | Thai Customs |
| **进口关税** | **0-30%** 不等，多数消费品 0-10% | Thai Customs |
| **VAT（增值税）** | **7%**（2024年起，原计划调至10%但暂缓） | Thai Customs |
| **免税额度** | **1,500泰铢（~US$45）** 以下免关税 | Thai Customs |
| **跨境电商综合税** | 关税 + 7% VAT（多数品类实际税负 7-15%） | 行业估计 |
| **清关效率** | 电子清关，多数商品 **1-3个工作日** | 行业估计 |

### 4.2 跨境电商监管政策

| 政策领域 | 详情 |
|----------|------|
| **电商注册** | 外国电商企业需在泰国商业部注册，年销售额超180万泰铢需注册VAT |
| **消费者保护** | 《消费者保护法》适用于跨境电商，7天无理由退货（部分品类） |
| **数据隐私** | PDPA（个人数据保护法）2022年生效，对标GDPR |
| **产品认证** | 食品/化妆品/电子产品需FDA/TISI等认证 |
| **外汇管制** | 跨境支付需通过授权银行，外汇汇出有一定限制 |
| **平台责任** | 平台需对卖家资质审核负责，假货/侵权产品平台连带责任 |

### 4.3 对中国卖家的政策环境

| 维度 | 评估 | 说明 |
|------|------|------|
| **关税优惠** | ⭐⭐⭐ | 中国-东盟FTA（ACFTA）提供关税减免，多数商品0-5% |
| **原产地规则** | ⭐⭐ | 需满足40%区域价值成分（RVC）才能享ACFTA优惠 |
| **跨境电商试点** | ⭐⭐⭐⭐ | 泰国政府积极推动跨境电商，有专门的电商园区和优惠政策 |
| **外汇管制** | ⭐⭐ | 利润汇出需通过正规渠道，有一定手续 |
| **政治关系** | ⭐⭐⭐⭐ | 中泰关系稳定，RCEP框架下贸易便利化持续推进 |
| **竞争监管** | ⭐⭐⭐ | 暂无针对中国卖家的歧视性政策 |

### 4.4 关键合规要求

1. **FDA认证**：食品、化妆品、药品、医疗器械需泰国FDA批准
2. **TISI认证**：电子产品、建材等需泰国工业标准认证
3. **进口许可证**：部分品类（如食品、保健品）需额外进口许可
4. **标签要求**：产品标签需包含泰语信息
5. **VAT注册**：年销售额超180万泰铢（~US$50,000）必须注册VAT

### 5.2 支付生态

| 支付方式 | 市场份额 | 特征 |
|----------|---------|------|
| **PromptPay（实时转账）** | **~40-45%** | 泰国全民级实时支付系统，银行间免费转账 |
| **TrueMoney Wallet** | ~15-20% | 泰国最大电子钱包之一 |
| **ShopeePay** | ~10-15% | Shopee生态内支付 |
| **COD（货到付款）** | **~30-40%** | 泰国电商重要特征，不可忽略 |
| **信用卡/借记卡** | ~10-15% | 普及率持续提升 |
| **BNPL（先买后付）** | ~5-8% | Atome/Lazada PayLater快速增长 |
| **Line Pay** | ~3-5% | Line生态内支付 |

#### 支付生态关键洞察

1. **PromptPay 是基础设施**：泰国央行推出的实时支付系统，覆盖所有银行账户，是电商支付的核心
2. **COD 仍占重要份额**：30-40%的订单使用货到付款，跨境卖家必须支持
3. **电子钱包碎片化**：无单一主导钱包，需接入多个支付渠道
4. **BNPL 快速增长**：Atome、Lazada PayLater 等推动客单价提升
5. **跨境支付**：需通过泰国授权银行，PayPal/Stripe 在泰国有限制

## 七、综合评估

### 7.1 SWOT 分析（中国卖家视角）
