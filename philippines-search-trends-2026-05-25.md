### 修饰词聚类分析

**价格-优惠群（占修饰词的60%+）**
- affordable、cheap、best price、discount、sale、promo
- **解读**: 菲律宾消费者极度价格敏感，"affordable"是跨品类最高频修饰词
- **行动**: 广告文案必须突出价格优势

**便利群**
- online、free shipping、COD、delivery
- **解读**: 免费配送是转化关键因子，COD在美妆/母婴中转化率比非COD高2-3倍
- **行动**: 免邮门槛设置，COD广告组合规

**信任群**
- original、authentic、FDA approved、original sealed
- **解读**: 假货泛滥导致消费者对"正品验证"需求强烈
- **行动**: Listing中突出认证标识、防伪验证

**品质群（增长中）**
- organic、eco-friendly、natural、k-beauty
- **解读**: 中高收入群体愿为品质支付溢价
- **行动**: SKU分层，基础款走量+品质款走利润

## 3. 搜索意图分类

| 意图类型 | 占比 | 代表搜索词 | 转化潜力 |
|---------|------|----------|---------|
| **Transactional（交易型）** | **42%** | buy X online, X price, X shop | ⭐⭐⭐⭐⭐ 最高 |
| **Commercial（商业比较型）** | 21% | best X brand, X vs Y, X review | ⭐⭐⭐⭐ 高 |
| **Informational（信息型）** | 21% | how to use X, what is X, X benefits | ⭐⭐⭐ 中 |
| **Navigational（导航型）** | 16% | X Shopee, X Lazada, X official store | ⭐⭐⭐ 中 |

> **解读**: 交易型意图占比最高（42%），说明菲律宾消费者搜索时已有较强购买意愿。蓝海品类（健康保健、宠物用品）购物者购买意愿更强，关键词和广告优化的优先层级。
