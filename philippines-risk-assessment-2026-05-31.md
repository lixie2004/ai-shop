## 二、四维评级标准说明

| 分数区间 | 风险等级 | 含义 |
|:-------:|:-------:|------|
| 1-3 | 🟢 低风险 | 风险可控，常规管理即可 |
| 4-6 | 🟡 中风险 | 需制定专项应对措施 |
| 7-8 | 🟠 较高风险 | 需重点管理，必须有缓解方案 |

| 维度 | 权重 | 评估重点 |
|:----|:----:|---------|
| **市场风险** | 25% | 需求真实性、市场容量、竞争强度、价格敏感度、趋势持续性 |
| **合规风险** | 25% | 强制认证要求、周期、费用、成分/标签限制、法规变动 |
| **财务风险** | 25% | 真实毛利率、净利润率、COD拒收损耗、汇率影响、盈亏平衡点 |
| **运营风险** | 25% | 物流适配性、退货率、库存管理、平台运营复杂度、供应链韧性 |

**综合风险分 = 市场×25% + 合规×25% + 财务×25% + 运营×25%**

## 五、关键发现与风险预警

### 5.1 🔴 严重发现

**发现 1：成本模型存在系统性偏差**

| 偏差类型 | 影响范围 | 严重程度 |
|:--------|:--------|:-------:|
| **服装 15% 关税完全未计入** | #6 Rashguard | 🔴 致命——净利率从 16% 降至可能亏损 |
| **大件物流成本严重低估** | #2 Cat Tree Tower、#5 Yoga Mat | 🟠 较高——实际成本可能高 50-100% |
| **选品清单毛利率系统性高估 5-8pp** | 全品类 | 🟡 中——影响所有 SKU 的预期利润 |
| **#10 护肤套装 FDA 认证成本被低估 2-3 倍** | #10 Korean Skincare | 🟠 较高——实际认证费用可能达 ₱15,000-30,000 |
| **#2 成本模型总成本可能偏低 20-30%** | #2 Cat Tree Tower | 🟠 较高——含关税后形势更严峻 |

**建议**：对 #2 Cat Tree、#5 Yoga Mat、#6 Rashguard 三个 SKU 的成本模型进行**独立复核确认**后再做最终决策。

**发现 2：FDA 认证风险是方案命门**

新方案 3 个 SKU 需 FDA 认证（#3 防晒霜 / #4 精华 / #10 护肤套装含 3 件），总费用可能达 ₱25,000-55,000（上轮方案的 FDA 需求为 0）。

| 认证项目 | SKU | 预估费用 (₱) | 预估周期 | 可合并认证 |
|:--------|:---:|:-----------:|:-------:|:---------:|
| FDA 化妆品通知（防晒霜） | #3 | 5,000-15,000 | 10-12 周 | 可共用进口商 |
| FDA 化妆品通知（精华液） | #4 | 5,000-10,000 | 8-10 周 | 可与 #3 共用 |
| FDA 化妆品通知（套装×3） | #10 | 15,000-30,000 | 10-16 周 | 可拆分单独认证 |
| **合计** | | **₱25,000-55,000** | | |

**关键建议**：
1. **Day 1 必须启动全部 FDA 认证流程**——认证周期 8-16 周，无认证无法上架
2. **#3 + #4 可共用 FDA LTO 挂靠进口商**——节省约 ₱5,000-10,000
3. **#10 强烈建议拆分为 3 个独立 SKU**——分别认证更灵活、可独立调价/促销、降低套装退货风险
4. 如不拆分，#10 的认证复杂度和成本将使方案可行性进一步下降

### 5.2 🟡 中等发现

**发现 3：宠物用品矩阵是方案核心亮点**

3 个宠物品类 SKU（#1 美容套装 + #9 环保玩具 + #2 猫爬架）形成产品矩阵：
- #1 + #9：零认证 + 低风险 + 高利润（20%+28%），可立即上架
- #2：高客单价 + 高利润潜力但物流挑战大，需修正成本模型后决策
- **宠物用品组合套装**（美容+清洁+玩具 = ₱399+₱249 = ₱648）可作为套装 SKU 上线，提升客单价和利润

**发现 4：运营复杂度被低估**

| SKU | 运营复杂度来源 | 影响 |
|:---:|:------------|:----|
| #2 Cat Tree | 大件运输/仅 Luzon/高 COD 拒收损失 | 需海外仓前置 |
| #6 Rashguard | 多色×多码 SKU 爆炸/高退货率/季节性 | 需精准尺码+季节库存管理 |
| #10 Skincare Set | 3 产品独立管理+套装不可拆零 | 需拆分或接受高风险 |
| #8 Blender | 电子售后/VAT 风险/竞争红海 | 需备售后成本池 |

## 八、声明与免责

1. 本评估基于用户提供的 2026-05-31 版本 3 份报告独立出具，数据未经独立核实。
2. 成本模型数据（净利率/物流成本/COD 损失等）来自 `philippines-cost-model-2026-05-31.md`，部分 SKU（尤其是大件和服装类）发现成本模型存在系统性偏差，**建议实际进入前逐 SKU 复核**。
3. FDA 认证周期为基于合规清单的行业估算，实际以 FDA Philippines 官方审批结果为准。FDA 对防晒霜的 SPF 声称有额外功效测试要求，周期可能比预期更长。
4. "本地配方"（#3 防晒霜）的差异化定位需补充实质支撑材料，否则难以转化为有效卖点。
5. FORM E 不适用（中国非东盟成员国）为菲律宾市场的结构性劣势，本轮方案仍有 3 个高利润 SKU 受此影响（护肤品类 15% 关税）。
6. VAT 免税额政策为评估基准，如菲律宾政府取消 <USD 50 免税额政策，整个方案的盈利模型需重新计算。
7. 本报告不构成投资决策的唯一依据，建议结合业务团队的实地调研、样品测试和试销数据综合决策。
