## 关键认证详解

### BPOM (食品药品管理局)
| 项目 | 详情 |
|------|------|
| **适用产品** | 化妆品、食品、补充剂、医疗器械 |
| **费用** | $2,000-4,000 |
| **周期** | 3-6个月（医疗器械4-8个月） |
| **要求** | 成分列表+检测报告+GMP |
| **有效期** | 5年 |
| **注意** | 需印尼本地代表(Agent) |

### Halal认证 (BPJPH/MUI) — ⚠️ 2026年10月17日强制生效
| 项目 | 详情 |
|------|------|
| **适用产品** | 食品、化妆品、个人护理、化学品、消费品及包装 |
| **费用** | $1,000-3,000 |
| **周期** | 2-4个月 |
| **要求** | 原料清真+生产过程审核 |
| **有效期** | 2-4年 |
| **⚠️ 强制日期** | **2026年10月17日** |

### SNI (印尼国家标准)
| 项目 | 详情 |
|------|------|
| **适用产品** | 电子产品、电器、玩具 |
| **费用** | $500-2,000 |
| **周期** | 1-3个月 |

## 进口监管要求（2025-2026更新）

### 关税与税收
| 项目 | 税率/要求 |
|------|----------|
| 进口关税 | 0-25%（MoF Reg. 4/2025简化为三档） |
| 增值税VAT | 11%（2025年维持） |
| 所得税(PPh22) | 7.5-10% |
| 免税额度 | $3/天/人（从$75降级） |
| 最低进口价 | $100/件（外国商家） |

### 跨境电商特殊要求
- PMSE牌照注册
- 数据本地化存储
- 印尼本地代表指定
- 产品标签印尼语
