## 一、新加坡合规框架概览

### 1.1 监管机构

| 机构 | 全称 | 监管范围 |
|------|------|---------|
| **HSA** | Health Sciences Authority | 保健品、化妆品、药品 |
| **IMDA** | Infocomm Media Development Authority | 无线/电子产品 |
| **SFA** | Singapore Food Agency | 食品安全 |
| **AVA** | (已并入SFA) | 宠物食品 |
| **Singapore Customs** | 新加坡海关 | 进口关税、GST |
| **PDPC** | Personal Data Protection Commission | 数据隐私 |

### 1.2 核心法规

| 法规 | 适用范围 | 关键要求 |
|------|---------|---------|
| **Health Products Act** | 保健品 | HSA注册 |
| **Consumer Protection (Safety Requirements) Act** | 消费品安全 | 安全标准 |
| **Telecommunications Act** | 无线设备 | IMDA认证 |
| **Wholesome Meat and Fish Act** | 食品 | SFA进口许可 |
| **GST Act** | 所有商品 | 9%消费税 |
| **PDPA** | 数据 | 个人数据保护 |

## 二、Top 10 产品合规要求

### 2.1 健康保健补剂（NMN胶囊、维D3+K2软糖、益生菌软糖）

| 项目 | 要求 |
|------|------|

| **申请周期** | 2-4 个月 |
| **申请费用** | SGD 200-500/产品 |
| **所需文件** | 产品配方、制造工艺、安全数据、功效声称依据、GMP证书 |

## 三、GST 合规要求

### 3.1 GST 注册门槛

| 条件 | 要求 |
|------|------|
| 年营业额 > SGD 100万 | 强制注册 |
| 年营业额 < SGD 100万 | 自愿注册 |
| 跨境电商 | 无论金额均需注册（2024年起） |

### 3.2 GST 征收规则

| 规则 | 说明 |
|------|------|
| **标准税率** | 9% |
| **零税率** | 出口商品 |
| **免税** | 金融服务、住宅物业 |
| **低值商品** | 2024年起取消免税门槛，所有进口商品均需缴纳9% GST |

### 3.3 平台代扣代缴

## 四、数据隐私合规（PDPA）

### 4.1 PDPA 核心要求

| 要求 | 说明 |
|------|------|
| **同意** | 收集个人数据需获得用户同意 |
| **目的限制** | 只能用于收集时的目的 |
| **数据保护** | 必须采取合理安全措施 |
| **跨境传输** | 需确保接收国提供同等保护 |

### 4.2 电商平台合规

| 场景 | 合规要求 |
|------|---------|
| 用户注册 | 隐私政策 + 同意收集 |
| 支付信息 | PCI DSS合规 |
| 营销邮件 | 需获得同意 + 退订选项 |
| 数据保留 | 不得无限期保留 |
