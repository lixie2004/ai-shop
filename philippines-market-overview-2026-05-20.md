### 1.3 东南亚区域排名

- 菲律宾已超越越南，成为东南亚**第三大**平台电商市场（仅次于印尼和泰国）

- 电商占整体零售额的 **5%**（2024年），对比中国（~30%）、印尼（~12%），渗透率仍低，增长空间巨大

- 网购人均年消费：2024年 **273美元** → 2025年预计 **302美元**

### 2.4 平台竞争格局分析

**Shopee（Sea Group）**：
- 菲律宾电商绝对霸主，55%份额，护城河来自高频购物习惯（年均11单/人）
- 2025年Q4推出 **Instant Mart** 计划，利用ShopeeFood基础设施实现4小时内送达
- 2026年初通知卖家 **J&T Express** 不再作为物流选项，整合至自有配送网络
- 自有支付（ShopeePay/SPayLater）占交易量约40%

**TikTok Shop（字节跳动）**：
- 增速最快的平台，FY 2025 GMV +53% YoY，从22%→29%份额
- Q4 2025季度GMV达20亿美元，单季份额35%，**超越Lazada成为第二大平台**
- 直播带货（Live-selling）是核心驱动力，美妆品类28%的GMV占比为各平台最高
- 冲动消费特征明显，客单价最低（PHP 198）

**Lazada（阿里巴巴）**：
- GMV从FY 2024的约45亿降至FY 2025的30亿（-34%），份额从28%→16%
- 战略转向品牌化/高端化，客单价最高（PHP 416）
- LazMall正品保障在电子品类仍有优势（电子占其GMV 26%）
- 面临Shopee规模压制 + TikTok Shop内容电商的双重夹击

**其他平台**：
- **Temu**（PDD Holdings）：2023年底进入菲律宾，以"工厂直供"超低价策略快速获客
- **Amazon**：在菲律宾流量排名第三（Shopee、Lazada之后），主要面向跨境购物
- **Carousell**：二手/长尾商品交易平台
- **Zalora**：时尚垂类平台

### 5.1 跨境电商规模

| 指标 | 数据 | 来源 |

| 跨境平台偏好 | Shopee、Lazada、Amazon、Carousell、Temu | PaymentsCMI |

### 5.2 关税与税收制度

**菲律宾关税制度概述**：

| 项目 | 详情 |
|------|------|
| **关税分类** | 进口商品分为4类：自由进口、管制、限制、禁止 |
| **关税税率** | 大部分商品适用 **0%~10%** 的MFN税率；东盟内部适用CEPT优惠税率（0%~5%） |
| **增值税（VAT）** | **12%**（进口环节征收，CIF价+关税为税基） |
| **消费税** | 对特定商品（烟酒、汽车、石油等）征收 |
| **进口许可** | 管制类商品需提前获得进口许可/清关证明 |
| **原产地证明** | FTA项下优惠关税需提供原产地证明 |

**跨境电商特殊政策**：

| 项目 | 详情 |
|------|------|
| **免税额度（De Minimis）** | 菲律宾**无明确的跨境电商免税门槛**（不像美国$800或欧盟€150），所有商业进口原则上均需缴纳关税和VAT |
| **个人物品免税** | 价值较低的个人自用物品可能享有一定免税待遇，但具体金额由海关裁量 |
| **BOC监管** | 菲律宾海关总署（Bureau of Customs, BOC）负责所有进口商品监管 |
| **清关流程** | 需提交提单/空运单、商业发票、装箱单、补充估值声明（SDV）等 |
| **税务登记** | 进口商需在BIR（税务局）注册，获取TIN |

### 5.3 跨境电商监管框架

| 法规/政策 | 内容 |
|-----------|------|
| **电子商务法（RA 8792）** | 2000年颁布，为电子交易提供法律框架 |
| **互联网交易法（ITA）** | 2023年12月颁布，建立电商运营规范，设立电商局（E-Commerce Bureau），创建在线争议解决平台 |
| **联合行政令 22-01** | 2022年发布，强化线上业务合规要求，保护消费者 |
| **电商路线图 2022** | DTI制定，推动MSME数字化 |
| **消费者保护** | 在线争议解决系统（PODRS）正在建设中 |
| **数据隐私** | 《数据隐私法》（RA 10173），由NPC监管，与GDPR和APEC CBPR对齐 |
| **知识产权** | 《知识产权法典》和《假冒产品特别法》要求电商平台执行IP权利 |

### 5.4 外资准入

| 要求 | 详情 |
|------|------|
| 外资持股比例 | 允许**100%外资持股**（通过设立菲律宾子公司） |
| 最低实收资本 | **$20万美元** |
| 注册要求 | 需向BIR、DTI、SEC注册 |
| 产品许可 | 特定产品需FDA等监管机构许可 |

### 5.5 跨境贸易关键挑战

- **关税不确定性**：缺乏明确的跨境电商免税门槛，低值包裹的关税处理存在灰色地带
- **清关效率**：BOC清关流程复杂，文件要求多，可能导致延误
- **VAT合规**：12% VAT增加了跨境商品成本
- **监管趋严**：Momentum Works报告指出，跨境平台面临的监管审查正在增加
- **物流瓶颈**：群岛地理导致跨境物流成本高、时效长

### 6.3 物流基础设施现状

**优势**：
- **平台自建物流**：Shopee的SPX Express已建立全国性分拣网络，2024年最大分拣中心日处理300万件
- **微履约中心**：主要平台在马尼拉等核心城市布局微履约中心，缩短配送距离
- **COD网络**：成熟的COD收款网络覆盖全国，支持货到付款模式
- **数字支付集成**：GCash/Maya与物流系统深度集成

**挑战**：

| 挑战 | 详情 |
|------|------|
| **地理障碍** | 7,600+岛屿，岛际运输依赖海运和空运，成本高、时效慢 |
| **交通拥堵** | 马尼拉等大城市严重拥堵，延长配送时间，增加燃油成本 |
| **基础设施不足** | 仅22,405座蜂窝基站（需7万座），道路网络在欠发达地区质量差 |
| **最后一公里成本** | 消费者期望低/免运费，但运营成本高，利润压力大 |
| **人力短缺** | 配送人员招聘和留存困难，劳动力市场竞争激烈 |
| **农村覆盖** | 偏远省份配送网络不完善，配送时效长 |

### 6.4 物流技术趋势（2025）

| 趋势 | 应用 |
|------|------|
| **自动化与机器人** | 仓库自动化分拣，减少人工依赖 |

### 7.1 SWOT 分析

| | 正面 | 负面 |

### 7.3 对跨境电商卖家的建议

1. **平台策略**：以Shopee为主力（55%份额），TikTok Shop为增长引擎（美妆/时尚品类必做），Lazada选择性布局（电子/品牌商品）
2. **移动优先**：所有listing、图片、checkout流程必须移动端优化
3. **直播带货**：菲律宾是东南亚直播购物渗透率最高的国家之一，美妆品牌尤其需要建立直播能力
4. **COD管理**：提供COD选项（尤其面向外省市场），但需管控退货率和现金流风险
5. **物流选择**：优先使用平台自有物流（SPX Express），外省市场考虑Ninja Van/J&T
6. **跨境合规**：关注BOC关税政策变化，确保VAT合规，利用FTA原产地优惠
7. **支付集成**：必须接入GCash（89%电子钱包份额），同时支持信用卡和COD
8. **大促节奏**：Q4（9.9、11.11、12.12）占全年GMV约30%+，需提前3个月备货和营销准备
