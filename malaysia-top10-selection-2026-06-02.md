## 筛选方法论

### 评估维度说明

| 维度 | 权重 | 评分标准 |
|------|------|---------|
| 月搜索量 | 30% | 5,000–10,000 = 5分 / 10,000–20,000 = 8分 / 20,000+ = 10分 |
| 预估毛利率 | 25% | 25–35% = 5分 / 35–45% = 8分 / 45%+ = 10分 |
| 竞争度 | 20% | 高分 = 竞争低（蓝海），低分 = 竞争高（红海） |
| 合规风险 | 15% | 低风险 = 10分 / 中风险 = 5分 / 高风险 = 0分（一票否决） |
| TikTok 内容杠杆 | 10% | 高热度 = 10分 / 中热度 = 5分 / 低热度 = 2分 |

### 硬性筛选条件（一票否决）

| 条件 | 说明 |
|------|------|
| ❌ 月搜索量 ≤ 5,000 | 需求不足，无法支撑销售 |
| ❌ 预估毛利率 ≤ 25% | 扣除平台佣金+物流+营销后利润空间不足 |
| ❌ 合规风险不可控 | NPRA 违禁成分/未注册产品/JAKIM 违规=直接排除 |
