## 一、执行摘要

### 1.1 核心发现

| 维度 | 关键数据 |
|------|----------|
| 市场规模 | 2025年GMV $35.5B，同比增速 **51.8%**（东南亚最快） |
| 平台格局 | Shopee 50% + Lazada 30% + TikTok Shop 21%（三强垄断98.8%） |
| 推荐品类 | 家居收纳、LED照明、宠物用品 |
| 首选SKU | 真空压缩袋、衣柜分层架、厨房收纳架、冰箱收纳盒 |
| 关键门槛 | TIS认证（3-6个月）、FDA备案（2-4个月） |
| 初期投入 | 约27,000 CNY（$3,800 USD） |
| 预期ROI | 6个月33%-200% |

### 1.2 进入建议

> **推荐以"先易后难"策略进入泰国市场**：
> - **第一阶段（0-3月）**: 4个无/低认证门槛SKU快速上线，验证市场
> - **第二阶段（3-6月）**: 推进TIS认证，上线LED照明品类
> - **第三阶段（6-12月）**: FDA备案，布局宠物用品品类

### 3.2 泰国特有消费特征

1. **租房文化盛行** → 可拆卸、便携式家居产品需求高
2. **热带气候** → 防晒、驱蚊、降温类产品需求持续
3. **宠物经济崛起** → 猫主人比例高，猫粮/猫砂/猫玩具需求旺盛
4. **通勤文化** → 便携水杯、收纳包、车载用品需求大
