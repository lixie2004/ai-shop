# 印度尼西亚用户需求挖掘报告

**报告日期**：2026年5月11日  
**分析师**：产品经理 / Emma XIE  
**报告类型**：用户画像与需求分析  
**目标市场**：印度尼西亚（Indonesia）

---

## 执行摘要

基于竞品评论分析、社交媒体舆情和消费者调研数据，本报告构建了印尼电商用户的四大核心画像，并提炼了S/A级品类的用户需求优先级。核心发现：印尼消费者是"价值敏感型"而非"价格敏感型"——他们愿意为品质+认证+服务支付溢价，但需要明确的信任信号。

## 一、用户画像

### 画像1：都市穆斯林女性（核心目标用户）

| 维度 | 描述 |
|------|------|
| **年龄** | 22-35岁 |
| **城市** | 雅加达、泗水、万隆、棉兰 |
| **职业** | 白领、创业者、全职妈妈 |
| **月收入** | IDR 5-15M ($300-950) |
| **月网购支出** | IDR 300-800K ($19-50) |
| **常用平台** | Shopee (日常) + TikTok Shop (发现) |
| **日均社媒** | 3.5+小时 |
| **核心需求** | 品质好、Halal认证、价格合理、款式时尚 |
| **购买决策因素** | 认证 > 评价 > 价格 > 品牌 > 物流 |

**典型场景**：
- 早上刷TikTok看到KOL推荐hijab → 点击Shopee链接 → 查看评论和BPOM认证 → 下单
- 斋月前1个月开始搜索"baju lebaran"（开斋节服装）
- 在社群（WhatsApp/IG）分享购物链接，信任朋友推荐

## 三、购买决策旅程

### 3.1 TikTok Shop 决策路径

```
发现（FYP推荐） → 兴趣（点击视频） → 信任（看评论+评分） → 决策（价格+认证） → 购买（一键下单）
     ↓               ↓                  ↓                      ↓
   KOL种草        产品展示          在线评论              低价+BNPL
   直播带货       使用教程          BPOM/Halal           免邮+闪购
```

### 3.2 Shopee 决策路径

```
搜索（关键词） → 浏览（列表页） → 比较（详情页） → 信任（评价+评分） → 购买
     ↓               ↓                  ↓                      ↓
  品类词+修饰词   主图+价格          详情+尺码表         评论+问答
```

### 3.3 关键决策因素权重

| 决策因素 | 权重 | 说明 |
|---------|------|------|
| 在线评价 | 30% | 4.5+评分产品转化率高出40% |
| 认证标识 | 25% | BPOM/Halal/SNI直接影响信任 |
| 价格 | 20% | 中价位（非最低价）最优 |
| 品牌 | 15% | 本土品牌信任度高于跨境 |
| 物流时效 | 10% | 2-3天达是基准线 |

## 五、用户需求总结

### 核心洞察

1. **信任 > 价格** — 印尼消费者愿意为认证和正品支付溢价
2. **品质 > 款式** — 面料/材质/耐用性比设计更重要
3. **认证 = 信任入口** — BPOM/Halal/SNI是购买决策的"通行证"
4. **社交驱动** — TikTok和KOL是产品发现的第一触点
5. **价值敏感** — 追求"value for money"，不是单纯低价

### 产品策略建议

| 品类 | 核心卖点 | 信任信号 | 内容策略 |
|------|---------|---------|---------|
| Modest Fashion | 品质面料+时尚设计 | 面料检测报告 | 试穿视频+OOTD |
| Halal Beauty | 认证齐全+成分透明 | BPOM+Halal双认证 | 成分解析+使用教程 |
| 母婴 | 安全认证+多功能 | SNI+无毒报告 | 妈妈社群+教育内容 |
| 家居收纳 | 尺寸精准+材质结实 | 承重测试视频 | 开箱+安装教程 |
| 手机配件 | 兼容性+品质保证 | 兼容性列表+保修 | 对比评测+使用场景 |
