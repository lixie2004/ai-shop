# Indonesia（印度尼西亚）跨境电商宏观环境分析简报

**报告日期**：2026年5月12日  
**编制**：趋势研究员  
**所属项目**：SEA-ENTRY-2026 东南亚市场进入战略  
**报告周期**：2025-2026 年度综合评估

---

## 执行摘要

Indonesia 是**东南亚最大的电商市场**，占东盟电商总GMV的 **52%以上**。2025年市场规模达 **US$90.35B**，预计2026年达 **US$104.21B**，2031年达 **US$212.58B**（CAGR 15.32%）。拥有 **2.84亿人口**（全球第四）、互联网用户 **2.29亿**（渗透率80.66%）、智能手机普及率极高。平台格局呈 **Shopee（53.22%）> TikTok Shop（27.37%）> Tokopedia（9.57%）> Lazada（9.09%）** 的梯级分布。消费者对价格敏感，COD占比约 **64%**，数字钱包占线上支付 **40.60%**。对中国卖家而言，Indonesia 是**东南亚绝对的人口红利市场**，但监管趋严（PMK 4/2025关税改革、GR 31/2023社交电商禁令）、物流碎片化、COD风险是三大核心挑战。

### 1.2 增速分析

- **Indonesia 是东南亚电商的绝对主力**：占东盟总GMV超52%，是新加坡（~5-6%）的近10倍
- **增长驱动因素**：
  - **直播电商爆发**：60%网购用户已通过直播购物，视频电商占线上GMV达20%（2022年<5%），推动GMV增长+3.2%
  - **二三线城市数字化**：二三线城市贡献50%电商GMV（2021年为40%），物流微枢纽网络降低配送成本12-18%
  - **金融科技推动**：BNPL（先买后付）推动AOV提升30-50%，BNPL贷款簿达IDR 35.14万亿（US$2.27B）
  - **政府政策**："Making Indonesia 4.0"推动1,900万家MSME上线，QR支付覆盖90%商家
  - **跨境卖家增长**：通过新加坡和中国网关的跨境卖家持续增加

## 三、平台竞争格局

### 3.1 平台市场份额（2025年APJII调查，N=8,700）

| 平台 | 用户访问率 | 排名 | 趋势 |

| **Tokopedia** | **9.57%** | **#3** | 守势，与Lazada胶着 |

### 3.2 全区域GMV视角（Momentum Works 2026）

| 平台 | 全区域GMV | 在Indonesia的地位 |
|------|----------|-----------------|
| Shopee（全区域） | US$83.2B | #1 |
| TikTok Shop（全区域，含Tokopedia） | US$45.6B | #2（TikTok-Tokopedia合并后协同效应显著） |
| Lazada（全区域） | US$18B | #3-4 |

### 3.3 各平台印度尼西亚市场深度分析

#### Shopee（Sea Ltd）— 绝对王者
- **用户覆盖率**：53.22%（男性54.03%，女性52.39%，全年龄段领先）
- **优势**：最大用户基础，ShopeePay生态，物流网络最广，Shopee Mall品牌旗舰店
- **Indonesia特色**：Shopee Live直播功能极受欢迎，免运费促销是核心获客手段
- **卖家视角**：流量最大但竞争最激烈，佣金费率2-6%

#### TikTok Shop（ByteDance + Tokopedia）— 颠覆者
- **用户覆盖率**：27.37%（从2024年的12.20%翻倍）
- **关键事件**：2024年与Tokopedia合并，整合后GMV快速增长
- **Indonesia特色**：直播电商GMV占平台20%，转化率是静态广告的3倍。Gen Z核心用户
- **监管风险**：2024年因GR 31/2023被暂停2个月，后通过与Tokopedia合并合规
- **卖家视角**：内容制作能力是关键门槛，适合美妆/时尚/食品等视觉驱动品类

#### Tokopedia（GoTo Group）— 本土巨头
- **用户覆盖率**：9.57%
- **优势**：印尼本土最大电商平台之一，GoTo生态（Gojek/GoPay/Tokopedia协同）
- **挑战**：被TikTok Shop快速追赶，市场份额受压
- **卖家视角**：本土信任度高，适合品牌型卖家

#### Lazada（Alibaba）— 守势
- **用户覆盖率**：9.09%（从17.54%大幅下降）
- **品牌策略**：转向品牌旗舰店模式，AI购物助手
- **挑战**：在印尼市场持续被Shopee和TikTok Shop挤压
- **卖家视角**：高端品牌定位，但流量持续下滑

#### Blibli — 小众本土玩家
- **份额**：<0.5%，持续边缘化

### 3.4 跨平台GMV集中度

- **Shopee + TikTok Shop** 合计覆盖超 **80%** 网购用户
- **四大平台**（Shopee + TikTok Shop + Tokopedia + Lazada）合计覆盖超 **99%** 网购用户
- 市场呈明显的"两超多强"格局

## 五、政策环境与外资准入限制

### 5.1 进口关税框架（PMK 4/2025新规）

| 政策维度 | 详情 | 影响 |
|----------|------|------|
| **免税额度** | **从US$75降至US$以下**（2025年3月5日生效） | 🔴 **重大利空**——跨境小包几乎不再免税 |
| **关税税率** | 0%/15%/25%（阶梯式，按商品类别） | 多数消费品15-25% |
| **VAT（增值税）** | **11%**（标准），奢侈品12% | 所有进口商品均需缴纳 |
| **所得税** | 5%（关税15%/25%商品）或0%（免税商品） | 增加综合税负 |
| **综合税负** | 多数品类：15-25%关税 + 11% VAT + 5%所得税 = **31-41%** | 对低价跨境商品影响巨大 |
| **ACFTA优惠** | 中国-东盟FTA提供0-5%关税（需满足原产地规则） | ✅ 利好正规进口 |
| **Batam例外** | Batam自由贸易区仍免税（但不适用外运至其他岛屿） | 部分卖家利用Batam做转口 |

### 5.2 跨境电商监管政策

| 政策 | 详情 | 影响 |
|------|------|------|
| **GR 31/2023** | 禁止外国商家在电商平台销售<US$100商品；禁止社交媒体直接交易 | 🔴 重大利空——低价跨境直邮模式被封堵 |
| **GR 80/2019** | 电商基本法：消费者保护、电子合同、税务合规 | 合规基础 |
| **BPOM认证** | 食品/化妆品/药品需BPOM批准 | 高门槛 |
| **SNI认证** | 部分产品需印尼国家标准认证 | 中等门槛 |
| **数据本地化** | 用户数据需在印尼境内存储 | 增加IT成本 |
| **负面投资清单** | 部分行业限制外资持股比例 | 需本地合作伙伴 |

### 5.3 对中国卖家的政策评估

| 维度 | 评估 | 说明 |
|------|------|------|
| **关税政策** | 🔴 不利 | US$3免税额度几乎消灭低价跨境小包模式 |
| **商品价值限制** | 🔴 不利 | GR 31/2023禁止<US$100商品跨境直售 |
| **ACFTA优惠** | ✅ 利好 | 正规进口可享0-5%关税，需满足原产地规则 |
| **数据合规** | 🟡 中等 | 数据本地化要求增加成本 |
| **政治关系** | 🟡 中等 | 中印关系总体稳定，但需关注民族主义情绪 |

### 5.4 关键合规清单

1. **进口许可证（API/NIB）**：需通过印尼进口商资质
2. **BPOM认证**：食品/化妆品/药品强制
3. **SNI认证**：部分电子产品/建材强制
4. **清真认证（BPJPH）**：食品/化妆品/药品在穆斯林市场越来越重要
5. **VAT注册**：进口商品需缴纳11% VAT
6. **标签要求**：产品标签需含印尼语信息
7. **本地合作伙伴**：外资电商通常需本地合作伙伴或设立PT公司

### 7.3 中国卖家入场策略建议

#### 阶段一：合规先行（0-6个月）

| 行动 | 详情 |
|------|------|
| **设立本地实体** | 设立PT PMA（外资公司）或与本地合作伙伴合作 |
| **获取进口资质** | API/NIB进口许可证 |
