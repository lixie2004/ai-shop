## 1. 成本建模方法论与假设

### 1.1 成本项定义

每个 SKU 包含 **10 项** 成本明细：

| # | 成本项 | 说明 |
|---|--------|------|
| ① | **采购成本（COGS）** | 含产品制造/采购、出厂包装、QC 费用（基于 1688/义乌采购 + OEM 报价估算） |
| ② | **头程物流（海/空运）** | 中国港口 → Port Klang/KLIA，含报关费、提单费、港口杂费 |
| ③ | **进口关税** | 马来西亚海关进口税率（基于 HS Code 适用税率范围） |
| ④ | **尾程配送（Courier）** | Shopee Express / J&T / Ninja Van 等末端配送至消费者（西马） |
| ⑤ | **平台佣金** | Shopee: 交易佣金 + 服务费 + 支付手续费（合计约 8-10%） |
| ⑥ | **广告营销费（Paid Ads）** | 站内广告（Shopee Ads / TikTok Shop Ads）按 ACoS 百分比计算 |
| ⑦ | **退货损耗率** | 基于品类退货率预留成本（含退货运费 + 报废/翻新损失） |
| ⑧ | **SST（销售与服务税）** | 马来西亚 SST 6%（大多数商品适用税率） |
| ⑨ | **仓储费** | 海外仓/本地仓月度仓储（按体积/件数计费） |
| ⑩ | **汇率波动准备金** | 人民币/美元兑林吉特汇率波动预留（2-3%） |

### 1.2 核心假设

| 参数 | 假设值 | 依据 |
|------|--------|------|
| 汇率 CNY/RMY | 1 CNY ≈ 0.66 RM / 1 RM ≈ 1.52 CNY | 2026年6月 Bloomberg 中间价 |
| 汇率 USD/RM | 1 USD = 4.70 RM | 2026年6月 BNM 参考价 |
| SST 税率（大多数商品） | 6% | Sales Tax Act 2018 |
| SST 税率（清真食品） | 0%（免税） | MIDA 税收激励 |
| 海运头程 | RM 2.5-4.5/kg（拼柜 LCL） | 深圳/宁波→Port Klang，2025 Q4 均价 |
| 空运头程 | RM 18-28/kg | KLIA 清关 |
| 平台综合费率（Shopee） | 8-10% | Shopee MY 2025 卖家政策 |
| 平台综合费率（Lazada） | 7-9% | Lazada MY 2025 卖家政策 |
| 本地仓储 | RM 0.15-0.40/件/月 | 西马第三方仓均价 |
| 平均退货率（美妆/保健） | 5-8% | 电商行业均值 |
| 平均退货率（家居/服装） | 8-12% | 电商行业均值 |
| 广告 ACoS（成熟品类） | 15-25% | Shopee Ads 行业基准 |
| 广告 ACoS（新品牌冷启动） | 25-40% | 前 3 个月平均 |

## 6. 汇率波动敏感性分析

### 6.1 假设场景

| 场景 | CNY/RM | USD/RM | 对采购成本影响 | 对净利影响 |
|------|--------|--------|--------------|-----------|
| **基准（当前）** | 0.66 | 4.70 | 基准 | 基准 |
| **人民币升值 5%** | 0.693 | 4.70 | 采购成本↓5% | 净利↑约5-8% |
| **人民币贬值 5%** | 0.627 | 4.70 | 采购成本↑5% | 净利↓约5-8% |
| **美元升值 5%** | 0.66 | 4.94 | 美元原料↑5% | 净利↓约1-3% |
| **美元贬值 5%** | 0.66 | 4.47 | 美元原料↓5% | 净利↑约1-3% |
| **极端：人民币+美元双贬10%** | 0.60 | 5.17 | 采购成本↑10-15% | 净利↓约12-20% |

### 6.2 关键洞察

1. **采购成本人民币计价部分**约占总成本 30-50%，汇率波动 5% 影响净利约 3-8%
2. **汇率波动准备金**（每SKU 0.5-1.2%）可覆盖正常波动（±2-3%）
3. **Halal猫零食**和**智能感应灯**汇率敏感度最高（电子元件/益生菌原料以美元/欧元计价）
4. 建议使用**远期结汇**锁定采购汇率，或在采购合同中约定汇率波动共担条款

### 8.2 数据局限性

1. **采购成本**为行业估算值，实际成本需以供应商正式报价为准（MOQ、工艺、材质差异可导致±20-30%波动）
2. **广告ACoS**为品类平均值，实际值受品牌知名度、产品差异化程度和投放策略影响
3. **物流费率**基于当前油价和运力状况，旺季（Q4）可能有10-20%上浮
4. **本模型不包含**企业所得税（24%），如以马来西亚注册公司运营，最终净利润需扣除所得税
5. **web_search**因DuckDuckGo bot-detection限制不可用，外部实时数据未能获取。本报告数据基于专业知识和行业基准
