## 1. 马来西亚消费者画像

### 核心用户群体

| 维度 | 描述 |
|------|------|
| **年龄分布** | 18-34岁占网购人群 65%+ |
| **性别** | 女性占美妆/时尚 70%+，男性占电子/健身 60%+ |
| **收入** | 中等收入为主（RM 3,000-8,000/月） |

|------|------|---------|------|

## 2. 各品类用户需求深度分析

### 🧴 清真美妆/护肤品

**用户画像**：25-40岁女性，月收入 RM 3,000-7,000，注重成分安全

**需求优先级排序**：
1. 🥇 **清真认证**（绝对前提，非谈判项）
2. 🥈 **成分安全透明**（无酒精、无动物衍生物）
3. 🥉 **适合热带气候**（不油腻、控油、清爽）
4. **美白/提亮**（65%消费者需求）
5. **防晒功能**（SPF30+）
6. **敏感肌友好**（无刺激配方）
7. **价格合理**（月消费 RM150-300）
8. **天然/有机成分**（54%偏好）

**好评关键词**：halal-certified, lightweight, non-greasy, brightening, gentle, effective
**差评关键词**：oily, fake halal, overpriced, skin irritation, 色差

### 👗 穆斯林时尚

**用户画像**：18-45岁女性，多元文化背景，社交活跃

**需求优先级排序**：
1. 🥇 **符合伊斯兰教义**（遮盖度适当）
2. 🥇 **透气舒适**（热带气候刚需）
3. 🥉 **时尚现代**（不老旧、不土气）
4. 尺码准确
5. 面料质量
6. 价格合理（MYR 20-50）
7. 色彩选择
8. 快速配送

**好评关键词**：cantik (beautiful), selesa (comfortable), berkualitas, trendy, sesuai
**差评关键词**：色差, 尺码不准, 面料闷热, 线头多

### 🐾 宠物用品

**用户画像**：25-45岁，城市白领，视宠物为家庭成员

**需求优先级排序**：
1. 🥇 **安全无毒**（食品级材质）
2. 🥇 **品质可靠**（不轻易损坏）
3. 🥉 **创新功能**（智能/自动）
4. 价格合理
5. 适合热带气候
6. 好看/可爱
7. 易清洁
8. 售后保障

### 💊 健康保健品

**用户画像**：30-55岁，健康意识强，中等收入

**需求优先级排序**：
1. 🥇 **清真认证**（穆斯林消费者必备）
2. 🥇 **真实效果**（可感知的变化）
3. 🥉 **成分安全**（无副作用）
4. NPRA注册（信任背书）
5. 价格合理
6. 剂型友好（易吞服）
7. 口味好（尤其软糖/口服液）
8. 品牌信任

### 🏠 家居收纳/厨房工具

**用户画像**：25-45岁，尤其新搬家人群和小户型住户

**需求优先级排序**：
1. 🥇 **实用性强**（真正解决问题）
2. 🥇 **安装简单**（无需专业工具）
3. 🥉 **颜值在线**（Instagram-worthy）
4. 材质安全（食品级）
5. 尺寸准确
6. 价格合理
7. 耐用性好
8. 售后保障

### ✨ 个人护理/美容工具

**用户画像**：20-40岁女性，关注美容护肤，喜欢尝试新品

**需求优先级排序**：
1. 🥇 **效果可见**（短期内有变化）
2. 🥇 **使用简单**（操作步骤少）
3. 🥉 **安全可靠**（无副作用）
4. 价格合理
5. 外观好看
6. 便携性
7. 电池/续航
8. 售后保障

## 3. 用户共性与差异

### 三大族群共性需求
| 需求 | 说明 |
|------|------|
| **清真认证** | 不仅穆斯林要求，非穆斯林也视为安全和质量标志 |
| **价格敏感** | 促销驱动消费，11.11/12.12 贡献 40%+ 美妆销售 |
| **社交证明** | 80% 看评价，48% 通过 KOL 发现产品 |
| **移动优先** | 72.67% 手机端购物，APP 体验至关重要 |
| **快速配送** | 期望 3-5 天内送达 |

### 跨文化差异
| 维度 | 马来消费者 | 华人消费者 | 印度裔消费者 |
|------|-----------|-----------|-------------|
| 语言偏好 | 马来语+英语 | 中文+英语 | 英语+马来语 |
| 支付方式 | DuitNow/GrabPay | Touch'n Go/FPX | 银行转账 |
| 促销敏感 | 高 | 中 | 极高 |
| 品牌忠诚 | 中 | 高 | 低 |
| 清真要求 | 绝对必备 | 加分项 | 视品类而定 |

## 4. 用户决策旅程

```
第1步：发现
  → TikTok/KOL (48%) > 搜索 (25%) > 社交平台 (15%) > 口碑 (12%)

第2步：研究  
  → 看评价 (80%) > 研究成分/配料 (58%) > 比价 (55%) > 看品牌官网 (20%)

第3步：考虑
  → 等待促销 (40%) > 查看清真认证 > 对比竞品 > 收藏购物车

第4步：购买
  → 促销期集中下单 > 移动端支付 > 选择快速配送

第5步：复购/推荐
  → 满意则复购 > 分享到社交媒体 > 留下评价
```
