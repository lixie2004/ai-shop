## 建模方法论与核心假设

### 成本模型公式

```
总成本 = 采购成本 + 头程物流 + 进口关税(0% CSFTA) + 新加坡GST 9%
         + 平台佣金 + 广告费(ACoS) + 尾程配送费 + 仓储费
         + 退货损耗 + 汇率波动预留 + 认证摊销 + 包装费

毛利额 = 零售价 - 总成本（不含广告费）
净利润 = 零售价 - 总成本（含广告费）

毛利率 = 毛利额 ÷ 零售价
净利率 = 净利润 ÷ 零售价
```

### 核心参数假设（2026年5月）

| 假设项 | 取值 | 来源/依据 |
|--------|------|-----------|
| **进口关税（CSFTA）** | 0%（凭FORM E） | 中国-新加坡自由贸易协定（CSFTA），绝大多数商品零关税 |
| **GST（消费税）** | 9% | 2024年起稳定在9%，2026年延续 |
| **GST对进口商品** | 9% ×（CIF价值 + 关税） | IRAS规定，低价值商品免税额S$400（2023年起取消） |

| **头程物流（空运）** | USD 2-5/kg ≈ S$2.70-6.70/kg | 1-3天 |
| **Shopee佣金（Non-Mall）** | 2-6% + 交易费2.18% | Shopee SG 2026费率表 |
| **Shopee佣金（Mall）** | 4-10% + 交易费2.18% | Shopee SG Mall费率 |

### 新加坡跨境进口税制说明

```
计算公式：
1. 关税 = CIF × 0%（CSFTA优惠，凭FORM E）
2. GST = (CIF + 关税) × 9%

注：新加坡GST对进口商品征收，门槛为S$400（低价值商品免税政策已于2023年取消）
```

**新加坡市场关键特征**：
- **无进口关税**（中国-新加坡FTA下绝大多数商品零关税）— 税负远低于印尼/越南/菲律宾
- **GST 9%** — 可抵扣（注册GST后），进口时先缴
- **新加坡是全球最开放的市场之一**，海关流程透明高效
- **高获客成本**（CPC S$2-5）但**高转化率**（新加坡网购渗透率>80%）
- **退货率低**（消费者成熟，退货文化不泛滥）

## 税务与合规优化建议

### 1. CSFTA原产地证（FORM E）— 强制执行

新加坡与中国有**中国-新加坡自由贸易协定（CSFTA）**，绝大多数商品凭FORM E享受零关税。虽然新加坡本身是低关税国家（MFN平均0%），但FORM E仍是合规必要文件。

### 2. GST注册与抵扣

| 策略 | 说明 |
|------|------|
| **GST注册门槛** | 年营业额超过S$100万需强制注册GST |
| **自愿注册** | 低于门槛也可自愿注册，抵扣进口GST |
| **进口GST抵扣** | 注册GST后，进口时缴纳的9%GST可在销售GST中抵扣 |
| **建议** | 如月销超过S$80K（约S$100万/年），建议注册GST以建立完整抵扣链条 |
