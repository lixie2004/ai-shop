# 🇵🇭 菲律宾电商成本模型分析报告

> **报告日期**: 2026-05-26  
> **汇率基准**: 1 USD ≈ 57 PHP

---

## 1. 全成本结构框架

```
┌─────────────────────────────────────────────────┐
│              售价 (Selling Price)                 │
├─────────────────────────────────────────────────┤
│  ① 采购成本 (COGS)                                │
│  ② 头程物流 (海运/空运 + 清关 + 拖车)            │
│  ③ 关税与进口税费 (Duty + VAT + 处理费)          │
│  ④ 尾程配送 (SPX/J&T/LBC 国内派送)               │
│  ⑤ 平台佣金 (Shopee/Lazada Commission)           │
│  ⑥ 平台服务费 (Transaction/Service Fee)          │
│  ⑦ 广告推广费 (Ad Spend / Marketing)             │
│  ⑧ 退货/售后损耗 (Return & Refund Reserve)       │
│  ⑨ 仓储/包装/运营费用                             │
│  ⑩ 支付手续费 (Payment Gateway Fee)              │
├─────────────────────────────────────────────────┤
│  = 总成本 → 售价 - 总成本 = 毛利/净利            │
└─────────────────────────────────────────────────┘
```

## 2. 关键参数假设

| 成本项 | 假设值 | 说明 |
|--------|--------|------|
| Shopee佣金 | 4-6% | 跨境卖家依品类浮动 |

| 支付手续费 | 2.0% | 含跨境支付通道费 |
| 汇率缓冲 | 1.5-2.5% | 汇率波动预留 |

## 5. 通用定价公式

```
保本售价 = (采购 + 物流 + 关税) ÷ (1 - 佣金% - 服务费% - ACoS% - 退货% - 支付%)
建议售价 = 保本售价 × (1.25 ~ 1.45)  ← 含20-31%目标净利率
```

### 定价示例（宠物玩具3件套）
```
保本售价 = (85+15+29) ÷ (1 - 4% - 2% - 15% - 3% - 2%) = 129 ÷ 0.74 = ₱174
建议售价 = ₱174 × 2.0 = ₱349（含45%毛利率）
```
