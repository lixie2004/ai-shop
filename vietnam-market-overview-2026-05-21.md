# 越南跨境电商宏观环境分析简报

**报告日期**：2026年5月21日  
**研究员**：趋势研究员（product-trend-researcher）  
**报告类型**：市场分析简报

---

## 摘要（Executive Summary）

越南是东南亚增长最快的电商市场之一，2024年电商GMV超过250亿美元，增速约20%，数字经济的GDP贡献率已达18.3%，位居东南亚第一。市场由Shopee和TikTok Shop双头主导，两者合计占四大平台GMV的93%以上。消费者高度移动化（96%通过手机购物），价格敏感度高，社交电商和直播购物正在重塑购买决策路径。政策环境趋严但方向明确——政府大力推动数字化转型的同时，通过税收合规和平台责任新规加强监管。对于跨境电商卖家而言，越南市场机会巨大但竞争正在快速升级，专业化运营能力将成为生存门槛。

---

## 一、电商市场规模（GMV）及增速

### 1.1 市场规模

| 指标 | 数值 | 来源 |
|------|------|------|
| 2024年电商市场总规模 | **250亿美元+** | 越南工贸部（MoIT） |
| 2024年四大平台GMV（Shopee/Lazada/TikTok Shop/Tiki） | **160亿美元** | Momentum Works |
| 2023年四大平台GMV | **138亿美元** | YouNet ECI |
| 2025年电商市场预期规模 | **280亿美元** | 越南工贸部预测 |
| 2025年数字经济GMV | **390亿美元** | Google/Temasek/Bain e-Conomy SEA 2025 |
| 2027年电商市场预测 | **440亿美元** | PCMI E-commerce Data Library |

### 1.2 增速分析

- **2024年增速**：四大平台GMV同比增长 **15.9%**（Momentum Works），整体电商市场增长约 **20%**（越南国家统计局）。
- **2025年H1增速**：四大平台在线零售额达 **202.3万亿越南盾（约79.6亿美元）**，同比增长 **41.52%**（Metric.vn）。2025年全年四大平台GMV预计达 **429.7万亿越南盾（约163.5亿美元）**，同比增长 **34.75%**。
- **2024-2027年CAGR**：预计 **23%**（PCMI）。
- **东南亚排名**：越南GMV总量排名第三（与菲律宾并列），仅次于印尼和泰国；增速排名第四，低于泰国（21.7%）、马来西亚（19.5%）和菲律宾（16.8%）。

### 1.3 增长驱动因素

- **宏观经济韧性**：2024年GDP增长7.55%，2025年目标8-10%，出口强劲和外资制造业投资是核心引擎。
- **数字基础设施**：移动宽带速度全球排名第37位（86.96 Mbps），固定宽带排名第35位（159.32 Mbps），家庭光纤覆盖率82.4%，已超2025年目标。
- **数字支付普及**：无现金交易年增50%+，活跃电子钱包账户达3000万，越南QR支付已与泰国、柬埔寨实现互操作。
- **AI高接受度**：81%的越南用户每天与AI工具互动，60%与AI聊天机器人对话，东南亚排名第一。

## 三、平台格局

### 3.1 四大平台GMV份额（2024年数据）

| 平台 | 2024年GMV份额 | 2025年H1份额 | 趋势 |
|------|-------------|-------------|------|
| **Shopee** | **65%** | **58%** | ↓ 份额被TikTok Shop蚕食，但仍保持第一 |
| **TikTok Shop** | **28%** | **39%** | ↑↑ 增速最快，2025年H1同比增长69% |
| **Lazada** | 约5-6% | 持续下滑 | ↓↓ 2025年H1销售额同比下降48% |
| **Tiki** | 约2-3% | 持续下滑 | ↓↓ 2025年H1销售额同比下降63% |

*数据来源：Momentum Works 2024年报告、Metric.vn 2025年H1报告*

### 3.2 平台竞争态势

**Shopee**：市场领导者，但面临TikTok Shop的强劲挑战。2025年H1仅增长16%，份额从63%降至58%。策略上加大物流投入，2025年6月推出全品类免运费政策（大件/特快除外）。

**TikTok Shop**：最大变量和增长引擎。2022年上线，2024年已占28%份额，2025年H1飙升至39%，同比增长69%。"娱乐+购物"模式深受越南消费者欢迎。2025年6月生日大促订单超2000万单，同比增长46%。

**Lazada & Tiki**：传统电商平台持续承压。Lazada通过与Pop Mart等品牌合作、举办Lazada Run等活动试图挽回颓势，但2025年H1销售额下降48%。Tiki下降63%，市场份额持续萎缩。

### 3.3 平台趋势判断

市场正在从"多平台并存"走向"Shopee + TikTok Shop"双头格局。传统货架式电商（Lazada、Tiki）面临生存危机，而内容驱动型电商（TikTok Shop）正在重新定义消费者购物路径。**信心等级：高**——这一趋势在2025年H1数据中已非常明确。

## 四、跨境电商政策环境

### 4.1 税收合规政策

**Decree 117（2025年7月1日生效）**：
- 所有电商和数字平台必须**代扣代缴**卖家的增值税（VAT）和个人所得税
- 适用于个人卖家和企业卖家
- 目的：堵住线上交易的税收漏洞，促进公平竞争

**税收增长数据**：2025年前5个月，线上企业纳税 **74.4万亿越南盾（约28.4亿美元）**，同比增长55%。

### 4.2 平台责任与消费者保护

**Decree 98 替代草案**（工贸部正在起草）：
- 电商平台需对卖家身份审核承担更大责任
- 如未能识别卖家或阻止假冒商品销售，平台将面临行政处罚
- 加强对贸易违规、假冒伪劣和消费者保护的执法力度

### 4.3 跨境电商进口政策

- 越南对跨境电商进口商品征收**进口关税 + 增值税（VAT 10%）+ 特别消费税**（如适用）
- 个人进口商品价值低于100万越南盾（约40美元）可享受免税额度
- 跨境电商平台需进行**海关申报**，并遵守越南产品标签和认证要求
- 政府正在推动**电子发票**制度，直播销售也纳入监管范围

### 4.4 数字化转型政策

- **国家数字化转型委员会**由总理直接主持，目标2025年数字经济贡献GDP超20%
- 已推出电子身份证（VNeID），5525万公民已激活
- 新颁布《数据法》《电信法》修订版，为AI、数据经济提供法律框架
- 目标：2030年电商无现金交易占比达80%

### 4.5 政策环境评估

越南政府的政策方向是**"鼓励发展 + 加强监管"**并行。一方面大力推动数字化转型、吸引AI和半导体投资；另一方面通过税收合规、平台责任和消费者保护法规提高市场准入门槛。对跨境电商卖家而言，合规成本将上升，但市场环境将更加规范。**信心等级：中高**——政策方向明确，但具体执行细则仍在完善中。

## 五、消费者画像与购物行为特征

### 5.1 人口统计特征

| 特征 | 描述 |
|------|------|
| 总人口 | 约1亿（2024年） |
| 中位年龄 | 约31岁 |
| 互联网用户 | 约7,800万（互联网渗透率约78%） |
| 智能手机用户 | 渗透率96%（用于网购） |
| 城市化率 | 约38%（城市人口集中在胡志明市和河内） |

### 5.2 购物行为特征

**移动优先**：
- 96%的消费者通过智能手机购物
- 70%的电商交易额来自移动设备
- 仅30%来自桌面端

**社交驱动**：
- 约50%的在线购物者尝试过**直播购物**
- 70%的互联网用户观看过直播，其中一半在直播中或直播后完成购买
- TikTok Shop已成为重要的产品发现渠道
- 购物决策越来越受短视频和社交媒体内容影响

**价格敏感**：
- 前三大购买驱动力：①低价和免运费 ②价格 ③快速配送
- 消费者平均每次购买涉及6.5个品类，说明比价行为普遍
- 促销活动期间消费集中度明显（如TikTok Shop生日大促订单增长46%）

**支付习惯**：
- 货到付款（COD）仍占主导：**67%**
- 电子钱包：18%
- 信用卡：8%
- 借记卡：3%
- 银行转账：3%
- 先买后付（BNPL）：2%

**购物频次**：
- 月均4次在线购物
- 约50%的消费者每周至少网购1次
- 平均订单价值：5.50-7美元（视频电商渠道）

### 5.3 消费者行为趋势

1. **"Shoppertainment"（购物娱乐化）**：消费者不再只是"搜索-购买"，而是通过短视频、直播、社交互动来发现和评估产品。TikTok Shop的爆发式增长就是这一趋势的最佳注脚。

2. **品牌信任度提升**：官方品牌店、高评分店铺表现越来越好，消费者越来越看重产品质量、溯源信息和售后服务。

3. **必需品线上化**：FMCG、食品饮料、家庭用品、健康个护等日常消费品正在加速向线上迁移。

4. **AI接受度极高**：81%用户每天使用AI工具，近60%与AI聊天机器人对话，对AI推荐和个性化体验接受度东南亚最高。

## 六、热门品类趋势

### 6.1 品类排名与增长

| 品类 | 排名 | 增长趋势 | 备注 |
|------|------|---------|------|
| **美妆个护** | #1 | 稳定增长 | 连续多年位居榜首 |
| **家居生活** | #2 | 稳定增长 | 与美妆并列前二 |
| **食品饮料** | #3 | ↑↑ 增速最高（65%） | 2025年超越家电升至第三 |
| **时尚运动** | #4 | 增长 | 卖家数量最多的品类 |
| **家庭护理** | #5 | 增长 | 日常必需品 |
| **母婴产品** | #6 | 增长 | 稳定需求 |
| **消费电子** | #7 | 增长 | 手机配件需求大 |
| **健康产品** | #8 | 增长 | 疫情后持续关注 |
| **家电** | #9 | ↓ 增速最低（8%） | 被食品饮料超越 |

*数据来源：Q&Me 2025年越南电商趋势报告、NielsenIQ Vietnam、PCMI*

### 6.2 视频电商品类表现

在TikTok Shop等视频电商渠道中，品类GMV排名：
1. **时尚配饰** — 领先
2. **美妆个护** — 紧随其后
3. **手机数码** — 第三
4. **日常用品** — 第四

视频电商平均订单价值：**5.50-7.00美元**。

### 6.3 品类趋势分析

**食品饮料是最大变量**：从第四跃升至第三，增速高达65%，说明越南消费者正在将日常食品采购从线下转向线上。这一趋势对跨境电商意味着——快消品（FMCG）的线上化窗口已经打开。

**美妆个护持续强势**：作为多年榜首品类，美妆个护在越南市场有深厚的消费者基础，且通过短视频/直播的种草效应持续放大。

**家电品类增速放缓**：仅8%增长，可能反映大件商品线上化面临物流和信任瓶颈。

## 九、风险与不确定性

1. **全球经济放缓风险**：概率中等。越南经济高度依赖出口，全球需求疲软可能影响消费者信心。
2. **平台政策突变风险**：概率中低。TikTok Shop在东南亚多国面临监管不确定性，越南可能跟进。
3. **汇率波动风险**：概率中等。越南盾对美元汇率波动可能影响跨境电商利润。
4. **竞争加剧导致利润率下降**：概率高。平台补贴战和免运费竞争将持续压缩卖家利润空间。
