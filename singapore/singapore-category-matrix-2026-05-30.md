## 重点品类深度分析

### 🏆 品类 1：保健品/膳食补充剂

| 维度 | 详情 |
|------|------|
| 市场需求 | 新加坡老龄化 + 健康意识强，保健品年消费超 US$500M |
| 高增长细分 | 功能性软糖 (+65% YoY)、护肝片、益生菌、胶原蛋白、NMN |
| 价格带 | US$25-80 (中高端) |
| 毛利率 | 45-60% |
| 竞争格局 | 本土品牌居多，中国跨境卖家占比 <15%，机会大 |
| 合规要求 | HSA 注册 + GMP 认证，周期 8-12 周 |
| 推荐切入点 | 功能性软糖、女性健康、肝脏保护 |

### 🏆 品类 2：宠物用品

| 维度 | 详情 |
|------|------|
| 市场需求 | 新加坡宠物拥有率 ~32%，年消费 US$300M+ |
| 高增长细分 | 智能喂食器/饮水机 (+50%)、环保猫砂 (+40%)、冻干零食 (+35%) |
| 价格带 | US$10-150 |
| 毛利率 | 35-55% |
| 竞争格局 | 品牌分散，头部品牌集中度 <30% |
| 合规要求 | SAVA 进口许可 (活体不需要，成品只需标准报关) |
| 推荐切入点 | 智能硬件、环保用品、中式零食 |

### 🏆 品类 3：运动户外装备

| 维度 | 详情 |
|------|------|
| 市场需求 | 跑步人口 25%+，健身房密度全球前列 |
| 高增长细分 | 瑜伽配件 (+35%)、跑步配件 (+25%)、骑行装备 (+20%) |
| 价格带 | US$15-100 |
| 毛利率 | 40-55% |
| 竞争格局 | Nike/Adidas 主导大件，配件市场分散 |
| 合规要求 | 标准 FCC/CE，无特殊认证 |
| 推荐切入点 | 瑜伽垫、弹力带、运动毛巾、跑步配件 |

### 品类 4：中式食品/零食

| 维度 | 详情 |
|------|------|
| 市场需求 | 华人占比 74%，中式零食需求刚需 |
| 高增长细分 | 螺蛳粉 (+60%)、自热火锅 (+50%)、地方特产 (+30%) |
| 价格带 | US$3-25 |
| 毛利率 | 25-40% |
| 竞争格局 | 竞争相对低，品牌集中度弱 |
| 合规要求 | SFA 食品进口许可，周期 3-6 周 |
| 推荐切入点 | 网红食品、地方特产、方便速食 |

### 品类 5：智能家居 IoT 小件

| 维度 | 详情 |
|------|------|
| 市场需求 | 新加坡 "智慧城市" 战略推动，智能家居渗透率 30%+ |
| 高增长细分 | 智能门窗传感器 (+45%)、智能插座 (+35%)、扫地机器人配件 (+30%) |
| 价格带 | US$15-80 |
| 毛利率 | 40-55% |
| 竞争格局 | 小米/涂鸦生态为主，自有品牌机会大 |
| 合规要求 | IMDA 认证，周期 4-8 周 |
| 推荐切入点 | 安防小件、节能控制器、智能照明 |

## 品类筛选决策

### ✅ "极佳" 品类 (综合评分 ≥ 4.0)
1. **保健品/膳食补充剂** — 高利润、高增长、竞争度低
2. **宠物用品** — 蓝海市场、增速最快、复购率高

### ✅ "精品" 品类 (综合评分 3.5-3.9)
3. **运动户外装备** — 需求明确、利润空间好
4. **中式食品/零食** — 华人刚需、竞争度低
5. **智能家居 IoT 小件** — 政策驱动、利润率高
6. **母婴用品** — 消费力强、品质敏感
7. **有机美妆护肤** — 增长快、差异化空间大

### ⚠️ "有条件进入" 品类 (综合评分 3.0-3.4)
8. **个护小家电** — 需产品差异化
9. **家居收纳 & 厨具** — 竞争中等，靠设计取胜
10. **消费电子配件** — 红海市场，需要极致性价比
