## 一、跨境电商基础合规

### 1.1 税务注册

| 项目 | 要求 | 说明 |
|------|------|------|
| **GST注册** | 年收入≥SGD 100万 | 低于此门槛由平台代扣代缴 |
| **GST税率** | 9% (CIF价值) | 所有进口商品 |
| **VAT申报** | 季度/月度 | 通过IRAS系统 |
| **公司实体** | 新加坡本地注册或跨境 | 建议注册新加坡本地公司 |

### 1.2 平台入驻要求

| 平台 | 基本要求 | 押金 | 验证材料 |
|------|---------|------|---------|
| **Shopee SG** | 身份证+银行账户 | S$500 | 营业执照(可选) |
| **Lazada SG** | 公司/个体户+银行账户 | S$1000 | 营业执照+法人证件 |
| **TikTok Shop** | 身份证+手机号 | S$300 | 营业执照(个人店) |

## 二、保健品合规 (HSA监管)

### 2.1 监管机构: Health Sciences Authority (HSA)

**适用范围**: 所有进口、制造、销售的健康补充剂/营养品

### 2.2 产品分类框架

| 分类因素 | 监管影响 |
|---------|---------|
| 成分列表 | 决定风险等级 |
| 预期用途 | 仅限结构/功能声称 |
| 产品声明 | 不得涉及疾病治疗 |
| 剂型 | 非无菌、口服为主 |
| 安全概况 | 需有安全使用历史 |

### 2.3 注册流程 (60个工作日)

| 步骤 | 内容 | 时间 |
|------|------|------|
| **Step 1** | 成分合规检查 + 声明验证 | 1-2周 |
| **Step 2** | 在线提交通知 (HSA portal) | 1周 |
| **Step 3** | 技术文件提交 | 1-2周 |
| **Step 4** | HSA审核 | ~60工作日 |
| **费用** | 无注册费 | — |

### 2.4 必备文件

| 文件 | 用途 |
|------|------|
| 制造商许可证/GMP证明 | 质量保证 |
| 分析证书(CoA) | 成分检测 |
| 成品规格书 | 批次一致性 |
| 稳定性数据(如适用) | 保质期支持 |
| 标签设计稿 | 合规审核 |
| 安全证据 | 风险说明 |

### 2.5 标签合规 (禁止 vs 允许)

| ✅ 允许的声明 | 🚫 禁止的声明 |
|-------------|-------------|
| 支持免疫功能 | 治疗感染 |
| 维持骨骼健康 | 治愈骨质疏松 |
| 促进消化健康 | 预防IBS |

### 2.6 2026年新增合规重点

- 重金属和污染物检测
- 植物成分溯源
- 益生菌菌株鉴定
- 跨境电商管控
- 误导性数字营销监管

### 2.7 保健品合规时间线

| 阶段 | 预估时间 | 费用(估算) |
|------|---------|-----------|
| 成分合规检查 | 2周 | SGD 500-1000 |
| GMP文件准备 | 4周 | SGD 2000-5000 (第三方审计) |
| HSA通知提交 | 1周 | 免费 |
| HSA审核 | 60工作日 | 免费 |
| 总时间 | **~4-5个月** | **SGD 2500-6000** |

## 三、护肤品合规

### 3.1 监管机构

- **HSA**: 部分含活性成分的产品
- **NEA (国家环境局)**: 化妆品注册
- **CCS (消费者保护)**: 标签规范

### 3.2 化妆品/护肤品注册要求

| 项目 | 要求 |
|------|------|
| 产品通知 | 通过NEA化妆品通知系统 |
| 成分限制 | 禁止28种成分(部分重金属、类固醇) |
| 标签要求 | 英文+成分INCI名+产地+净含量 |
| 安全证明 | 建议第三方实验室检测 |
| 动物测试 | **禁止** (2020年起全面禁止) |

### 3.3 时间线

| 阶段 | 预估时间 |
|------|---------|
| NEA化妆品通知 | 1-2周 |
| 第三方安全检测 | 2-4周 |
| 标签合规审核 | 1周 |
| **总计** | **~4-6周** |

## 四、智能家居/宠物智能设备合规

### 4.1 多监管机构架构

| 机构 | 监管范围 |
|------|---------|
| **IMDA** | 电信/IoT设备连通性、安全标签 |
| **CSA** | IoT网络安全标准 |
| **Enterprise SG** | Safety Mark认证(电气安全) |
| **EMA** | 电气系统和能源设备 |
| **PDPC** | 数据隐私(PDPA法案) |

### 4.2 关键认证清单

| 认证 | 监管方 | 是否强制 | 适用产品 |
|------|-------|---------|---------|
| **IMDA Type Approval** | IMDA | ✅ 强制 | 智能门锁(含RF)、路由器、IoT设备 |
| **Safety Mark** | Enterprise SG | ✅ 部分强制 | 智能开关、智能插座、电源相关 |
| **CLS (IoT注册)** | CSA | ✅ 鼓励 | 所有IoT设备(提升消费者信任) |
| **PSB (电气安全)** | IMDA/PSB | ✅ 部分 | 涉及电气安全的产品 |
| **PDPA合规** | PDPC | ✅ 强制 | 所有收集个人数据的设备 |

### 4.3 PDPA数据隐私要求

| 要求 | 内容 |
|------|------|
| 数据保护 | 设备收集的用户数据需加密 |
| 用户同意 | 明确告知数据收集目的 |
| 数据存储 | 本地或新加坡数据中心 |
| 漏洞披露 | 72小时内报告安全事件 |

### 4.4 智能家居合规时间线

| 阶段 | 预估时间 | 费用(估算) |
|------|---------|-----------|
| IMDA Type Approval | 4-8周 | SGD 2000-5000 |
| Safety Mark | 4-8周 | SGD 1500-4000 |
| CLS IoT注册 | 2-4周 | SGD 500-2000 |
| PDPA合规评估 | 2周 | SGD 1000-3000 |
| **总计** | **~3-4个月** | **SGD 5000-14000** |

## 七、合规优先级建议

### 7.1 进入顺序建议

| 优先级 | 品类 | 理由 |
|--------|------|------|
| **P0 (立即启动)** | 护肤品 + 宠物食品 | 合规周期最短(4-6周) |
| **P1 (3个月内)** | 宠物智能设备 | 合规周期3-4个月但利润率高 |
| **P2 (5个月内)** | 保健品 | 合规周期4-5个月但市场大 |
| **P3 (观望)** | 智能门锁 | 合规成本高+竞争激烈 |

### 7.2 合规红线 ⚠️

1. **HSA认证未完成前，保健品绝不上架**
2. **IMDA Type Approval未完成前，IoT设备绝不上架**
3. **PDPA数据隐私不合规前，所有联网设备禁止运营**
4. **化妆品成分未审核前，护肤品禁止上架**
5. **GST未完成注册前，平台代扣代缴需确认**
