## 2. 蓝海品类划分

- **蓝海**（指数 ≥ 40 且增长趋势为 ↑）：
  1. **环保可持续**（指数 49）
  2. **宠物**（指数 48）
  3. **母婴**（指数 42）
  4. **婚礼礼品**（指数 40）
  5. **健康美容**（指数 36）— 兼具高潜力与相对中等竞争
- **机会**（指数 30‑39）：
  - 运动户外、学习玩具、美妆工具、家居生活、个人护理
- **红海**（指数 < 30）：
  - 消费电子、时尚服饰、食品饮料、智能家居、车载配件

## 4. 行动路线图（建议）

1. **快速试点**：选取 **宠物** 与 **环保可持续** 两个品类，各投入 1‑2 人，进行 4 周 PoC（产品采购、物流、TikTok Shop 直播）。
2. **平台布点**：
   - Shopee：主打宠物与母婴日常消耗品，利用平台优惠券提升复购。
   - TikTok Shop：聚焦健康美容、环保可持续，依赖短视频带货。
   - Lazada：锁定高客单价的婚礼礼品与智能家居小件。
3. **合规准备**：提前完成 HSA（化妆品/健康产品）与 SG Food Agency（食品类）认证，避免清关延误。
4. **物流策略**：选择 **DHL eCommerce** 与 **SingPost** 的跨境小包方案，预计 3‑5 天送达新加坡主城，成本约 SGD 3‑5/件。
5. **监测 KPI**：
   - 月 GMV 增长 ≥ 15%
   - 复购率 ≥ 20%（宠物、母婴类）
   - TikTok Shop 直播转化率 ≥ 4%
