# 新加坡跨境电商合规检查清单

> **报告日期**: 2026-05-30  
> **目标市场**: Singapore  
> **编制依据**: HSA (卫生科学局)、SFA (食品局)、IMDA (资讯通信媒体发展局)、Enterprise Singapore

## 新加坡合规监管概览

| 监管机构 | 管辖范围 | 适用品类 |
|---------|---------|---------|
| **HSA** | 保健品、医疗器械、药品 | 保健品 |
| **SFA** | 食品、宠物食品 | 中式食品、宠物零食 |
| **IMDA** | 无线通信设备、IoT | 智能家居 |
| **SPF/CNB** | 法律执行 | 所有品类 |
| **PDPC** | 数据保护 | 所有品类 (涉及用户数据) |

## SKU 1：功能性软糖维生素

| 合规项 | 详情 |
|--------|------|
| **认证名称** | HSA 健康补充剂通报 (Health Supplement Notification) |
| **认证类型** | 通报制 (非审批制，但需合规) |
| **认证费用** | ~S$800-2,000 / SKU (代理服务费) |
| **认证周期** | 8-12 周 |
| **有效期** | 持续有效 (需年度更新) |
| **风险等级** | ⚡ 中等 |
| **具体要求** | ① 成分必须在 HSA 允许清单内 ② 标签需英文 ③ 不得含药物成分 |
| **缓解方案** | 选品时确认成分符合 HSA 清单；与本地合规代理商合作 |

### 额外要求
- 标签要求：产品名称、成分表、净含量、生产批号、保质期、注意事项
- 广告限制：不得宣称"治疗"、"治愈"等医疗功效
- 进口申报：需 HSA 通报号 + 进口许可

## 合规行动路线图

### 优先级建议
1. **立即启动** (Week 1-2): 
   - SKU 5 (NMN) — 周期最长，需提前准备检测报告
   - SKU 1 (软糖维生素) — 确认 HSA 成分清单合规性

2. **逐步推进** (Week 3-6):
   - SKU 2, 8 (智能设备) — IMDA 认证，使用已认证模组可加速
   - SKU 6, 7 (食品类) — SFA 进口许可

3. **快速启动** (Week 1-4):
   - SKU 3, 4, 9, 10 (低风险品类) — 检测报告准备后即可上架

## ⚠️ 重点提醒

1. **HSA 红线**: 保健品中不得添加药物成分，违者刑事处罚
2. **SFA 红线**: 进口食品必须申报，未申报视为违法
3. **IMDA 红线**: 使用未经认证的无线设备销售将被罚款 (最高 SGD 10,000)
4. **PDPC 红线**: 收集用户数据需遵守 PDPA (个人数据保护法)
5. **GST 合规**: 年跨境销售额超 SGD 100 万需注册 GST
