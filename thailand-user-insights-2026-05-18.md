## 6. 购买决策路径

```
发现产品 → 搜索比价 → 查看评价 → 确认正品 → 检查运费 → 下单 → 收货 → 使用 → 评价
  ↓           ↓          ↓          ↓          ↓
 TikTok/     Shopee/    评论+评分   品牌/Facebook  免费配送
 Instagram   Lazada              官方认证    为前提
```

### 关键决策节点

1. **发现阶段**: TikTok 短视频 + KOL 推荐最有效
2. **比较阶段**: Shopee/Lazada 搜索 + 比价
3. **信任阶段**: 评价数量+质量、官方认证、品牌背书
4. **转化阶段**: 免费配送 + 促销 + 限时优惠
5. **复购阶段**: 产品质量 + 客服体验 + 会员优惠
