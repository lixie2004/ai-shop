## 建模方法论与假设

### 成本模型公式

```
总成本 = 采购成本 + 头程物流 + 进口关税 + 进口SST增值税 
         + 平台佣金 + 广告费 + 尾程配送费 + 退货损耗 
         + 汇率波动预留 + 认证摊销 + 仓储费 + 支付手续费

毛利额 = 零售价 - 总成本（不含广告费）
净利润 = 零售价 - 总成本（含广告费）

毛利率 = 毛利额 ÷ 零售价
净利率 = 净利润 ÷ 零售价
```

## 成本项定义与取值标准

### 1. 采购成本（含OEM/ODM费用）
- 包含：原材料、生产、包装、OEM/ODM模具分摊
- 计价货币：CNY（人民币），按汇率1 MYR = 1.65 CNY折算
- 基于500-2000件小批量试单价格

### 2. 头程物流（中国→马来西亚）
- 海运（普货）：RM 2.5-4.0/kg（含港口操作费）
- 海运（敏感货/液体/粉状）：RM 4.0-8.0/kg
- 空运（普货）：RM 12-18/kg
- 空运（敏感货）：RM 18-25/kg
- 含：出口报关、进口清关、港口到仓运输

### 3. 进口关税
- 按HS Code分类
- 适用ASEAN-China FTA优惠税率（需提供FORM E原产地证）
- 无FTA税率：Most Favored Nation (MFN)税率

### 4. 进口增值税（SST/LPGT）
- 低价值商品税（LPGT）：CIF价值≤RM 500，税率10%
- SST销售税：标准税率10%（2024年至今）
- 计算基数 = CIF价值（货值+运费+保险）+ 关税

### 5. 平台佣金
- 按平台加权平均：Shopee 60% + TikTok 25% + Lazada 15%
- 加权佣金率 ≈ 5.0%（含固定费用分摊）

### 6. 广告费
- 基于品类平均ACoS（广告花费/销售额）
- 新品类冷启动阶段采用中高值
- 成熟品类采用行业基准值

### 7. 尾程配送费
- 西马（Peninsular Malaysia）：RM 4.0-7.0/单（J&T / City-Link）
- 东马（Sabah/Sarawak）：RM 8.0-15.0/单
- 加权平均（90%西马 + 10%东马）

### 8. 退货损耗
- 含：退货率 ×（退货物流 + 翻新/报废成本）
- 美妆个护：3-5%
- 食品保健品：2-4%
- 电子电器：3-6%
- 家居收纳：2-4%
- 手机配件：2-4%
- 服装配饰：5-8%

### 9. 汇率波动预留
- 按总成本（除人民币计价采购成本外）的3%计提
- 人民币→MYR兑换时锁定风险

### 10. 认证费用（JAKIM/SIRIM/NPRA等）
- JAKIM清真认证：RM 5,000-15,000（首次，3年有效）
  - 按3年摊销 × 预计年销量
- SIRIM认证（电子类）：RM 8,000-20,000（含测试）
  - 按3年摊销 × 预计年销量
- NPRA产品备案：RM 500-2,000/产品
- MAQIS进口检疫：RM 200-500/批次
- 食品级测试报告：RM 2,000-5,000

### 11. 仓储费
- 马来西亚海外仓（第三方）：RM 0.50-1.50/件/月
- 按30天周转计算

### 12. 支付手续费
- 线上支付网关：2.0-2.5%
- FPX（马来西亚本地网银）：1.5-2.0%
- 加权平均：2.2%

### 4. 转让定价与实体架构

**建议架构**：
```
中国贸易公司（卖方）
    ↓ 出口价（FOB/CNF）
香港中间贸易公司（可选，利润池）
    ↓ 转移定价
马来西亚本地公司（进口商+电商运营）
    ↓
Shopee/Lazada/TikTok Shop（本地卖家账户）
```

**优势**：
- 马来西亚本地公司运营可避免"跨境卖家"标签（Shopee跨境佣金高于本地）
- 本地公司可注册SST并抵扣进项税
- 香港中间公司可递延马来西亚所得税（需有经济实质）
- 合理转移定价将利润留存低税率管辖区
