## 四、跨境电商政策与关税环境

### 4.1 税率总览

| 税种 | 税率 | 说明 |
|------|------|------|
| **GST（商品和服务税）** | **9%** | 2024年1月1日从8%上调至9% |
| **进口关税** | **0%** | 绝大多数消费品零关税 |
| **消费税** | 特定商品 | 烟酒/石油等特定品类 |

### 4.2 低价值商品（LVG）GST规则

新加坡自2023年1月1日起实施**低价值商品GST征收**制度：

| 规则要点 | 详情 |
|---------|------|
| **低价值商品定义** | 销售价值 **≤S$400**（约$300 USD）的商品 |
| **征税方式** | 对GST注册卖家的≤S$400商品，在**销售点征收GST** |
| **适用条件** | 通过空运或邮政进口的商品 |
| **CIF免税额** | 非GST注册卖家，CIF≤S$400仍免GST |
| **多件商品** | 每件≤S$400的单品分别计税，合并发货不影响 |
| **GST注册门槛** | 年应税营业额>S$100万需注册 |

### 4.3 跨境电商进口成本计算示例

**场景1：低值商品（含GST）**
| 项目 | 金额 |
|------|------|
| 商品价格 | S$380 |
| 运费+保险 | S$25 |
| GST（9% × S$380） | S$34.20 |
| 进口关税 | S$0 |
| **买家实际支付** | **S$414.20** |

**场景2：高值商品（>S$400）**
| 项目 | 金额 |
|------|------|
| 商品价格 | S$500 |
| 运费+保险 | S$30 |
| CIF价值 | S$530 |
| 进口关税 | S$0（多数品类） |
| GST（9% × S$530） | S$47.70 |
| **买家实际支付** | **S$577.70** |

### 4.4 跨境电商核心合规要点

| 要求 | 详情 |
|------|------|
| **GST注册** | 年跨境销售额>S$100万需注册 |
| **海关申报** | 所有进口商品需通过TradeNet系统申报 |
| **限制商品** | 武器、药品、食品、动植物等需许可证 |
| **HSA管制** | 健康/保健品需HSA注册（Health Sciences Authority） |
| **知识产权** | 商标/专利侵权零容忍，海关可扣留仿冒品 |
| **消费者保护** | Lemon Law（柠檬法）对缺陷商品提供退换保障 |
| **数据保护** | PDPA（个人数据保护法）严格规范客户数据使用 |

### 4.5 新加坡作为区域物流枢纽

| 基础设施 | 数据 |
|---------|------|
| **港口吞吐量** | 4,010万TEU（2025年），**全球第二**（仅次于上海） |
| **机场客运量** | 6,850万人次（2025年），连接**400+城市** |
| **海运连接** | **600+港口**，120+国家 |
| **仓储** | 保税仓库（无关税存储直到销售） |
| **区域配送** | 48小时到胡志明市，72小时到雅加达 |

> 📌 核心优势：卖家可将库存集中至新加坡保税仓，零关税存储，按需配送至东南亚各国。新加坡+马来西亚占东盟在线零售额1/3，但仅占区域人口8%。
