## 第一章：市场概览

### 1.1 电商市场规模与增速

| 年份 | GMV（十亿美元） | 同比增速 | 数据来源 |
|------|----------------|---------|---------|
| 2024 | $56.5 | — | Google-Temasek-Bain |
| 2025 | ~$71 | ~14% | SellerCraft |
| 2026（预测） | $100+ | ~15-19% | Mordor Intelligence |
| 2031（预测） | $212.58 | CAGR 15.32% | Premium Report |

**关键数据点**：
- 印尼是东南亚 **最大电商市场**，占区域GMV约44%
- 2024-2031年CAGR预计15.32%
- 社交电商占GMV **40-45%**，TikTok Shop从12%飙升至27%

### 1.2 消费力水平

| 指标 | 数据 | 说明 |
|------|------|------|
| 总人口 | 2.83亿 | 全球第四大人口国 |
| 中位年龄 | 30岁 | 年轻化人口结构 |
| 互联网用户 | ~2.21亿 | 渗透率81%+ |
| 社交媒体用户 | ~1.86亿 | TikTok 1.25亿+ |
| 穆斯林人口 | 87%（~2.46亿） | 全球最大穆斯林市场 |
| 中产阶级 | 5,200万+ | 持续扩大 |
| 人均GDP | ~$4,800 | 2025年估计 |
| COD占比 | 73.7% | 货到付款仍是主流 |

### 1.3 平台格局

| 平台 | 市场份额 | 特点 |
|------|---------|------|
| **Shopee** | ~38-53% | 最大平台，全品类覆盖 |

**关键趋势**：
- Shopee + TikTok Shop 合计占60%+份额
- TikTok Shop从12%飙升至27%，是最大变量
- 内容电商占GMV 25%，持续上升

### 1.4 跨境电商政策环境

| 政策 | 详情 |
|------|------|
| 进口关税 | 0-30%不等，取决于HS编码 |
| 增值税（VAT） | 11%（2025年起） |
| 所得税（PPh 22） | 跨境电商适用 |
| 低值商品免税 | 已取消（2025年生效） |
| Halal强制认证 | 2026年10月17日生效 |
| SNI认证 | 电子产品强制（Permenperin 7/2025） |
| BPOM注册 | 食品/保健品/化妆品强制 |
| 农业部FBU | 宠物食品进口审批（1-3年） |

**2026年关键法规变更**：

| 法规 | 生效时间 | 影响 |
|------|---------|------|
| Halal认证强制化（BPJPH） | 2026年10月17日 | 食品/保健品/化妆品 |
| Permendag 16/2025 进口新规 | 2025年8月已生效 | 所有进口商品 |
| SNI数字化（SIINas平台） | 2026年全面实施 | 电子产品 |
| Permenperin 7/2025 家电SNI | 2025年1月发布 | WiFi插座等 |

### 1.5 消费者行为特征

**购物偏好**：
- 价格敏感（35%）> Review/口碑（25%）> 物流（15%）> 认证/安全（15%）> 品牌（10%）
- COD占比73.7%，信任基础薄弱
- 社交电商驱动，TikTok日均使用45-60分钟
- 大促节点：Ramadan/开斋节 > 11.11 > 12.12 > 8.17（独立日）

**搜索行为特征**：

| 修饰词 | 印尼语 | 占比 | 适用品类 |
|--------|-------|------|---------|
| 便宜 | murah | 35% | 全部品类 |
| 折扣 | diskon | 30% | 全部品类 |
| 正品 | original / asli | 20% | 保健品、美妆、母婴 |
| 认证 | BPOM / SNI / Halal | 15% | 保健品、食品、电子 |
| 包邮 | gratis ongkir | 20% | 全部品类 |

### 1.6 市场趋势与机会

| 趋势 | 描述 | 对跨境卖家的意义 |
|------|------|----------------|
| **Halal强制认证** | 2026年10月生效 | 提前认证可建立壁垒 |
| **TikTok Shop爆发** | 份额从12%升至27% | 必须布局内容电商 |
| **内容电商深化** | 占GMV 25% | 短视频+直播是核心场景 |
| **EV市场爆发** | 增速50%+/年 | EV充电器/配件是蓝海 |
| **女性健身增长** | 居家健身需求上升 | 女性专用健身产品供给不足 |
