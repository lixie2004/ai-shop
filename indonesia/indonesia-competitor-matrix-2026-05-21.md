# 印度尼西亚市场竞品矩阵分析

> **报告日期**：2026-05-21  
> **目标市场**：印度尼西亚（Indonesia）  
> **分析平台**：Shopee Indonesia / Tokopedia / TikTok Shop Indonesia / Lazada Indonesia  
> **分析品类**：健康补剂、宠物用品、清真食品、母婴用品、家居收纳、穆斯林时尚、美妆个护、户外运动  
> **数据来源**：各平台前台数据抓取、第三方数据工具估算、社交媒体舆情交叉验证

---

## 一、分析框架说明

### 竞品筛选标准
- 各品类选取 **Top 5 竞品 SKU**（按销量+评价综合排名）
- 覆盖不同价格带（低端/中端/高端）
- 兼顾本土品牌与国际品牌

### 指标定义
| 指标 | 说明 |
|------|------|
| 价格带 | 印尼盾（IDR），标注美元换算（1 USD ≈ 16,300 IDR） |
| 销量估算 | 基于平台月销量显示及第三方工具估算（取区间中值） |
| 评价评分 | 5分制，取平台前台显示评分 |
| 差评率 | 1-2星评价占比，基于抽样评价内容分析 |
| 品牌类型 | 本土品牌 / 中国品牌 / 国际品牌 |
