5. 物流基础设施

地理挑战

  • 17,000+ 岛屿，物流成本高、时效差异大
  • Java 岛集中60%人口，外岛增长潜力大但基础设施薄弱

主要物流服务商

  • J&T Express（极兔）— 中资背景，性价比强
  • JNE — 全国覆盖最广
  • SiCepat — 次日达服务受欢迎
  • Pos Indonesia — 国营邮政，覆盖偏远地区
  • Shopee Xpress / Lazada Logistics — 平台自建物流
  • Gojek/Grab — 同城即时配送

物流成本与趋势

  • 物流成本约占商品价值的 15-25%
  • Tier-2/3城市配送时效持续改善
  • 冷链物流发展初期，生鲜电商机会大
  • 进口清关流程复杂，时效不确定
