## 建模方法论与核心假设

### 成本模型公式

```
总成本 = 采购成本 + 头程物流 + 进口关税(BM) + 增值税(PPN 11%)
         + 所得税预提(PPh 22) + 平台佣金 + 广告费 + 尾程配送费
         + 退货损耗 + 汇率波动预留 + 认证摊销 + 仓储费 + 支付手续费

毛利额 = 零售价 - 总成本（不含广告费）
净利润 = 零售价 - 总成本（含广告费）

毛利率 = 毛利额 ÷ 零售价
净利率 = 净利润 ÷ 零售价
```

### 印尼跨境进口税制说明

印尼对进口商品实行三级税制（CIF为计税基础）：

```
计算公式：
1. 关税(BM) = CIF × 关税税率
2. 增值税(PPN) = (CIF + BM) × 11%
3. 所得税(PPh 22) = (CIF + BM) × 7.5%（无API）/ 2.5%（有API）
```

### 2026年税制关键更新

- **PPN 12%政策**：2025年1月Prabowo总统将12%限定于奢侈品，多数商品通过DPP Nilai Lain公式维持有效税率**11%**（2026年延续）
- **Coretax系统**：2025年1月起所有PKP实体强制使用Coretax平台进行电子发票和申报
- **PPh 22无变化**：有API 2.5%，无API 7.5%

**行动建议**：建议在印尼注册本地PT公司并申请API。长期来看，PPh 22从7.5%降至2.5%可节省约2/3的所得税预提。

### 3. 实体架构建议

```
中国贸易公司（卖方/制造商）
    ↓ FOB/CNF出口
印尼本地公司 PT PMA（进口商+电商运营）
    ↓ API持有者 + PKP（VAT注册纳税人）
Shopee / TikTok Shop / Tokopedia（本地卖家账户）
```

**优势**：
- 本地PT可注册API，PPh 22从7.5%降至2.5%
- 本地PKP可抵扣PPN进项税（进口PPN可抵扣销售PPN）
- 避免"跨境卖家"标签，佣金结构更优
- 提升印尼消费者信任（本地卖家信任度更高）
