## 1. 执行摘要

印度尼西亚是东南亚最大的数字经济体，2024年电商GMV突破**750亿美元**，预计2026年超**1000亿美元**，年均增速约**19%**。作为全球第10大电商市场，印尼拥有**2.22亿互联网用户**、**6570万网购用户**和极高的移动互联网渗透率。

经过11步系统化选品流程，本报告最终推荐进入**6个低合规门槛SKU**作为首批产品，同步启动清真认证流程储备高壁垒品类。

### 核心发现

| 维度 | 数据 |
|------|------|
| 电商GMV 2024 | US$75 Billion |
| 电商增速 CAGR | 15-19% |
| 首选平台 | Shopee（首发）+ TikTok Shop（增长最快） |
| 首批SKU数 | 6个（低合规门槛） |
| 平均毛利率 | **32.5%** |
| 首年月GMV目标 | $15,000（M12） |
| 合规门槛 | 低合规SKU直接进入，高壁垒品类并行认证 |

| 维度 | 描述 |

- **跨境友好度**: 平台代扣代缴，无需本地注册

## 10. 风险监控与退出机制

### 10.1 风险红线

| 红线 | 阈值 | 触发动作 |
|------|------|---------|
| 毛利率 < 15% | 任SKU | 停售+重新定价 |
| 广告ACOS > 毛利率80% | 任Campaign | 暂停广告+优化 |
| 退货率 > 15% | 月维度 | 停售+质检 |
| 差评率 > 10% | 任SKU | 下架+供应商沟通 |
| 清真认证被拒 | 任意 | 暂停该品类 |
| 月GMV < 预测50% | 连续2月 | 暂停广告+复盘 |

### 10.2 退出机制

| 条件 | 动作 |
|------|------|
| 3个月内未达盈亏平衡 | 缩减SKU至Top 3 |
| 6个月仍未盈利 | 全品类退出 |
| 合规/法律问题 | 立即暂停+律师介入 |
| 政策重大变化 | 评估影响+调整策略 |
