## 一、用户画像

### 核心用户画像

| 维度 | 描述 |
|------|------|
| **年龄** | 18–35岁为主（Gen Z + 千禧一代），中位年龄30.4岁 |
| **性别** | 女性占比~55%（美妆/时尚/家居），男性~45%（电子/运动/宠物） |
| **地域** | 爪哇岛占60%（雅加达/泗水/万隆），苏门答腊15%，其他25% |
| **收入** | 中产阶级为主（月入IDR 500–1500万 / $315–945） |
| **教育** | 大专及以上学历占网购用户~65% |
| **设备** | 手机购物占69.4%，几乎无桌面购物习惯 |
| **日均上网** | 4.5小时（含社交媒体2.5小时） |
| **日均TikTok使用** | 60分钟+ |

| 类型 | 占比 | 特征 | 偏好品类 |
|------|:----:|------|---------|
| **价格猎手** | 35% | 最低价导向，比价频繁，COD偏好 | 配饰/日用品 |

## 四、购买决策路径

### 典型决策流程

```
发现需求 → 社交媒体/TikTok种草 → Shopee搜索比价 → 
查看评价(≥4星) → 确认清真/BPOK认证 → 对比价格 → 
下单(电子钱包优先) → 收货验货 → 评价/分享
```

### 关键转化节点

| 节点 | 转化率影响因素 | 优化建议 |
|------|-------------|---------|
| 发现→种草 | KOL内容质量 | 投资TikTok内容 |
| 种草→搜索 | 品牌词SEO | 优化标题关键词 |
| 搜索→浏览 | 主图+价格 | 高质量主图+竞争力定价 |
| 浏览→加购 | 评价+认证 | 积累好评+展示认证 |
| 加购→支付 | 支付方式 | 支持主流电子钱包 |
| 支付→复购 | 物流+品质 | 快速发货+品质稳定 |
