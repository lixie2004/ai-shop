### 1.4 跨境政策环境

| 政策 | 影响 |
|------|------|
| **跨境电商免税门槛 $3/件**（2023年从$75骤降） | 几乎所有跨境包裹需缴税 |
| **清真认证法** — 2026年10月17日食品/化妆品/保健品强制 | 无Halal认证将无法合法销售 |
| **BPOM注册** — 食品/化妆品/药品强制 | 无认证=不信任，无法上架 |
| **VAT 11%** | 合规成本上升 |
| **数据本地化** | 需印尼境内存储用户数据 |
| **纺织品进口配额** | 成衣/纺织品进口受限 |

### 1.5 关键判断

印尼是**东南亚最具战略意义的电商市场**。$900亿+规模、15%+增速、2.84亿人口、60%直播参与率——任何东南亚战略都绕不开。但**监管环境也是东南亚最复杂的**：$3免税门槛、清真认证强制、BPOM注册、纺织品配额等多重壁垒。入场策略必须兼顾市场机会和合规成本。
