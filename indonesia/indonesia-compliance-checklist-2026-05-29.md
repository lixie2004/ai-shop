## 1. 印尼合规全景概览

### 1.1 关键法规变更

| 法规 | 生效日期 | 核心变更 | 影响范围 |
|:----|:--------|:---------|:---------|
| **PMK 4/2025** | 2025-03-05 | 进口免税门槛从 **$75降至$3**；关税改用渐进税率（0%/15%/25%） | 所有进口商品 |
| **BPJPH Halal强制** | 2026-10-17 | 化妆品、消费品、化学品必须Halal认证方可流通 | No.7, No.9 |
| **PerBPOM 25/2025** | 2026 | 化妆品成分技术要求更新，监管收紧 | No.7, No.9 |
| **SDPPI 2026** | 2026-01-15 | RLAN设备需本地实验室或MRA伙伴国实验室测试报告 | No.1, No.2, No.5, No.8, No.10 |
| **Permendag 16/2025** | 2025 | 进口许可证API-U/API-P分类细化 | 所有SKU需考虑 |

### 1.2 $100 最低限价影响

根据跨境电商政策，单次进口申报金额 **≥$100** 方可通过正规跨境电商渠道。PMK 4/2025进一步将免税门槛从$75降至**$3**，意味着几乎每单进口都需缴纳进口关税和增值税（11% VAT）。

**套装组合策略**（Top 10中8个SKU已通过套装跨越$100门槛）可降低单件合规分摊成本。

### No.1 智能宠物喂食器套装（含摄像头+WiFi）

#### 准入认证要求

| 认证类型 | 是否必需 | 说明 |
|:--------|:--------|:-----|
| **SDPPI** | ✅ **强制** | 含WiFi模块（2.4GHz/5GHz）和摄像头无线传输功能，必须取得SDPPI认证 |
| **SNI** | ❌ 非强制 | 宠物品类食品接触部分无强制SNI，但建议可出具食品接触材料合规报告 |
| **Halal** | ❌ 不适用 | 宠物用品不在此阶段Halal强制范围内 |

#### 认证费用估算（USD）

| 项目 | 金额 | 备注 |
|:----|:----|:-----|
| SDPPI政府规费（本地测试报告） | ~$430 | IDR 7,000,000 |
| SDPPI认证代理/测试服务费 | $1,500–$3,000 | 含实验室测试协调 |
| 商标注册（DGIP印尼） | $300–$600 | R Mark，周期18个月，建议早启动 |
| **合计估算** | **$2,230–$4,030** | |

#### 认证周期

| 阶段 | 预估周期 |
|:----|:--------|
| 技术文档准备 | 2–4 周 |
| 样品送测（本地实验室） | 4–6 周 |
| SDPPI审查与发证 | 2–4 周（10–30工作日） |
| **总计** | **8–14 周** |

#### 风险评级：🟡 **中**

- WiFI模块频段需适配印尼2.4GHz频段规范
- 2026年起新规要求RLAN设备的测试报告须来自本地实验室或印尼MRA伙伴国实验室，海外报告可能不被认可
- 证书有效期3年，需到期前续期

#### 成分/材料限制

- 食品接触料仓/碗：建议使用食品级PP/不锈钢，材料需提供食品接触安全证明（非强制SNI，但BPOM可能抽检食品接触材料）
- 锂电池（如有）：须通过UN38.3认证，运输需符合IATA/DGR

#### 标签和包装要求

- 产品标签须包含印尼语产品名称、制造商/进口商信息、SDPPI认证编号
- SDPPI标签格式：`SDPPI + 证书编号 + PLG.ID`
- 警告语需印尼语："Peringatan"格式
- 包装上须注明WiFi频段和功率信息
- 建议：全部用户界面和说明书使用印尼语

#### 进口许可证要求

- 进口商须持有 **API-U（普通进口商许可证）** 或 **API-P（生产进口商许可证）**
- 需NIB（商业登记号）+ NPWP（税号）
- 通过正规海关申报（HS编码：8424.89 / 8517.62）

#### 替代方案

- 若WiFi模块认证周期过紧，可暂时移除WiFi模块，以"基础版非联网喂食器"先行上市，后再做含WiFi版本升级
- 与本地SDPPI认证代理（如JTEN、TÜV SÜD）合作可加速流程

### No.2 安防摄像头+门锁套装

#### 准入认证要求

| 认证类型 | 是否必需 | 说明 |
|:--------|:--------|:-----|
| **SDPPI** | ✅ **强制** | 摄像头含WiFi/蓝牙模块；智能门锁含蓝牙/WiFi/NFC模块 |
| **SNI** | ❌ 非强制 | 安防类产品目前非SNI强制范畴 |
| **PDPA合规** | ✅ **建议** | 印尼PDPA 2024年生效，涉及人脸/视频数据采集需合规 |

#### 认证费用估算（USD）

| 项目 | 金额 | 备注 |
|:----|:----|:-----|
| SDPPI摄像头认证 | ~$2,000–$4,000 | 含实验室测试 |
| SDPPI门锁认证（独立申请） | ~$2,000–$4,000 | 含蓝牙/NFC测试 |
| 套装组合可能享受同系列折扣 | - | 同一品牌/系列产品可批量申请降价 |
| PDPA合规咨询 | $500–$2,000 | 隐私政策+数据声明 |
| **合计估算** | **$4,500–$10,000** | |

#### 认证周期

| 阶段 | 预估周期 |
|:----|:--------|
| SDPPI技术文档+测试 | 4–8 周 |
| SDPPI审查与发证 | 2–4 周 |
| PDPA合规部署 | 2–4 周（可并行） |
| **总计** | **8–12 周** |

#### 风险评级：🟡 **中**

- 两个设备需各自取得SDPPI证书，增加时间和成本
- 数据隐私敏感——PDPA规定视频数据必须有明确的用户同意机制和数据存储声明
- 2026 SDPPI新规需本地测试报告

#### 成分/材料限制

- 内置锂电池：须UN38.3认证
- 本地存储方案（SD卡/NAS）优于云端存储，以降低PDPA风险

#### 标签和包装要求

- 所有电子设备标签须含SDPPI证书编号
- 安防摄像头须注明"仅供安防用途"印尼语声明
- 隐私声明须印在包装或附在随附材料中
- 印尼语说明书至少包含操作说明、隐私政策和频段信息

#### 进口许可证要求

- API-U 或 API-P
- HS编码：摄像头 8525.89 / 门锁 8301.40
- 若含加密功能，可能需要额外科技部审批（较少见，但建议确认）

#### 替代方案

- 分阶段上市：先上摄像头单品，门锁延后2-3个月
- 若SDPPI周期过长，可考虑与已获SDPPI认证的ODM厂商合作，直接使用其认证编号（须确认copy control许可）

### No.3 轻量安全座椅+推车套装

#### 准入认证要求

| 认证类型 | 是否必需 | 说明 |
|:--------|:--------|:-----|
| **SNI** | ✅ **强制** | 儿童安全座椅属于SNI强制产品范畴 |
| **印尼商标** | ✅ **必需** | SNI申请前置条件（R Mark或受理回执） |
| **SDPPI** | ❌ 不适用 | 纯机械产品，无无线模块 |
| **Halal** | ❌ 不适用 | 母婴大件不在当前Halal强制范围内 |

#### 认证费用估算（USD）

| 项目 | 金额 | 备注 |
|:----|:----|:-----|
| 印尼商标注册（DGIP） | $300–$600 | 周期18个月，回执可用 |
| SNI产品测试（本地实验室） | $2,000–$4,000 | 安全座椅碰撞测试费用较高 |
| ISO 9001 QMS审计 | $1,000–$2,000 | 工厂审计费用 |
| SNI认证代理服务费 | $2,000–$4,000 | 含SIINAS系统操作 |
| 年度监督审计 | $1,000–$2,000/年 | 持续成本 |
| **首年合计** | **$5,300–$12,600** | |
| **后续年维持** | **~$2,000–$4,000/年** | |

#### 认证周期

| 阶段 | 预估周期 |
|:----|:--------|
| 商标注册受理 | 1–3 个月（获回执后即可启动下一步） |
| 技术文档准备 | 4–6 周 |
| 样品测试 | 6–10 周（碰撞测试较慢） |
| 工厂审计 | 3–4 周 |
| SNI CoC + SPPT审批 | 2–4 周 |
| **总计（不含商标全流程）** | **15–24 周（约4–6个月）** |
| **全流程含商标** | **18–24 个月** |

#### 风险评级：🔴 **高**

- 全流程长，SNI 2025新规下认证难度提升
- 需指定印尼本地官方代表（IOR），且IOR只能代理一个品牌
- 工厂须通过ISO 9001审计——无此基础将严重延迟
- 安全座椅的碰撞测试是高风险项——不通过需重新设计
- 后续年审须持续维护

#### 成分/材料限制

- 面料：需通过色牢度、甲醛含量、阻燃性测试（印尼热带气候下要求更高）
- 塑料件：须符合SNI相关材料标准
- 防霉面料需提供微生物测试报告

#### 标签和包装要求

- 产品须印SNI标志 + 证书编号
- 标签须印尼语：产品名称、型号、制造日期、安全警告、使用限制
- 安全座椅须注明适用体重/年龄范围（印尼语）
- 安装说明须包含印尼语图示版
- 包装须印SNI认证号

#### 进口许可证要求

- API-P（生产进口商，更适合此品类）或API-U
- 需"进口批准函"（PI – Persetujuan Impor）——特定安全产品需额外申请
- 所有货物须进入指定本地仓库，不得直接门到门配送
- 500+ 台时可能需额外SPPT配额管理

#### 替代方案

- **最高风险SKU**：建议以OEM/ODM模式与已取得SNI认证的印尼本地制造商合作，直接使用其认证
- 或作为长期战略项目（12个月+）分阶段推进，不适合快速上市
- 推车部分可独立先行上市（推车非SNI强制），安全座椅后行

### No.4 Gorpcore户外服装套装

#### 准入认证要求

| 认证类型 | 是否必需 | 说明 |
|:--------|:--------|:-----|
| **SNI** | ❌ 非强制 | 普通服装无SNI强制要求 |
| **Halal（2026）** | ✅ **2026-10-17起强制** | 服装/纺织品被列入消费品强制Halal类别（Phase 2） |
| **纺织品合规** | ✅ **建议** | 面料安全标准（甲醛/偶氮染料）可能被抽检 |

#### 认证费用估算（USD）

| 项目 | 金额 | 备注 |
|:----|:----|:-----|
| Halal认证（BPJPH + MUI） | $1,500–$4,000 | 纺织品路径 |
| 面料安全测试（可选） | $500–$1,500 | 甲醛/偶氮/Nickel等 |
| 印尼商标注册（建议） | $300–$600 | 建议注册保护品牌 |
| **合计估算** | **$2,300–$6,100** | |

#### 认证周期

| 阶段 | 预估周期 |
|:----|:--------|
| Halal认证：审核+审计+发证 | 8–12 周 |
| 商标注册（可选） | 12–18 个月 |
| **总计（仅必需项）** | **8–12 周** |

#### 风险评级：🟢 **低**

- 服装品类准入门槛在Top 10中最低
- 2026年10月前还可无Halal认证销售（但有风险：届时需撤回未认证产品）
- 主要风险来自尺码适配和面料色牢度（非合规风险，是产品风险）

#### 成分/材料限制

- 偶氮染料：须符合欧盟REACH或类似标准（印尼海关可能抽检）
- 甲醛含量：婴幼儿接触类有上限，但成人户外服影响较小
- 防晒UPF50+声明：建议有第三方测试报告支持（非强制但有利于市场营销和避免虚假声明争议）

#### 标签和包装要求

- 服装标签须印尼语标注：面料成分、洗涤说明、尺寸、产地
- 印尼法规要求纺织品的标签须注明：`Komposisi Bahan`（面料成分）、`Cara Perawatan`（护理说明）
- 从2026年10月起，包装须展示Halal标志（如适用）
- 穆斯林友好设计建议注明（长袖、宽松等描述非强制但有利于市场）

#### 进口许可证要求

- API-U 普通进口商许可证即可
- HS编码：6203 / 6204（男装/女装外套类）
- 纺织品进口税率：PMK 4/2025下约15-25%（需确认具体HS编码）

#### 替代方案

- 暂无高难度认证障碍，无需替代方案
- 建议：在2026年10月前启动Halal认证，确保不中断销售

### No.5 宠物GPS追踪器

#### 准入认证要求

| 认证类型 | 是否必需 | 说明 |
|:--------|:--------|:-----|
| **SDPPI** | ✅ **强制** | 含GPS/北斗接收器 + 蓝牙/WiFi（如有）通信模块 |
| **SNI** | ❌ 非强制 | 宠物电子设备无SNI强制 |
| **Halal** | ❌ 不适用 | 非食品/化妆品/消费品范畴 |

#### 认证费用估算（USD）

| 项目 | 金额 | 备注 |
|:----|:----|:-----|
| SDPPI认证（本地报告） | ~$2,000–$3,500 | 含无线频段测试 |
| GPS/北斗射频测试 | 含在上项中 | |
| **合计估算** | **$2,000–$3,500** | |

#### 认证周期

| 阶段 | 预估周期 |
|:----|:--------|
| 文档准备+送测 | 4–6 周 |
| SDPPI审查发证 | 2–4 周 |
| **总计** | **6–10 周** |

#### 风险评级：🟢 **低**

- 认证路径清晰，SDPPI已有成熟方案
- GPS/北斗双模的技术标准不构成额外障碍
- 风险点：Signal在偏远群岛覆盖有限——但这属产品性能问题而非合规问题

#### 成分/材料限制

- 内置锂电池：须UN38.3认证和空运/MSDS文件
- 防水IP67：需在文档中说明测试标准（IEC 60529）
- 若设备支持eSIM/eMTC，需运营商合作（更复杂）

#### 标签和包装要求

- SDPPI标签须物理印刷在设备外壳或包装上
- 印尼语说明书须包含：频段信息、电池安全提示、防水等级说明
- 跟踪器的数据隐私声明（定位数据收集说明）——按PDPA推荐

#### 进口许可证要求

- API-U
- HS编码：8517.62（通信设备）或 8526.91（无线电导航辅助设备）

#### 替代方案

- 无重大障碍，无需替代方案
- 注意：单品$45-55低于$100门槛，需套装组合销售或本地仓备货配送

### No.6 防潮实木家具系列

#### 准入认证要求

| 认证类型 | 是否必需 | 说明 |
|:--------|:--------|:-----|
| **SNI** | ❌ 非强制 | 普通家具非SNI强制类别 |
| **木材进口合规** | ✅ **必需** | 印尼对木材进口有严格的原产地证明和检疫要求 |
| **Halal** | ❌ 不适用 | 家具不在当前Halal强制范围内 |

#### 认证费用估算（USD）

| 项目 | 金额 | 备注 |
|:----|:----|:-----|
| 木材原产地证明 + SVLK合规 | $500–$1,500 | 印尼木材合法性验证系统 |
| 植物检疫证书 | $100–$300 | 出口国出具 |
| 防潮涂层安全测试（可选） | $300–$800 | 确保甲醛/挥发性有机物合规 |
| **合计估算** | **$900–$2,600** | |

#### 认证周期

| 阶段 | 预估周期 |
|:----|:--------|
| 木材证明文件准备 | 2–4 周 |
| 检疫认证 | 1–2 周（出口国流程） |
| **总计** | **3–6 周** |

#### 风险评级：🟢 **低**

- 家具准入门槛低，无强制产品认证
- 核心风险在物流（17,000岛屿配送）和运输破损——非合规风险

- HS编码：9403.30（木制家具）
- 木制品的进口关税：PMK 4/2025下可能为15-25%区间
- 11% VAT
- 需通过指定港口清关（主要是Tanjung Priok或Tanjung Perak）
- ⚠️ **家具进口**受Permendag 29/2024限制较严，大宗家具可能需要配额和额外进口批准

#### 替代方案

- 考虑本地采购木材/使用印尼本地藤编材料，从中国仅进口五金件和设计图纸，在印尼本地组装
