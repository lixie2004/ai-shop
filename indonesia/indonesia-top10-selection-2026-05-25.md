## 1. Screening Methodology

### 三维评分模型
| Dimension | Weight | Scoring Criteria |
|-----------|--------|------------------|
| **市场吸引力** (Market Attractiveness) | 40% | Search volume, growth rate, market size |
| **竞争度** (Competition Level) | 30% | Number of sellers, brand concentration, price war intensity |
| **合规可行性** (Compliance Feasibility) | 30% | Halal/BPOM/SNI requirements, certification timeline, import restrictions |

### 筛选门槛
| Threshold | Value | Rationale |
|-----------|-------|-----------|
| 月搜索量 | >5,000 | Sufficient demand signal |
| 预估毛利率 | >25% | Viable after all costs |
| 合规风险 | 中低 | No mandatory certifications >12 weeks |
| 头部集中度 | <70% | Not a winner-take-all market |

## 4. Phased Entry Strategy

### Phase 1 (Month 1-3): 低门槛快速验证
- Halal Sunscreen SPF50
- Clean Beauty Halal Lip Tint
- Men's Halal Skincare Set
- Baby Mosquito Repellent

**理由**: BPOM + Halal认证周期较短（8-12周），可快速上架验证市场反应

### Phase 2 (Month 4-6): 中等资质要求
- Functional Gummy Supplements (BPOM注册进行中)
- Collagen Drink (Halal)
- Halal Pet Food
- Montessori Toy Set

**理由**: 需要更长认证周期，Phase 1期间完成注册

### Phase 3 (Month 7-9): 技术/智能产品
- Gaming Phone Cooler
- AI Baby Monitor

**理由**: 需要SDPPI认证 + 更高客单价，需先建立品牌信任

## 6. Risk Warnings

| Risk | Mitigation |
|------|------------|
| Halal认证周期可能超预期 | 提前6个月启动认证，选择有经验的本地代理 |
| MoT Reg 31/2023限制跨境低价商品 | 定价 >USD 100 或使用本地仓 |
| 2026年10月保健品强制Halal | 立即启动Halal认证，不可延迟 |
| 汇率波动 (IDR) | 定价留出15%缓冲，定期调整 |
| 平台政策变化 | 多平台运营，避免单一平台依赖 |
