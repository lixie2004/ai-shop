## 建模方法论与假设

### 成本模型公式

```
总成本 = 采购成本 + 头程物流 + 进口关税 + 进口增值税(PPN) 
         + 所得税预提(PPh) + 平台佣金 + 广告费 + 尾程配送费 
         + 退货损耗 + 汇率波动预留 + 认证摊销 + 仓储费 + 支付手续费

毛利额 = 零售价 - 总成本（不含广告费）
净利润 = 零售价 - 总成本（含广告费）

毛利率 = 毛利额 ÷ 零售价
净利率 = 净利润 ÷ 零售价
```

### 关键假设（2026年5月）

| 假设项 | 取值 | 依据 |
|--------|------|------|
| **进口关税（BM）** | 0-25%（按HS Code） | 印尼2025年MFN平均8.1%，2025年PMK No.4/2025简化分类 |
| **增值税（PPN）** | 11%（2022年上调） | 印尼增值税标准税率，计划2025年逐步升至12% |
| **所得税预提（PPh 22）** | 2.5-7.5%（有进口许可证API） | 无API的进口商PPh 22为7.5% |
| **所得税预提（PPh 22无API）** | 7.5% | 跨境电商常见情形 |
| **低价值商品免税额** | US$3以下 | PMK No.199/2019，极低门槛 |
| **IDR/USD汇率** | 16,400 | 2026年5月参考 |
| **CNY/IDR汇率** | 2,250 | 1 CNY ≈ 2,250 IDR |
| **汇率波动预留** | 5% | 基于IDR年化波动率（较MYR更高） |
| **Shopee印尼佣金** | 5.0-7.0%（跨境卖家） | 2026年标准费率，Star/Star+卖家4.25-6.0% |
| **Tokopedia佣金** | 3.0-5.5%（Power Merchant） | 2026年标准费率 |
| **TikTok Shop佣金** | 3.0-4.5% | 2026年新卖家费率 |
| **支付手续费** | 2.5%（加权） | COD 1.5% + 线上支付3.0% 加权 |
| **ACoS（广告支出占比）** | 15-25%（按品类） | 印尼跨境行业基准 |
| **COD占比** | 65% | 印尼COD支付占60-70% |

### 印尼跨境进口税制说明

印尼对跨境电商商品实行三级税制：

```
CIF价值 → 进口关税(BM) → 增值税(PPN 11%) → 所得税(PPh 22)

计算公式：
1. 关税(BM) = CIF × 关税税率
2. 增值税(PPN) = (CIF + BM) × 11%
3. 所得税(PPh 22) = (CIF + BM) × 7.5%（无API）
```
