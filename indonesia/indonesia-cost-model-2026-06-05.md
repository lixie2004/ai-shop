## 一、成本模型方法论

### 1.1 成本结构总览

每个 SKU 的全成本模型包含以下 10 大成本类别：

```
┌──────────────────────────────────────────────────────┐
│                 全成本结构瀑布图                       │
│                                                      │
│  销售收入 (GMV)                                      │
│  ├─ ① 采购成本 (COGS)                                │
│  │   → 出厂价 + 包装 + 质检                           │
│  ├─ ② 头程国际物流                                    │
│  │   → 中国仓库 → 印尼港口/保税仓                      │
│  ├─ ③ 进口关税 (Customs Duty)                        │
│  │   → 基于 HS 编码的从价税率                          │
│  ├─ ④ 进口增值税 (Import VAT / PPN)                  │
│  │   → 11% × (CIF + 关税)                            │
│  ├─ ⑤ 印尼尾程配送 (Last-Mile)                       │
│  │   → Jakarta 圈内 / 爪哇岛 / 外岛                   │
│  ├─ ⑥ 平台佣金 + 交易手续费                           │
│  │   → 佣金(3-5%) + 交易费(2.18%) + 服务费            │
│  ├─ ⑦ 广告与营销 (Advertising)                       │
│  │   → Shopee Ads CPC + 内容营销+KOL                  │
│  ├─ ⑧ 认证与合规 (Certification)                     │
│  │   → BPOM / Halal / SNI 年化摊销                    │
│  ├─ ⑨ 退货与损耗 (Returns & Shrinkage)               │
│  │   → 品类退货率 × 单件成本                           │
│  └─ ⑩ 企业所得税预留 (Corporate Tax Provision)        │
│      → 22% 企业所得税率 (中小企业优惠后约 11-16.5%)    │
│                                                      │
│  ═══════════════════════════════════                  │
│  = 税后净利润                                         │
└──────────────────────────────────────────────────────┘
```

### 1.2 关键假设说明

| 参数 | 假设值 | 依据 |
|------|--------|------|
| **汇率** | 1 USD = 16,200 IDR | 市场报告基准汇率 |
| **VAT (PPN)** | 11% | 印尼 2024 年起适用税率 |
| **企业所得税** | 16.5%（有效税率） | 年应税所得 < 500 亿 IDR 部分 11%，超过部分 22%；此处按中型跨境卖家加权 |
| **平台佣金 (Shopee)** | 3-5%（按品类） | Shopee Indonesia 2025 费率表 |
| **交易手续费** | 2.18% | Shopee 统一费率 |
| **头程物流** | $3-6/kg（空运经济） | 中国→印尼跨境物流行情 |
| **尾程配送 (Jakarta圈)** | Rp 15,000-25,000/件 | JNE/J&T 电商小件报价 |
| **尾程配送 (爪哇岛)** | Rp 25,000-35,000/件 | 区域差异 |
| **广告费率 (Shopee Ads CPC)** | Rp 300-800/点击 | 品类竞争度差异 |
| **广告占GMV比** | 5-12% | 新品期高至 12%，成熟期降至 5% |
| **退货率** | 2-8% | 品类差异（美妆高、收纳低） |
| **运营人员分摊** | 按SKU数均摊 | 单人多品运营模式 |

### 1.3 平台费率明细（Shopee Indonesia，2025-2026）

| 费用项目 | 费率 | 说明 |
|---------|------|------|
| 基础佣金 | 2-6% | 按品类，美妆/个护 4-6%，家居/收纳 3-4%，宠物 3-4% |
| 交易手续费 | 2.18% | 支付处理，按成交额 |
| 技术服务费 | 0-5% | 部分品类豁免，印尼暂未全面征收 |
| Coins Cashback（可选） | 3-5% | 影响搜索排名，实际多为必选 |
| Free Shipping（可选） | 3-5% | 影响转化率，实际多为必选 |
| **平台综合费率** | **10-15%** | 含上述所有费用 |
