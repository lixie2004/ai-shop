## 关键认证详解

### BPOM (食品药品管理局)
| 项目 | 详情 |
|------|------|
| **适用产品** | 化妆品、食品、补充剂 |
| **费用** | $2,000-4,000 |
| **周期** | 3-6个月 |
| **要求** | 成分列表+检测报告+GMP |
| **有效期** | 5年 |
| **注意** | 需印尼本地代表(Agent) |

### Halal认证 (BPJPH/MUI)
| 项目 | 详情 |
|------|------|
| **适用产品** | 食品、化妆品、服饰 |
| **费用** | $1,000-3,000 |
| **周期** | 2-4个月 |
| **要求** | 原料清真+生产过程审核 |
| **有效期** | 2-4年 |
| **注意** | 2024年起强制(分阶段) |

### SDPPI (无线设备)
| 项目 | 详情 |
|------|------|
| **适用产品** | 有无线功能的电子设备 |
| **费用** | $1,500-3,000 |
| **周期** | 2-3个月 |
| **要求** | 射频测试+型号核准 |
