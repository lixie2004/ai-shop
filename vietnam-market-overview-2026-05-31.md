## 六、跨境电商政策环境

### 6.1 电商法（2025年通过，2026年7月1日生效）

越南首部专门《电商法》于**2025年12月10日**正式通过，2026年7月1日生效，重大变化包括：

| 新规 | 内容 |
|------|------|
| **平台分类** | 四类：直销平台、中介平台、社交电商、综合平台 |
| **卖家实名** | 卖家必须提供完整身份/企业信息，不得匿名经营 |
| **商品标签** | 必须完整展示保质期、生产批号等强制信息 |
| **直播监管** | 平台须验证主播身份，保存直播音视频数据备查 |
| ** affiliate营销** | KOL/KOC身份须验证，平台须管控违规链接 |
| **算法透明** | 平台须披露推荐算法逻辑 |
| **外国平台** | 使用越南语或".vn"域名的外国平台须在越南设立实体或指定代表，并缴纳保证金 |

### 6.2 关税与清关政策

| 政策 | 详情 |
|------|------|
| **低值免税取消** | 2025年起取消小额商品进口免税，**所有进口商品均需缴税** |
| **VAT税率** | 大多数进口商品适用**5%–10%**增值税 |
| **进口关税** | 根据商品类别和原产国不同，一般**0%–30%**；FTA下可享优惠税率 |
| **小额豁免（草案）** | 单笔≤VND 200万（≈$80）可免物理检验和进口关税，但年度累计≤VND 9600万（≈$3,900）/进口商/年 |
| **关税计算方式** | CIF价值 → 进口关税 → 其他税 → VAT（基于CIF+关税+其他税） |
| **HS编码** | 须准确分类，错误分类可能导致多缴关税或延误 |
| **FTA优惠** | ACFTA、ATIGA、VJEPA等协定下可享优惠税率，需原产地证书（CO） |
