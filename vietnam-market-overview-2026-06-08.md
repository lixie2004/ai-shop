### 3.2 各平台深度分析

#### 🟠 Shopee（Sea Limited）：守擂者

| 维度 | 详情 |
|------|------|
| 市场地位 | 越南电商绝对龙头，56%份额 |
| 增长态势 | 增速显著放缓至+16% YoY，Q3进一步降至+4% |
| 核心优势 | 物流网络最广（Shopee Express）、用户基数最大、购物习惯最成熟、品类最全 |
| 品牌化贡献 | 品牌商店（Mall）仅占3.4%店铺数但贡献28.7%营收 |
| 跨境卖家意义 | **流量基本盘**，适合走量型卖家，但增速放缓、竞争激烈 |

#### 🔵 TikTok Shop（字节跳动）：颠覆者

| 维度 | 详情 |
|------|------|
| 市场地位 | 第二大平台，份额从29%飙升至39-41% |
| 增长态势 | **+69% YoY**（2025全年），H1更高达**+148% YoY** |
| 核心优势 | "Shoppertainment"模式、AI算法推荐、短视频/直播内容驱动 |
| 直播电商规模 | 预计2026年达$110亿 |
| 跨境卖家意义 | **增长最快渠道**，但需要短视频/直播内容能力 |

#### 🟢 Lazada（阿里巴巴）：持续萎缩

- 销售下降48%，份额缩至2-3%
- 阿里战略重心转移，对越南市场投入减少

#### 🟡 Tiki：边缘化

- 销售下降63%，份额缩至1-2%
- 从曾经的越南第二大平台滑落至边缘位置

#### 其他

- **Sendo**：已转型为Sendo Farm，专注农产品
- **Amazon**：越南市场存在感极低，主要在出口端（越南卖家→Amazon全球）
- **本土独立站**：品牌DTC模式正在萌芽

### 3.3 关键判断

越南电商市场已进入**双寡头时代**，Lazada和Tiki的溃败速度超出所有人预期。对于跨境卖家：**Shopee是基本盘，TikTok Shop是增长引擎**，两者必须同时布局。品牌化是核心趋势——仅3.4%的官方品牌店铺贡献了28.7%的营收，认证正品店铺的溢价能力极强。

## 四、跨境进口政策与关税环境

### 4.1 关税体系

| 维度 | 详情 |
|------|------|
| **关税表** | Circular No.31/2022/TT-BTC及2025年修订版（Decree 199/2025） |
| **优惠协定** | 18个优惠贸易协定：CPTPP、RCEP、EVFTA（欧越自贸）、ASEAN等 |
| **跨境电商免税额度** | 单次VND 100万（~$40），年度VND 200万（~$80） |
| **跨境电商进口税率** | 行邮税，根据品类一般10%-70%不等 |
| **增值税（VAT）** | 标准10%（部分品类5%或0%） |
| **特别消费税（SCT）** | 针对烟酒、汽车、大排量摩托车等 |
| **中国→越南关税优势** | ASEAN-China FTA（ACFTA），多数商品0-5%关税 |

### 4.2 2025年关键政策变化

| 政策 | 生效/预计时间 | 核心内容 | 对跨境卖家的影响 |
|------|:------------:|-----------------|-----------------|
| **平台税务扣缴令（Decree相关）** | **2025年7月1日** | 电商平台须代扣代缴个人卖家的VAT和个税 | ⚠️ 合规成本上升，跨境卖家需关注 |
| **电商法草案** | 2025年10月提交审议 | 外国卖家须在越设代表处或指定本地代表 | ⚠️ 可能增加运营门槛 |
| **进口商品标准** | 新规要求 | 进口商品须符合越南技术标准和安全要求 | ⚠️ 部分品类需额外认证 |
| **跨境电商海关程序法令草案** | 拟订中 | 规范跨境电商物品报关、风险管理等 | 📋 持续跟踪 |
| **CPTPP关税修订（Decree 13/2025）** | 2025年1月21日 | 对特定国家商品增加原产地条件 | ⚠️ 影响转口贸易路径 |

### 4.3 跨境贸易规模

| 指标 | 2024 | H1 2025 | 2027目标 |
|------|------|---------|---------|
| 跨境进口GMV | $5.645亿 | $2.85亿 | — |
| 跨境进口增速 | **+43%** | +7% | — |
| 跨境出口（B2C） | $3.5亿（2023） | — | **$55亿**（政府目标） |
| 跨境进口件数 | 3.241亿件 | 1.64亿件 | — |

### 4.4 关键判断

2025年是越南跨境电商监管的关键转折年。三大趋势：
1. **平台代扣税已落地**（7月生效）：跨境卖家税务合规成本增加
2. **电商法即将通过**：可能要求外国卖家设立本地实体
3. **跨境进口增速放缓**（+7% vs 去年+43%）：部分受监管收紧影响

对中国卖家而言，**中国-东盟自贸协定（ACFTA）** 提供了关税优势，且中越陆运跨境物流成本远低于海运/空运。建议提前做好税务合规安排。

### 6.4 关键判断

越南消费者呈现**年轻化、移动化、内容驱动**三大核心特征：

1. **Gen Z主导**（53.4%）：内容消费习惯强，品牌忠诚度低但尝鲜意愿高
2. **品牌化加速**：消费者越来越倾向选择正规渠道（官方店贡献28.7%营收），品牌溢价空间大
3. **环保意识觉醒**：75%消费者关注可持续品牌，ESG定位可能成为差异化卖点
4. **跨境需求旺盛**：进口商品2024年增长43%，显示越南消费者对海外商品有强烈需求

| **ViettelPay** | Viettel电信旗下，政府背书 | 建议接入 |
| **ShopeePay** | Shopee生态内支付 | Shopee卖家自动接入 |
| **银行转账** | 大促期间仍广泛使用 | 接受 |
| **货到付款（COD）** | 仍占一定比例但下降中 | 注意现金流和退货率 |
| **VietQR** | 全国统一QR码，跨行扫码便捷 | 关键支付基础设施 |
| **BNPL（先买后付）** | 开始渗透越南市场 | 新兴趋势 |

### 7.3 仓储基础设施

| 维度 | 现状 |
|------|------|
| **平台自营仓** | Shopee/TikTok Shop/Lazada均在胡志明和河内设有分仓 |
| **第三方仓储** | 快速发展，第三方云仓服务成熟 |
| **保税仓** | 越南正在推进跨境电商保税仓模式 |
| **中国边境仓** | 广西凭祥/东兴等口岸边境仓为跨境卖家提供前置备货 |
| **建议策略** | 中国卖家可利用中越陆运，从边境仓直发越南，物流成本最低 |

### 7.4 关键判断

越南基础设施处于**快速改善期**但仍存短板：
- **物流**：主要城市已较好，但偏远地区仍慢。**与中国接壤是独特优势**
- **支付**：数字钱包渗透率高，MoMo+ZaloPay双巨头。COD占比下降中
- **仓储**：平台仓+第三方仓+边境仓三层体系。**建议中国卖家考虑中越边境仓前置备货**
