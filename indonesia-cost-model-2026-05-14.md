## 一、成本模型框架

### 1.1 全成本构成

```
售价 = 采购成本 + 头程物流 + 仓储费 + 平台佣金 + 广告费 + 
       尾程配送 + 退货损耗 + 汇率波动 + 税费 + 利润
```

### 1.2 通用成本参数

| 成本项 | 费率/金额 | 说明 |
|--------|----------|------|
| **平台佣金** | 3-6% | Shopee: 3-4%, TikTok Shop: 4-6% |
| **支付手续费** | 1-2% | 数字钱包/COD 手续费 |
| **头程物流** | $0.5-3/kg | 海运/空运/快递 |
| **仓储费** | $0.5-2/件/月 | 平台仓/第三方仓 |
| **尾程配送** | $0.5-2/件 | 平台物流/第三方 |
| **广告费 (ACOS)** | 15-30% | 新品期30%，成熟期15-20% |
| **退货损耗** | 5-15% | 服饰最高，电子最低 |
| **汇率波动** | 2-5% | IDR/USD 波动 |
| **进口关税** | 0-15% | 品类不同税率不同 |
| **VAT** | 11% | 印尼增值税 |
| **所得税** | 2-5% | 企业所得税/预扣税 |
