## 一、评估方法论

### 评分标准（1-5分）

| 分值 | 含义 |
|------|------|
| 🟢 1-2 | 低风险 — 可直接推进 |
| 🟡 2.5-3.5 | 中等风险 — 需关注并制定缓解方案 |
| 🔴 4-5 | 高风险 — 必须给出缓解方案，否则不建议进入 |

### 综合评级规则

| 条件 | 评级 |
|------|------|
| 所有维度 ≤ 2 | 🟢 低 |
| 任一维度 3-3.5 但无 ≥ 4 | 🟡 中 |
| 任一维度 ≥ 4 | 🔴 高 |
| 需BPOM认证（4-8月）且毛利率 < 25% | 至少 🟡 中 |
| 需Halal认证且2026年10月前无法完成 | 🔴 高 |
