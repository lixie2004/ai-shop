```

                    │  （品类矩阵+竞品矩阵）  │

                               │ 通过
                    ┌──────────▼──────────┐
                    │   第三层：合规验证     │
                    │  合规风险可控         │
                    │  （进入门槛 ≤ 7/10）  │

                    ┌──────────▼──────────┐
                    │   Top 10 选品清单    │
                    └─────────────────────┘
```

### 综合评分公式

```
选品得分 = (搜索量指数 × 0.30) + (毛利率指数 × 0.30) + (竞争度逆向 × 0.20) + (增长潜力 × 0.20)
```

## 关键结论

### 1. 宠物用品是绝对核心
Top 10中有3个宠物产品（#1喂食器、#2猫砂、#3保健零食），全部为P0优先级。这是新加坡跨境电商**竞争度最低、增长最快、利润最优**的赛道，蓝海窗口期预计1-2年，必须立即入场。

### 2. 健康补充是利润引擎
#4胶原蛋白和#5益生菌合计月毛利预估$25,720，是Top 10中利润贡献最大的品类组合。本土化适配（亚洲人体质+热带气候）是核心差异化。

### 3. 母婴产品是确定性机会
#7环保尿裤月搜索量25,000-35,000，是Top 10中搜索量最高的产品。安全是绝对底线，但一旦建立信任，品牌忠诚度极高。

### 4. 家居产品是利润担当
#6香薰蜡烛毛利率50-65%，是Top 10中最高的。设计驱动+视觉种草是核心打法。

### 5. 平台策略：Shopee走量 + Lazada建信任 + TikTok种草
不同品类需要不同的平台组合策略，但核心逻辑一致：**内容种草（TikTok）→ 信任背书（Lazada旗舰店）→ 转化走量（Shopee）→ 品牌沉淀（独立站）**。
