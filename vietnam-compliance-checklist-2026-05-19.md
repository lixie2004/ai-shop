## 越南电商合规框架

### 监管机构

| 机构 | 职责 |
|------|------|
| **MOIT（工贸部）** | 电商运营许可、平台监管 |
| **MOH（卫生部）** | 化妆品、保健品、食品注册 |
| **MIC（信息通信部）** | 电信设备认证 |
| **NAFIQD（食药监局）** | 化妆品进口注册 |
| **GSTD（标准计量质量局）** | 产品标准和技术法规 |
| **税务局** | VAT、进口税、所得税 |

### 关键法规

| 法规 | 生效时间 | 关键要求 |
|------|---------|---------|
| 新电商法 | 2026年7月 | 外国卖家需设立代表处或授权代表 |
| 税收代扣代缴法令 | 2025年7月 | 平台代扣增值税和个人所得税 |
| 化妆品管理法 | 持续更新 | 进口化妆品需NAFIQD备案 |
| 食品安全法 | 持续更新 | 食品/保健品需MOH注册 |

## 税务合规

### 税率一览

| 税种 | 税率 | 说明 |
|------|------|------|
| 进口关税 | 0-30% | 根据品类，RCEP优惠税率可降至0-5% |
| VAT | 8% | 进口环节+销售环节 |
| 企业所得税 | 20% | 如在越南设立实体 |
| 个人所得税 | 5-35% | 平台代扣代缴 |

### RCEP 优惠
- 中越之间 RCEP 协定生效，多数商品关税逐步降至 0-5%
- 需申请原产地证书（Form RCEP）
