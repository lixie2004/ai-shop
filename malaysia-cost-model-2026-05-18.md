# 附录：成本模型方法论

## 参数来源

| 参数 | 数据来源 |
|------|---------|
| Shopee Malaysia 2026 Q2费率 | digitalinasia.com 平台费用对比报告（2026年4月更新） |

## 计算公式

```
CIF价值 = 采购成本 + 头程物流
总成本 = CIF价值 + 进口关税 + SST(8%) + 平台佣金 
       + 交易费 + 技术服务费 + 广告费 + 尾程配送 
       + 退货损耗 + 仓储费 + 汇率缓冲 + 包装耗材 
       + 认证分摊

综合毛利率 = (售价 - 总成本) / 售价 × 100%

净利率 = (售价 - 总成本 - 企业所得税) / 售价 × 100%
       = 综合毛利率 - CIT
```

## 风险声明

1. **平台费率变动风险**：Shopee Malaysia 2026年2月新增5%技术服务费，综合费率已达17-20%。预计费率仍有上升趋势，建议预留2-3%弹性空间。
2. **汇率风险**：MYR对USD年波动3-5%（2026年5月3.95，年初最高4.07），模型中已预留1%缓冲。
3. **SST政策风险**：2024年SST从6%上调至8%，不排除进一步上调可能。
4. **Halal认证风险**：JAKIM认证周期3-6个月，NPRA化妆品通知需2-3个月，认证期间无法销售。
5. **竞争风险**：模型假设当前竞争度不变，实际入场后竞品跟风可能推高广告费。
6. **本模型为理论测算**，实际毛利率受库存周转、季节波动、平台促销力度、退货率波动等多因素影响。建议每季度更新一次成本模型。
