## 一、合规框架总览

### 1.1 印尼进口电商监管体系

| 监管机构 | 职责 | 跨境电商关联 |
|----------|------|-------------|
| **BPOM** | 食品药品监管 | 美妆/健康/食品强制注册 |
| **BKPM** | 投资协调 | 外资电商准入 |
| **SNI** | 国家标准 | 部分产品强制认证 |
| **SDPPI** | 无线电监管 | 无线通信产品 |
| **MUI** | 清真认证 | 推荐（非强制进口） |
| **海关** | 进口管理 | 关税+免税额度 |
| **DJBC** | 税务总局 | VAT/所得税代扣 |

### 1.2 跨境进口监管关键词

| 政策 | 内容 | 生效时间 |
|------|------|---------|
| **PMK 4/2025** | 进口免税额度降至$3 | 2025年3月 |
| **PMK 37/2025** | 平台代扣增值税 | 2025年 |
| **BPOM条例** | 进口化妆品强制注册 | 持续 |
| **SNI强制清单** | 部分产品需SNI认证 | 持续 |

|------|------|

| **适用法规** | BPOM条例第1054/MENKES/PER/XI/2015号 |

## 四、合规行动计划

### Phase 1：立即启动（0-1个月）

| 行动 | SKU | 负责人 | 预计完成 |
|------|-----|--------|---------|
| 指定BPOM当地代理商 | 防晒霜/精华/维生素C | 合规团队 | 1个月 |
| 启动BPOM注册申请 | 防晒霜/精华/维生素C | 代理商 | 1个月提交 |
| 启动SDPPI认证 | TWS耳机 | 合规团队 | 1个月 |
| 启动SNI认证 | 充电宝 | 合规团队 | 1个月 |

### Phase 2：认证等待期（1-6个月）

| 行动 | 说明 |
|------|------|
| 手机壳/头巾/收纳/厨具 | 无需认证，可直接上架销售 |
| 密切跟踪BPOM审批进度 | 每月跟进 |
| 准备清真认证申请 | 防晒霜/精华/维生素C |
| 备选方案 | 若BPOM被拒，调整配方/包装重新提交 |

### Phase 3：全面合规（6-12个月）

| 行动 | 说明 |
|------|------|
| 所有SKU完成认证 | 确保合规上架 |
| 建立认证维护机制 | 续期+变更管理 |
| 合规档案建立 | 证书+检测报告+批次记录 |

## 五、风险红线提醒

| 红线 | 后果 | 预防 |
|------|------|------|
| 无BPOM销售化妆品/保健品 | 海关扣留+罚款+店铺关闭 | 认证完成前不上架 |
| 无SDPPI销售无线设备 | 没收+罚款 | 认证完成前不上架 |
| 含违禁成分(汞/氢醌) | 刑事起诉 | 第三方检测 |
| 虚假halal认证 | 品牌危机+法律风险 | 仅使用MUI认证 |
| 逃税(VAT未申报) | 账号冻结+补税+罚款 | 合规申报 |
