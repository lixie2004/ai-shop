## 一、成本模型框架

### 1.1 全成本核算项（≥8项）

| 成本项 | 说明 | 估算基准 |
|--------|------|---------|
| 1. 采购成本 | 出厂价（含包装） | 供应商报价 |
| 2. 头程物流 | 中国→越南（海运/空运） | 重量/体积计费 |
| 3. 进口关税 | 越南进口税率 | HS编码对应 |
| 4. 增值税(VAT) | 越南增值税 | 10% CIF+关税 |
| 5. 平台佣金 | Shopee/Lazada/TikTok | 3-6% |
| 6. 广告投放 | PPC/社交媒体 | 销售额的10-20% |
| 7. 尾程配送 | 越南国内配送 | 重量/区域 |
| 8. 退货损耗 | 退货率×单件成本 | 品类不同 |
| 9. 支付手续费 | 支付处理费 | 1-2% |
| 10. 汇率波动 | VND/CNY汇率 | 预留2-3% |

## 四、成本优化建议

1. **批量采购**：集中品类采购可降低10-15%采购成本
2. **海运优先**：非紧急品类一律海运，降低头程成本50%+
3. **广告优化**：新品期ACOS控制在15%以内，成熟期降至10%
4. **套装化**：组合销售提升客单价，摊薄固定成本
5. **本地仓**：销量稳定后转海外仓，降低尾程成本30%
