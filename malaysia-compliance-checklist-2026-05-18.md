# 马来西亚市场合规检查清单

> **报告日期**: 2026-05-18  
> **目标市场**: 马来西亚  
> **适用产品**: Top 10 选品清单

---

## 一、马来西亚合规体系概览

### 1.1 监管机构

| 机构 | 缩写 | 职责 |
|------|------|------|
| 马来西亚皇家海关 | RMCD | 进口关税、SST、清关 |
| 国家药品监管局 | NPRA (MDA) | 药品、化妆品、保健品注册 |
| 伊斯兰发展局 | JAKIM | Halal认证 |
| 标准与工业研究协会 | SIRIM | 产品认证、标准 |
| 通讯与多媒体委员会 | MCMC | 无线设备、电信产品 |
| 国内贸易与消费者事务部 | KPDNHEP | 消费者保护、产品安全 |

### 1.2 核心法规

| 法规 | 适用范围 |
|------|---------|
| Sales Tax Act 2018 | SST销售税 |
| Customs Act 1967 | 进口关税 |
| Cosmetic Control Guidelines | 化妆品注册 |
| Communications and Multimedia Act 1998 | 无线设备 |
| Trade Descriptions Act 2011 | 产品标签、描述 |
| Consumer Protection Act 1999 | 消费者权益 |
| Personal Data Protection Act 2010 | 数据隐私 |

## 二、各SKU合规要求

### #1 自动宠物喂食器 (电子产品)

| 项目 | 要求 |
|------|------|
| **认证要求**: | CE认证(EMC+LVD)、FCC(如含WiFi) |
| **SIRIM认证**: | 非强制，但建议 |
| **MCMC认证**: | 如含WiFi/蓝牙，需MCMC型式认证 |
| **认证费用**: | CE: RM 3,000-8,000; MCMC: RM 2,000-5,000 |
| **认证周期**: | CE: 4-8周; MCMC: 6-12周 |
| **风险评级**: | 🟡 中 |
| **合规建议**: | 提前准备CE和MCMC认证；产品需有英文/马来文说明书 |

---

### #2 模块化厨房收纳架 (金属制品)

| 项目 | 要求 |
|------|------|
| **认证要求**: | 无强制认证 |
| **材质安全**: | 不锈钢材质需符合食品接触安全标准(如接触食品) |
| **标签要求**: | 英文/马来文标签，含材质、产地、使用说明 |
| **认证费用**: | 无 |
| **认证周期**: | 无 |
| **风险评级**: | 🟢 低 |
| **合规建议**: | 确保材质安全，避免有害物质(铅、镉)；提供清晰安装说明 |

---

### #3 Halal认证保湿面霜 (化妆品)

| 项目 | 要求 |
|------|------|
| **认证要求**: | **NPRA化妆品通知** (强制) + **JAKIM Halal认证** (强烈建议) |
| **NPRA注册**: | 所有化妆品必须在NPRA备案(Notification) |
| **Halal认证**: | JAKIM Halal认证，需审核原料和生产流程 |
| **认证费用**: | NPRA: RM 500-1,000/产品; Halal: RM 5,000-15,000 |
| **认证周期**: | NPRA: 2-4周; Halal: 3-6个月 |
| **风险评级**: | 🔴 高 |
| **合规建议**: | **必须**先完成NPRA注册才能销售；Halal认证强烈建议(溢价15-20%)；成分表需完整披露 |

---

### #4 穆斯林运动头巾 (纺织品)

| 项目 | 要求 |
|------|------|
| **认证要求**: | 无强制认证 |
| **标签要求**: | 英文/马来文标签，含材质、洗涤说明、产地 |
| **纤维成分**: | 需准确标注纤维成分百分比 |
| **认证费用**: | 无(可选SIRIM认证RM 2,000-5,000) |
| **认证周期**: | 无 |
| **风险评级**: | 🟢 低 |
| **合规建议**: | 确保材质标注准确；避免使用宗教敏感图案；提供详细尺码表 |

---

### #5 防摔手机壳 (塑料制品)

| 项目 | 要求 |
|------|------|
| **认证要求**: | 无强制认证 |
| **材质安全**: | 如宣称"环保/可降解"，需有相关认证 |
| **认证费用**: | 无(可选环保认证RM 3,000-10,000) |
| **认证周期**: | 无 |
| **风险评级**: | 🟢 低 |
| **合规建议**: | 避免虚假环保宣称；确保产品与描述一致 |

### #6 GPS宠物追踪器 (电子产品/无线设备)

| 项目 | 要求 |

| **SIRIM认证**: | 建议 |
| **认证费用**: | CE: RM 5,000-10,000; MCMC: RM 3,000-8,000 |
| **认证周期**: | CE: 6-10周; MCMC: 8-16周 |
| **风险评级**: | 🔴 高 |
| **合规建议**: | 无线设备认证周期长，需提前6个月准备；需英文/马来文说明书 |

### #7 真空压缩袋套装 (塑料制品)

|------|------|

| **风险评级**: | 🟢 低 |

### #8 加厚瑜伽垫 (TPE材质)

| 项目 | 要求 |
|------|------|
| **认证要求**: | 无强制认证 |
| **材质安全**: | 需符合REACH/ROHS(如出口欧盟)；马来西亚无强制 |
| **认证费用**: | 无(可选REACH测试RM 2,000-5,000) |
| **认证周期**: | 无 |
| **风险评级**: | 🟢 低 |
| **合规建议**: | 确保无有害化学物质(邻苯二甲酸酯、甲醛)；标注材质和厚度 |

## 三、税务合规

### 3.1 SST销售税

| 项目 | 要求 |
|------|------|
| **注册门槛** | 年销售额超过RM 500,000 |
| **销售税率** | 5-10%(因品类而异) |
| **服务税率** | 8%(2025年7月起) |
| **申报频率** | 每2个月 |
| **跨境销售** | 低价值商品也需缴纳SST |

### 3.2 进口关税

| 品类 | 税率 |
|------|------|
| 电子产品 | 多数免税 |
| 化妆品 | 5-10% |
| 纺织品 | 5-15% |
| 塑料制品 | 5-20% |
| 金属制品 | 5-15% |

### 3.3 电子发票

- 2025年7月起分阶段实施B2B电子发票
- 跨境电商卖家需关注合规要求

## 四、产品标签要求

### 4.1 强制标签内容

| 项目 | 要求 |
|------|------|
| **产品名称** | 英文或马来文 |
| **成分/材质** | 完整列表 |
| **原产地** | "Made in China" 等 |
| **进口商信息** | 名称、地址 |
| **使用说明** | 英文或马来文 |
| **警告标识** | 如适用 |
| **净含量/规格** | 数字+单位 |

### 4.2 特殊品类标签

| 品类 | 额外要求 |
|------|---------|
| 化妆品 | 成分表(INCI)、批号、有效期 |
| 电子产品 | 电压、功率、安全警告 |
| 食品接触材料 | 食品级标识 |
| 纺织品 | 纤维成分、洗涤说明 |
