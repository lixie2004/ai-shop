## 菲律宾合规框架速览

### 1. 核心监管机构

| 机构 | 简称 | 主管范围 |
|------|:----:|---------|
| 食品药品管理局 | **FDA Philippines** | 食品、化妆品、药品、医疗器械、家用化学品注册 |
| 标准局 | **BPS (DTI)** | 强制产品认证：PS 质量认证（国产）/ ICC 进口清关认证（进口） |
| 国家电信委员会 | **NTC** | 无线通信设备型式批准（Type Approval） |
| 畜牧业局 | **BAI (DA)** | 动物饲料/宠物食品注册与进口许可 |
| 海关局 | **BOC** | 进口清关、关税征收、进口许可证 |
| 伊斯兰 Halal 认证机构 | **IDCP / HDIP / IMAN** | Halal 认证（自愿，多家认证机构并存） |
| 环境管理局 | **EMB (DENR)** | 化学品合规 |

### 2. 主要合规分类

| 产品类型 | 所需认证 | 主管机构 |
|---------|---------|---------|
| 化妆品 | FDA CPN（产品通知） + 需指定菲律宾 RP | FDA CCRR |
| 食品/宠物食品 | FDA CPR（产品注册） + BAI 注册（宠物食品） | FDA + BAI |
| 家用化学品/清洁产品 | FDA CPN 或 CPR | FDA |
| 插电类电子产品 | **BPS ICC 认证** | BPS (DTI) |
| 无线/RF 设备（蓝牙/WiFi/4G） | **NTC Type Approval** | NTC |
| 一般消费品 | 无强制认证（如纺织品、背包、手机壳） | — |
| Halal 产品 | Halal 证书（**非强制**但市场需要） | IDCP/HDIP/IMAN 等 |

### 3. 进口通用要求

| 要求 | 法规/依据 | 说明 |
|------|---------|------|
| 进口商登记 | BOC 进口商登记 | 须在 BOC 系统注册 |
| 原产地证明 | Form AF/Form E（ASEAN FTA） | 享 ASEAN 0关税需提供 |
| 标签语言 | **英语/菲律宾语**（英语即可接受） | 较灵活，不强制当地语言 |
| 海关编码 | HS Code（ASEAN Harmonised Tariff Nomenclature） | 按 HS Code 确定税率 |
| 进口许可证 | 部分商品需 SPS/ICC 许可 | 见具体要求 | 视品类而定 |

### 4. 税务合规（2026年）

| 税种 | 税率 | 说明 |
|------|:----:|------|
| 进口关税（Import Duty） | 0-30%（MFN） | 按 HS Code 征收 |
| ASEAN 优惠关税 | **0-5%** | 东盟成员国进口享优惠 |
| VAT（增值税） | **12%** | 按（货价+关税）×12% 计算 |
| 最低免税限额（De Minimis） | PHP 10,000（约 $180） | 低于此值的个人包裹免税 |

### 5. FDA 注册体系

| 品类 | 注册类型 | 费用（PHP） | 周期 | 有效期 |
|------|---------|:----------:|:----:|:-----:|
| 化妆品 | CPN（产品通知） | ~500/产品 | 3-10 个工作日 | 2年 |
| 食品 | CPR（产品注册） | ~2,000-3,000/产品 | 1-3 个月 | 1-5年 |
| 家用化学品 | CPN 或 CPR | ~500-2,000/产品 | 2-6 周 | 2年 |
| **LTO（经营许可证）** | 所有品类前置许可 | ~4,000-6,000 | 2-3 个月 | 2年 |

### 6. BPS ICC 认证（进口商品强制认证）

- **适用对象**：列入强制认证目录的进口商品（87+品类）
- **核心要求**：每批次/每提单须申请 ICC 清关证书
- **涉及品类**：家用电器（电扇、搅拌机、电熨斗）、灯具、电线电缆、LPG 钢瓶、开关插座等
- **周期**：2-6 周（取决于产品类别和测试状态）
- **费用**：PHP 5,000-30,000+（含测试费）

### 7. Halal 认证现状

- ❗**⚠️ 注意**：菲律宾 **目前没有** 面向国内销售的强制 Halal 认证法律（用户提示中提到的"2026年10月强制"实际是**印度尼西亚**的政策，不是菲律宾）
- 菲律宾 Halal 认证为**自愿**性质，由以下机构提供：
  - **IDCP Halal** — 国内最大，含 98 个穆斯林组织
  - **HDIP** — Halal Development Institute of the Philippines
  - **IMAN** — 其他私立认证机构
- **市场考量**：菲律宾约 10-15% 穆斯林人口（主要位于南部棉兰老岛），Halal 认证是**市场差异化工具**而非法律强制性要求
- **出口导向**：菲律宾 Halal 出口促进（RA 10817）针对出口商，不强制内销

### SKU 6：Halal 美白精华

**产品定位**：具有"美白/提亮"功效的精华液 + Halal 认证

#### 🔴 特殊风险：功效宣称边界问题

**"美白"（Whitening）在菲律宾 FDA 属于高风险宣称**。根据 ASEAN Cosmetic Directive 和 FDA Philippines：
- "提亮"（Brightening）、"焕亮"（Radiant）、"均匀肤色"（Skin tone even）→ 可接受为化妆品宣称
- "美白"（Whitening）、"淡斑"（Anti-melasma）、"去黑色素"（Anti-melanin）→ 易被划分为**药品**，需走药品注册（Drug/OTC）而非化妆品通知
