## 一、模型假设与框架

### 1.1 通用假设

| 假设项 | 基准值 | 说明 |
|--------|--------|------|
| **汇率（USD/CNY）** | 7.25 | 2026年5月即期汇率中间价 |
| **汇率准备金率** | 3% | 覆盖人民币升值3%的汇兑损失 |
| **头程物流（海运）** | 商品成本的 8-15% | 按品类体积/重量差异化 |
| **亚马逊平台佣金** | 15% | 大部分品类标准佣金率（含宠物、户外、家居、DIY） |
| **个护/健康品类佣金** | 8% | 健康补充剂、个护仪器适用低佣金档 |
| **FBA 配送费** | 按件计费 | 基于尺寸/重量，详见各SKU |
| **FBA 月度仓储费** | $0.87/立方英尺/月 | 非旺季（1-9月）基准费率 |
| **广告费占收入比（ACoS）** | 20-30% | 新品期取高值，成熟期取低值 |
| **退货损耗率** | 3-8% | 按品类差异化（电子产品高、耗材低） |
| **退货处理成本** | $3-8/件 | 含检测/翻新/不可售损失 |
| **美国进口关税** | 按品类税率 | 基于HTS编码估算 |
| **企业所得税（联邦+州）** | 25-28% | 联邦21% + 州税4-7% |
| **月度销量假设** | 500 件/月 | 基准场景，用于计算月度利润 |
| **分析周期** | 12 个月 | 年度滚动 |

### 1.2 成本结构框架（8+ 项）

每个 SKU 按以下成本项逐层拆解：

```
  零售价（MSRP）
  ├── 1. 采购成本（COGS）
  ├── 2. 头程物流（海运/空运到FBA仓）
  ├── 3. 平台佣金（Amazon Referral Fee）
  ├── 4. FBA 仓储费（月度仓储 + 配送费）
  ├── 5. 广告费（PPC / ACoS）
  ├── 6. 退货损耗（退货率 × 单件处理成本）
  ├── 7. 汇率准备金（汇兑风险缓冲）
  ├── 8. 税费（进口关税 + 所得税）
  ├── 9. 合规成本（FDA注册/检测分摊）*
  └── 10. 包装/售后/杂项*
  ─────────────────────────
  = 净利润
```

> *标注为部分SKU适用的附加成本项，确保每个SKU ≥ 8项。*

### 1.3 场景定义

| 场景 | 定义 | 关键变量调整 |
|------|------|-------------|
| **乐观** | 新品表现超预期 | 销量+30%，ACoS-5pp，退货率-2pp |
| **基准** | 按行业均值预测 | 所有假设取基准值 |
| **悲观** | 市场竞争加剧 | 售价-10%，销量-20%，ACoS+5pp，退货率+2pp |
