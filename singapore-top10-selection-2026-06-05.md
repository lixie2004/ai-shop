## 1号：中端 RFID 多猫防抢食喂食器

| 维度 | 详情 |
|------|------|
| **产品名称** | SmartPet RFID 自动喂食器 — 多猫版（4L 料仓 + RFID 识别） |
| **品类** | 宠物智能设备 |
| **预估毛利率** | **58%** |
| **月搜索量估算** | ~8,500（"automatic cat feeder Singapore" + "multi cat feeder"） |
| **竞争度评分** | 🟢 **低（3/10）** — 高端仅 SureFeed（$350），中端无 RFID 方案 |
| **合规风险** | ✅ **低** — 无需特殊认证，CE/FCC 即可 |
| **推荐定价** | SGD $169（比 SureFeed 便宜 52%，比 Xiaomi 贵 20%） |
| **推荐平台** | Shopee Mall > Lazada Mall > Pet Lovers Centre 线下 |
| **目标毛利/月** | 300 台 × $98 利润 = **$29,400** |

### 入选理由（4 重逻辑）

**需求侧 — P0 痛点（用户洞察报告）**  
多猫家庭防抢食被列为跨品类排名第 1 的 P0 需求。新加坡 30%+ 养宠家庭有 ≥2 只猫，而目前市面仅 SureFeed（$350）一款 RFID 方案，$140-200 中端价位段完全空白。

**竞品侧 — 真空地带**  
对比现有 SKU 矩阵：SmartPaw（$69）无 RFID、Xiaomi（$140）无 RFID、Catlink（$179）有 RFID 但仅 Pro 版且体验差。中端 RFID 喂食器直接锁定 1.5 万潜在家庭的需求。

**供给侧 — 可落地性**  
核心组件（RFID 读取模块 + 螺旋下料器 + 陶瓷碗）在中国供应链中高度成熟，ODM 方案成本约 RMB 180-220（SGD $35-42），BOM 可控。

**利润侧 — 价格锚定**  
以 $169 定价，对比 $69 的入门款和 $350 的高端款，性价比叙事成立。毛利率 58% 在宠物品类属健康水平。

**验证逻辑**

用户洞察报告识别"HDB 门锁兼容性指南"为 P2 需求，但实际购买行为显示"不知道能不能装"是转化最大阻碍——5-8% 差评来自"买回来和门不匹配"。这在新加坡 HDB 住户中是高频心理障碍。

**利润侧**  
智能门锁 BOM 约 $65-75（含指纹模块 + WiFi 模组 + 锁体 + 安装件），以 $169 定价，毛利率 48%。安装简易度意味着降低退货率（目前智能锁退货率 8-12%，主要来自安装失败）。
