## 一、电商市场规模（GMV）及增速

### 1.1 市场规模总览

| 年份 | GMV（十亿美元） | 同比增速 | 数据来源 |
|------|----------------|---------|---------|
| 2023 | ~$9.86 | — | 根据2024年倒推 |
| 2024 | $13.82（四平台）/ ~$16（全渠道） | +40.4%（四平台） | YouNet ECI / Vietnam News |
| 2025 | $16.35（四平台）/ $26-28（全口径线上零售） | +34.75%（四平台） | Metric.vn / VCCI |
| 2026（预测） | $20+（四平台） | 预计20-25% | 行业综合预测 |

**关键数据点**：
- 2024年四大平台（Shopee、TikTok Shop、Lazada、Tiki）合计GMV达 **$13.82B**，同比增长40.4%，远超越南整体零售业9%的增速
- 2025年四大平台GMV达 **VND429.7万亿（约$16.35B）**，同比增长34.75%
- 2025年越南线上零售总额达 **$26-28B**（含社交电商、品牌官网等全口径）
- 2025年Shopee + TikTok Shop合计占越南整体零售市场（$216.3B）的近 **8%**，较2024年的6.5%进一步提升
- 2025年售出商品超 **36亿件**，同比增长15%+
- YouNet ECI预测2024-2028年CAGR可达 **35%**（乐观情景），2028年总交易额有望达 **$49.9B**

### 1.2 东南亚区域地位

越南是东南亚 **第三大电商市场**（仅次于印尼$56.5B、泰国$23.5B），与菲律宾（$16B）并列，领先于马来西亚（$11.5B）。

| 国家 | 2024年GMV | 增速 | 区域份额 |
|------|----------|------|---------|
| 印尼 | $56.5B | +5% | 44% |
| 泰国 | $23.5B | +22% | 18% |
| **越南** | **~$16B** | **+40%** | **~12%** |
| 菲律宾 | $16B | — | ~12% |
| 马来西亚 | $11.5B | +20% | 9% |
| 新加坡 | $4.9B | — | 4% |

> **跨境电商视角**：越南40%+的增速在东南亚主要市场中位居第一，是增量红利最显著的市场。

## 二、消费力水平

### 2.1 人口与互联网基础

| 指标 | 数据 | 说明 |
|------|------|------|
| 总人口 | ~1.01亿 | 2025年估计 |
| 互联网用户 | ~7,600-7,900万 | 互联网渗透率约75-78% |
| 社交媒体用户 | ~7,620万 | 几乎等于互联网用户数 |
| 智能手机渗透率 | ~70-75% | 移动端购物占主导 |
| 日均在线订单量（东南亚区域） | 4360万件/天 | 越南是主要贡献市场之一 |

### 2.2 经济与消费力

| 指标 | 数据 | 说明 |
|------|------|------|
| 2025年GDP增速 | ~6.5-7% | 东南亚增速最快的经济体之一 |
| 人均GDP | ~$4,500-4,800 | 2025年估计 |
| 人均年网购支出 | ~$160 | 2024年估计（$16B/1亿人） |
| 零售市场总规模 | $216.3B | 2025年，电商渗透率约12-13% |
| 网购渗透率 | ~55-60% | 互联网用户中网购比例 |

### 2.3 消费者画像

**Gen Z（18-26岁）**：
- 占网购人群的53.4%，是最活跃的网购群体
- 偏好品类：时尚、美妆、个人护理
- 51%受社交媒体趋势影响购买决策
- 55%会参考微网红（1万-10万粉丝）的评测
- 2023-2028年线上消费预计从$4.5B增至$20.3B

**Millennials（27-44岁）**：
- 占网购人群的46.6%
- 平均订单价值 **$125**，显著高于Gen Z
- 偏好品类：时尚、美妆、家用电器、消费电子
- 82%重视工作-生活平衡，愿为品质服务付溢价
- 2023-2028年线上消费预计从$6.9B增至$29.7B
- 前三大购买决策因素：退换货政策 > 便捷支付 > 灵活配送时间

**高频购物习惯**：
- 62.8%的数字消费者每周至少网购一次
- 月收入≥3000万VND（~$1,200）的消费者对促销依赖度低，按需购买

> **跨境电商视角**：越南消费者年轻化、社交驱动特征明显。Gen Z是增量主力，Millennials是客单价主力。跨境卖家需差异化定位。

## 三、平台格局

### 3.1 四大平台GMV份额演变

| 平台 | 2023年份额 | 2024年份额 | 2025年Q3份额 | 2024年GMV | 趋势 |
|------|-----------|-----------|-------------|----------|------|
| **Shopee** | 64% | 64-65% | 56% | ~$10.4B | ↓ 份额被TikTok蚕食 |
| **TikTok Shop** | 25% | 28-29% | 41% | ~$4.5B | ↑↑ 增速最快 |
| **Lazada** | 8% | 6% | 3% | ~$1.0B | ↓ 持续萎缩 |
| **Tiki** | 2% | 1% | <1% | ~$0.2B | ↓↓ 边缘化 |

**关键趋势**：
- **Shopee + TikTok Shop 双寡头格局确立**：合计占97%市场份额

- Shopee 2025年H1增长仅16%，份额从63%降至58%
- Lazada和Tiki合计份额从10%萎缩至3%以下
- Mall店铺（认证卖家）仅占店铺总数的2.64%，却贡献了Shopee+TikTok Shop **35.7%**的销售额

## 四、跨境电商政策环境

### 4.1 进口关税

| 政策 | 详情 |
|------|------|
| 跨境电商进口免税门槛 | 单次交易 ≤ 100万VND（~$40）免关税（2024年政策） |
| 一般进口关税 | 0-40%不等，取决于HS编码和原产地 |
| 增值税（VAT） | 标准税率10%，跨境电商进口适用 |
| 特别消费税 | 针对特定品类（化妆品、电子产品等） |
| 中越自贸协定 | 中国原产商品享受优惠税率（RCEP/ACFTA框架） |

### 4.2 电商法规（重大变化）

**2025年电商法（2025年7月生效）**：

| 条款 | 要求 | 对跨境卖家的影响 |
|------|------|----------------|
| 外资卖家许可 | 外国企业向越南销售须获得工贸部（MoIT）许可 | 需设立越南代表处或指定授权代表 |
| 本地化合规 | 进口商品须符合越南产品标准、技术要求和安全法规 | 需提前了解产品合规要求 |
| 卖家身份认证 | 通过VNeID系统验证在线卖家身份 | 提高卖家门槛，利好合规卖家 |
| 直播责任 | 直播带货主播须对宣传内容负责 | 达人合作需规范 |
| 平台责任 | 平台须向监管机构共享卖家信息 | 信息透明度提高 |
| 进口商品清单 | 须符合政府批准的跨境电商进口商品清单 | 部分品类可能受限 |

**2025年7月税收新规**：
- 电商平台须代扣代缴个体卖家的 **VAT和个人所得税**
- 适用于国内外所有卖家
- 平台按月申报缴纳

### 4.3 外资限制与合规要求

| 领域 | 政策 |
|------|------|
| 外资持股比例 | 电商领域外资可100%持股（2020年放开） |
| 代表处要求 | 跨境卖家须在越南设立代表处或指定授权代表 |
| 产品标准 | 须符合越南国家标准（TCVN），部分品类需认证 |
| 知识产权 | 越南知识产权局（IP Vietnam）负责商标/专利注册 |
| 数据本地化 | 部分数据须存储在越南境内 |

### 4.4 跨境电商进出口数据

| 指标 | 2024年 | 2025年H1 | 趋势 |
|------|--------|---------|------|
| 跨境进口电商销售额 | VND14.2万亿（$564.5M） | VND7.5万亿（$285M） | +43% YoY（2024） |
| 跨境进口商品量 | 3.24亿件 | 1.64亿件 | +38% YoY（2024） |
| B2C出口电商 | $3.5B（2023） | — | 预计2028年达$5.8B |

> **跨境电商视角**：越南跨境电商监管趋严是确定性趋势。2025年电商法要求外资卖家获得许可并符合本地标准，这提高了合规门槛，但也 **淘汰了大量不合规的中小卖家**，利好有合规能力的品牌型跨境卖家。

## 五、消费者行为特征

### 5.1 购物偏好

| 维度 | 特征 |
|------|------|
| 购物动机 | 社交驱动（51% Gen Z因社交趋势购买）、促销敏感、品质追求提升 |
| 品牌偏好 | 国际品牌（Apple、Samsung、L'Oréal）主导高端，中国品牌快速崛起 |
| 信任因素 | Mall店铺/认证卖家占比仅2.64%但贡献35.7%销售额 → **信任溢价显著** |
| 内容偏好 | 短视频 > 图文；直播带货转化率最高；微网红（1万-10万粉）信任度最高 |
| 购物时段 | 午休（12:00-13:00）和晚间（20:00-22:00）为高峰 |
| 大促节点 | 9.9、10.10、11.11、12.12、Tet（越南新年）为全年最大促销季 |

### 5.2 支付方式

| 支付方式 | 市场份额（估计） | 特点 |
|---------|----------------|------|
| **COD（货到付款）** | ~60-70% | 仍是主流，信任基础 |
| **电子钱包** | ~15-20% | MoMo、ZaloPay、ShopeePay快速增长 |
| **银行转账** | ~10% | 大额订单偏好 |
| **信用卡/借记卡** | ~5-8% | 城市年轻群体增长中 |
| **BNPL（先买后付）** | <5% | 新兴，年轻群体试用中 |

> **跨境电商视角**：COD占比极高意味着跨境卖家需选择支持COD的物流方案，同时注意退货率管理。电子钱包增长迅速，接入MoMo/ZaloPay可提升转化。

### 5.3 物流期望

| 维度 | 消费者期望 | 行业现状 |
|------|-----------|---------|
| 配送时效 | 河内/胡志明市内1-2天，跨省3-5天 | Shopee/Lazada基本达标 |
| 运费敏感度 | 极高，"免运费"是最大转化因子 | 平台普遍提供免运补贴 |
| 物流追踪 | 实时追踪为基本期望 | 主流平台已覆盖 |
| 退货便利性 | 7天无理由退货成为标配 | Mall店铺退货率更低 |
| 主要物流商 | GHN、GHTK、J&T Express、Ninja Van、Viettel Post | 竞争激烈，价格战持续 |

### 5.4 热门品类（跨境机会品类）

| 品类 | 热度 | 跨境机会 | 竞争度 |
|------|------|---------|--------|
| 时尚女装/配饰 | ★★★★★ | 中国供应链优势大 | 高 |
| 美妆个护 | ★★★★★ | 日韩/中国美妆受欢迎 | 中高 |
| 消费电子/配件 | ★★★★☆ | 中国3C供应链绝对优势 | 高 |
| 家居生活 | ★★★★☆ | 中国家居供应链优势 | 中 |
| FMCG/快消 | ★★★★☆ | 日用品需求大 | 中 |
| 母婴用品 | ★★★☆☆ | 品质要求高，信任门槛高 | 中 |
| 运动户外 | ★★★☆☆ | 增长潜力大 | 中低 |
| 宠物用品 | ★★☆☆☆ | 新兴品类，蓝海 | 低 |

## 六、2025-2026年市场趋势和机会

### 6.1 核心趋势

| 趋势 | 描述 | 对跨境卖家的意义 |
|------|------|----------------|
| **1. Shoppertainment（购物娱乐化）** | TikTok Shop引领，短视频+直播成为核心购物场景 | 必须布局内容电商，短视频/直播能力成为核心竞争力 |
| **2. 市场集中度提升** | 卖家数量下降7.5%，向头部集中；Mall店铺贡献35.7%销售额 | 品牌化/认证化是方向，小卖家生存空间压缩 |
| **3. 跨境进口增长** | 2024年跨境进口+43%，2025年H1持续增长 | 越南消费者对中国商品需求旺盛，跨境进口窗口期 |
| **4. 监管趋严** | 2025年7月电商法生效，税收代扣代缴 | 合规成本提高，但淘汰不合规竞争者，利好品牌卖家 |
| **5. 中国品牌回归** | 中国消费品牌以升级产品和本地化策略重返东南亚 | 既是竞争者也是参照——本地化是成功关键 |
| **6. AI赋能** | Momentum Works预测AI可为东南亚电商额外创造$131B交易额 | AI选品、客服、广告优化将成为竞争壁垒 |
| **7. 社交电商深化** | Zalo（国民级IM）+ TikTok + Facebook多平台社交购物 | 需建立多平台社交矩阵 |
| **8. FMCG线上化** | FMCG超越家电成为第二大品类，+62% YoY增长 | 快消品线上化是最大增量机会之一 |

### 6.2 跨境卖家机会矩阵

| 机会领域 | 机会描述 | 优先级 | 进入难度 |
|---------|---------|--------|---------|

### 6.3 风险提示

| 风险 | 描述 | 应对策略 |
|------|------|---------|
| **合规风险** | 2025年电商法要求外资许可+产品合规 | 提前完成产品认证，设立越南代表处 |
| **关税风险** | 进口关税政策可能调整 | 关注RCEP/ACFTA优惠税率，合理HS编码 |

| **平台政策风险** | 平台规则频繁变化 | 多平台布局，不过度依赖单一平台 |

## 七、总结与建议

### 越南市场核心结论

1. **高增长确定性**：越南是东南亚增速最快的电商市场（40%+），预计2028年达$49.9B
2. **双寡头格局**：Shopee + TikTok Shop占97%份额，必须双平台布局
3. **年轻化驱动**：Gen Z和Millennials是主力，社交电商是核心场景
4. **合规门槛提高**：2025年电商法是双刃剑——提高成本但淘汰弱者
5. **跨境进口窗口期**：中国商品在越南需求旺盛，跨境进口持续高增长

### 下一步行动建议

| 优先级 | 行动 | 时间框架 |
|--------|------|---------|
| P0 | 确定目标品类（建议：美妆/时尚/消费电子/家居） | 1-2周 |
| P0 | 研究目标品类的产品认证要求 | 1-2周 |
| P1 | 评估Shopee跨境店和TikTok Shop跨境店入驻条件 | 2-3周 |
| P1 | 供应链成本核算（含关税、物流、VAT） | 2-3周 |
| P2 | 竞品分析（各品类TOP卖家分析） | 3-4周 |
| P2 | 制定本地化策略（Listing翻译、视觉、定价） | 3-4周 |
