# 🇸🇬 新加坡合规检查清单 — Top 10 SKU 准入分析

**评估日期**：2026-06-13  
**目标市场**：新加坡（电商渠道 + 跨境准入）  
**核心法规背景**：
- **HSA（卫生科学局）**：保健品注册（Health Supplement Registration）、化妆品通报（Cosmetic Product Notification）
- **SFA（新加坡食品局，原 AVA 职能已并入 SFA）**：宠物食品进口许可与注册
- **MUIS（新加坡伊斯兰理事会）**：Halal 认证（全球黄金标准之一）
- **IRAS（国内税务局）**：GST 9%（2024年起），低价值商品 GST 已对进口 B2C 征收
- **新加坡海关**：进口许可证（TradeNet）、HS 编码分类、关税
- **PDPA（个人数据保护法）**：电商用户数据收集与处理合规
- **消费者保护（公平交易法 CPG）**：产品标签、广告宣称限制

---

## 总体摘要

| 维度 | 评估 |
|------|------|
| 电商准入门槛 | 🟢 **低** — 无电商卖家许可证要求，无 MoIT 前置审批 |
| 高风险品类 | 保健品（HSA 注册 3-6 个月）、宠物食品（SFA 进口许可） |
| 低风险品类 | 家居收纳、运动服饰、瑜伽垫（无强制认证） |
| 税务通用要求 | GST 9%（2024年1月起）+ 进口关税（多数消费品 0%） |
| 数据合规 | PDPA 已生效，电商需隐私政策+同意机制 |
| 标签语言 | 英语（官方要求），马来语在 Halal 产品上为加分项 |

### 与越南/马来西亚对比关键差异

| 维度 | 新加坡 | 越南 | 马来西亚 |
|------|--------|------|----------|
| 电商许可证 | ❌ 不需要 | ✅ MoIT 许可证 | ❌ 不需要 |
| 保健品注册 | HSA 注册（3-6月） | MOH 注册（6-12月） | NPRA 注册（9-18月） |
| 宠物食品 | SFA 进口许可（3-6月） | DVS 进口许可 | DVS + JAKIM（如 Halal） |
| Halal 认证 | MUIS（2-4月） | 不强制 | JAKIM（3-6月） |
| 化妆品 | HSA 通报制（1-2月） | MOH 通告（3-6月） | NPRA 通知（1-3月） |
| GST/VAT | 9% | 10% | SST 5-10% |
| 关税 | 多数消费品 0% | 0-30% | 0-25% |

**结论：新加坡是东南亚合规门槛最低的跨境市场之一，但 HSA/SFA 注册周期不可忽视。**

### 🥇 #1 — HDB适配防潮收纳套装

| **认证名称** | **无强制认证** |
| **主管机构** | 不适用 |

| **认证周期** | **0 周**（可立即上架） |
| **进口关税** | HS 3924.90（塑料家居用品）：**0%**（新加坡多数塑料制品无关税） |
| **GST** | **9%**（进口时缴纳，跨境 B2C 平台可能代扣） |
| **标签要求** | 英语标签：产品名称、材质、尺寸、使用说明、进口商/制造商信息 |
| **产品标准** | 无强制标准，但建议符合 SS 582（新加坡塑料安全标准，非强制） |
| **PDPA 影响** | 低 — 不涉及用户数据收集 |
| **其他要求** | 产品责任保险（建议）、消费者保护法合规（退换货政策） |

### 🏠 风险评级：🟢 低

| 风险点 | 说明 | 缓解措施 |
|--------|------|----------|
| 材质安全 | 如含 BPA 等有害物质可能被消费者投诉 | 提供材质安全检测报告（SGS/Intertek） |
| 尺寸不符 | HDB 适配是核心卖点，尺寸偏差将导致退货 | 出厂前抽样实测新加坡标准柜体 |
| 竞品抄袭 | 无认证壁垒意味着易被模仿 | 申请外观设计专利（IPOS，费用约 S$250） |

### 💡 合规建议

1. **立即上架**：零认证壁垒，OEM 打样后可直接通过 Shopee/Lazada 上架
2. **标签合规**：确保英语标签包含产品名称、材质（PP+ABS）、尺寸、进口商信息
3. **差异化保护**：通过 IPOS 注册外观设计（周期 2-3 周，费用约 S$250）
4. **品质管控**：建议做 SGS 材质安全检测（费用约 S$300-500），作为营销信任背书

### 🥇 #2 — 小颗粒维生素C+锌

| 维度 | 详情 |
|------|------|
| **认证名称** | **HSA 保健品注册（Health Supplement Registration）** |
| **主管机构** | Health Sciences Authority (HSA) — 保健品与化妆品司 |
| **适用法规** | Health Supplements Regulations 2005（Cap. 138A, Rg 2） |
| **认证费用** | HSA 申请费：**S$400-800**（视产品类别）<br>产品检测（第三方实验室）：**S$1,500-3,000**（重金属、微生物、有效成分含量）<br>顾问/代理费（如需）：**S$2,000-5,000**<br>**总计约 S$4,000-9,000（约 US$3,000-6,700）** |
| **认证周期** | **3-6 个月**（HSA 审核）+ 1-2 个月（产品检测）= **总计 4-8 个月** |
| **进口关税** | HS 2106.90（膳食补充剂）：**0%** |
| **GST** | **9%** |
| **标签要求** | 英语标签：产品名称、成分表（含含量）、每日建议用量、批号、有效期、制造商信息、HSA 注册编号、"此产品不用于诊断、治疗或预防任何疾病"免责声明 |
| **成分限制** | 维生素C 每日剂量 ≤ 1,000mg（HSA 上限）；锌 ≤ 25mg/日；禁止含处方药成分 |
| **宣称限制** | 不允许宣称治疗/预防疾病功能；允许的保健宣称需有科学依据（如"维生素C 支持免疫系统"） |
| **PDPA 影响** | 中 — 如通过 App/网站收集用户健康数据需 PDPA 合规 |

### 💊 风险评级：🟡 中高

| 风险点 | 说明 | 缓解措施 |
|--------|------|----------|
| HSA 审核延迟 | 资料不全会导致反复补正，周期延长至 8-12 个月 | 提前准备完整技术文件（成分分析、稳定性测试、GMP 证书） |
| 成分超标 | 维生素C/锌含量超过 HSA 上限将被拒绝 | 确保配方符合 HSA 允许剂量范围 |
| 宣称违规 | "增强免疫力"等宣称可能被认定为药品宣称 | 仅使用 HSA 允许的保健宣称用语 |
| 无 GMP 证书 | 海外工厂需提供 GMP 或 ISO 22000 证书 | 选择已有 GMP/ISO 22000 认证的 OEM 工厂 |

### 💡 合规建议

1. **立即启动 HSA 注册**：这是最长的先导路径，建议在 OEM 打样阶段同步准备
2. **配方预审**：提交前请 HSA 认可的顾问做配方合规预审（费用约 S$500-1,000）
3. **工厂资质**：确保 OEM 工厂持有 GMP 或 ISO 22000 证书（HSA 要求）
4. **标签预审**：HSA 对标签要求严格，建议提前按 HSA 模板制作
5. **替代方案**：如 HSA 周期过长，可考虑先以"食品"类别上架（需符合 Food Regulations），但保健宣称受限

### 🥉 #3 — 热带气候防晒霜（无酒精+不泛白）

| 维度 | 详情 |
|------|------|
| **认证名称** | **HSA 化妆品产品通报（Cosmetic Product Notification, CPN）** |
| **主管机构** | Health Sciences Authority (HSA) — 保健品与化妆品司 |
| **适用法规** | Health Products (Cosmetic Products — ASEAN Cosmetic Directive) Regulations |
| **认证费用** | HSA 通报费：**S$0**（通报制，无官费）<br>产品检测（SPF 测试）：**S$2,000-4,000**（人体 SPF 测试，需 ISO 24444 认证实验室）<br>安全评估报告：**S$1,000-2,000**（需持证安全评估师）<br>顾问/代理费：**S$1,500-3,000**<br>**总计约 S$4,500-9,000（约 US$3,300-6,700）** |
| **认证周期** | SPF 测试：**4-6 周**<br>安全评估：**2-3 周**<br>HSA 通报审核：**2-4 周**<br>**总计约 2-3 个月** |
| **进口关税** | HS 3304.99（防晒化妆品）：**0%** |
| **GST** | **9%** |
| **标签要求** | 英语标签：产品名称、成分表（INCI 命名）、SPF 值、PA 等级、净含量、批号、有效期、制造商/进口商信息、"避免接触眼睛"等安全用语 |
| **成分限制** | 需符合 ASEAN Cosmetic Directive 禁用/限用成分清单；防晒剂种类和浓度需符合新加坡允许列表 |
| **SPF 宣称** | SPF 宣称必须有测试报告支持；SPF50+ 是新加坡允许的最高宣称 |
| **宣称限制** | "不泛白""无酒精"等宣称需有证据支持；不允许宣称"100%防晒"等绝对化用语 |

### 🌿 风险评级：🟡 中

| 风险点 | 说明 | 缓解措施 |
|--------|------|----------|
| SPF 测试不达标 | 实际 SPF 低于宣称值是最常见问题 | 配方定型后先做预测试（费用约 S$500），确认达标后再做正式测试 |
| 成分不合规 | 某些防晒剂（如氧苯酮）在新加坡受限 | 确保配方使用新加坡允许的防晒剂 |
| 安全评估不合格 | 成分安全性数据不足 | 选择有充分安全数据的成熟原料 |
| 无酒精宣称验证 | "无酒精"需提供成分分析报告 | 确保配方完全不含乙醇/变性酒精 |

### 💡 合规建议

1. **SPF 测试优先**：这是最长路径，配方定型后立即送检（建议选择 SGS 或 Intertek 新加坡实验室）
2. **安全评估师**：需由 HSA 认可的安全评估师出具报告（新加坡本地顾问）
3. **通报制优势**：与保健品注册不同，化妆品是通报制（Notification），审核周期短得多
4. **ASEAN 互认**：如已在马来西亚 NPRA 完成通报，新加坡 HSA 可加速审核
5. **替代方案**：如 SPF 测试周期过长，可先上架"护肤乳液"（无 SPF 宣称），后续补充 SPF 宣称

### #4 — 中端天然无谷猫粮

| 维度 | 详情 |
|------|------|
| **认证名称** | **SFA 宠物食品进口许可（Pet Food Import Licence）** + **产品注册** |
| **主管机构** | Singapore Food Agency (SFA) — 动物与兽医司（Animal & Veterinary Service, AVS） |
| **适用法规** | Wholesome Meat and Fish Act（宠物食品相关条款）、SFA 进口许可制度 |
| **认证费用** | SFA 进口许可申请费：**S$200-500**<br>产品检测（营养分析+微生物）：**S$1,500-3,000**<br>工厂审核（SFA 可能要求）：**S$3,000-8,000**（含差旅）<br>顾问/代理费：**S$2,000-5,000**<br>**总计约 S$7,000-16,500（约 US$5,200-12,300）** |
| **认证周期** | 进口许可申请：**4-8 周**<br>产品检测：**2-4 周**<br>工厂审核（如需）：**4-8 周**<br>**总计约 3-6 个月** |
| **进口关税** | HS 2309.90（宠物食品）：**0%**（从 ASEAN/RCEP 国家进口） |
| **GST** | **9%** |
| **标签要求** | 英语标签：产品名称、成分表（按含量降序排列）、营养分析保证值（粗蛋白、粗脂肪、粗纤维、水分）、喂食指南、净重、批号、保质期、制造商/进口商信息、SFA 注册编号 |
| **成分限制** | 禁止含牛海绵状脑病（BSE）风险动物成分（反刍动物肉骨粉）；禁止含某些抗生素/激素 |
| **营养标准** | 建议符合 AAFCO（美国饲料管理协会）或 FEDIAF（欧洲宠物食品工业联合会）标准 |
| **适口性** | 无强制要求，但适口性差将导致高退货率 |

### 🐾 风险评级：🟡 中高

| 风险点 | 说明 | 缓解措施 |
|--------|------|----------|
| SFA 工厂审核 | 海外工厂可能需要接受 SFA 审核（疫情期间多为文件审核） | 确保工厂有出口到新加坡的经验，持有 HACCP/ISO 22000 |
| 成分溯源 | SFA 要求完整的原料来源证明 | 建立原料溯源体系，每批原料有供应商证明 |
| 营养不达标 | 粗蛋白等指标不符合标签宣称 | 出厂前每批做营养分析检测 |
| 适口性风险 | 新加坡猫口味偏好可能与欧美不同 | 在新加坡做小规模试吃测试（50-100 只猫） |

### 💡 合规建议

1. **选择有新加坡出口经验的工厂**：这是最关键的一步——有经验的工厂熟悉 SFA 要求
2. **AAFCO 标准**：确保配方符合 AAFCO 猫食品营养标准（SFA 认可）
3. **预检**：发货前做第三方检测（营养+微生物+重金属），费用约 S$1,500-2,000
4. **进口许可**：需新加坡注册公司作为进口主体（如无本地公司，可通过第三方进口代理）
5. **替代方案**：如 SFA 周期过长，可考虑通过已有 SFA 许可的进口商合作（借牌进口），但利润空间会被压缩

| **PDPA 影响** | 低 — 不涉及用户数据收集 |

### 🏋️ 风险评级：🟢 低

| 风险点 | 说明 | 缓解措施 |
|--------|------|----------|

1. **立即上架**：零认证壁垒，面料打样后可直接上架
2. **面料检测**：建议做 OEKO-TEX Standard 100 认证（费用约 S$500-1,000），作为营销信任背书
3. **UPF 测试**：如宣称 UPF 50+，需有测试报告（AS/NZS 4399 标准）
4. **尺码标注**：亚洲版型是差异化卖点，建议提供详细的厘米尺码表

### #6 — 低糖胶原蛋白软糖

| 维度 | 详情 |
|------|------|
| **认证名称** | **HSA 保健品注册（Health Supplement Registration）** |
| **主管机构** | Health Sciences Authority (HSA) |
| **适用法规** | Health Supplements Regulations 2005 |
| **认证费用** | HSA 申请费：**S$400-800**<br>产品检测：**S$2,000-4,000**（有效成分含量、重金属、微生物、糖含量验证）<br>顾问/代理费：**S$2,000-5,000**<br>**总计约 S$4,400-9,800（约 US$3,300-7,300）** |
| **认证周期** | **3-6 个月**（HSA 审核）+ 1-2 个月（产品检测）= **总计 4-8 个月** |
| **进口关税** | HS 2106.90（膳食补充剂）：**0%** |
| **GST** | **9%** |
| **标签要求** | 英语标签：产品名称、成分表、每日建议用量、批号、有效期、HSA 注册编号、"此产品不用于诊断、治疗或预防任何疾病"免责声明 |
| **成分限制** | 胶原蛋白肽无明确上限；维生素C ≤ 1,000mg/日；生物素 ≤ 5,000mcg/日；赤藓糖醇作为甜味剂需符合 Food Regulations |
| **宣称限制** | 不允许宣称"美容/抗衰老"等可能被视为药品/化妆品宣称的功能；允许"支持皮肤健康"等温和宣称 |
| **低糖宣称** | "低糖"宣称需符合 Singapore Food Regulations 定义（糖含量 ≤ 5g/100g 固体） |

### 💊 风险评级：🟡 中高

| 风险点 | 说明 | 缓解措施 |
|--------|------|----------|
| HSA 审核延迟 | 与 #2 维生素C 类似 | 同步启动 HSA 注册 |
| 胶原蛋白来源 | 如使用动物源胶原蛋白，需说明来源（牛/鱼/猪） | 建议使用鱼源胶原蛋白（素食友好+无宗教限制） |
| 低糖宣称合规 | 赤藓糖醇替代后需验证实际糖含量 | 送检验证糖含量符合"低糖"定义 |
| 明胶 vs 果胶 | 如使用明胶（动物源），需说明来源；植物基果胶更合规 | 使用植物基果胶（已在选品中明确） |

### 💡 合规建议

1. **与 #2 维生素C 同步启动 HSA 注册**：可共用顾问/代理，降低边际成本
2. **配方优化**：使用鱼源胶原蛋白 + 植物基果胶，避免宗教/素食合规问题
3. **低糖验证**：送检验证糖含量，确保"低糖"宣称有数据支持

## 关键合规提醒

### ⚠️ 必须做

1. **HSA 注册先行**：保健品注册周期最长（4-8 个月），必须在 OEM 打样阶段同步启动
2. **SFA 进口主体**：宠物食品需新加坡注册公司作为进口主体，如无本地公司需找进口代理
3. **Halal 原料溯源**：Halal 认证的核心是原料溯源，从采购环节就要建立 Halal 原料供应链
4. **英语标签**：所有产品标签必须包含英语信息（新加坡官方要求）
5. **GST 合规**：确保进口时正确申报 GST，保留所有进口单据

### ⚠️ 禁止做

1. **未经 HSA 注册销售保健品**：最高罚款 S$10,000 或监禁 6 个月（或两者并罚）
2. **未经 SFA 许可进口宠物食品**：货物可能被扣押+罚款
3. **虚假 Halal 宣称**：未经 MUIS 认证使用 Halal 标志是违法行为
4. **虚假 SPF 宣称**：无测试报告支持 SPF 宣称可能被 HSA 处罚
5. **婴儿产品过度宣称**：婴儿产品的安全宣称需有充分证据支持
