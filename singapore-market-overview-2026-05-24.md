## 4️⃣ 跨境进口政策与关税环境

### 4.1 GST（消费税）

| 项目 | 详情 |
|------|------|
| **GST 税率** | 9%（2024 年起，2023 年为 8%） |
| **征收范围** | 几乎**所有进口商品**，**无免税门槛**（de minimis = 0） |
| **低值商品（≤SGD 400）** | 由海外供应商通过 OVR（海外供应商注册）制度在结账时收取 |
| **高值商品（>SGD 400）** | 进口时由海关征收 |
| **注册门槛** | 年营业额 > SGD 1百万（强制注册） |
| **非注册企业** | 进口 GST 不可抵扣，为实际成本 |

### 4.2 关税与禁限

| 项目 | 详情 |
|------|------|
| **FTA 网络** | 27 个 FTA，多数商品进口关税为 0 |
| **应税商品** | 仅 4 类：烟草、酒精、石油制品、机动车 |
| **跨境电商综合税** | 进口 GST 9% + 可能的产品特定认证费用 |
| **特殊认证** | 食品（SFA）、化妆品（HSA）、电子产品（IMDA） |
| **FTA 原产地优惠** | ASEAN、中国-新加坡 FTA 等可享受优惠税率 |

### 4.3 合规要点

- **所有进口商品均需缴纳 9% GST**，成本结构必须包含
- 跨境卖家需注册 OVR 或在新加坡注册 GST
- 食品、化妆品、电子产品需额外认证
- 新加坡被评为全球最高效海关之一（DNV 领先海运城市排名第一）

## 6️⃣ 市场机会与风险摘要

### ✅ 机会
1. **高消费力**：人均 GDP US$88K，AOV SGD$185，品质溢价空间大
2. **移动端优先**：92%+ 智能手机渗透，Shopee/Lazada 移动流量占 80%+
3. **多元文化**：适合差异化品类（华人/马来/印度消费群体需求各异）
4. **物流高效**：全球最佳物流基础设施，当日达/次日达普及
5. **跨境友好**：多数商品零关税，FTA 网络完善

### ⚠️ 风险
1. **GST 成本**：9% 进口 GST 必须计入成本，无免税门槛
2. **竞争激烈**：Shopee+Lazada 占 88%，已建立壁垒
3. **高运营成本**：人力/租金/物流成本高于东南亚其他国家
4. **小市场**：仅 590 万人口，GMV 天花板相对有限
5. **认证要求**：食品/化妆品/电子品需额外 SFA/HSA/IMDA 认证
