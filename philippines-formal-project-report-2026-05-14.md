## 执行摘要

菲律宾是**东南亚增长最快的数字经济体**，2026年电商市场规模达 **PHP 2.1万亿（~$37.5B）**，增速 **18%**。菲律宾已超越越南成为**东南亚第三大平台电商市场**。

### 核心发现

| 维度 | 结论 |
|------|------|
| **市场规模** | $37.5B（2026），+18% YoY，东南亚第三 |
| **平台格局** | Shopee 42-55% / Lazada 16-28% / TikTok Shop 15-29% |
| **重点品类** | 宠物用品（S级）、健康保健（S级，+45%增速）、户外装备（S级）、手工DIY（S级）、个护小家电/家居收纳/厨房小家电（A级） |
| **Top 10 SKU** | ✅ Go × 5 / ⚠️ 有条件 Go × 2 / ❌ No-Go × 3 |
| **推荐保留** | DIY工具套装、太阳能户外灯、模块化收纳盒、草本补充剂、胶原蛋白补充剂 |
| **建议移除** | 防晒霜SPF50（-4.3%净利率）、便携式破壁机（-2.6%）、中端无线耳机（-3.4%） |
| **最高风险** | LED美容面罩（FDA分类不确定 + 高退货率） |
| **最高收益** | DIY工具套装（25.2%净利率）、太阳能户外灯（24.1%净利率） |
| **关键变化** | COD从45%降至11%，数字支付（GCash/Maya）成主流64% |
| **内容电商** | 占平台GMV 32%，+25%跨境销售增速 |

### 一句话结论

**优先推进5个低风险高收益SKU，移除3个亏损SKU，2个高风险SKU需附加条件。核心问题是3个SKU净利率为负，在比索贬值环境下必须果断砍掉。**

## 四、社交媒体舆情分析

### 关键发现

- **9000万+社媒用户**，日均使用3-4小时（全球最高之一）
- **"Budol"文化**：菲律宾特有"被种草冲动购物"文化
- **"Hiyang"护肤文化**：消费者用"hiyang"（合肤质）衡量产品有效性
- **从报复性消费到"选择疲劳"**：消费者对品牌更加挑剔
- **双重消费人格**：77%认为"有钱买贵的东西很重要"，但48%同时追求极简

### 消费者痛点（≥5个）

1. **价格敏感** — 追求性价比，不喜欢"太贵"的品牌
2. **信任危机** — 对陌生品牌不信任，需要社交证明
3. **物流焦虑** — 7641岛屿导致配送不确定性
4. **产品质量** — 假货泛滥，对正品要求高
5. **COD依赖** — 虽然降至11%，但农村地区仍需COD
6. **本地化不足** — 不喜欢非本地化的产品描述

### 竞品卖点分析（≥3个）

1. **Shopee**：低价+促销+物流网络
2. **TikTok Shop**：内容驱动+直播带货+社交推荐
3. **Lazada**：品牌正品+LazMall保证
