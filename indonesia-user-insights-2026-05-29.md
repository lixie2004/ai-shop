## 一、市场背景与消费特征

### 1.1 核心市场数据

| 维度 | 数据 |
|------|------|
| 人口 | 2.83亿，中位年龄30岁 |
| 互联网用户 | 2.21亿+，渗透率81%+ |
| 社交媒体用户 | 1.86亿+，TikTok 1.2亿+ |
| 电商GMV（2024） | $56.5B，东南亚最大 |
| 穆斯林人口占比 | 87%（Halal认证是刚需） |
| 中产阶级 | 5,200万+ |
| 平台格局 | Shopee ~38% / Tokopedia ~23% / TikTok Shop ~20% |
| 社交电商占比 | 40-45% |
| 购买决策因素 | 价格(35%) > 口碑(25%) > 物流(15%) > 认证(15%) > 品牌(10%) |
| TikTok Shop定价特征 | 92%销售额来自<$10商品 |

### 1.2 关键市场动态

- **PMK 4/2025**：进口免税门槛从$75降至$3，跨境直邮模式受冲击，本地仓成唯一选择
- **Halal 2026强制认证**：化妆品Halal认证从加分项变为入场券
- **KOC信任度>头部KOL**：5k-50k粉丝的中腰部KOC信任度更高、互动率更好、成本更低
- **消费链路**：TikTok种草 → TikTok Shop/Tokopedia下单 → WhatsApp社群复购
- **最大消费季**：斋月(Ramadan)+开斋节(Lebaran)，提前2个月备货

## 四、用户画像

### 4.1 画像一：城市年轻养宠族（Gen Z + 千禧一代）

| 维度 | 特征 |
|------|------|
| **年龄** | 22-32岁 |
| **性别** | 女65% / 男35%（女性占主导） |
| **月收入** | Rp5-15jt ($312-938) |
| **城市层级** | Jabodetabek（大雅加达）、泗水、万隆、棉兰 |
| **职业** | 白领/自由职业/大学生 |
| **养宠类型** | 猫（80%）> 狗（15%）> 其他（5%） |
| **购物平台** | TikTok Shop（60%）> Shopee（30%）> Tokopedia（10%） |
| **日均社媒使用** | 3-4小时 |
| **核心消费动机** | 宠物人性化——把宠物当"孩子"，愿意为宠物生活质量付费 |
| **价格敏感度** | 🟡中等——愿意为品质多付30-50%，但追求性价比 |
| **品牌认知** | 对Xiaomi/PetKit有品牌认知，但更信任KOC真实使用分享 |
| **内容偏好** | TikTok萌宠视频+智能设备开箱+好物推荐 |
| **购物决策路径** | TikTok种草 → 看Shopee评论 → KOC推荐验证 → 下单 |

**核心需求**：
- 智能喂食器/饮水机，但要求印尼语App + 品质可靠（$12-31）
- 适配印尼猫体型的项圈/背带/猫窝
- 宠物冷却垫（热带气候刚需）
