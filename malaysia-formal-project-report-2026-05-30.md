## 1. 市场概览

### 1.1 电商市场规模

| 指标 | 数值 |
|------|------|
| 2025年电商 GMV | **$200 亿**（RM 833 亿） |
| 2024年增速 | **+19.5%**（东南亚第二快） |
| 2027年预测 | **$235 亿**（CAGR ~16%） |
| 人均 GDP | **$12,140**（东南亚第二） |
| 互联网渗透率 | **97.7%** |
| 网购渗透率 | **87.8%** |
| 日均上网时长 | **8 小时 17 分** |

### 1.2 平台格局

| 平台 | 市场份额 | 特点 |
|------|---------|------|
| **Shopee** | ~55-60% | 主导地位，全品类覆盖 |
| **Lazada** | ~20-25% | 中高端定位 |
| **TikTok Shop** | ~15%+（增长最快） | 内容电商，社交驱动 |
| **PG Mall** | ~5% | 本土平台 |

### 1.3 核心市场特征

- **多元族群**：马来族 69% / 华族 23% / 印度族 7%，需差异化策略
- **清真认证是第一信任门槛**：JAKIM 认证对穆斯林消费者是购买决策必要条件
- **"研究型购买"模式**：购买前看评测、对比价格、查看评价
- **社交电商爆发**：TikTok Shop 2025年 GMV 同比增长 200%+
- **数字支付成熟**：FPX 29.8% / 银行转账 28.6% / COD 17.8%
- **SST 改革**：2025年7月起低价值进口商品（≤RM500）征收 10% 销售税

## 4. 舆情与消费者洞察

### 4.1 核心发现

| # | 发现 | 涉及品类 |
|---|------|---------|
| 1 | **"信任赤字"**是跨品类最大障碍 | 全部品类 |
| 2 | **"湿热气候适配"**是马来西亚独有刚需 | 时尚、宠物、母婴 |
| 3 | **"认证 = 信任货币"** — NPRA/SIRIM/JAKIM 是决策门槛 | 补剂、母婴、智能 |
| 4 | **"研究型购买 + 社群驱动"** — KOC > KOL | 全部品类 |
| 5 | **清真认证是最大的差异化蓝海** | 补剂、宠物、美妆 |
| 6 | **"小户型 + 租房经济"**催生空间适配需求 | 智能、母婴、宠物 |
| 7 | **"宠物人性化"**支出不设上限 | 宠物 |
| 8 | **开斋节消费脉冲**是穆斯林时尚核心引擎 | 时尚 |
