## 建模方法论与假设

### 成本模型公式

```
总成本 = 采购成本 + 头程物流 + 进口关税 + 进口GST 
         + 平台佣金 + 广告费 + 尾程配送费 + 退货损耗 
         + 汇率波动预留 + 认证摊销 + 仓储费 + 支付手续费

毛利额 = 零售价 - 总成本（不含广告费）
净利润 = 零售价 - 总成本（含广告费）

毛利率 = 毛利额 ÷ 零售价
净利率 = 净利润 ÷ 零售价
```

### 关键假设（2026年5月）

| 假设项 | 取值 | 依据 |
|--------|------|------|
| **GST税率** | **9%** | 2024年1月从8%上调至9%，2026年维持9% |
| **进口关税** | 0%（绝大多数消费品） | 新加坡是自由港，除烟酒汽车等外，绝大多数商品零关税 |
| **低价值商品GST豁免** | SGD 400以下免GST | 但2023年起已取消，**所有进口商品均征GST** |
| **海外空运/海运** | 海运CNY 1.5-3/kg / 空运CNY 5-12/kg | 中国→新加坡航线成熟，运价稳定 |
| **SGD/USD汇率** | 1 SGD ≈ 0.75 USD | 2026年5月，SGD为亚洲最强货币之一 |
| **SGD/CNY汇率** | 1 SGD ≈ 5.35 CNY | 2026年5月参考 |
| **汇率波动预留** | 2% | SGD是低波动货币（年均波动<3%） |
| **Shopee SG佣金** | 5.0-8.5%（按品类） | Shopee SG跨境佣金，时尚6.5%+/电子4.5%+ |
| **Lazada SG佣金** | 4.5-7.5%（按品类） | Lazada SG标准费率 |

| **支付手续费** | 2.0%（加权） | PayNow 0.5% + 卡支付2.5% + GrabPay 1.8% |
| **ACoS（广告支出占比）** | 12-20%（按品类） | 新加坡广告成本低于印尼/马来 |
| **尾程配送费** | SGD 2.0-5.0/单 | Shopee Xpress / J&T SG / Ninja Van |

### 新加坡市场关键特征

| 特征 | 说明 | 成本影响 |
|------|------|---------|
| **零关税自由港** | 除烟酒汽车外，所有进口商品0关税 | 进口关税成本=0 |
| **GST 9%** | 2024年上调后，所有进口商品+本地销售均征 | 进项税可抵扣（需注册GST） |
| **高物流效率** | 全岛任何地点次日达 | 尾程配送费低但服务要求高 |
| **高退货率（服饰）** | 服饰退货率15-20%，非服饰3-5% | 需预留退货损耗 |
| **高CPC** | 新加坡CPC SGD 0.5-2.0，高于东南亚其他国家 | 广告成本占比中高 |
| **PayNow即时支付** | 新加坡本地银行即时转账 | 支付手续费低至0.5% |
| **HDB组屋配送** | 70%+人口住HDB组屋，配送需上楼 | 尾程有额外楼层费用 |
| **高合规要求** | HSA/AVA/SFA/IMDA等监管严格 | 认证成本高但可形成壁垒 |

**结论**：一旦年销售额超过SGD 100万必须注册GST。但即使低于门槛，**建议主动注册GST**——可回收所有进口GST进项税，净效果是降低2-3%的成本负担。

### 2. 零关税优势利用

新加坡是所有Top 10 SKU的零关税天堂。与马来西亚（10% SST）和印尼（11% PPN + 7.5% PPh）相比，新加坡的税负优势极为显著：

| 市场 | 关税 | 增值税 | 所得税预提 | 合计税负 |

| 🇸🇬 新加坡 | 0% | 9%（可抵扣） | 0% | **0-9%** |
| 🇲🇾 马来西亚 | 0-20% | 10%（LPGT/SST） | 0% | 10-30% |
| 🇮🇩 印度尼西亚 | 0-25% | 11%（PPN） | 7.5%（PPh） | 18.5-43.5% |

### 4. 实体架构建议

```
中国贸易公司（卖方）
    ↓ FOB出口
新加坡本地公司（进口商+电商运营）
    ↓ GST注册
Shopee/Lazada/TikTok Shop SG（本地卖家账户）
```

**优势**：
- 新加坡公司注册GST后，所有进口GST可全额抵扣
- 新加坡企业所得税17%，且有税收优惠计划（SGD 10万以下免税）
- 本地卖家在Shopee SG的佣金费率低于跨境卖家1-2%
- 新加坡消费者对本地公司信任度更高
