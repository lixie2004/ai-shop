摘要

越南电商市场2025年GMV约$163.5亿（4,297万亿越南盾），同比增长34.75%，预计2028年将达到$499亿（CAGR 35%）。市场已进入双头垄断格局（Shopee 53% + TikTok Shop 44% = 97%份额）。本报告通过12步分析流程，最终推荐Top 10选品方向，其中宠物用品以9.2分综合评分位居第一，其次是智能家居和运动户外。

  • Millennials（46.6%电商用户）：周均1-3次购买，客单价$125，偏好家电/3C/家居

第六部分：结论与建议

结论

  1. 越南电商市场正处于高速增长期（CAGR 35%），是东南亚最具潜力的电商市场之一
  2. 双头垄断格局（Shopee 53% + TikTok Shop 44%）已形成，建议双平台布局
  3. 宠物用品为最佳切入品类（蓝海、高利润、低竞争）
  4. 智能家居和运动户外为第二梯队推荐品类

建议

  1. 从宠物用品和家居装饰切入，快速验证市场
  2. 3个月内根据数据反馈调整品类结构
  3. 关注TikTok Shop直播电商红利
  4. 提前准备认证资质（特别是宠物食品和智能家居）
