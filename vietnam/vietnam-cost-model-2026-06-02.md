# 越南 Top 10 SKU 全成本模型与毛利测算报告

**报告日期**: 2026-06-02  
**分析口径**: 跨境直邮 + 本地仓（Shopee FBS / TikTok Shop 本地履约）双模式  
**汇率基准**: 1 USD = 25,000 VND（2026年6月估算）  
**货币单位**: 除特别说明外，成本与售价均为 VND

---

## 一、成本模型通用假设

| 参数 | 数值 | 说明 |
|------|------|------|
| 采购地 | 中国大陆（广东/浙江/义乌） | 1688/阿里国际站 |
| 头程物流 | 中国→越南跨境物流 | 海运拼箱/空运小包 |
| 进口关税 | 0% - 10% | 按HS编码，大部分消费品5-10% |
| 进口 VAT | 10% | 越南标准增值税率 |
| 平台佣金（Shopee） | 5.5% - 6.5% | 含交易手续费，Shopee Mall略高 |
| 平台佣金（TikTok Shop） | 5.0% - 6.0% | 含技术服务费 |
| 本地履约（FBS/3PL） | 按重量阶梯 | 0.5kg以下 ~12,000 VND |
| 广告费率 | 8% - 15% | 占GMV比例，新店铺较高 |
| 退货损耗率 | 2% - 8% | 按品类差异 |
