## 四、合规检查清单

### 4.1 企业资质要求

| 项目 | 要求 | 说明 |
|------|------|------|
| 营业执照 | ✅ 必须 | 中国企业营业执照 |
| 进出口权 | ✅ 必须 | 海关编码备案 |
| 食品经营许可证 | ✅ 食品类 | 宠物食品需MARD认证 |
| 品牌授权书 | ✅ 品牌类 | 品牌方授权 |
| 产品质量认证 | ✅ 部分品类 | 视产品类别 |

### 4.2 产品认证要求

| 品类 | 认证要求 | 认证机构 |
|------|---------|---------|
| 宠物食品 | MARD注册 | 越南农业与农村发展部 |
| 宠物用品 | 无特殊要求 | 无 |
| 智能家居 | CR mark（安全）+ MIC（无线电） | 越南信息通信部 |
| 小家电 | CR mark | 越南信息通信部 |
| 美妆护肤 | 化妆品备案 | 越南卫生部门 |
| 母婴用品 | 质量安全认证 | 越南标准计量局 |
| 数码配件 | CR mark | 越南信息通信部 |
| 服饰鞋包 | 无特殊要求 | 无 |

### 4.3 关税与税务

| 项目 | 税率 | 说明 |
|------|------|------|
| 进口关税 | 0-25% | 视产品类别和原产地 |
| 增值税（VAT） | 10% | 标准税率 |
| 消费税 | 0-50% | 特定品类 |
| 跨境电商税 | 减免政策 | 小额包裹（<$200）可能减免 |

### 4.4 合规流程

```
企业资质准备 → 产品认证 → 海关备案 → 进口清关 → 国内配送
     ↓              ↓            ↓            ↓            ↓
  营业执照       CR/MARD      海关编码      报关          3PL物流
  进出口权       MIC/卫生     税务登记      缴税
  品牌授权      化妆品备案
```
