# 越南跨境电商宏观环境分析简报

**编制日期**：2026年5月16日  
**数据来源**：Mordor Intelligence、DataReportal Digital 2025、World Bank、IMF、越南工贸部(MOIT)、Google-Temasek-Bain e-Conomy SEA 2024  
**报告编制**：OWL 趋势研究员

---

## 摘要（Executive Summary）

越南是东南亚增长最快的电商市场之一，2025年电商GMV预计达 **$27.7B**，2026年预计达 **$33.6B**，2026-2031年CAGR高达 **21.08%**。互联网渗透率78.8%，社交媒体渗透率75.2%，智能手机普及率约71%，移动端贡献71%以上电商收入。

**核心判断**：越南正处于电商渗透率快速提升的黄金窗口期。年轻人口（中位年龄33.4岁）、快速壮大的中产阶级（预计2026年占人口26%）、政府强力推动数字化转型，三大引擎叠加使越南成为跨境卖家不可忽视的战略市场。但物流基础设施仍存短板，农村宽带覆盖不足，跨境政策仍在完善中。

## 四、跨境电商政策环境

### 4.1 电商法律框架

| 法规 | 内容 | 影响 |
|------|------|------|
| **Decree 52/2013/ND-CP** | 电子商务基本法 | 规范电商活动，保护消费者权益 |
| **Decree 52修订草案（2020年二稿）** | 跨境电商监管 | 要求境外平台在越南注册/报备 |
| **CPTPP（全面与进步跨太平洋伙伴关系协定）** | 自由贸易协定 | 减免关税，简化海关程序 |
| **RCEP（区域全面经济伙伴关系协定）** | 区域贸易协定 | 扩大跨境贸易便利化 |
| **Circular 78（电子发票指令）** | 电子发票强制要求 | 提升交易透明度，增强消费者信任 |
| **个人数据保护法** | 数据保护 | 类似GDPR，要求企业合规处理用户数据 |

### 4.2 跨境电商监管要点

**境外平台合规要求**（Decree 52修订草案）：
- 拥有越南域名或面向越南用户的境外电商平台，须在越南注册/报备
- 须指定越南境内的代表机构或授权代表
- 须定期向越南主管部门报告经营活动
- 须遵守越南消费者保护法规

**进口关税与税收**：
- 越南海关对跨境电商进口征收进口关税+增值税（VAT 10%）
- CPTPP/RCEP框架下，多数商品关税逐步降至0-5%
- 跨境电商小包裹享受一定免税额度（具体金额需根据最新海关公告确认）
- 2024年跨境进口包裹达3.241亿件，价值$5.68亿，同比增长43%

### 4.3 政策趋势

1. **监管趋严**：越南政府正加强对跨境电商的监管，要求境外平台合规运营
2. **关税优惠持续**：CPTPP和RCEP为跨境卖家提供长期关税减免红利
3. **无现金社会推进**：政府目标2025年无现金支付占比超50%
4. **数据本地化**：个人数据保护法要求企业存储越南用户数据在境内
5. **电子发票强制**：Circular 78要求所有电商交易开具电子发票

### 4.4 美国关税政策影响

- 2025年美国最高法院裁定特朗普关税政策违宪，对越南出口美国的商品是重大利好
- 越南作为全球供应链转移的最大受益国之一，对美出口持续增长
- 但全球贸易政策不确定性仍高，需持续关注

## 五、物流基础设施

### 5.1 物流市场概况

| 指标 | 现状 |
|------|------|
| 主要运输方式 | 公路运输（卡车）为主，城市内摩托车配送 |
| 一线城市配送时效 | 1-2天（胡志明市、河内） |
| 二线城市配送时效 | 2-4天（岘港、芹苴、海防、芽庄） |
| 农村地区配送时效 | 3-7天 |
| 冷链覆盖率 | 主要集中在胡志明市和河内 |
| 生鲜损耗率 | 25-30%（果蔬类） |

### 5.2 物流基础设施进展

**最后一公里配送扩张**：
- 物流商正在二线城市建设城市整合中心，配送时间缩短最高40%
- 摩托车配送在密集城区仍是最有效方式
- 环保车辆和数字路线规划正在推广

**主要物流服务商**：
| 服务商 | 类型 | 特点 |
|--------|------|------|
| GHN (Giao Hàng Nhanh) | 第三方物流 | 覆盖广，价格优 |
| GHTK (Giao Hàng Tiết Kiệm) | 第三方物流 | 性价比高 |
| J&T Express | 第三方物流 | 中资背景，快速扩张 |
| Ninja Van | 第三方物流 | 东南亚网络 |
| Shopee Express | 平台自有 | Shopee生态内 |
| Lazada Logistics (LEX) | 平台自有 | Lazada生态内 |
| TikiNOW | 平台自有 | Tiki自建，2小时达 |
| Vietnam Post | 国有邮政 | 覆盖最广，偏远地区 |
| DHL Express | 国际快递 | 跨境首选 |
| FedEx | 国际快递 | 跨境首选 |

### 5.3 物流挑战

1. **冷链物流碎片化**：冷库集中在胡志明市和河内，二线城市以外覆盖不足
2. **农村配送成本高**：山区和偏远地区配送成本是城市的2-3倍
3. **退货率高**：推高整体履约成本，对跨境卖家影响更大
4. **基础设施缺口**：十大结构性缺口，从劳动力技能到IT追溯系统
5. **海关清关效率**：跨境电商包裹量激增，海关处理能力面临压力

| 场景 | 估算成本 |

| 中国→越南（海运整柜） | $800-1,500/20尺柜 |

| 平台 | 类型 | 用户规模 | 特点 |

### 6.3 支付趋势

1. **无现金支付加速**：2024年初无现金交易笔数增长63.3%，金额增长41.45%

3. **BNPL崛起**：2025年BNPL GMV达$4.05B，提升客单价30-40%
4. **COD持续下降**：随着数字支付普及和消费者信任提升，COD占比持续降低
5. **国家人口数据库对接支付**：简化结账流程，提升转化率

### 6.4 跨境支付注意事项

- 越南央行对外汇管制较严格，跨境收款需通过合规渠道
- 建议接入本地支付网关（MoMo、ZaloPay、VNPay）提升转化率
- 越南盾（VND）汇率波动需关注，建议采用美元定价+VND显示
- 跨境结算周期通常为T+7至T+14
