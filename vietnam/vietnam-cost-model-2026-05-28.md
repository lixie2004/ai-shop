## 1. 分析框架与方法论

### 1.1 全成本模型结构

每个SKU的成本模型覆盖 **10 项成本要素**，从出厂到最终交付全链路：

```
出厂成本 → 出口报关 → 头程物流 → 进口关税 → 进口VAT →
仓储入库 → 平台佣金 → 广告费 → 尾程配送 → 退货损耗 → 汇率波动
```

### 1.2 计算公式体系

| 指标 | 公式 |
|------|------|
| CIF价 | 出厂采购成本 + 头程物流 |
| 进口关税 | CIF价 × 关税税率（RCEP优惠税率） |
| 进口VAT计税基础 | CIF价 + 关税 |
| 进口VAT | (CIF价 + 关税) × 8% |
| 落地成本 | CIF价 + 关税 + 进口VAT + 仓储费 + 尾程配送 |
| 平台扣点 | 售价 × 平台佣金率 |
| 广告费 | 售价 × ACoS（广告费占销售额比重） |
| 退货损耗 | 售价 × 退货率（含COD拒收全损场景） |
| 汇率风险准备金 | 售价 × 2.5% |
| **综合毛利率** | **(售价 - 落地成本) / 售价 × 100%** |
| **综合净利率** | **(售价 - 落地成本 - 平台佣金 - 广告费 - 退货损耗 - 汇率损失) / 售价 × 100%** |

### 1.3 建模方法论

1. **采购成本**：基于中国工厂出厂价（含包装），按品类和规格估算
2. **头程物流**：深圳/广州 → 河内/胡志明，专线物流 ￥30-35/kg
3. **关税税率**：基于HS编码估算、RCEP优惠、越南最惠国税率
4. **VAT**：越南标准VAT税率 8%（2025年后恢复正常税率）
5. **退货率**：综合COD（67%占比）和预付费订单，按品类调整
6. **汇率波动**：CNY/VND 年度波动率约 3-5%，按 2.5% 计提风险准备金

## 2. 越南市场宏观参数假设

### 2.1 基础税率与费用

| 参数 | 数值 | 备注 |
|------|------|------|
| RCEP优惠关税 | 0%-5% | 大部分消费品3年内降至0-5% |
| 最惠国关税（MFN） | 5%-15% | HS编码依赖，无RCEP时适用 |
| 增值税（VAT） | 8% | 标准税率（2025起恢复正常8%） |
| 平台佣金 | 5%-8% | Shopee 6%、Lazada 5%、TikTok Shop 8% |
| 平台支付手续费 | 1%-2% | 随平台政策浮动 |
| COD占比 | 67% | 越南跨境电商市场特征 |
| 退货率（平均） | 15%-20% | COD订单>30%，预付费约8%-12% |
| 汇率（CNY/VND） | 1:3,450 | 2026年5月参考 |
| 汇率波动风险准备金 | 2.5% | 年度波动率中值 |

### 2.2 物流成本参数（中国→越南）

| 运输方式 | 时效 | 单价 | 适用场景 |
|----------|------|------|----------|
| 国际快递（DHL/FedEx） | 3-5天 | ￥50-70/kg | 高价值/急件 |
| 专线物流 | 5-8天 | ￥30-35/kg | 中小包裹主流方式 |
| 海运拼箱（LCL） | 15-25天 | ￥80-120/m³ | 大件/批量备货 |
| 海外仓干线（海运+陆运） | 20-30天 | ￥15-20/kg | 备货至胡志明/河内仓 |

### 2.3 COD退货影响结构

| 退货场景 | 占退货比 | 估算依据 |
|----------|---------|----------|
| COD拒收 | 40% | COD订单约12%-15%被拒收 |
| 已收货退货 | 30% | 品类依赖，服饰类最高 |
| 质量问题退货 | 20% | 产品描述不符/运输损坏 |
| 其他（改单/重复等） | 10% | 系统误差 |
