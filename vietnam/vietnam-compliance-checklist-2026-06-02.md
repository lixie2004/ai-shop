# 越南 Top 10 SKU 合规检查清单

**报告日期**: 2026-06-02  
**目标市场**: Vietnam（越南社会主义共和国）  
**适用法规**: 2026年7月1日生效的《电子商务法2025》（Law No. 122/2025/QH15）  
**数据来源**: QCVN技术法规、Circular 06/2011/TT-BYT、Law on Animal Husbandry 2018、Decree 43/2017/ND-CP、CR Mark认证体系、越南电商法2025

## 一、越南合规框架总览

### 1.1 核心监管体系

| 监管领域 | 主管机构 | 核心法规 |
|---------|---------|---------|
| 产品准入认证 | 科技部（MOST）- STAMEQ | QCVN技术法规 + CR Mark认证 |
| 化妆品/美妆工具 | 卫生部（MOH）- DAV | Circular 06/2011/TT-BYT |
| 宠物食品 | 农林部（MARD） | Law on Animal Husbandry 2018 + Circular 04/2020/TT-BNNPTNT |
| 电子电器 | 科技部（MOST）+ 信息通信部（MIC） | QCVN 4:2009/BKHCN（安全）+ QCVN（EMC） |
| 电商运营 | 工贸部（MOIT） | Law on E-Commerce 2025（2026.7.1生效） |
| 进口税务 | 财政部 + 海关总署 | Law on Customs 2019 + VAT Law |

### 1.2 2026年7月电商法生效后的关键变化

| 变化项 | 旧规（Decree 52/2013） | 新规（Law on E-Commerce 2025） | 对跨境卖家的影响 |
|-------|----------------------|-------------------------------|----------------|
| 卖家身份 | 可匿名运营 | **必须实名验证**，公示企业/个人注册信息 | 跨境卖家需通过平台完成电子身份认证 |
| 产品信息 | 无强制披露 | **必须公示**质量、用途、相关许可证 | 所有SKU需准备完整产品资料 |
| 直播销售 | 无监管 | 平台需验证主播身份，**保存直播音视频**，设置投诉渠道 | TikTok Shop直播需合规 |
|  affiliate营销 | 无监管 | 平台需验证推广者身份，**管控违规链接** | 达人带货需签署合规协议 |
| 外国平台 | 无本地要求 | **必须设立法人或指定代表**，缴纳保证金 | Shopee/TikTok需有越南实体 |
| 低值免税 | VND 100万（~$40） | **已取消**，所有进口均需征税 | 跨境小包不再免税 |
| 处罚力度 | 行政罚款 | 可**封号、下架、刑事追责** | 违规成本大幅上升 |

### SKU #1：猫砂（Cát vệ sinh cho mèo）

| 分析维度 | 详情 |
|---------|------|
| **产品性质** | 宠物日常用品（非食品、非药品） |
| **准入认证** | ❌ 无需CR Mark认证 |
| **产品标准** | 无强制性国家标准；建议符合企业标准（TCVN自愿性） |
| **成分/材料限制** | 膨润土/豆腐猫砂无特殊限制；含除臭颗粒的需确保化学成分安全（无甲醛、重金属超标） |
| **税务合规** | 进口关税：约5-10%（HS 2508/3307）；VAT：10% |
| **标签要求** | 产品名称、成分、使用方法、生产日期/保质期、原产国、进口商信息（Decree 43/2017） |
| **电商法合规** | 需公示产品信息和质量声明；2026.7后需实名开店 |
| **风险评级** | 🟢 **低** |

**合规替代方案**：无认证门槛，无需替代。建议优先选择豆腐猫砂（环保卖点），避免含化学除臭剂的膨润土猫砂（成分争议风险）。

**认证费用/周期**：无认证费用。

### SKU #2：瑜伽垫（Thảm yoga）

| 分析维度 | 详情 |
|---------|------|
| **产品性质** | 运动器材/家居用品（非电子） |
| **准入认证** | ❌ 无需CR Mark认证 |
| **产品标准** | 无强制性国家标准；建议符合ISO 10993（生物相容性，如含PVC） |
| **成分/材料限制** | TPE材质：环保无异味，推荐；NBR/PVC材质：需确保不含邻苯二甲酸酯（Phthalates）等有害物质 |
| **税务合规** | 进口关税：约5-10%（HS 3926/4016）；VAT：10% |
| **标签要求** | 材质说明、厚度、尺寸、使用警告、原产国、进口商信息 |
| **电商法合规** | 需公示材质和安全性声明；厚度/密度参数需准确标注 |
| **风险评级** | 🟢 **低** |

**合规替代方案**：若PVC/NBR材质无法提供环保认证，建议全面切换为TPE材质（可宣称"环保无毒"，差异化卖点）。

**认证费用/周期**：无认证费用。第三方检测报告（SGS/Intertek）约$200-400/款，周期2-3周。

### SKU #3：美妆刷套装（Bộ cọ trang điểm）

| 分析维度 | 详情 |
|---------|------|
| **产品性质** | 美妆工具（非化妆品本身，不属于Circular 06/2011监管范围） |
| **准入认证** | ❌ 无需化妆品备案；❌ 无需CR Mark认证 |
| **产品标准** | 无强制性国家标准 |
| **成分/材料限制** | 刷毛材质需明确标注（动物毛 vs 合成纤维）；动物毛需确保无疫病风险；手柄材质（塑料/金属/木材）无特殊限制 |
| **税务合规** | 进口关税：约10-15%（HS 9603）；VAT：10% |
| **标签要求** | 材质成分、使用场景、原产国、进口商信息 |
| **电商法合规** | 需公示刷毛材质来源（动物毛需合规证明）；2026.7后直播推广需合规 |
| **风险评级** | 🟡 **中** |

**风险说明**：动物毛刷具涉及动物检疫问题（虽非强制），且2026电商法对直播推广美妆产品监管趋严，虚假宣传风险较高。

**合规替代方案**：建议优先使用**合成纤维刷毛**（超细纤维/尼龙），完全规避动物检疫和伦理争议，且成本更低、更易标准化。

**认证费用/周期**：无认证费用。如需动物毛检疫证明，约$100-200，周期1-2周。

### SKU #5：车载手机支架（Giá đỡ điện thoại ô tô）

| 分析维度 | 详情 |
|---------|------|
| **产品性质** | 汽车配件（机械结构为主，若含电子功能如无线充电则需额外认证） |
| **准入认证** | ⚠️ **基础款（纯机械）：无需认证**；⚠️ **含电子功能（无线充电/USB充电）：需CR Mark认证**（QCVN 4:2009/BKHCN 安全 + EMC） |
| **产品标准** | 机械结构无强制标准；电子功能需符合QCVN安全及EMC要求 |
| **成分/材料限制** | 塑料/金属材质无特殊限制；电子元件需符合RoHS（无铅、无汞等） |
| **税务合规** | 进口关税：约10-15%（HS 8517/8708）；VAT：10% |
| **标签要求** | 适用车型/手机尺寸、材质、原产国、进口商信息；含电子功能需标注CR Mark |

- 如需电子功能，可考虑与已获CR Mark认证的越南本地品牌合作（OEM贴牌）。

**认证费用/周期**：纯机械款：无。含电子功能款：CR Mark安全+EMC认证约$2,000-5,000，周期2-3个月（含测试+工厂审核）。

### SKU #6：宠物自动喂食器（Máy cho ăn tự động cho thú cưng）

| 分析维度 | 详情 |
|---------|------|
| **产品性质** | 宠物电子电器（含电机、电路、可能含Wi-Fi/APP功能） |
| **准入认证** | 🔴 **必须CR Mark认证**（QCVN 4:2009/BKHCN 安全 + QCVN EMC） |
| **产品标准** | 电气安全（绝缘、过热保护、漏电保护）；EMC（电磁干扰/抗干扰） |
| **成分/材料限制** | 食品接触部分（食盆）需符合食品接触材料标准；塑料外壳需阻燃等级达标 |
| **税务合规** | 进口关税：约10-15%（HS 8509/8543）；VAT：10% |
| **标签要求** | 额定电压/功率、安全警告、原产国、进口商信息、**CR Mark标识** |
| **电商法合规** | 需公示电气安全认证证书；直播推广需强调安全使用 |
| **风险评级** | 🔴 **高** |

**合规替代方案**：
- 方案A：**选择已有CR Mark认证的供应商**（中国出口越南的成熟品牌），直接贴牌销售，认证成本由上游承担。

- 方案C：**通过越南本地进口商代理认证**（授权代表模式），分摊认证成本。

**认证费用/周期**：CR Mark安全认证约$1,500-3,000；EMC认证约$1,000-2,000；总计约$2,500-5,000，周期2-4个月（含越南本地实验室测试+可能的工厂审核）。

### SKU #7：美容仪（Máy làm đẹp）

| 分析维度 | 详情 |
|---------|------|
| **产品性质** | 美容电子设备（含射频/LED/微电流等功能，可能涉及医疗器械分类） |
| **准入认证** | 🔴 **必须CR Mark认证**（QCVN 4:2009/BKHCN 安全 + QCVN EMC）；⚠️ **若宣称医疗功效（如"治疗痤疮""抗衰老"），可能被归类为医疗器械，需额外向卫生部申请医疗器械注册** |
| **产品标准** | 电气安全（IEC 60335系列）；EMC（IEC 61000系列）；若为医疗器械需符合越南医疗器械法规 |
| **成分/材料限制** | 接触皮肤部分需符合生物相容性要求（ISO 10993）；LED/射频元件需符合辐射安全标准 |
| **税务合规** | 进口关税：约10-20%（HS 8543/9019）；VAT：10% |
| **标签要求** | 额定电压/功率、安全警告、使用方法、禁忌症（如有）、原产国、进口商信息、**CR Mark标识** |
| **电商法合规** | **高风险**：2026电商法对有条件经营品类（美容仪可能涉及医疗器械）要求提供完整许可证；直播推广严禁虚假医疗功效宣传 |
| **风险评级** | 🔴 **极高** |

**合规替代方案**：
- 方案A（推荐）：**选择基础款美容仪**（仅含LED光疗/冷敷功能，不宣称医疗功效），按普通电器管理，仅需CR Mark认证。
- 方案B：**切换为非电子美容工具**（如美容滚轮/刮痧板/面膜仪（非电子）），完全规避电器认证和医疗器械风险。
- 方案C：若坚持销售高端美容仪，**必须聘请越南本地医疗器械注册代理**，预计额外增加$5,000-15,000注册费用及3-6个月周期。

**认证费用/周期**：
- CR Mark安全+EMC认证：约$3,000-6,000，周期2-4个月。
- 医疗器械注册（如适用）：约$5,000-15,000，周期3-6个月。
- **合计最高可达$8,000-21,000，周期5-10个月。**

> ⚠️ **重要变化**：越南已于2025年取消VND 100万（~$40）的低值免税门槛，**所有跨境进口包裹均需申报纳税**，无论金额大小。

### 4.2 跨境电商税务合规要点

1. **DDP模式（完税后交货）**：建议采用，由卖家承担关税和VAT，避免买家拒收。
2. **平台代扣**：Shopee/TikTok Vietnam可能对跨境订单代扣VAT，需确认平台政策。
3. **本地仓库**：通过越南3PL仓发货（海外仓模式），按一般贸易报关，税率确定但可批量清关降低成本。
4. **RCEP优惠**：若从中国进口，可利用RCEP原产地证享受部分商品关税减免（需确认具体HS编码）。

## 五、跨境电商清关流程要点

### 5.1 小包直邮清关（跨境B2C）

```
订单生成 → 包裹发运 → 抵达越南海关 → 电子申报（e-Customs）
    → 系统审单（风险分级）→ 低风险：快速放行 / 高风险：开箱查验
    → 缴纳税费（关税+VAT）→ 尾程配送（GrabExpress/GHN/GHTK）
```

**关键要点**：
- 需通过越南海关**VNACCS/VCIS**系统电子申报
- 每个包裹需附带**商业发票**（含HS编码、价值、原产国）
- 2026电商法生效后，平台需向海关提供**卖家真实身份信息**
- 低值免税取消后，**每单均需缴税**，建议DDP模式

### 5.2 一般贸易清关（B2B/海外仓补货）

```
签订进口合同 → 委托报关行 → 提交清关单证 → 海关查验（如需）
    → 缴纳税费 → 提货至仓库 → 分销/上架
```

**必备单证**：
- 商业发票（Commercial Invoice）
- 装箱单（Packing List）
- 提单/空运单（B/L or AWB）
- 原产地证（C/O，可享RCEP/ATIGA优惠税率）
- 产品认证证书（CR Mark/检测报告等，如适用）
- 进口许可证（如适用，如宠物食品需农林部备案）
