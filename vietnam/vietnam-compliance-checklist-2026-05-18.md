## 越南跨境电商合规框架速览

### 1. 进口通用要求（所有实体商品适用）

| 要求 | 法规依据 | 说明 |
|------|---------|------|
| 原产地证明（C/O） | 越南海关法 | 享受优惠关税需提供 Form E / Form D 等 |
| 商业发票（Commercial Invoice） | Circular 38/2015/TT-BTC | 越南语或英文，含 HS Code、单价、总价 |
| 装箱单（Packing List） | 同上 | 详细列明每箱商品内容 |
| 越南语标签 | Decree 43/2017/ND-CP + 111/2021/ND-CP | 所有进口商品必须附越南语标签，含：品名、产地、成分、规格、制造商、进口商信息 |
| 进口商登记 | 企业须持有效的 Enterprise Registration Certificate | 跨境卖家需委托或设立越南实体作为进口商 |
| 海关商品编码（HS Code） | Circular 31/2022/TT-BTC | 关税按 HS Code 征收，MFN 税率 0-35% |

### 2. Directive 8598/BCT-TMDT（2024年10月26日）

- **适用范围**：所有在越南运营的跨境电商平台（Temu, Shein, 1688 等均受影响）
- **核心要求**：
  - 跨境电商平台必须在越南完成商业登记
  - 平台需对入驻商家商品合规性负责
  - 未登记平台将被封锁域名 + 切断支付通道
  - 平台需建立消费者投诉处理机制
- **对卖家的影响**：选择合规平台入驻；或自建网站需向 MoIT 备案

### 3. 新电商法（2026年7月1日生效）

- 外国电商平台使用越南语、.vn 域名或达到一定交易量 → 须在越南设立法人实体或指定授权代表
- 算法推荐透明度要求
- 平台需代扣代缴税款

### 4. 跨境数字服务税（2025年起）

- 外国供应商（无越南常设机构）通过电商/数字平台在越南开展业务 → **VAT 税率从 5% 升至 10%**
- 必须通过越南税务 portal 登记并申报
- 平台运营商承担代扣代缴义务

#### 🔴 防晒霜 UV 过滤剂高风险提醒

越南严格采用 ASEAN Cosmetic Directive Annex VII（UV 过滤剂清单）。**已确认的禁用 UV 过滤剂或高受限成分**：
- 4-Methylbenzylidene camphor (4-MBC) — 在越南可能受限
- Benzophenone-3 (Oxybenzone) — 浓度限值严格（最大 6%）
- Octocrylene — 部分市场在重新评估中
- Homosalate — 浓度受限（最大 10%）

⚠️ 必须由认证实验室出具防晒产品成分合规报告。

| 项目 | 评级 |

| 建议入市路径 | 先单独推洁面乳+保湿乳（不含防晒），待流程跑通后再上线防晒霜。不要一次性上架套装。 |
