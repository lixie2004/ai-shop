## 一、成本模型方法论

### 1.1 成本构成

| 成本类别 | 说明 | 计算方式 |
|----------|------|----------|
| 采购成本 | 产品出厂价 | 供应商报价 |
| 头程物流 | 中国→越南运输 | 按重量/体积计费 |
| 关税 | 进口关税 | 按HS编码税率 |
| 平台佣金 | Shopee/TikTok Shop | 按平台费率 |
| 支付手续费 | 支付渠道费用 | 按交易金额 |
| 广告成本 | 平台广告投放 | 按ACOS估算 |
| 仓储成本 | 海外仓/本地仓 | 按存储量计费 |
| 售后成本 | 退货/损耗/客服 | 按退货率估算 |
| 包装成本 | 产品包装 | 按单件计费 |
| 其他成本 | 认证/工具/杂费 | 按实际情况 |
