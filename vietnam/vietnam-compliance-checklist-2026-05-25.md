### 🥇 第1名：胶原蛋白果冻（零食化保健）

| 项目 | 内容 |
|------|------|
| **产品类别** | Thực phẩm bảo vệ sức khỏe (TPBVSK) — 保健食品 |
| **产品形态** | 果冻条/零食化（非传统胶囊/片剂，但含功能性宣称仍归TPBVSK） |
| **主管机构** | Cục An toàn thực phẩm (VFA) — 越南食品管理局，隶属卫生部 |

#### 准入认证

| 项目 | 内容 |
|------|------|
| **认证名称** | Giấy tiếp nhận đăng ký bản công bố sản phẩm（产品登记准入证书） |
| **法律依据** | Decree 46/2026/NĐ-CP（替代Decree 15/2018）；Circular 43/2014/TT-BYT |
| **办理费用** | 政府规费：~2,000,000 - 5,000,000 VND（$80 - $200） |
| | 检测费用（稳定性+成分）：$800 - $2,000（视检测项数） |
| | 第三方服务费（代理注册）：$1,500 - $3,000 |
| | **合计预估：$2,500 - $5,200** |
| **办理周期** | 法定：21个工作日（VFA审核） |
| | 实际（含材料准备+补正）：**45 - 90天** |
| **证书有效期** | 无固定到期日，产品配方/生产厂/标签变更需重新注册 |
| | （Decree 15下已备案产品在12个月过渡期内有效） |

#### 关键材料要求

1. **Certificate of Free Sale (CFS)** — 原产国政府出具，需公证+领事认证
2. **GMP证书** — 生产厂必须持有有效的GMP认证（Decree 46强制要求）
3. **稳定性研究报告** — 长期+加速稳定性测试
4. **科学证据** — 支持所有功能宣称的文献/实验数据
5. **授权书** — 越南本地代表/进口商的正式授权
6. **产品检测报告** — 12个月内的第三方检测
7. **越南语标签草案** — 必须包含"本品非药品，不能替代药物治疗"声明

#### 风险评级：🔴 **高**

| 风险因素 | 说明 |
|---------|------|
| Decree 46新规 | 2026年4月刚生效，VFA审核标准尚在磨合期，可能延误 |
| GMP要求 | 中国ODM工厂需持有国际认可GMP证书，非所有工厂具备 |
| 果冻形态风险 | 零食化保健品可能被质疑是否属于TPBVSK，需提前与VFA确认分类 |
| 功能宣称审查 | 任何"美容/美白/抗衰老"宣称均需科学证据支撑，审查严格 |
| 标签翻译错误 | 越南语标签规格严格，翻译不当直接导致被拒 |

#### 🔄 替代方案（高风险应对）

| 方案 | 说明 | 成本影响 | 周期影响 |
|------|------|---------|---------|
| **方案A：找越南本地OEM** | 在越南找已持有GMP+TPBVSK注册资质的工厂代工，可在越南境内完成注册 | 多$0.5-1.0/包 | 缩短至30-45天 |
| **方案B：先以普通食品入市** | 不宣称保健功能，以"胶原蛋白果冻零食"名义按普通食品自申报（Decree 46简化路径），后续再升级为TPBVSK | 无需注册费，仅需自申报 | 15-21天 |
| **方案C：与越南经销商合作** | 越南进口商持已有注册编号，以贴牌/OEM形式使用其注册编号快速入市 | 分成15-25% | 30天 |

| 项目 | 内容 |
|------|------|

| **法律依据** | Decree 46/2026/NĐ-CP（食品自申报）；QCVN 12-1:2011/BYT（食品接触塑料）；Circular 34/2011/TT-BYT |
| **办理费用** | 食品自申报政府规费：~100,000 - 300,000 VND（$4 - $12） |
| | QCVN 12-1检测（迁移物、重金属、感官）：$300 - $800 |
| | 第三方服务费（代理申报）：$300 - $600 |
| | **合计预估：$604 - $1,412** |
| **办理周期** | 自申报：7-15工作日（提交后即时生效） |
| | QCVN检测送样：7-14天 |

1. **QCVN 12-1:2011/BYT** — 塑料食品接触材料检测（总迁移量、高锰酸钾消耗量、重金属、蒸发残渣等）
2. **食品自申报** — 提交至省级卫生厅（Sở Y tế）
3. **越南语标签** — 标注材质、使用温度、用途、制造商、进口商
4. **PPSU材质证明** — 需提供PPSU原料符合食品级标准的声明

#### 风险评级：🔴 **中偏高风险**

| 风险因素 | 说明 |
|---------|------|
| Decree 46新规 | 2026年4月起食品接触材料归MOH/VFA管理（此前归BCT），可能导致申报流程不明确 |
| PPSU测试复杂 | 欧盟标准EN 14350与越南QCVN 12-1测试标准不完全一致，可能需要重复测试 |
| 婴儿用品敏感 | 海关可能加强抽查，需提前准备好QCVN检测报告（不能仅靠供应商自证） |

| **办理费用** | 测试费（指定越南实验室）：$500 - $1,500（视测试项目） |
| | 型号认证费：$300 - $800 |
| | 工厂审查费（如需）：$500 - $1,000 |
| | 越南代理商/服务费：$500 - $1,000 |
| | CR Mark标签费：$50 - $100 |
| | **合计预估：$1,850 - $4,400** |
| **办理周期** | 样品测试：10-20工作日 |
| | 文件审核+工厂审查（如需）：10-15工作日 |
| | 证书签发：5-10工作日 |
| | **总周期：30 - 60天** |
| **证书有效期** | 3-5年（需周期性监督审核） |
| **特别注意** | ⚠️ **摩托车头盔是越南强制CR Mark产品，无认证不可合法进口和销售** |

#### 关键材料与要求

1. **QCVN头盔技术标准** — 冲击吸收、穿透抵抗、保持系统（系带）强度等
2. **ABS外壳+EPS缓冲层** — 材料必须符合QCVN的力学性能要求
3. **指定实验室测试** — 必须在STAMEQ认可的越南境内实验室测试
4. **工厂审核（Type 5方案）** — 批量生产需通过工厂质量体系审查
5. **越南语标签** — 制造商、型号、尺寸、检验标识、有效期等

#### 风险评级：🔴 **高**

| 风险因素 | 说明 |
|---------|------|
| **强制CR Mark** | 无CR Mark即无法清关，不可变通 |
| 越南指定实验室 | 必须送样到越南STAMEQ认可实验室，跨境送样+时差增加复杂度 |
| 工厂审核要求 | 中国工厂需接受越南认证机构的远程或现场审查 |
| 头型差异 | 亚洲头型（偏圆）vs 欧美头型（偏长），QCVN测试需使用越南标准头模 |
| 批量一致性 | 每一生产批次需保持一致，否则监督审核可能不通过 |
| 2025新交法 | Road Traffic Safety Law 36/2024（2025年1月生效）大幅提高无证头盔罚款 |

#### 🔄 替代方案（高风险应对）

| 方案 | 说明 | 成本影响 | 周期影响 |
|------|------|---------|---------|
| **方案A：找越南本地头盔OEM** | 与已有CR Mark认证的越南头盔厂合作贴牌，直接使用其认证编号 | 多$1-2/个（贴牌费） | 10-15天即可上市 |
| **方案B：通过越南进口商备案** | 越南进口商以自身名义申请CR Mark（使用你的产品做测试），你做ODM供应商 | 分成12-18% | 30-45天 |
| **方案C：中国预检+越南正式检** | 在中国按QCVN标准预检，合格后再正式送越南测，减少一次拒检风险 | +$300-500 | 45-60天 |
| **方案D（最推荐）：** | **方案A + 方案B组合** — 初期用越南OEM快速铺货，同期推进自有品牌CR Mark认证 | 初期多$1-2/个 | 长期最优 |

## 📋 合规检查清单（通用 — 适用所有SKU）

### 阶段一：产品准入检查 ⬜

- [ ] 确认产品HS编码（越南海关4位/6位/8位准确分类）
- [ ] 查询产品是否属于QCVN强制目录（CR Mark要求）
- [ ] 查询产品是否需要VFA/省级卫生厅备案或注册
- [ ] 查询产品是否需要特殊许可（如保健品、食品、化妆品）
- [ ] 确认进口商/代理人具备相应资质
- [ ] 确认中国供应商具备出口资质（出口食品备案等）

### 阶段二：文件准备 ⬜

- [ ] 商业发票（Commercial Invoice）— 越南语/英语
- [ ] 装箱单（Packing List）
- [ ] 提单（Bill of Lading / Airway Bill）
- [ ] 原产地证（C/O）— 享受FTA关税优惠时需要
- [ ] Certificate of Free Sale（保健品/食品适用）
- [ ] GMP证书（保健品适用）
- [ ] 产品检测报告（分品类准备：食品/保健品/食品接触材料）
- [ ] QCVN检测报告（CR Mark产品适用）
- [ ] 越南语标签样本（所有进口到越南的消费品均需）
- [ ] 使用说明书越南语版（电子产品/机械类需）
- [ ] 授权书/进口委托书

### 阶段三：产品与标签合规 ⬜

- [ ] 越南语标签包含以下要素：
  - [ ] 产品名称（越南语）
  - [ ] 制造商名称和地址
  - [ ] 进口商名称和地址
  - [ ] 原产地（"Sản xuất tại Trung Quốc"）
  - [ ] 成分/材质清单（越南语）
  - [ ] 净含量
  - [ ] 生产日期和保质期
  - [ ] 使用和储存说明
  - [ ] 警告标识（如适用）
- [ ] 功能性宣称有科学证据支持
- [ ] 标签中英文转越南语翻译准确（建议由越南母语者审核）
- [ ] 包装符合越南环境法规（Decree 08/2022限塑要求）

### 阶段四：进口清关 ⬜

- [ ] 与越南报关行/海关代理签订服务协议
- [ ] 通过越南贸易门户（vietnamtradeportal.gov.vn）确认关税税率
- [ ] 确认适用FTA：ACFTA（中国-东盟FTA，关税优惠）
- [ ] 准备Form E（中国-东盟FTA原产地证）以享受关税减免
- [ ] 确认VAT税率（标准10%，部分商品2025.7-2026.12期间降至8%）
- [ ] 了解进口食品检验分级（绿色/黄色/红色通道）
- [ ] 确保进口食品剩余保质期≥60%
- [ ] 预留海关查验缓冲时间（+3-7天）

### 阶段五：上市后合规 ⬜

- [ ] 建立越南语客户服务（Shopee/Tiki在线回复）
- [ ] 准备产品召回预案
- [ ] 监控差评中的合规投诉（成分安全、标签问题）
- [ ] 定期检查法规更新（Decree/Circular变更）
- [ ] 维护进口记录+产品检测报告存档（至少3年）
- [ ] 保健品/食品: 每半年检查VFA发布的新监管通知
- [ ] 建立月度合规审计制度
