> **评分方法**：
> - **市场潜力**依据 GMV 占比、增长趋势、搜索热度。
> - **竞争度**依据平台已有卖家数量、品牌集中度、进入壁垒。
> - **进入门槛**分为 **低**（仅需常规备案即可）、**中**（需特定认证或冷链）、**高**（需长链供应链、严格监管）。
> - **适合平台**基于平台用户画像与内容属性（TikTok Shop 适合短视频驱动、Shopee 适合大流量通路）。

## 2. 潜力‑竞争度四象限矩阵

## 3. 品类分级建议
| 级别 | 说明 | 包含品类 |
|------|------|----------|
| **S‑类** | 高潜力 + 低竞争，建议 **优先布局**，先行者优势显著 | 智能家居、健康与保健、环保可持续产品、订阅制盒装、绿色清洁用品、手工艺/本土创意、教育玩具/STEAM、男性护肤/个人护理（视竞争细分进一步评估） |
| **A‑类** | 高潜力 + 中/高竞争，需要 **品牌差异化、内容营销** 才能突破 | 消费电子、美妆护肤、时尚服装、食品饮料（健康类）、家居生活、母婴用品、宠物用品、户外/运动装备 |
| **B‑类** | 中潜力 + 低竞争，可做 **小规模实验** 或 **细分市场** 切入 | 手工艺/本土创意、绿色清洁用品、教育玩具（若资源受限） |
| **C‑类** | 低潜力或高竞争，**不建议投入** 或放在后期观察 | 汽车/电动车配件、游戏周边（竞争激烈且增速一般） |

## 4. 行动路线图（建议 6‑12 个月）
1. **Q1‑Q2**：快速落地 S‑类品类
   - 在 TikTok Shop 投入短视频种草（智能家居、订阅盒、环保包装）。
   - 与本地环保组织、KOL 合作提升可信度。
2. **Q2‑Q3**：同步推进 A‑类核心品类的品牌化运营
   - 设立本地化品牌旗舰店，针对消费电子、女性美妆执行 **官方授权+本土化客服**。
   - 开启 **Shopee 大促**（9.9、11.11）以及 **TikTok Shop 直播** 双渠道促销。
3. **Q3‑Q4**：评估 B‑类实验结果，决定是否扩大规模。
   - 通过小批量入库测试手工艺、绿色清洁的用户转化率。
4. **全年**：持续监测平台竞争格局（Shopee vs TikTok Shop），并关注 **RCEP 关税** 带来的成本变化。
