# 市场分析简报：越南跨境电商宏观环境

**报告日期**：2026年6月7日  
**分析员**：趋势研究员  
**信心等级**：高（多数据源交叉验证）

---

## 1. 摘要

越南是东南亚增长最快的电商市场之一，2025年四大主流平台GMV达约165亿美元（约429万亿越南盾），同比增长约35%。市场正从"野蛮增长"进入"精耕细作"阶段——Shopee（56%）与TikTok Shop（41%）形成双头垄断格局，合计占97%份额。2026年7月1日《电商法》正式生效，合规门槛显著提升，对中国跨境卖家既是挑战（税收/海关收紧）也是机遇（劣质卖家出清，品牌化卖家受益）。

## 4. 平台格局：从"四强争霸"到"双头垄断"

### 4.1 市场份额（2025年全年，Metric数据）

| 平台 | 市场份额 | 趋势 | 定位 |
|------|---------|------|------|
| **Shopee** | ~56% | 稳中有降 | 搜索型购物、信任交易、物流网络成熟 |
| **TikTok Shop** | ~41% | 快速上升（从2024年29%→41%） | 社交电商/娱乐购物（Shoppertainment） |
| **Lazada** | ~2% | 持续萎缩 | 正品跨境差异化（与天猫/Gmarket合作） |
| **Tiki** | ~1% | 持续萎缩 | 本土玩家，份额被严重挤压 |

### 4.2 关键趋势

- **双头垄断已成定局**：Shopee + TikTok Shop 合计97%市场份额，Lazada和Tiki合计仅3%
- **TikTok Shop增速惊人**：一年内份额从29%飙升至41%，3个季度内提升约6个百分点
- **Mall店效应显著**：仅占店铺数2.64%的官方旗舰店贡献了35.7%的销售额
- **活跃卖家数下降但总GMV增长**：2025年H1超7,000家店铺退出，但GMV仍增长23%——劣质卖家被出清
- **直播/短视频是核心增长引擎**：TikTok Shop GMV同比增长超过140%（2025年）

### 4.3 平台策略分化

- **Shopee**：优化物流分区定价（SLS），精细化最后一公里成本管理
- **TikTok Shop**：2026年4月起提高联盟卖家准入门槛，从"流量驱动"转向"质量驱动"
- **Lazada**：聚焦跨境差异化，整合7,000+国际商家（天猫、Gmarket），配送缩至~7天，推出30天退货政策

## 5. 政策与监管环境

### 5.1 里程碑：《电子商务法》（Law on E-Commerce No. 122/2025/QH15）

| 事项 | 内容 |
|------|------|
| 通过时间 | 2025年12月10日 |
| 生效时间 | **2026年7月1日** |
| 替代法规 | 取代Decree 52/2013/ND-CP |
| 性质 | 全面、统一的电商法律框架 |

### 5.2 核心变化

1. **税收合规全面收紧**
   - 取消"定额税"（lump-sum tax），转向申报制纳税
   - 卖家须保留完整账簿、为每笔交易开电子发票
   - 平台须实时共享卖家数据给税务机关
   - 超过阈值（约5亿越南盾）的平台可代扣代缴或标记审计

2. **跨境商品免税门槛取消**
   - 低价值进口商品增值税豁免将被严格取消
   - 海关审查加强，对跨境卖家查验更严

3. **平台责任扩大**
   - 平台须对商品来源真实性负责
   - 严厉打击假货、空包、欺诈投诉
   - 直播带货纳入监管范围

4. **跨境电商专章**
   - 新法设专章规范涉外电商活动
   - 跨境电商进出口的法律框架首次完整建立

### 5.3 政策走向判断

越南电商正在从"放任式增长"转向"规范监管+出口激励"的双轨模式。短期（2026 H2）合规成本显著上升，但中长期有利于规范经营的跨境卖家。

## 7. 机会与威胁分析

### 机会

| 机会 | 说明 | 时间窗口 |
|------|------|---------|
| **社交电商增量** | TikTok Shop占41%仍在增长，直播带货尚未到天花板 | 6-12个月 |
| **合规出清红利** | 新法淘汰大量不合规小卖家，为品牌化卖家腾出空间 | 12-18个月 |
| **品类结构性增长** | 健康、童装、家居增速>40%；美妆个护为第一大品类 | 持续 |
| **跨境进口需求** | Lazada整合天猫/Gmarket，跨境消费信心提升 | 12个月 |
| **数字支付升级** | 移动钱包>3500万用户，COD比例下降中 | 6-12个月 |

### 威胁

| 威胁 | 说明 | 紧迫度 |
|------|------|--------|
| **合规成本飙升** | 2026年7月新法生效，税收/海关/平台监管全面收紧 | ⚠️ 1个月内 |
| **双头垄断议价能力** | Shopee+TikTok占97%，卖家无替代渠道，广告成本上升 | 持续 |
| **低价跨境模式终结** | 免税门槛取消、代扣代缴，低价小包模式利润被压缩 | 1-3个月 |
| **劣质卖家出清潮** | 虽然利好规范卖家，但短期可能伴随价格战和流量波动 | 3-6个月 |
| **基础设施压力** | 包裹量年增30%，物流履约能力面临考验 | 6-12个月 |

## 8. 对中国跨境卖家的建议

### 短期行动（现在 - 2026年9月）

1. **合规优先**：立即梳理越南税务和海关合规要求，确保7月1日前完成注册
2. **调整定价策略**：将增值税和合规成本纳入定价模型
3. **评估平台选择**：Shopee覆盖广泛用户、TikTok Shop社交增量大、Lazada跨境差异化
4. **关注品类机会**：美妆个护（第一大品类）、健康保健（增速>40%）、家居用品

### 中期策略（2026年Q4 - 2027年）

1. **品牌化路线**：建立品牌旗舰店（Mall店），利用合规出清窗口建立品牌认知
2. **布局海外仓**：实现2-5天本地配送，提升转化率和平台评分
3. **内容电商能力**：培养直播/短视频运营能力，TikTok Shop仍是最大增量来源
4. **供应链优化**：降低单品成本以应对税负增加和广告成本上升

- 新电商法实施细则和执行力度仍有变数
- TikTok Shop在美国的监管风险可能间接影响越南市场策略
- 全球经济下行风险可能影响越南出口和消费信心
