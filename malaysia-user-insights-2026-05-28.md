## 一、用户画像

### 1.1 核心用户群

| 维度 | 画像 |
|------|------|
| **年龄** | 18-40 岁（主力 25-35 岁） |
| **性别** | 女性 65% / 男性 35% |
| **地理位置** | 西马 Klang Valley > Penang > Johor Bahru > 其他 |
| **收入水平** | 中等收入（月均 RM 3,000-8,000） |
| **设备** | 移动端 90%+ |
| **语言** | 马来语（主要）+ 英语 + 中文 |
| **平台偏好** | Shopee > TikTok Shop > Lazada |
| **支付方式** | 电子钱包 > 银行卡 > BNPL |

### 1.2 消费心理特征

| 特征 | 描述 |
|------|------|
| **价格敏感** | 高度价格敏感，喜欢优惠券和免邮 |
| **社交驱动** | TikTok/IG 影响购买决策 |
| **品质觉醒** | 越来越重视品质而非纯粹低价 |
| **清真意识** | 穆斯林消费者对清真认证要求高 |
| **品牌意识上升** | 从纯低价转向品牌信任 |
