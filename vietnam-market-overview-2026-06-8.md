> Shopee + TikTok Shop 合计占 **~97%** 市场份额，双寡头格局已形成。

### 3.2 各平台深度分析

#### 🟠 Shopee：守擂者

- **市场地位**：越南电商绝对龙头，56%份额
- **增长态势**：增速显著放缓至+16% YoY（2025 H1），面临TikTok严重侵蚀
- **核心优势**：物流网络最广（Shopee Express）、用户基数最大、购物习惯最成熟
- **新动态**：品牌商店（Mall）贡献了Shopee + TikTok Shop总营收的**28.7%**，但仅占总店铺数的3.4%
- **跨境卖家意义**：流量基本盘，但增速放缓、竞争激烈

#### 🔵 TikTok Shop：颠覆者

- **市场地位**：第二大平台，份额从29%飙升至39-41%
- **增长态势**：**+69% YoY**（2025全年），H1 2025更是**+148% YoY**
- **核心优势**："Shoppertainment"（购物+娱乐）模式，直播带货转化率极高
- **直播电商**：越南近**50%消费者**通过直播购物，63%的在线购物者参与直播购物
- **直播电商市场规模**：预计2026年达**$110亿**
- **跨境卖家意义**：**增长最快的渠道**，但需要短视频/直播内容能力

#### 🟢 Lazada & Tiki：边缘化

- **Lazada**：销售下降48%，持续萎缩
- **Tiki**：销售下降63%，从曾经的越南第二大平台滑落至边缘位置
- **Sendo**：已转型为Sendo Farm，专注农产品
- **启示**：越南市场已进入**双寡头时代**，中小平台生存空间极小

## 四、跨境进口政策与关税环境

### 4.1 关税政策

| 维度 | 详情 |
|------|------|
| **一般进口关税** | 依据2025年关税表（Circular No.31/2022/TT-BTC及后续修订） |
| **优惠关税** | 越南签署18个优惠贸易协定（CPTPP、RCEP、EVFTA等） |
| **跨境电商进口免税额度** | 单次交易免税额VND 100万（约$40），年度免税额VND 200万（约$80） |
| **跨境电商进口关税** | 一般适用行邮税，税率根据品类10%-70%不等 |
| **增值税（VAT）** | 标准税率10%（部分品类5%或0%） |
| **特别消费税（SCT）** | 针对烟酒、汽车等特定品类 |

### 4.2 跨境电商监管新政（2025关键变化）

| 政策 | 生效时间 | 核心内容 |
|------|---------|---------|
| **电商法草案** | 2025年10月提交审议 | 要求在越销售的外国企业获得工贸部许可、设立代表处或指定本地法定代表 |
| **外国商品标准** | 新规要求 | 进口商品须符合越南产品标准、技术法规和安全要求 |
| **平台税务扣缴令** | **2025年7月1日** | 国内外电商平台须为个人卖家代扣代缴VAT和个人所得税 |
| **卖家实名制** | 持续推进 | 卖家须提供准确身份信息，平台须核实并配合税务稽查 |
| **跨境电商海关程序法令草案** | 拟订中 | 规范跨境电商商品的报关、风险管理、完税价格认定等程序 |
| **CPTPP优惠关税修订** | 2025年1月21日 | 对原产于特定国家的商品增加条件限制 |

### 4.4 关键判断

越南跨境电商监管正在快速收紧，2025年是**政策关键年**。三大趋势：

1. **平台税务代扣代缴**（7月已生效）：跨境卖家需关注合规成本上升
2. **电商法即将出台**：外国卖家可能需要在越设立实体或指定代表
3. **进口标准趋严**：商品须符合越南技术标准和原产地规则

对于中国卖家，短期合规压力大，但**中国连续多年是越南最大贸易伙伴**，供应链优势+地理位置优势（陆路可达）仍是核心竞争力。建议提前做好税务合规和本地代表安排。

4. **跨境购物热情高**：进口商品2024年增长43%，显示越南消费者对海外商品有强烈需求

## 七、基础设施（物流、支付、仓储）

### 7.1 物流基础设施

| 指标 | 数据/状态 |
|------|----------|
| **物流市场地位** | 全球增长最快的物流市场之一 |
| **国家战略** | 2025年10月政府批准《2025-2035物流服务业发展战略》 |
| **物流成本** | 占GDP约16-18%（持续下降中） |
| **主要物流商** | Shopee Express、Giao Hang Nhanh(GHN)、Giao Hang Tiet Kiem(GHTK)、J&T Express、Ninja Van、Viettel Post |
| **仓储** | 第三方仓储快速发展，各大平台均在越南建立分仓（Shopee在胡志明市和河内均有大型仓） |
| **配送时效** | 主要城市（胡志明/河内）通常1-3天可达，偏远地区3-7天 |
| **地理优势** | 与中国接壤，陆路跨境物流成本远低于海运/空运 |
| **挑战** | 最后一公里配送在偏远农村仍困难；退货逆向物流尚不成熟 |
| **Shopee物流** | 自建Shopee Express，是越南最大的电商物流网络之一 |
| **J&T Express** | 快速增长，Shopee生态深度绑定 |

**物流市场关键发展**：
- 越南政府将物流列为国家重点发展领域
- 2025-2035战略明确：降低物流成本至GDP的12-14%
- 国内外玩家加速布局（DHL、FedEx、顺丰等均在越南有业务）
- 跨境陆路物流因中越边境口岸升级而效率提升

| **平台自营仓** | Shopee/TikTok Shop/Lazada均在胡志明和河内设有分仓 |
| **第三方仓储** | 快速发展，如Lazada的FBL仓、第三方云仓等 |
| **保税仓** | 越南正在推进跨境电商保税仓模式 |
| **中国边境仓** | 广西/云南口岸边境仓为跨境卖家提供前置备货 |
| **建议** | 中国卖家可利用中越陆运，从中国边境仓直发越南，物流成本最低 |

### 7.4 关键判断

越南基础设施处于**快速改善期**但仍存在明显短板：

- **物流**：主要城市已较好（1-3天），但偏远地区仍慢。与中国接壤是**独特优势**——陆运跨境成本远低于海运
- **支付**：数字钱包渗透率高（MoMo 3,100万+用户），COD占比下降中。跨境卖家需接入MoMo + ZaloPay + VNPay三大主流支付方式
- **仓储**：平台仓+第三方仓+边境仓形成三层体系。建议中国卖家考虑**中越边境仓前置备货**模式
