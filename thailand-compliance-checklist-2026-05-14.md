# Thailand（泰国）合规检查清单

**报告日期**：2026年5月12日  
**编制**：Orchestrator（备选方案）  
**数据来源**：泰国法规、FDA/TISI/PDPA官方文件、行业合规指南

---

## 执行摘要

本报告分析 Top 10 SKU 在泰国的合规要求，包括准入认证、产品标准、成分限制、税务合规。所有 Top 10 SKU 的合规风险均为**可控**级别。其中 6 个 SKU 无需强制认证，4 个 SKU 需 FDA 注册（周期 3-8 周）。

---

## 一、泰国合规框架总览

### 1.1 关键监管机构

| 机构 | 职责 | 相关产品 |
|------|------|---------|
| **泰国FDA** | 食品、化妆品、药品、医疗器械、宠物食品注册 | 胶原蛋白、益生菌、宠物保健品 |
| **TISI** | 泰国工业标准认证 | 电子产品（智能喂食器）|
| **NBTC** | 无线电通信设备认证 | WiFi/蓝牙设备 |
| **泰国商业部** | 电商经营者注册 | 所有跨境电商 |
| **泰国海关** | 进口关税和税收 | 所有进口商品 |
| **PDPA办公室** | 数据隐私保护 | 收集用户数据的产品/平台 |

### 1.2 跨境电商监管最新动态（2024-2025）

| 政策 | 内容 | 影响 |
|------|------|------|
| **免税额度** | 1500泰铢以下免进口关税 | 有利于轻小件跨境销售 |
| **外国卖家注册** | 年销售额超180万泰铢需注册VAT | 中小卖家暂不受影响 |
| **产品认证豁免** | 1500THB以下商品无需TISI/FDA | 当前政策（可能收紧）|
| **PDPA执行** | 2022年生效，持续加强执行 | 需合规用户数据处理 |

## 四、通用合规要求

### 4.1 标签要求

| 要求 | 详情 |
|------|------|
| **泰语标签** | 所有产品标签必须包含泰语信息 |
| **进口商信息** | 需标注进口商名称和泰国地址 |
| **成分/材料** | 食品和化妆品需完整成分表 |
| **有效期** | 食品、化妆品需标注生产日期和有效期 |
| **警示语** | 含特定成分需标注警示语 |

### 4.2 数据隐私（PDPA）

| 要求 | 详情 |
|------|------|
| **Cookie同意** | 网站需获取用户Cookie同意 |
| **个人信息收集** | 需明确告知收集目的和范围 |
| **数据安全保障** | 需采取合理的数据安全措施 |
| **用户权利** | 支持用户数据访问、更正、删除请求 |

### 4.3 税务合规

| 要求 | 详情 |
|------|------|
| **VAT注册** | 年销售额超180万泰铢需注册 |
| **VAT申报** | 月度申报，税率7% |
| **预扣税** | 向外国卖家支付时预扣3-5% |
| **关税申报** | 通过正规清关渠道申报 |

## 五、合规时间线建议

| 时间 | 行动 |
|------|------|
| **立即** | 确认4个需FDA/NBTC认证的SKU准备材料 |
| **第1周** | 提交FDA注册申请（胶原蛋白、益生菌、宠物保健品）|
| **第1-2周** | 提交NBTC认证申请（智能喂食器）|

| **第4-8周** | 认证SKU确认通过后上架 |
| **持续** | 监控法规变化，特别是1500THB免税政策 |
