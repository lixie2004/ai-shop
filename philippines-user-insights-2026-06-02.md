## 用户画像详细

### 画像 1：Gen Z 美妆探索者（核心画像）

```
┌─────────────────────────────────┐
│  Gen Z 美妆探索者               │
│  年龄：18-25岁                   │
│  性别：女性为主（80%）           │
│  地点：大马尼拉、宿务            │
│  职业：大学生/职场新人           │
│  月消费力：PHP 2,000-4,000       │
│  平台：TikTok Shop > Shopee      │
│  购物频率：每月 3-5 次           │
└─────────────────────────────────┘
```

**行为特征**：
- 每天刷 TikTok 3+ 小时
- 跟随 K-Beauty 趋势
- 购买前会看至少 3 个视频测评
- 价格敏感但愿意为品质买单
- 冲动消费 + 大促囤货

**需求优先级**：
1. 适合湿热天气的控油产品
2. 高性价比 K-Beauty 套装
3. 正品保障
4. 直播互动体验
5. 社交分享价值

**内容偏好**：
- 30秒护肤教程
- Before/After 对比
- "Get ready with me" 形式
- 成分解析（简短版）

```

└─────────────────────────────────┘
```

**行为特征**：
- 关注健康趋势

- 偏好国际品牌或认证品牌

- 更信任 LazMall/Shopee Mall

**需求优先级**：
1. 科学验证的健康补充剂
2. 方便的复购机制
3. 透明成分和认证
4. 套装/组合优惠
5. 使用指导和跟踪

**内容偏好**：
- 详细的成分说明
- 用户见证（视频）
- 医生/专家推荐
- 对比分析

## 差评关键词分析（跨品类）

### 高频差评词

| 差评词 | 出现品类 | 频次 | 改进机会 |
|--------|---------|------|---------|
| "cheap quality" / "mababa ang kalidad" | 手机配件、宠物用品 | 极高 | 品质控制+选品标准 |
| "not original" / "fake" | 美妆个护 | 高 | 品牌认证+防伪标签 |
| "didn't work" / "walang epekto" | 健康补充剂 | 中高 | 效果预期管理 |
| "slow delivery" | 全品类 | 中高 | 本地仓+物流优化 |
| "wrong item received" | 全品类 | 中 | 质检+包装改善 |
| "too small" / "different from picture" | 手机配件、宠物用品 | 中 | 准确展示+尺寸说明 |
