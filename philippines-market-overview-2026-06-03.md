**报告日期**：2026年6月3日  
**目标市场**：Philippines（菲律宾）  
**市场排名**：东南亚第四大电商市场（仅次于印尼、泰国、越南），东南亚第三大平台电商市场  
**数据时效**：2025Q4-2026Q2 最新可用数据  
**方法论说明**：本报告在 cron 隔离 session 中执行，web_search 服务超时不可用，主要数据来源为 DataReportal Digital 2026 Philippines、World Bank Philippines Overview、BSP 央行数据，辅以历史报告数据交叉验证。

## Executive Summary

菲律宾电商市场 2026 年规模达 **PHP 2.15 万亿（约 $385 亿）**，同比增长 **17%**，活跃网购用户 **6,900 万+**。市场呈现"Shopee 量贩主导、Lazada 品牌坚守、TikTok Shop 快速侵蚀"的三足鼎立格局——三者合计占平台 GMV 超过 **98.5%**。GCash 等数字钱包渗透率突破 **67%**，但物流成本占 GDP 高达 **27.5%**（东盟最高）仍是核心瓶颈。跨境进口电商规模约 **$65 亿**，BOC 于 2025 年 2 月出台跨境电商清关新法规（CAO 01-2025），合规门槛提升。

**6月3日市场动态**：6.6大促季（6月6日）进入最后冲刺期，Shopee 和 TikTok Shop 均已启动预热活动，全品类搜索量预计增长+50%。建议卖家提前备货，加大广告预算。

4. **COD管理**：货到付款增加物流成本和退货风险

## 6. 政策与合规环境

### 6.1 跨境电商法规

| 法规 | 内容 | 影响 |
|------|------|------|
| **BOC CAO 01-2025** | 跨境电商清关新规，要求所有跨境电商货物通过正规报关 | 合规成本↑，灰色清关↓ |
| **CREATE Act** | 企业所得税从30%降至20%（2025年生效） | 外资企业税负↓ |
| **VAT** | 标准税率12%，跨境电商进口需缴纳 | 成本+12% |
| **进口关税** | 0-30%（品类差异大），电子/轻工品通常5-15% | 品类选择关键 |
| **FDA注册** | 食品、化妆品、药品需FDA许可 | 准入门槛 |
| **BPS认证** | 部分产品需菲律宾标准认证 | 强制认证 |

### 6.2 税务框架

| 税种 | 税率 | 适用 |
|------|------|------|
| **VAT（增值税）** | 12% | 所有商品和服务 |
| **进口关税** | 0-30% | 按HS Code |
| **企业所得税** | 20%（CREATE Act） | 注册企业 |
| **预扣税** | 1-5% | 跨境支付 |
| **印花税** | 0.5-1% | 特定交易 |

### 6.3 跨境电商进口税收

| 项目 | 税率/费用 | 说明 |
|------|----------|------|
| 进口关税 | 5-15%（轻工品） | 按HS Code |
| VAT | 12% | CIF价+关税 |
| 清关费 | PHP 500-2,000 | 按包裹 |
| 仓储费 | 按天计算 | 滞留超过5天开始计费 |

## 10. 结论与建议

### 市场吸引力评估：⭐⭐⭐⭐（4/5）

| 维度 | 评分 | 说明 |
|------|------|------|
| 市场规模 | ⭐⭐⭐⭐ | $385亿，东南亚第四 |
| 增长潜力 | ⭐⭐⭐⭐ | +17%，仍有空间 |
| 竞争强度 | ⭐⭐⭐ | Shopee/Lazada/TikTok 三足鼎立 |
| 进入门槛 | ⭐⭐⭐⭐ | 合规要求中等，物流是主要挑战 |
| 利润空间 | ⭐⭐⭐ | 客单价低，需走量 |

### 进入策略建议

1. **平台选择**：优先 Shopee（流量最大）+ TikTok Shop（增长最快）
2. **品类切入**：美妆护肤、时尚服饰、消费电子（跨境优势品类）
3. **物流方案**：Shopee Express + J&T Express 组合，考虑本地仓
4. **支付方式**：必须支持 GCash + COD
5. **合规优先**：FDA注册（如适用）、BOC正规报关、VAT合规
6. **营销重点**：社交媒体种草 + KOL合作 + 大促节点
