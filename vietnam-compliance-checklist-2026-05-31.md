## 一、越南进口合规框架速览

| 品类 | 主管机构 | 核心要求 | 参考法规 |
|------|---------|---------|---------|
| 美妆/化妆品 | 越南卫生部 (MOH) | 产品备案通知 (Thông báo) | Decree 15/2018/ND-CP |
| 食品/保健品 | 越南卫生部 (MOH) / 食品局 (VFA) | 进口许可证 + 卫生安全证书 | Law on Food Safety 2010, Decree 15/2018 |
| 宠物食品 | 越南畜牧业局 (DARD) | 动物检疫 + 产品注册 | Law on Animal Husbandry |
| 玩具 | 越南标准计量质量局 (STAMEQ) | QCVN 3:2019/BKHCN 安全标准 | Circular 15/2018/TT-BKHCN |
| 家居/收纳 | 无强制认证 | 自愿性 TCVN 标准 | — |
| 保健品 | 越南卫生部 (MOH) | 保健品注册证书 (Giấy công nhận) | Decree 15/2018/ND-CP |

### SKU #5: 防晒霜 SPF50+ (Sunscreen SPF50+)

| 维度 | 详情 |
|------|------|
| **品类** | 化妆品（防晒类） |
| **准入认证** | **化妆品备案通知 (Thông báo sản phẩm mỹ phẩm)** — 强制 |
| **产品标准** | 符合 QCVN 7:2016/BYT（化妆品安全标准） |
| **成分限制** | 防晒剂需符合越南允许使用的防晒剂清单（Annex 2 of Decree 15/2018）；禁用成分包括汞、铅、氢醌等 |
| **税务合规** | 进口关税：约 10-15%（HS 3304.30 美容化妆品）；增值税 10% |
| **认证名称** | 化妆品备案通知 (Thông báo sản phẩm mỹ phẩm) |
| **认证费用** | $500-2,000/款（含检测费、备案代理费） |
| **认证周期** | **3-9 个月**（检测 2-4 月 + 备案审核 1-5 月） |
| **风险评级** | 🟡 **中** |

**合规检查清单**：
- [ ] **备案前**：准备产品配方表（含所有成分及含量）
- [ ] **备案前**：选择越南认可实验室进行安全检测（微生物、重金属、防晒剂含量）
- [ ] **备案中**：通过越南本地代理商/持有人提交备案申请
- [ ] **备案后**：获取备案通知编号，标注于产品包装
- [ ] **包装**：越南语标签（品名、成分全表、使用方法、保质期、生产商信息、备案编号）
- [ ] **进口**：每次进口需申报化妆品备案编号
- [ ] **成分**：确保防晒剂在越南允许清单内（如 Avobenzone、Octocrylene 等需逐一核对）
- [ ] **广告**：不得宣称医疗功效（如"治疗晒伤"等）

**替代方案**：
| 方案 | 描述 | 适用场景 |
|------|------|---------|
| **方案 A** | 先以"护肤品"（非防晒宣称）类目测试市场，同步启动防晒备案 | 想快速验证市场 |
| **方案 B** | 选择已有越南备案的韩国/日本品牌，以分销/授权方式进入 | 不想承担备案成本和时间 |
| **方案 C** | 与越南本地化妆品工厂合作 OEM，利用其已有备案资质 | 长期品牌化策略 |

| **产品标准** | 符合越南宠物食品标准（TCVN 11815 参考） |
| **成分限制** | 禁止使用违禁药物残留、重金属超标、黄曲霉毒素超标 |
| **税务合规** | 进口关税：约 5-10%（HS 2309.90 宠物食品）；增值税 10% |

| **认证周期** | **8-16 周**（检疫审批 4-8 周 + 每批次检疫 1-2 周） |
| **风险评级** | 🟡 **中** |

**合规检查清单**：
- [ ] **出口前**：中国供应商需获得越南认可的出口资质
- [ ] **出口前**：每批次提供兽医卫生证书（Veterinary Health Certificate）
- [ ] **进口时**：申请动物检疫审批（Giấy phép nhập khẩu động vật）

- [ ] **包装**：越南语标签（品名、成分、保质期、储存条件、生产商、进口商）
- [ ] **成分**：确保不含越南禁用的动物源性成分（如某些肉类来源）
- [ ] **检测**：每批次提供黄曲霉毒素、重金属检测报告
- [ ] **注册**：产品配方需在越南畜牧业局备案

**替代方案**：
| 方案 | 描述 | 适用场景 |
|------|------|---------|
| **方案 A** | 选择越南本地宠物食品品牌代工，以自有品牌销售 | 降低进口检疫门槛 |
| **方案 B** | 先以"宠物零食"（非冻干）类目测试，冻干品类后续追加 | 降低首批合规复杂度 |
| **方案 C** | 与越南进口商合作，由其承担检疫和注册责任 | 轻资产进入 |
