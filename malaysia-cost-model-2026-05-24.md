## 一、税务与费用基准假设

| 成本类别 | 假设依据 |
|----------|---------|
| **进口关税** | 化妆品/护肤品 (HS 3304): 0-10%，取均值 5%；食品/保健品 (HS 2106/1905): 0-8%，取均值 5% |
| **SST 销售税 (进口环节)** | 化妆品/非必需消费品: 10%；加工食品: 5% 或 10%（视分类） |
| **SST 服务税 (平台佣金)** | 平台佣金需额外加 8% SST |
| **Shopee 平台费用** | 交易费 2.12%（含 SST）+ 佣金 2.12%（≥100单卖家）+ 5% 技术支持费（2026.2 起）+ 可选营销项目 3-7% |
| **Lazada 平台费用** | Marketplace 佣金 3-6% + 订单处理费 5% + 8% SST 附加于佣金 |
| **广告费用 (ACoS)** | Shopee/Lazada 站内广告: 占 GMV 8-15%（新品期取 12%） |
| **物流费用 (最后一公里)** | 西马 RM5-8/单，东马 RM8-15/单（取西马均值 RM6/单） |
| **仓储/拣货/打包** | 第三方仓 RM3-5/单（含包材） |
| **支付通道费** | 已含于平台交易费中 |
| **Halal 认证摊销** | JAKIM 认证费用 + 年审，按 12 月摊销 |
| **NPRA 注册摊销** | 化妆品/保健品注册费，按 24 月摊销 |
| **退货/售后损耗** | 电商行业均值 3-5%，取 4% |
| **汇兑/跨境采购损耗** | 中国进口按 USD/MYR 汇率 + 1% 汇损 |

## 四、税务优化策略总结

### 4.1 进口环节优化

| 策略 | 适用 SKU | 预估节税 |
|------|---------|---------|
| 本地 OEM/灌装（原料进口 vs 成品进口） | 1, 4, 5, 8 | SST 节省 3-5% 货值 |
| 利用 ASEAN 原产地证 (ATIGA Form D) 零关税 | 东盟采购的原料 | 关税 0%（vs 5%） |
| 本地采购原料（Tongkat Ali, Kacip Fatimah） | 5, 10 | 免关税 + 简化 SST |
| 申请制造业免税（原料/包装材料 SST 豁免） | 1, 4, 7（若本地设厂） | SST 全额豁免 |

### 4.2 平台费用优化

| 策略 | 节费幅度 | 说明 |
|------|---------|------|
| Shopee Mall/LazMall vs Marketplace | 视品类 | Mall 佣金率更高但转化率 +20-30% |
| 仅参加必需营销项目 | 省 3-7% | "可选"项目对排名影响需权衡 |
| 自建官网 DTC (Shopify/SiteGiant) | 省 8-12% | 支付通道费 2-3% 替代平台总费率 12%+ |
| 使用平台 FBL 物流补贴 | 物流费减 20-30% | 满足时效要求可享补贴运费 |

### 4.3 合规注意事项

- **SST 注册门槛**: 年营业额 > RM500,000 必须注册 SST，每月/每两月申报
- **NPRA 合规**: 护肤品/保健品必须在 NPRA 备案/注册，违规销售可罚款最高 RM50,000
- **JAKIM Halal 认证**: 面向穆斯林消费者的食品/化妆品/保健品强制要求
- **转让定价文档**: 若涉及跨境关联交易（如 IP 授权费），需准备同期文档
- **电商税务**: 平台代扣代缴趋势加强，建议咨询本地税务顾问确认申报义务
