## 重点品类深度分析

### 🏆 Tier 1：优先进入（综合评分 ≥ 7.0）

#### 1. 美妆个护（综合 7.6）
| 维度 | 分析 |
|------|------|
| 市场规模 | 占泰国电商 **43.2%**，第二大品类 |
| 增速 | 年增约 15-20%，TikTok直播驱动显著 |
| 热门子品类 | 精华液、面霜、防晒霜、唇膏、香水、身体乳 |
| 价格带 | THB 99-599（约US$3-17）最畅销 |
| 竞争格局 | 韩系/日系品牌主导，中国新锐品牌快速崛起 |
| 跨境机会 | ✅ 中国美妆供应链优势明显，价格竞争力强 |
| 关键风险 | 需泰国 FDA 认证，成分限制严格 |
| 建议策略 | 主打性价比+韩系风格包装，TikTok种草+Shopee转化 |

## 关键发现

1. **宠物用品是最大蓝海**：竞争度仅5/10，跨境适配度9/10，复购率极高
2. **美妆个护是利润王者**：43.2%市场占比+高利润率+TikTok直播红利
3. **家居收纳被低估**：低竞争+低壁垒+稳定需求，适合新手卖家
4. **健康保健增速最快**：20-25%年增，但需注意FDA认证周期
5. **食品饮料门槛最高**：FDA+物流双重壁垒，建议后期进入
