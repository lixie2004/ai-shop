# 马来西亚跨境电商宏观环境分析简报

**报告日期**：2026年5月18日  
**分析师**：投资研究员  
**覆盖范围**：马来西亚电商市场规模、消费力、平台格局、跨境政策、物流基础设施、消费者行为、支付生态  
**数据来源**：Mordor Intelligence、DataReportal (Digital 2024: Malaysia)、U.S. Trade.gov、Reuters Institute、马来西亚央行(BNM)、MDEC、PayNet、Opensignal、MCMC

---

## 执行摘要

马来西亚是东南亚最具吸引力的跨境电商目标市场之一。2025年电商市场规模预计达 **121.8亿美元**（约567亿令吉），2024年实际GMV约为 **106.2亿美元**（约494亿令吉），年增速约 **14.7%**。互联网渗透率高达 **97.4%**，智能手机驱动的交易占比 **72.67%**，数字钱包支付占比已达 **34.92%**。Shopee、Lazada 和 TikTok Shop 三足鼎立，TikTok Shop 在经历2024年短暂监管暂停后强势回归。跨境进口政策方面，马来西亚对低值进口商品免征关税（≤500令吉），但自2024年1月起对跨境电商征收 **10% 销售税（SST）**。物流基础设施在马来半岛较为成熟，但东马（沙巴、砂拉越）仍面临较高的末端配送成本。

## 一、电商市场规模（GMV）及增速

### 市场规模

| 指标 | 数值 | 说明 |
|------|------|------|
| 2024年GMV | ~106.2亿美元（~494亿令吉） | 基于Mordor Intelligence模型推算 |
| 2025年GMV（预测） | 121.8亿美元（~567亿令吉） | Mordor Intelligence, 2026年1月更新 |
| 2031年GMV（预测） | 231.1亿美元 | CAGR 13.67%（2026-2031） |
| 2025年政府GMV目标 | 1,650亿令吉（~355亿令吉增量） | MyDIGITAL蓝图目标，实际进度有差距 |

### 增长驱动因素

- **智能手机渗透率超90%**，5G覆盖率达82.4%（2025年Q3，Opensignal数据），移动端交易占比72.67%
- **数字钱包快速普及**：2024年电子钱包交易额达311亿令吉（约72亿美元），占在线支付34.92%
- **直播电商爆发**：TikTok Shop直播GMV占比达平台总GMV的22%，转化率是传统搜索的3-5倍
- **政府政策推动**：国家电商战略路线图2.0（NESR 2.0）和MyDIGITAL蓝图降低了110万家中小企业触网门槛
- **B2B电商加速**：2025年7月强制电子发票政策推动企业采购线上化，B2B板块CAGR预计15.63%

### 品类结构

| 品类 | 2025年收入份额 | 增速 |
|------|-------------|------|
| 时尚服饰 | 26.67% | 稳定 |
| 食品饮料 | ~15% | CAGR 16.46%（最快） |
| 消费电子 | ~18% | 稳定 |
| 美妆个护 | ~12% | 快速增长 |
| 家居生活 | ~10% | 稳步增长 |

### 增长制约因素

- **城市市场趋于饱和**：巴生谷、槟城、柔佛新山的电商渗透率已超85%，获客成本（CAC）同比上升23%
- **网络诈骗**：2024年MCMC记录超7万起网络诈骗案，损失超18亿令吉，购物车放弃率达68%
- **东马物流瓶颈**：沙巴、砂拉越的空运成本居高不下，制约了B2B电商的全国性扩张

### 消费特征

- **中等收入群体扩大**：马来西亚约40%家庭属于中等收入群体（M40），月收入4,850-10,959令吉，是跨境电商的核心消费人群
- **年轻人口红利**：18-34岁人口占比28.5%，数字化程度高，对跨境品牌接受度强
- **消费升级趋势**：从价格敏感型向品质驱动型转变，对国际品牌、有机/清真认证产品需求上升
- **通胀温和**：2024年CPI约2.0-2.5%，消费信心总体稳定

### 消费分层

| 群体 | 月收入范围 | 占比 | 消费特征 |
|------|-----------|------|---------|
| T20（高收入） | ≥10,960令吉 | 20% | 品牌导向，跨境购物频繁 |
| M40（中等收入） | 4,850-10,959令吉 | 40% | 性价比+品质兼顾，主力消费群体 |
| B40（低收入） | <4,850令吉 | 40% | 价格敏感，促销驱动，COD偏好 |

## 三、平台格局

### 三大平台市场份额

马来西亚电商市场呈 **Shopee > Lazada > TikTok Shop** 的三足鼎立格局，但TikTok Shop增速最快。

| 平台 | 估计市场份额（2025年） | 核心优势 | 主要挑战 |
|------|----------------------|---------|---------|
| **Shopee（Sea Group）** | ~45-50% | 价格竞争力强、物流网络密、游戏化运营、农村渗透 | 补贴依赖，盈利压力大 |
| **Lazada（阿里系）** | ~25-30% | 品牌旗舰店生态、天猫国际跨境通道、品质心智 | 用户增长放缓，创新不足 |
| **TikTok Shop** | ~15-20% | 内容驱动、直播电商、Gen Z用户粘性 | 监管风险（2024年曾暂停2个月），合规成本上升 |
| **其他**（Amazon SG、Zalora、PG Mall等） | ~5-10% | 垂直品类、品牌差异化 | 规模有限 |

### 平台动态

**Shopee**：
- 马来西亚最大电商平台，2025年预计投入约35亿令吉（约8.1亿美元）用于运费补贴和返现
- Shopee Express自有物流覆盖马来半岛主要城市，当日达/次日达覆盖率领先
- 跨境卖家可通过Shopee Cross-Border项目入驻，中国卖家占比显著

**Lazada**：
- 阿里系技术中台支撑，Lazada Logistics在吉隆坡、槟城设有大型仓储中心
- 品牌旗舰店（LazMall）是差异化卖点，吸引国际品牌入驻
- 与天猫国际的跨境供应链整合是独特优势

**TikTok Shop**：
- 2024年因监管问题暂停约2个月，月损失约2亿美元GMV
- 恢复后通过本地化注册重获Gen Z用户，直播GMV占比达22%
- 转化率是传统搜索的3-5倍，但内容审核合规成本上升
- 与Tokopedia的整合（GoTo集团）增强了供应链能力

### 社交电商

- 约 **33%** 的马来西亚人仍通过Facebook、Instagram等社交媒体直接购物
- WhatsApp Business是中小卖家重要的私域销售渠道
- 网红/KOL营销在美妆、时尚品类中转化率突出

## 四、跨境进口政策和关税

### 关税政策

| 政策 | 详情 |
|------|------|
| **进口关税** | 马来西亚平均适用关税约5.8%（WTO数据），多数电子产品、纺织品关税0-10% |
| **低值商品免税** | 进口价值 ≤500令吉（~107美元）的商品免征进口关税 |
| **跨境电商销售税（SST）** | 自 **2024年1月1日** 起，通过跨境电商平台进口的商品征收 **10% 销售税**（原为6% SST，电商单独设定为10%） |
| **商品及服务税（GST）** | 目前未恢复GST，仍采用SST制度；但政府已讨论2026-2027年可能重新引入GST（税率可能为6-8%） |
| **自由贸易协定** | 马来西亚是RCEP、CPTPP成员，与中国、澳大利亚、日本、韩国等有优惠关税安排 |

### 跨境电商税收要点

1. **10% SST**：适用于所有通过电商平台跨境进口的商品，由平台代扣代缴
2. **进口税**：特定品类（如酒精、烟草、汽车、特定农产品）仍需缴纳进口税，税率因品类而异
3. **电子发票（e-Invoice）**：2025年7月起强制实施，年营收超1亿令吉的企业已先行执行，2026年将扩展至所有企业
4. **数字服务税（DST）**：自2020年起对数字服务提供商征收6%数字服务税，影响跨境SaaS和数字平台

### 清关与合规

- **MY Customs预清关系统**：2024年上线后，来自中国和新加坡的包裹清关时间缩短约30%
- **清真认证**：食品、化妆品、药品等品类需获得JAKIM清真认证方可在马来西亚销售
- **FDA等效审批**：医疗器械、保健品需向NPRA（国家药品监管局）注册
- **个人数据保护**：PDPA 2010要求跨境数据传输需获得用户同意

### 政策风险提示

- **GST恢复风险**：若2026-2027年恢复GST（预计6-8%），叠加现有10% SST，跨境电商综合税负可能上升
- **数据本地化趋势**：马来西亚政府正在讨论更严格的数据本地化要求，可能增加跨境运营成本
- **TikTok Shop监管**：2024年的暂停事件表明，社交媒体电商面临较高的政策不确定性

## 五、物流基础设施

### 总体评估

马来西亚物流基础设施在东盟处于中上水平，马来半岛（西马）较为发达，东马（沙巴、砂拉越）相对落后。

### 关键物流指标

| 指标 | 详情 |
|------|------|
| **LPI排名** | 全球第26位（世界银行2023年物流绩效指数） |
| **主要港口** | 巴生港（Port Klang，全球第12大集装箱港）、丹戎帕拉帕斯港（PTP）、槟城港 |
| **国际机场** | 吉隆坡国际机场（KLIA）、槟城国际机场、亚庇国际机场 |
| **公路网络** | 马来半岛高速公路网完善，南北大道（PLUS）贯通全境 |
| **铁路** | KTM通勤铁路覆盖巴生谷，ETS电动列车连接主要城市 |

### 主要物流服务商

| 类型 | 代表企业 | 特点 |
|------|---------|------|
| **国家队** | Pos Malaysia（马来西亚邮政） | 覆盖最广，包括东马偏远地区，但时效较慢 |
| **国际快递** | DHL、FedEx、UPS | 时效快（1-3天），价格高，适合高价值商品 |
| **电商自建物流** | Shopee Express、Lazada Logistics | 与平台深度整合，运费补贴力度大 |
| **第三方物流** | J&T Express、Ninja Van、City-Link、GDex | 价格适中，时效2-5天，跨境电商首选 |
| **同城配送** | GrabExpress、Lalamove | 即时配送，适合生鲜和紧急订单 |

### 物流成本

| 路线 | 参考时效 | 参考运费（小包裹） |
|------|---------|-------------------|
| 中国→马来半岛（空运） | 3-7天 | ~15-30令吉/公斤 |
| 中国→马来半岛（海运） | 10-20天 | ~3-8令吉/公斤 |
| 马来半岛→东马 | 3-7天（空运） | ~25-50令吉/公斤 |
| 马来半岛同城 | 1-3天 | ~5-12令吉 |

### 物流痛点

1. **东马配送成本高**：沙巴、砂拉越的末端配送成本是西马的2-3倍
2. **岛屿覆盖难**：纳闽、热浪岛等偏远岛屿配送时效不稳定
3. **退货物流**：逆向物流体系不完善，退货成本占商品价值15-25%
4. **仓储资源紧张**：巴生谷地区优质仓储空间供不应求，租金上涨

### 政府物流政策

- **数字自由贸易区（DFTZ）**：在吉隆坡国际机场附近设立电商清关中心，简化跨境包裹通关
- **国家物流政策（NLP）**：2025年更新版，目标将物流成本从GDP的13%降至10%
- **MyDIGITAL基础设施投资**：2025年前投入约210亿令吉用于5G和数字基础设施建设

## 六、消费者购物习惯和偏好

### 数字人口画像

| 指标 | 数值（2024年1月） |
|------|-----------------|
| 互联网用户 | 3,359万（渗透率97.4%） |
| 社交媒体用户 | 2,868万（渗透率83.1%） |
| 移动互联网连接 | 4,455万（渗透率129.2%） |
| 中位年龄 | 31.0岁 |
| 城市化率 | 78.9% |

### 购物行为特征

**1. 移动优先**
- 72.67%的电商交易通过智能手机完成
- 渐进式Web应用（PWA）加载时间<3秒是转化率的关键
- 移动App使用时长：Shopee用户日均使用约25分钟

**2. 社交驱动**
- 48%的马来西亚人使用TikTok获取新闻（Reuters Institute 2025），为东南亚最高
- 约1/3消费者通过社交媒体平台（Facebook、Instagram、WhatsApp）直接购物
- KOL/网红推荐是美妆、时尚品类最重要的购买决策因素

**3. 促销敏感**
- 9.9、11.11、12.12等大促期间GMV可达日常的5-8倍
- 免运费是影响购买决策的首要因素（超过折扣本身）
- 返现（Cashback）和积分体系对用户留存效果显著

**4. 品类偏好**
- **时尚服饰**：最大品类（26.67%），偏好快时尚和穆斯林时尚（Modest Fashion）
- **食品饮料**：增长最快（CAGR 16.46%），健康食品、进口零食需求旺盛
- **美妆个护**：韩流美妆、清真认证护肤品增长突出
- **消费电子**：手机配件、智能家居设备需求稳定
- **母婴产品**：进口奶粉、婴儿用品跨境需求强劲

**5. 跨境购物偏好**
- 马来西亚跨境购物主要来源国/地区：**中国、新加坡、日本、美国、韩国**
- 跨境购物驱动因素：价格优势（42%）、品类丰富度（35%）、品质信任（23%）
- 跨境购物障碍：运费（38%）、配送时效（28%）、退货困难（22%）、关税不透明（12%）

**6. 穆斯林消费特色**
- 马来西亚约61.3%人口为穆斯林，清真（Halal）认证是食品、化妆品、药品的准入门槛
- 穆斯林时尚（Modest Fashion）是独特细分市场，全球领先
- 斋月（Ramadan）期间电商GMV通常增长30-50%，是全年最重要的营销节点

### 购物旅程

```
社交媒体种草 → 平台搜索比价 → 查看评价/直播 → 加入购物车 → 
等待促销/免运 → 下单支付 → 追踪物流 → 收货评价 → 复购/分享
```

## 七、支付生态

### 支付格局概览

| 支付方式 | 估计份额（2025年） | 趋势 |
|---------|-------------------|------|
| **数字钱包（DuitNow QR等）** | 34.92% | ↑ 快速增长 |
| **银行转账（FPX）** | ~25% | → 稳定 |
| **信用卡/借记卡** | ~20% | → 稳定 |
| **货到付款（COD）** | ~10% | ↓ 下降中 |
| **先买后付（BNPL）** | ~5-7% | ↑↑ 最快增长（CAGR 17.18%） |
| **其他** | ~3-5% | — |

### 关键支付基础设施

**1. DuitNow QR（马来西亚国家二维码支付标准）**
- 由PayNet（马来西亚国家支付网络）运营
- 2024年处理交易额达311亿令吉（约72亿美元）
- 覆盖260万商户点
- 已与新加坡（PayNow）、泰国（PromptPay）、印尼（QRIS）实现跨境二维码互通
- 2025年指令要求所有持牌钱包在2026年中前接入DuitNow

**2. FPX（Financial Process Exchange）**
- 马来西亚主要的在线银行转账支付系统
- 支持20+家银行实时转账
- 是电商大额支付的首选方式
- 2024年处理交易量超8亿笔

**3. 主要数字钱包**

| 钱包 | 运营方 | 特点 |
|------|-------|------|
| **Touch 'n Go eWallet** | TNG Digital（蚂蚁集团合资） | 市场份额最大，交通+零售场景全覆盖 |
| **GrabPay** | Grab | 与Grab生态深度整合，外卖+出行+支付 |
| **Boost** | Axiata | 电信运营商背景，话费充值+零售 |
| **ShopeePay** | Sea Group | Shopee平台内闭环支付，线下扩展中 |
| **MAE by Maybank** | 马来亚银行 | 银行系数字钱包，金融服务完善 |

**4. 先买后付（BNPL）**

| 平台 | 特点 |
|------|------|
| **Atome** | 新加坡进入马来西亚，时尚品类为主 |
| **Grab PayLater** | Grab生态内，与GrabPay无缝衔接 |
| **Shopee PayLater** | Shopee平台内，用户基数大 |
| **AEON Credit** | 传统零售金融背景 |

BNPL市场预计以 **CAGR 17.18%** 增长至2031年，是增长最快的支付细分领域。

**5. 跨境支付**
- 信用卡仍是跨境交易首选（约80%跨境交易使用信用卡）
- PayPal、Stripe、Worldpay等国际支付网关被广泛接受
- 中国跨境卖家通常通过PingPong、Payoneer、万里汇等收款
- DuitNow QR与东南亚邻国的跨境二维码互通促进了旅游消费

### 支付趋势

1. **无现金加速**：BNM目标2026年无现金交易占比达60%+

## 八、综合评估与建议

### SWOT分析

| | 正面 | 负面 |
|---|------|------|
| **内部** | **优势**：高互联网渗透率、年轻人口、多元文化市场、清真经济枢纽 | **劣势**：城市市场趋于饱和、东马物流瓶颈、网络诈骗率高 |
| **外部** | **机会**：直播电商增长、BNPL普及、RCEP关税优惠、DuitNow跨境互通 | **威胁**：GST恢复风险、TikTok监管不确定性、CAC持续上升 |

### 跨境电商进入建议

1. **平台选择**：优先入驻Shopee（流量最大）+ TikTok Shop（增长最快），Lazada作为品牌补充
2. **品类策略**：重点关注时尚服饰（穆斯林时尚）、美妆个护（清真认证）、食品饮料（进口零食）
3. **物流方案**：西马使用J&T/Ninja Van等第三方物流，东马考虑Pos Malaysia或DHL；在巴生谷设海外仓可提升时效
4. **支付接入**：必须支持DuitNow QR、FPX和主流数字钱包；BNPL可提升客单价15-25%
5. **合规要点**：关注10% SST代扣代缴、清真认证（JAKIM）、NPRA产品注册、e-Invoice合规
6. **营销节点**：斋月（Ramadan）、Hari Raya、9.9、11.11、12.12是核心大促
7. **本地化**：马来语+英语双语运营，尊重穆斯林文化，提供COD选项（尤其东马）

### 风险提示

- **税收政策变化**：GST恢复可能增加5-8%的税负
- **汇率风险**：令吉对美元近年波动较大，需做好汇率对冲
- **监管收紧**：跨境电商可能面临更严格的产品标准和数据合规要求
- **竞争加剧**：平台补贴战持续，独立站模式在马来西亚获客成本较高
