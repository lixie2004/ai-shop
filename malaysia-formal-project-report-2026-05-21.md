## 第一章 市场概览

### 1.1 摘要

马来西亚是东南亚第三大电商市场，2024年平台电商GMV约 **$115–128亿美元**（Momentum Works口径），以 **19.5% 的同比增速**领跑东南亚。Shopee以约 **52% 的 GMV 份额**保持绝对领先，TikTok Shop快速追赶至约 **$20亿美元**（+100% YoY），Lazada约 **$10–12亿美元**。马来西亚拥有东南亚最高的互联网渗透率（97.4%）、成熟的支付基础设施和不断扩大的中产阶级，是跨境电商进入东南亚的战略要地。

### 1.6 跨境电商政策环境

| 税种 | 税率 | 说明 |
|------|------|------|
| **SST（销售与服务税）** | 标准10%/减让6%/零税率 | 2025年7月起扩展至数字服务 |
| **公司所得税（居民）** | 15–24%（累进） | 中小企业前RM600K享优惠 |
| **公司所得税（非居民）** | 24%（固定） | 仅对马来西亚来源收入征税 |
| **进口关税** | 因产品/原产地而异 | 有FTA的国家可能享零关税 |

**政策趋势**：马来西亚正在收紧对跨境电商的税收合规要求。SST扩展至跨境数字服务、低价值商品税（LVG Tax）等新规增加运营成本，但合规门槛提升有利于规范化运营的品牌卖家。

### 1.7 SWOT分析

| | 内容 |
|------|------|
| **优势** | 东南亚最高互联网渗透率（97.4%）；成熟金融体系（银行覆盖率88%）；多语言环境降低进入门槛；良好物流基础设施 |
| **劣势** | 人口规模有限（3,449万）；价格竞争激烈；物流最后一公里在非城市区域仍有挑战；税收合规成本上升 |
| **机会** | 社交电商（TikTok Shop）高速增长期；BNPL和数字钱包推动新消费群体入网；直播电商渗透率有提升空间；中产阶级扩大带动消费升级 |
| **威胁** | SST扩展和LVG税增加跨境成本；Shopee一家独大导致平台议价能力不对称；TikTok Shop内容电商规则变化快；合规审查趋严 |
