## 3. 平台格局（2025 年末）

| 平台 | 市场份额 | 核心策略 | 趋势 |
|------|----------|----------|------|
| **Shopee** | **≈ 55-60%** | 跨品类 + 直播 + 促销补贴 | 仍居首位，但面临 TikTok 挑战 |
| **Lazada** | **≈ 25-30%** | 阿里生态 + 物流优势 | 稳居第二，品牌化路线 |
| **TikTok Shop** | **≈ 8-12%** | 社交 + 短视频 + 内容电商 | **增速最快**，2026 预计突破 10% |
| **GoShop/Zalora** | ≈ 2-5% | 垂直/本地 | 细分市场 |

**Shopee + Lazada 合计 ≈ 70% GMV**，但 TikTok Shop 以内容电商模式快速抢占年轻用户。Instagram Shopping 也在马来西亚持续演进。

**趋势**：直播购物 GMV 激增（+1.4% CAGR 贡献），社交电商成为核心增量。平台间补贴战加剧，客户获取成本持续攀升。

## 4. 跨境进口政策 & 监管环境

| 政策 | 内容 | 影响 |
|------|------|------|
| **进口关税** | 平均 **6.5%** | 相对低，鼓励进口 |
| **SST（服务税）** | 2025 年 7 月改革，5-10% 按品类 | 增加跨境成本，尤其服饰、化妆品 |
| **免值门槛** | **无低值免税** | 所有进口需清关 |
| **海关预清关** | 通关时间缩短 **30%** | 利好时效敏感型卖家 |
| **外资持股** | 100% 允许 | 便利外资企业设立 |
| **电子发票** | 2025 年 7 月强制推行 | 推动传统零售商上线 |
| **数据本地化** | 必须在本土存储 | 合规成本略升 |
| **主要采购来源** | 中国（最大，占跨境 70%+） | 中国商品优势明显 |

**监管趋势**：
- **NESR 2.0 + MyDIGITAL** 政策蓝图降低 114.8 万企业上线门槛
- **东盟数字贸易便利化** 措施简化跨境流程
- **数据保护和消费者隐私** 法规趋严
- **清真认证** 是在马来西亚（穆斯林占 60%+）销售食品、化妆品、药品的必备条件

## 5. 热门品类趋势

| 品类 | 线上份额 | CAGR | 关键趋势 |
|------|----------|------|----------|
| **时尚服装** | **26.67%**（第一） | 8-10% | 穆斯林时尚、可持续面料、正品企划 |
| **电子/数码** | **≈ 31%** | 4-6% | 5G 设备、可穿戴、配件 |
| **食品饮料** | 10-12% | **16.46%**（最快） | 健康食品、零食跨境、清真食品 |
| **健康美妆** | 15-18% | 12-14% | 清真认证、有机护肤、功能性美妆 |
| **家居生活** | 8-10% | 5-7% | 智能家居、环保家具 |
| **母婴/玩具** | ≈ 12% | 7-9% | 安全认证、品牌化 |

**跨境卖家核心机会**：
1. **清真认证产品**：穆斯林人口 60%+，清真认证是食品、化妆品、药品的刚需门槛，也是差异化壁垒
2. **穆斯林时尚**：东南亚最大的穆斯林时尚市场之一，结合清真认证可形成强差异化
3. **食品饮料**：增速最快（16.46% CAGR），中国零食、健康食品需求旺盛
4. **电子配件**：5G 推动设备升级，配件需求持续增长
5. **可持续/环保产品**：年轻消费者环保意识提升，可持续品牌有溢价空间

**风险提示**：
- 高端化妆品与医药类监管严格
- 东马物流成本高，配送不平衡
- SST 改革增加税负

## 6. 跨境卖家战略建议

| 维度 | 建议 |
|------|------|
| **首选平台** | Shopee（流量最大）+ TikTok Shop（增速最快） |
| **支付方式** | 必须接入 DuitNow QR + 数字钱包 + BNPL |
| **物流** | 西马优先，东马评估成本后决策 |
| **认证** | 食品/化妆品/药品必须获取清真认证（JAKIM） |
| **定价策略** | 中等偏高定价 + 促销敏感（11.11/12.12/斋月） |
| **内容策略** | 短视频 + 直播是核心获客渠道 |
| **差异化** | 清真认证 + 本地化包装 + 马来语/英语双语 |
