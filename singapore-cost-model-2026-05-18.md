### 税费参数

| 税项 | 税率 | 说明 |
|------|------|------|
| 进口关税 | 0% | 新加坡为自由港，绝大多数消费品免税 |
| GST（商品及服务税） | 9% | 2024年起从8%升至9%；低价值商品（≤$400）也需缴纳 |
| 企业所得税 | 17% | 按净利润征收，首10万SGD享75%免税 |

### 物流成本基准

| 运输方式 | 成本 | 适用场景 |
|----------|------|---------|
| 空运（中国→新加坡） | $5–8/kg | 标准跨境空运，时效3–5天 |
| 海运拼箱（LCL） | $1–2/kg | 大批量重货，时效10–14天 |
| 海运整箱（FCL 20ft） | $1,200–$2,500 | 整柜大批量 |
| 尾程配送（Shopee Logistik） | SGD 0.8–2.0/单 | 按重量，J&T/Speedpost |
| 尾程配送（Lazada Fulfillment） | SGD 1.0–2.5/单 | 按重量+区域 |

### 通用成本假设

| 成本项 | 假设值 | 说明 |
|--------|--------|------|
| 退货损耗率 | 5–8% | 按品类；电子/服装取高值 |

# 附录：成本模型方法论

## 参数来源

| 参数 | 数据来源 |
|------|---------|
| Shopee Singapore 2026 Q2费率 | digitalinasia.com 平台对比 + getonecart.com 费率表 |

| 尾程配送费 | Shopee Logistik + J&T 标准费率 |

| 汇率基准 | 2026年5月 SGD/USD 0.74 |

## 计算公式

```
总成本 = 采购成本 + 头程物流 + GST + 平台佣金 + 交易费 
       + 技术服务费 + 广告费 + 尾程配送 + 退货损耗 
       + 仓储费 + 汇率缓冲 + 包装耗材 + 认证分摊 + 促销分摊

综合毛利率 = (零售价 - 总成本) / 零售价 × 100%

净利率 = (零售价 - 总成本 - 企业所得税17%) / 零售价 × 100%

盈亏平衡月销量 = 固定成本 / (单件毛利 × 6个月回收期)
```

## 风险声明

1. **GST上调风险**：新加坡GST从7%→8%→9%（2024年），未来可能继续上调至10%。
2. **平台费率上升风险**：Shopee 2026年2月新增5%技术服务费，费率仍有上升趋势。
3. **竞争风险**：新加坡市场小（人口560万），高毛利SKU（#8、#6）可能快速吸引跟风者。
4. **汇率风险**：SGD对CNY/USD波动约3–5%，模型中已预留1%缓冲。
5. **本模型为理论测算**，实际毛利率受库存周转、季节波动、平台促销力度、退货率波动等多因素影响。建议每季度更新一次成本模型。
