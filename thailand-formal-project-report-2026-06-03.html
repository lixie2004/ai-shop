3.2 泰国特有消费特征

  1. 租房文化盛行 → 可拆卸、便携式家居产品需求高
  2. 热带气候 → 防晒、驱蚊、降温类产品需求持续
  3. 宠物经济崛起 → 猫主人比例高，猫粮/猫砂/猫玩具需求旺盛
  4. 通勤文化 → 便携水杯、收纳包、车载用品需求大
