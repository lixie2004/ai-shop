| # | 痛点 | 严重程度 | 具体表现与评论高频词 |

## 七、Shopee vs TikTok Shop 评论行为对比

### 7.1 平台行为差异总览

| 维度 | Shopee | TikTok Shop |

| **评论深度** | 文字较长，平均20-50字，会详细描述使用体验 | 短平快，平均5-15字，更情绪化 |

| **差评特征** | 详细描述问题，会写"为什么不好" | 情绪化表达，"ไม่ดี"（不好）、"อย่าซื้อ"（别买） |

| **价格敏感度** | 评论中经常提到价格，"แพงไป"（太贵了） | 价格提及较少，更关注"好不好用" |
| **信任机制** | 店铺评分+商品评分+历史销量 | 视频内容+主播信任+实时互动 |

### 8.2 品类优先级排序

```
P0（优先进入）: 家居收纳 + 宠物用品
  ├── 家居收纳：低合规门槛、差异化空间大、"不打孔"定位可快速建立优势
  └── 宠物用品：高增速、情感驱动、复购率高、本地化空间大

P1（有条件进入）: LED照明 + 个人护理
  ├── LED照明：需要TIS认证+本地仓储，但利润空间大、竞争相对温和
  └── 个人护理：需要FDA认证+本地化配方，利润最高但合规成本最高

P2（谨慎进入）: 手机配件
  └── 手机配件：容量极大但竞争极度激烈，需要强供应链+极致性价比
```
