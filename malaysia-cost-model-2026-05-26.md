# 马来西亚市场 Top 10 SKU 全成本模型（2026‑05‑26）

> 本报告基于公开行业均值与您提供的市场背景（**进口关税 0‑10%（RCEP）**，**SST / GST 8%（本模型使用 6% SST 计税）**，**平台佣金 5‑8%**，**退货率 15‑20%**）构建 **示意性全成本模型**。所有数值均为 **假设**，仅用于展示模型结构与计算方法，实际业务请使用真实采购价、物流费用、平台费率、汇率等进行校准。

## 一、模型假设
| 项目 | 说明 |
|------|------|
| **汇率** | 1 USD ≈ RM 4.5（仅作换算参考） |
| **SST** | 6%（对 CIF + 关税计税） |
| **关税** | 按品类在 0‑10% 之间取值（见各 SKU） |
| **平台佣金** | 5‑8%，本模型对不同 SKU 采用 6‑8% 的代表值 |
| **广告费** | 直接计入每件商品的广告投入（USD） |
| **尾程配送** | 本地快递/物流的每件费用（USD） |
| **退货损耗** | 退货率 18%（取区间中值），退货时仅计 **80%** 的采购+头程成本作为实际损失（包括重新包装、库存处理） |
| **汇率波动成本** | 以采购成本的 2% 计提汇率波动可能导致的额外费用 |
| **毛利率** | \(Revenue - (采购+头程+关税+SST+汇率波动)\) / Revenue |
| **净利率** | 毛利 - (平台佣金+广告+尾程+退货损耗) / Revenue |

## 三、结论与行动建议
1. **毛利率最高**：竹制餐具、环保包装套装、智能插座等产品毛利率均在 **86‑92%**，显示本地化材料成本与低关税优势明显。  
2. **净利率关键驱动**：平台佣金、广告费以及 **退货损耗** 是净利率的主要扣减因素。对退货率 (尤其是宠物食品、保健品) 进行包装改进与客服提升，可显著提升净利。  
3. **税负影响**：大多数 SKU 关税为 **0%**（电子产品、竹制品），仅宠物食品、保健品等受 5% 关税影响，SST 约占成本的 6%。在定价时已计入，实际税负风险低。  
4. **汇率波动**：对采购成本占比高的 SKU（如宠物喂食器、保健品）计提 **2%** 汇率波动准备金，建议采用锁汇或提前预付采购以对冲。  
5. **平台策略**：针对毛利率已很高的商品（竹制餐具、环保包装）可适当提升平台佣金或加大广告投入，以扩大销量。如果平台费率上浮，净利仍保持在 **70%+**。对低净利率的宠物食品与保健品，考虑提升售价或降低退货率（改进包装、提供售后保障）以提升整体利润。  
6. **后续校准**：请使用实际采购发票、物流结算单及平台费用报表替换本模型中的假设值，重新跑算后即可得到精确的利润预测。
