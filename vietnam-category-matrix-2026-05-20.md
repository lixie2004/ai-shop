## 一、越南电商市场宏观概览

### 1.1 市场规模与增速

| 指标 | 数值 | 同比 |
|------|------|------|
| 2025年四平台GMV | VND 429.7万亿（$16.35B） | +34.75% |
| 2025年全口径线上零售 | $26-28B | +35% |
| 2025上半年GMV | VND 222.1万亿 | +23% |
| 2026年预测 | 双位数增长 | CAGR 35%（乐观） |
| 日均线上消费 | $44M/天 | +34.8% |

### 1.2 平台格局

| 平台 | 市场份额（2025） | 增速 | 定位 |
|------|-----------------|------|------|
| **Shopee** | ~56% | +16% | 全品类低价+活动驱动 |
| **TikTok Shop** | ~41% | +69%~148% | 内容+社交电商 |
| **Lazada** | ~3% | 稳定 | 品质+品牌 |
| **Tiki** | <1% | 下滑 | 精品+快速配送 |

> ⚠️ **关键变化**：TikTok Shop在2025年Q1同比+113.8%，市场份额从23%升至35%，正在快速蚕食Shopee份额。双寡头格局加剧，卖家必须双平台运营。

### 1.3 消费者画像

| 维度 | 数据 |
|------|------|
| 人口 | 1.01亿（2025） |
| 中位年龄 | 31岁 |
| 35岁以下占比 | 60% |
| 人均GDP | ~$4,717（2025） |
| 月均收入 | $200-$600 |
| 网购渗透率 | ~76% |
| 社交电商购物占比 | 50% |
| 社交媒体用户 | 7,620万 |
| 日均社媒时长 | 2.5-3小时 |

### 1.4 关键趋势

1. **Shoppertainment（购物娱乐化）**：短视频+直播成为核心购物场景，TikTok Shop爆发式增长
2. **社交电商主导**：50%消费者通过社交平台完成购买决策
3. **价格敏感+品质升级并存**：消费者既追求性价比，又愿意为品质付溢价
4. **"成分党"崛起**：美妆、食品、保健品领域消费者越来越关注成分和认证
5. **KOC > 头部KOL**：5k-50k粉丝的微网红转化率更高
6. **COD仍占主导**：60-70%订单使用货到付款，电子钱包（MoMo/ZaloPay）快速增长
7. **监管趋严**：2025年7月电商法生效，外资卖家需获MoIT许可+VNeID认证

### 1.5 政策与合规

| 领域 | 要求 |
|------|------|
| 电商法（2025.7生效） | 外资卖家需MoIT许可+VNeID认证 |
| VAT | 标准税率10% |
| 进口关税 | RCEP框架下优惠税率 |
| 产品认证 | 电子产品需CR认证，食品需FDA认证 |
| 跨境进口 | 2024年+43%，持续增长 |
| 平台佣金 | 部分品类上调至9% |

## 二、候选品类全景扫描（18个品类）

### 品类评估方法论

每个品类从6个维度评估：
- **潜力评分（1-10）**：市场容量×增长趋势×需求强度
- **竞争度评分（1-10）**：10=极度竞争，1=几乎无竞争
- **进入壁垒**：低/中/高（认证要求、供应链复杂度、资金门槛）
- **增速预估**：基于平台数据和行业报告的年均增长率
- **利润率预估**：扣除所有成本后的净利润率区间
- **跨境适配度（1-5）**：中国供应链优势×物流可行性×合规难度
