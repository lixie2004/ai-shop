## 税务与关税（通用参考）

| 税项 | 税率 | 说明 |
|:----:|:---:|------|
| **PPh 22 进口所得税** | 7.5% (拥有API) / 10% (无API) | 进口清关时预缴 |
| **BM 进口关税** | 0 - 30% | 视HS编码而定，化妆品/食品通常 5-20% |
| **PPN 增值税** | **12%**（2026年从11%上调） | 2026年1月生效，进口+销售环节均适用 |

| **MoT 31/2023 跨境限制** | 跨境电商进口商品最低价≥**$100** | 低于$100的商品限制跨境直邮，需通过印尼贸易商或本地仓 |
| **BMTP 额外保障税** | 最高可达50% | 特定消费品（化妆品/纺织品/电子产品）可能适用 |

### 1. Halal Sunscreen SPF50+（30-50ml）— 美妆个护

| 项目 | 内容 |
|------|------|
| **产品分类** | Kosmetik (Cosmetics) — 防晒属于高风险化妆品 |
| **HS 编码参考** | 3304.99（护肤品/防晒） |
| **主管机构** | BPOM（化妆品处）+ BPJPH（清真事） |

#### 准入认证

| 项目 | 内容 |
|:----|:-----|
| **认证名称** | **BPOM Cosmetic Notification (NA编号)** |
| 法律依据 | BPOM Reg 22/2017; BPOM Reg 18/2024（新标签广告规定，2025年11月强制执行） |
| 审核费用（政府） | IDR 1,000,000 - 3,000,000 / 款（≈ $62 - $185） |
| 第三方代理费用 | $500 - $1,500（含本地代表服务） |
| SPF in-vivo 检测 | $800 - $2,000（人体防晒指数测试） |
| 稳定性/微生物检测 | $300 - $800 |
| **合计估计费用** | **$1,662 - $4,485** |
| 办理周期 | **3 - 6 个月**（资料完整 + BPOM周转） |
| NA编号有效期 | **5 年**（可续期） |

#### 清真认证 — ⚠️ 2026年强制

| 项目 | 内容 |
|:----|:-----|
| 主管机构 | **BPJPH**（BPOM管注册，BPJPH管清真——两者完全独立但必须并行） |
| 法律依据 | Law 33/2014（清真产品保障法）；PP 39/2021；**BPJPH Circular 7/2025** |
| 强制节点 | ⚠️ **2026年10月**起所有化妆品必须持有清真认证方可上市（含进口品） |
| 费用（一般企业） | IDR 5,000,000 - 20,000,000（$300 - $1,200）+ 审查/差旅费 |
| 办理周期 | 4 - 8 个月（含LPH工厂审查 + MUI法特瓦会审议） |
| 有效期 | **4 年**（续期需在到期前3个月办理） |
| 关键 | 必须与BPOM注册**同步推进**，不能先注册后补清真 |

#### 成分限制

| 项目 | 内容 |
|:----|:-----|
| BPOM禁限成分 | 超过1,600种禁用物质（汞、氢醌、维A酸、激素类固醇、违禁色素） |
| 防晒剂 | 必须为ASEAN化妆品指令附录VII清单中的防晒剂（不可使用未列明的成分） |
| SPF宣称上限 | SPF 50+（超出可能被归为药品） |
| 酒精 | 清真版本如需认证：乙醇含量必须为零或<1%（技术用途），且来源非酒精饮料 |
| BPOM Reg 18/2024 | 含防晒成分必须标注警告语（SPF值保留±10%容差） |

#### 标签要求（BPOM Reg 18/2024）

| 项目 | 要求 |
|:----|:-----|
| 语言 | **全部使用印尼语**（外语仅允许作为辅助，不得大于印尼语字号） |
| 必须包含 | 产品名称、用途、成分表（降序）、制造商/进口商信息、批次号、有效期、NA编号 |
| 警告标识 | 防晒霜需标注：使用前摇匀，远离眼睛 ISO 警告 |
| 2D条码 | 推荐使用，有利于追溯 |
| NA编号 | "NA#############" 格式，印在包装显著位置 |
| 清真标识 | 2026年10月起须有官方清真标志（无认证不可印） |

#### 风险评级：🟡 **中**

| 风险因素 | 说明 |
|:--------|:-----|
| SPF人体测试 | 费用高（$800-2,000），需提前2-3个月安排 |
| 清真认证4-8个月 | 若现在启动-BPOM 3月+BPJPH 4月=7月，盯紧2026年10月强制期限 |
| Reg 18/2024过渡期 | 2025年11月15日前必须完成新标标签更新 |

#### 🔄 替代方案

| 方案 | 说明 | 影响 |
|:----|:-----|:----|
| **方案A：BPOM + BPJPH 并行申请** | 化妆品NA注册与清真认证同时推进，节省总体时间2-3个月 | 复杂但时间最优 |
| **方案B：本地OEM代工** | 使用已有BPOM+清真认证的印尼OEM工厂生产，贴己品牌 | +$0.5-1/支，时间省50% |
| **方案C：分阶段申报** | 先注册SPF15/30（低风险非SPF5款）简化程序 | 功能缩小但加快上市 |

### 3. Functional Gummy Supplements（60粒）— 保健品

| 项目 | 内容 |
|:----|:-----|
| **产品分类** | Suplemen kesehatan (Health Supplements) |
| **主管机构** | **BPOM（保健品处）** + BPJPH（清真） |
| **法律框架** | BPOM Reg 11/2020；BPOM Reg 6/2025（新稳定性测试要求）；Decree 70/2025（成分分类修订） |

#### 准入认证

| 项目 | 内容 |
|:----|:-----|
| **认证名称** | **BPOM Supplement Registration（营养补充剂注册）** |
| **编号格式** | **POM SD##########**（保健品专用，不同于化妆品NA编号） |
| 政府费用 | IDR 5,000,000 - 20,000,000（$300 - $1,200） |
| 稳定性测试（Zone IV 30°C/65%RH） | $1,000 - $2,500 |
| 微生物+重金属+成分检测 | $500 - $1,200 |
| 第三方代理/本地代表 | $1,500 - $4,000 |
| **合计估计费用** | **$3,300 - $8,900** |
| 办理周期 | **6 - 12 个月**（保健品审批慢于化妆品） |
| 有效期 | 5 年 |

#### 清真认证（⚠️ 软糖是清真高风险品）

| 项目 | 内容 |
|:----|:-----|
| **强制性** | **⚠️ 强制**（2026年10月起，含食品级补充剂） |
| 明胶问题 | 明胶是动物源成分，必须100%清真来源（牛明胶/鱼明胶）、工厂须隔离非清真生产线 |
| ⚠️ 猪明胶 | **绝对禁止**——印尼对猪源成分零容忍，发现即面临产品召回+刑事起诉 |
| **强烈建议** | 使用**果胶(pectin)**或**卡拉胶(carrageenan)**替代明胶 |
| 费用 | 见防晒霜部分 +$300 - $1,200 + 4-8个月 |

#### 成分限制（BPOM保健品指南）

| 成分 | 每日上限 | 说明 |
|:----|:-------:|:-----|
| 维生素C | ≤1,000mg | 常规保健品含量OK ✅ |
| 维生素D | 各维生素按BPOM参考 | 需确认不超上限 |
| 糖/糖醇 | 包装上必须标注糖含量 | 印尼含糖监管呈收紧趋势 |
| 功能性成分 | 必须列入BPOM允许清单 | 不可使用未批准的功能性原料 |
| 明胶 | 清真版本不可用猪源 ❌ | **果胶替代为最优解** |

#### 标签要求

| 项目 | 内容 |
|:----|:-----|
| 语言 | **全部印尼语**（禁止仅标注英语） |
| 必须标注 | BPOM/POM编号、成分表、剂量说明、保质期、制造商+进口商、警告语 |
| 功能宣称 | 仅允许"维持健康/支持免疫功能"——**不得宣称治疗/预防疾病**（否则归为药品） |
| 清真标识 | 已通过清真认证的标志，2026年10月起强制 |
| 形态标注 | "Permen jeli suplemen kesehatan"（保健果冻糖） |

#### 风险评级：🔴 **高**

| 风险因素 | 说明 |
|:--------|:-----|
| ⚠️ **BPOM注册6-12个月** | 东南亚属中长周期；如提交材料不完整可能延长至18个月 |
| ⚠️ **明胶清真红线** | 猪明胶在印尼是法律和宗教双重红线——绝对不可使用 |
| ⚠️ **稳定性测试新规** | BPOM Reg 6/2025增加稳定性条件要求，软糖高湿稳定性是已知难点 |
| ⚠️ **2026年10月前必须清真** | 时间紧迫：BPOM 6-12月+BPJPH 4-8月，须现在立即启动 |
| 分类模糊 | "功能性软糖"可能被归为补充剂或加工食品，分类错误重头再来 |

#### 🔄 替代方案

| 方案 | 说明 | 影响 |
|:----|:-----|:----|
| **方案A（最推荐）**：**果胶替代明胶** | 使用植物基果胶从根本上解决清真合规问题 | 成本+$0.1-0.3/粒，但长久安全 |
| **方案B：先走加工食品路径** | 以"含维生素的果味软糖零食"名义走BPOM **MD编号**（加工食品），1-2个月入市 | 不可宣称保健功能 |
| **方案C：印尼OEM代工** | 使用已有BPOM POM+清真认证的印尼软糖OEM厂 | +$0.3-0.8/袋，时间节省50% |
| **方案D：新加坡注册转进口引证** | 先在新加坡HSA注册，再以已注册产品引证至BPOM | 缩短约3-4个月 |

### 4. Baby Mosquito Repellent（20片贴/100ml）— 母婴用品

| 项目 | 内容 |
|:----|:-----|
| **产品分类** | Kosmetik或Bahan Berbahaya（驱蚊产品分类存在争议） |
| **主管机构** | **BPOM（化妆品处 或 药品处 — 视宣称而定）** |
| **法律框架** | BPOM Reg 22/2017；若含DEET > 20%可能被归为药品 |

#### 准入认证

| 项目 | 内容 |
|:----|:-----|
| **路径A：归化妆品** | BPOM Cosmetic Notification（NA编号），适用不含药物成分的天然驱蚊产品 |
| **路径B：归药品** | BPOM Drug Registration（危险品类），DEET等合成驱蚊剂需药品注册 |
| 化妆品路径费用 | $1,000 - $3,000（同化妆品标准） |
| 药品路径费用 | $3,000 - $8,000（需临床试验数据/毒理报告） |
| **化妆品路径周期** | 3 - 6 个月 |
| **药品路径周期** | 12 - 24 个月 |

#### 成分限制

| 成分 | 限制 |
|:----|:-----|
| DEET | ≤20%（2岁以上儿童）；≤10%（2-12岁儿童） |
| 香茅油/citronella | 属于天然驱蚊剂，可走化妆品路径 |
| 派卡瑞丁(picaridin) | ≤20%，成分需提前确认清单 |
| 对婴儿适用宣称 | 若宣称"适合0-12月婴儿"——抬高风险评级至药品级 |

#### 风险评级：🟡 **中**

| 风险因素 | 说明 |
|:--------|:-----|
| 分类不确定性 | 天然vs合成驱蚊剂决定了BPOM化妆品路径(3月)vs药品路径(12-24月)的巨大差异 |
| 婴儿安全审查 | 婴儿产品即使在化妆品路径也会面临更严格的成分审查 |
| 天然成份 | 若使用香茅油+柠檬桉油等天然驱蚊成分，可走化妆品路径 |

#### 🔄 替代方案

| 方案 | 说明 | 影响 |
|:----|:-----|:----|
| **方案A：纯天然配方走化妆品** | 使用approved天然驱蚊剂（香茅油/柠檬桉油/薄荷油），走BPOM化妆品通知路径 | 3-5个月 |
| **方案B：明确标注"2岁以上儿童使用"** | 不宣称适合婴儿，不触及高风险药品分类 | 法律路径更清晰 |

### 5. Halal Pet Food / Treats（400g-1kg）— 宠物食品

| 项目 | 内容 |
|:----|:-----|
| **产品分类** | Pakan hewan / Makanan hewan peliharaan |
| **主管机构** | **BPOM**? **BKP? KEMENTAN?** — ⚠️ **归属非常复杂** |

#### 法律框架——监管部门归属争议

| 机构 | 管辖 | 对宠物食品 |
|:----|:----|:----------|
| BPOM | 用于人类的食品/药品 | 宠物食品非人类食品——**BPOM一般不直接管** |
| BKP / KEMENTAN | 饲料/动物饲料（农业部畜牧总局） | **最可能的监管方**——宠物饲料归农业部的饲料处 |
| BPJPH | 清真认证 | 宠物食品虽非人类食品，但销售环节涉及清真商家，可自愿申请 |
| 当前无统一监管 | 宠物食品进口相对宽松——无强制BPOM预批 | 但各港口海关检查标准不一 |

#### 准入认证

| 项目 | 内容 |
|:----|:-----|
| **路径A** | 按**饲料/动物饲料**注册到农业部（Direktorat Pakan）——需饲料注册号 |
| **路径B** | 按**加工食品**注册到BPOM（如果产品标注食用且外观像人类零食） |
| 农业路径费用 | $500 - $2,000（注册IDR 2-5M + 检测） |
| BPOM路径费用 | $2,000 - $5,000（若被强行归为食品，需补充剂注册） |
| 周期 | 农业路径：2 - 4 个月；BPOM路径：6-12个月 |
| **当前实务** | 大多数宠物食品在印尼以品牌分销形式存在，**未强制预注册**，但海关可能扣押要求提供成分分析 |

#### 清真认证

| 项目 | 内容 |
|:----|:-----|
| 宠物食品的清真要求 | **法律上对宠物食品无强制性清真认证**（仅人类食用品强制） |
| 市场实务 | 宠物食品市场品牌化程度低，"Halal Pet Food"是差异化卖点但无强制 |
| 清真肉源 | 如含鸡肉/牛肉/鱼肉成分，建议自愿获得MJAKIM/BPJPH认证 |
| **法务建议** | 可**自愿申请**清真认证以建立品牌信任；不含肉源的素食零食无需清真 |

#### 标签要求

| 项目 | 内容 |
|:----|:-----|
| 印尼语/双语 | 成分表、保质期、储存条件 |
| 成分说明 | 标注蛋白质/脂肪/纤维含量（如饲料路径，按农业标准） |
| 清真标识 | 仅当获得认证后方可使用，严禁假冒宣称 |
| "Untuk hewan peliharaan" | 必须标注"仅供宠物食用"避免被误认为人类食品 |

#### 风险评级：🔴 **高**

| 风险因素 | 说明 |
|:--------|:-----|
| ⚠️ **监管归属不明确** | 宠物食品在印尼的监管空白导致不同海关/港口执法标准不一 |
| ⚠️ **海关扣留风险** | 部分港口（如泗水）要求宠物食品必须有BPOM注册号，否则直接退运 |
| ⚠️ **BPOM/BKP来回划分** | 产品可能被甩到两个机构之间，退回重送 |

| **最低价格 $100** | 跨境电商直接对消费者的商品价格不得低于$100/件 | 防晒霜($7-10)、护肤套装($15-25)、唇油($5-8)、驱蚊贴($3-5)——**全部低于$100** |
| **正面清单** | 跨境只能进口正面清单上的品类 | 需逐项核对 |
| **解决方案** | **注册印尼本地公司+使用本地仓库**，走B2B进口模式 | 全部10个SKU |

### 2. BPJPH Circular 7/2025 — 清真认证强制时间表

| 阶段 | 强制对象 | 时间 |
|:----|:---------|:----|
| 第一阶段 | 食品、饮品 | 2024年10月起 |
| 第二阶段 | 保健品、传统药品、OTC | 2026年10月起 |
| 第三阶段 | 化妆品 | 2026年10月起 |
| 第四阶段 | 所有消费品（服装/家居/玩具） | 2028年起 |

👉 **SKU 1、2、3、8、10** 必须在2026年10月前获得清真认证

### 3. BPOM Reg 18/2024 — 化妆品新标签/广告规定

- **2025年11月15日截止**：所有已在市化妆品必须更新标签符合新标
- 印尼语为主、2D条码推荐、特殊警告语（AHA/SPF/水杨酸）
- SKU 1、2、10受影响

### 4. 2026年增值税上调

PPN从11%→**12%**（2026年1月起），进口成本额外增加约1%

## 📋 完整合规检查清单

### 阶段一：公司主体与前置要求 ⬜

- [ ] 在印尼注册**本地法人实体** (PT PMA 或 PT) — 所有10个SKU都需要
- [ ] 取得**NIB**（营业执照）和对应KBLI代码
- [ ] 注册**OSS系统**账户
- [ ] 取得**NPWP**（税务登记号）
- [ ] 获得**API**（进口许可证编号）
- [ ] 确认HS编码并查询BM关税税率
- [ ] 确认BMTP额外保障税是否适用
- [ ] 注册SDPPI/SNI/Biaya（按品类）
- [ ] 注册调查清真申报系统

### 阶段二：产品注册/认证 ⬜

#### BPOM化妆品（SKU 1,2,10）
- [ ] 确认产品配方不含BPOM禁限成分（1,600+种禁用物质）
- [ ] 准备PIF (Product Information File)
- [ ] 准备CFS（原产国自由销售证明）— 印尼大使馆/领事馆公证认证
- [ ] 准备GMP证书（ISO 22716或等效标准）
- [ ] 提交Notifkos e-Registration
- [ ] 获取NA编号
- [ ] 新标符合检查（BPOM Reg 18/2024）

#### BPOM保健品（SKU 3, 8）
- [ ] 确认产品分类补充剂/加工食品
- [ ] 确认活性成分在BPOM允许成分清单
- [ ] 完成稳定性测试（30°C/65%RH，Zone IV条件）
- [ ] 完成微生物+重金属检测
- [ ] 准备科学证据支持功能宣称（保健品路径）
- [ ] 确认工厂GMP（PIC/S优先）
- [ ] 通过e-Registration提交
- [ ] 获取POM SD编号（保健品）或MD编号（加工食品）

#### BPJPH清真认证（SKU 1,2,3,5,8,10）
- [ ] ⚠️ 确认强制适用时间表（2026年10月）
- [ ] 注册MySihalal/BPSPH系统
- [ ] 建立Halal Assurance System (HAS 23000)
- [ ] 明确所有原料的100%清真来源并收集认证
- [ ] 如含明胶 → **果断替换为果胶**（SKU 3）
- [ ] 如含胶原蛋白 → 鱼胶原或清真牛胶原（SKU 8）
- [ ] 如含蜡质/油脂 → 确认植物源（SKU 10）
- [ ] 安排LPH工厂审查
- [ ] 获取清真证书 + Halal标志

#### SNI强制认证（SKU 6,7,9）
- [ ] 确认适用SNI标准号
- [ ] 在印尼指定**唯一官方代表**
- [ ] 商标进行印尼注册（至少R商标受理函）
- [ ] 工厂认证ISO 9001
- [ ] 安排SNI测试（可在国外认可实验室预检）
- [ ] 安排工厂审查（如Type 5方案）
- [ ] 获取CoC(符合性证书) + SPPT(市场准入证)
- [ ] 印制SNI标志和CoC编号

#### SDPPI认证（SKU 7,9）
- [ ] 确认是否含无线通信功能（蓝牙/WiFi/NFC）
- [ ] 确认SDPPI频段技术参数与FCC/CE差异
- [ ] 送样到SDPPI认可实验室
- [ ] 获取Sertifikat SDPPI

### 阶段三：标签与包装 ⬜

- [ ] 标签**以印尼语为主**（外语字号不得大于印尼语）
- [ ] 产品名称（印尼语）
- [ ] 制造商名称和地址
- [ ] 进口商/分销商名称和地址（印尼法人）
- [ ] 原产国（"Dibuat di Tiongkok" / "Impor dari Tiongkok"）
- [ ] 成分表（含量降序，INCI名称+印尼语）
- [ ] 净含量（公制单位）
- [ ] 批次号
- [ ] 生产日期和有效期
- [ ] 使用说明（印尼语）
- [ ] 警告语/注意事项
- [ ] **NA编号**（化妆品）/ **POM SD编号**（保健品）/ **MD编号**（加工食品）— 印在包装
- [ ] **SNI标志**+CoC编号（玩具SNI强制产品）
- [ ] **SDPPI标签**（电子/电信设备）
- [ ] **Halal标志**（已获认证的2026年10月起强制）
- [ ] 2D条码（推荐）
- [ ] 防晒霜：SPF值+警告语（Reg 18/2024）
- [ ] 含AHA/水杨酸：特别警告语
- [ ] 保健品："补充剂"声明，非药品
- [ ] 宠物食品："仅供宠物食用"声明

### 阶段四：进口清关 ⬜

- [ ] 确认MoT Reg 31/2023跨境限制（≥$100）
- [ ] 确认贸易方式：B2B进口（跨境电商直邮受限）
- [ ] 准备Form E（中国-东盟FTA优惠关税）
- [ ] 计算进口关税总额：BM + PPh 22 + PPN 12%
- [ ] 准备清关文件：发票、箱单、提单、COO、认证副本
- [ ] 确认Port of Entry（雅加达Tanjung Priok / 泗水Tanjung Perak）
- [ ] 预分配进口报关行

### 阶段五：上市后合规 ⬜

- [ ] 在Shopee/Tokopedia/Lazada上架（需要BPOM编号）
- [ ] 建立印尼语客户服务体系

- [ ] 监督清真认证续期（每4年）
- [ ] BPOM化妆品通知续期（每5年）/ 保健品注册续期（每5年）
