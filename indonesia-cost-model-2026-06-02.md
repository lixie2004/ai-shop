## 五、核心结论

### 5.1 成本结构关键发现

```
发现① 物流成本是最大变量
  → 占全成本 5-15%（vs 中国国内 3-5%）
  → 大件产品物流成本是大敌

发现② De Minimis $3 实际影响有限
  → 关税已被平台代收+定价包含
  → 真正影响是合规成本增加

发现③ Halal/BPOM认证是最大前期投入
  → 单品分摊$5-20/件（初期）
  → 销量>3,000件/月后可有效摊薄

发现④ 退货成本不可忽视
  → 时尚类退货率12% = 吃掉大部分利润
  → 改善尺码信息+试穿内容=最直接降退货方式

发现⑤ 广告ACOS是利润的决定性因素
  → ACOS每降低5% = 利润率提升3-5个百分点
  → 新品期ACOS 30-35%，成熟期需降至20-25%
```

> **分析师核心观点**：印尼跨境电商的成本结构本质是"高固定成本（认证+物流）+ 高变动成本（广告+退货）"。最赚钱的SKU不一定是最高毛利的，而是**认证成本可控+退货率低+广告ACOS优化空间大**的品类。LED照明和宠物智能用品是"低难度+高利润"的最佳切入点。Halal认证产品虽然利润率高，但前期$5,000-8,000的认证费用是进入壁垒——非常适合已有销量基础的卖家升级，不适合新卖家首单测试。
