## 品类评估方法论

### 评分维度（各 1-10 分）

| 维度 | 权重 | 数据来源 |
|------|------|---------|
| **市场需求** (Market Demand) | 30% | 搜索量、平台品类占比、增长率 |
| **竞争度** (Competition) | 25% | 头部卖家集中度、品牌垄断程度 |
| **利润空间** (Profit Margin) | 20% | 采购成本、平台费用、物流成本估算 |
| **跨境可行性** (Cross-border Feasibility) | 15% | 物流复杂度、合规门槛、关税 |
| **趋势强度** (Trend Momentum) | 10% | TikTok/社媒热度、YoY 增长 |

### 质量门禁
- 品类矩阵 ≥ 10 个品类
- 每个品类含潜力评分和竞争度评分

## 品类深度分析

### 1. 美妆个护（K-Beauty 工具+护肤品）⭐⭐⭐⭐⭐

| 维度 | 详情 |
|------|------|
| **市场需求** | TikTok Shop 美妆 GMV 占比 28%，全品类最高。菲律宾消费者对 K-Beauty 和 J-Beauty 有强品牌忠诚度。 |
| **竞争度** | 中高。大量本土卖家和中国卖家。差异化空间在品牌化而非价格战。 |
| **利润空间** | 30–50%（Ecomobi 数据）。LED 面罩、刮痧板、精华套装等高毛利。 |
| **跨境可行性** | 中。化妆品需 FDA 认证，但非处方护肤品门槛较低。 |
| **趋势强度** | 极强。TikTok #skincare 内容持续爆款，单条视频可清空 50 件库存。 |
| **推荐平台** | **TikTok Shop + Shopee** |
| **建议 SKU** | LED 面罩、Gua Sha 工具、防晒霜、精华套装、护肤礼盒 |

### 2. 健康营养补充剂 ⭐⭐⭐⭐⭐

| 维度 | 详情 |
|------|------|
| **市场需求** | 菲律宾是东南亚补充剂采用率最高的市场之一。维生素、胶原蛋白、免疫增强剂是常青热销品。 |
| **竞争度** | 中等。白标/自有品牌空间大。 |
| **利润空间** | 40–60%（Ecomobi 数据），是最高毛利品类之一。 |
| **跨境可行性** | 中。需 FDA 认证，但补充剂类相对标准。 |
| **趋势强度** | 强。健康意识持续提升，复购率高（月均复购）。 |
| **推荐平台** | **Shopee + Shopify** |
| **建议 SKU** | 胶原蛋白粉、维生素 C、免疫增强剂、瘦身茶、益生菌 |

### 3. 手机/电子配件（差异化）⭐⭐⭐⭐

| 维度 | 详情 |
|------|------|
| **市场需求** | 菲律宾人日均手机使用 9+ 小时，手机配件是常年 TOP5 品类。 |
| **竞争度** | 高。通用手机壳/数据线已被中国卖家以 ₱50–99 价格主导。必须走差异化路线。 |
| **利润空间** | 25–40%。3合1无线充电座、MagSafe 配件、内容创作配件（环形灯）利润较好。 |
| **跨境可行性** | 高。电子产品进口限制少，物流便利。 |
| **趋势强度** | 中高。内容创作者经济推动环形灯/麦克风等配件需求。 |
| **推荐平台** | **Shopee + Lazada + TikTok Shop** |
| **建议 SKU** | 3合1无线充电座、MagSafe 配件、USB-C 集线器、环形灯套装 |

### 4. 时尚服饰（快时尚）⭐⭐⭐⭐

| 维度 | 详情 |
|------|------|
| **市场需求** | 全平台 TOP1 品类（Shopee 27%/TikTok 26%）。 |
| **竞争度** | 高。但快时尚更替快，新款式持续有需求。 |
| **利润空间** | 中低。价格竞争激烈，但通过差异化设计/定制可提升。 |
| **跨境可行性** | 中高。服装进口关税较低，物流成本可控。 |
| **趋势强度** | 强。社交媒体驱动时尚趋势变化快。 |
| **推荐平台** | **Shopee（走量）+ TikTok Shop（引流）** |
| **建议 SKU** | 休闲服装、运动休闲装、配饰 |

### 5. 家居收纳/小型家电 ⭐⭐⭐

| 维度 | 详情 |
|------|------|
| **市场需求** | 18% 市场份额（从 12% 跳升）。混合办公模式推动家居改善需求。 |
| **竞争度** | 中。本土品牌和跨境卖家竞争。 |
| **利润空间** | 中。小型收纳品利润较好，大件家电物流成本高。 |
| **跨境可行性** | 中低。大件物流是瓶颈。 |
| **趋势强度** | 中。可持续和环保设计加分。 |
| **推荐平台** | **Shopee + Lazada** |
| **建议 SKU** | 小型收纳盒、空间整理器、小型厨房电器 |

### 6. 母婴用品 ⭐⭐⭐

| 维度 | 详情 |
|------|------|
| **市场需求** | 稳定需求。菲律宾家庭规模大，母婴消费持续。 |
| **竞争度** | 中。国际品牌主导高端，跨境可切入中端。 |
| **利润空间** | 中高。安全座椅、婴儿推车等高单价品类利润好。 |
| **跨境可行性** | 中。需安全认证。 |
| **趋势强度** | 稳定。 |
| **建议 SKU** | 婴儿喂养用品、婴儿护理、母婴玩具 |

### 7. 宠物用品 ⭐⭐⭐

| 维度 | 详情 |
|------|------|
| **市场需求** | 增长中。菲律宾宠物经济正在兴起。 |
| **竞争度** | 低。蓝海市场。 |
| **利润空间** | 高。宠物食品和用品利润率好。 |
| **跨境可行性** | 中高。宠物配件进口限制少。 |
| **趋势强度** | 上升中。社交媒体宠物内容推动。 |
| **建议 SKU** | 宠物玩具、宠物美容工具、宠物服装 |

## 蓝海品类识别

基于"低竞争 + 高增长"标准：

| 蓝海品类 | 理由 | 窗口期 |
|---------|------|--------|
| **宠物用品** | 市场起步阶段，本土供应不足 | 6–12个月 |
| **白标健康补充剂** | 40–60% 毛利，复购率高，品牌化空间大 | 12–18个月 |
| **K-Beauty 工具** | 差异化明显，直播转化高 | 6–12个月 |
| **内容创作者配件** | 新兴需求，环形灯/麦克风等 | 6个月 |
