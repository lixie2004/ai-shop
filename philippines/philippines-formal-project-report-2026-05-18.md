## 1. 执行摘要

### 核心结论

菲律宾作为东南亚第三大电商市场（2025年GMV约 **$22B**，CAGR 15%），呈现出**年轻化、价格敏感、社交驱动**的鲜明特征。本报告通过市场宏观、品类矩阵、搜索热点、竞品分析、用户洞察、成本建模、合规审查、风险评估九层分析，最终筛选出 **10 个推荐 SKU**，并给出分阶段入场策略。

### 关键发现

| # | 发现 | 影响 |
|---|------|------|
| 1 | Shopee 占电商流量 **55%**，TikTok Shop 增速 **+53% YoY** | 双平台策略：Shopee 走量，TikTok 做品牌 |
| 2 | **"affordable"** 跨品类搜索占比 28%，价格敏感度极高 | 定价策略必须锚定中低端 |
| 3 | 穆斯林人口 **1,100 万+**（10-15%），Halal 美妆品牌空白 | Halal 认证是差异化工具（菲律宾自愿认证） |

### 2.4 消费者画像

| 维度 | 特征 |
|------|------|
| **价格敏感度** | 极高 — "affordable" 搜索占比 28% |
| **决策因素** | 价格(35%) > 品牌信任(20%) > 评论(20%) > 物流(15%) > 认证(10%) |
| **支付偏好** | GCash(89%份额) + PayMaya + COD(~15%) |
| **Gen Z** | 占人口 ~30%，日均 TikTok 60-90 分钟，60%+ 受社媒影响购买 |
| **Millennials** | 占人口 ~25%，月均电商消费 ₱1,500-5,000 |
| **穆斯林人口** | 10-15%（1,100 万+），Mindanao 地区集中 |
| **宠物家庭** | 67% 养狗 / 43% 养猫 |
| **住房自有率** | ~80%，DIY 需求强劲 |

### 2.5 关键趋势

1. **Halal 市场扩张**：穆斯林人口 1,100 万+，Halal 美妆/食品需求增长（注：菲律宾 Halal 认证目前为自愿性质）
2. **物流升级**：跨海物流合作，平均递送时效缩短 30-40%
3. **数字支付渗透**：GCash + PayMaya，数字支付占比 50-55%
4. **环保意识觉醒**：Shopee "eco-friendly" 搜索量增长 35%+
5. **TikTok Shop 爆发**：2025 年增速 +53%，美妆品类占 28% GMV
