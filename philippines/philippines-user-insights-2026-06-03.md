# 菲律宾用户需求挖掘报告

**报告日期**：2026年6月3日  
**目标市场**：Philippines  
**数据来源**：竞品评论分析、社交媒体舆情、消费者行为数据

---

## Executive Summary

基于竞品评论、Q&A 数据和社交媒体分析，挖掘菲律宾消费者真实需求。核心发现：

1. **价格是第一决策因素**，但品质担忧是最大转化障碍
2. **社交证明极其重要**：72% 购买前查看 Review
3. **物流体验是复购关键**：配送速度和包裹完好性
4. **本地化是差异化核心**：尺码、语言、审美适配

## 五、用户决策旅程

### 5.1 购买决策流程

```
发现 → 研究 → 比较 → 决策 → 购买 → 收货 → 评价 → 复购
  ↓       ↓       ↓       ↓       ↓       ↓       ↓
社交    查看    比价    信任    支付    物流    分享
媒体    评价    平台    建立    方式    体验    推荐
```

### 5.2 各阶段关键影响因素

| 阶段 | 关键因素 | 数据来源 |
|------|---------|---------|
| **发现** | KOL推荐、平台推荐、搜索 | TikTok 60%, Shopee 25%, Google 15% |
| **研究** | Review数量/质量、图片、视频 | 平均查看 8-12 条评价 |
| **比较** | 价格、运费、促销 | 平均比较 3-5 个listing |
| **决策** | 信任信号（认证、销量、评分） | 评分 > 4.5 转化率 +40% |
| **购买** | 支付方式、COD选项 | GCash 45%, COD 25% |
| **收货** | 配送速度、包裹完好 | 期望 3-5 天 |
| **评价** | 产品符合预期 | 满意→好评率 65% |
| **复购** | 整体体验、客服 | 复购率 ~25% |
