## 详细合规分析

### #1 美白+防晒二合一乳霜 — 🔴 高风险

| 项目 | 详情 |
|------|------|
| **监管机构** | FDA (Food and Drug Administration) |
| **认证名称** | CPR (Certificate of Product Registration) |
| **法规依据** | FDA Act of 2009, AO No. 2020-0018 |
| **费用** | USD 500-1,500（含代理费） |
| **周期** | 3-6个月 |
| **要求** | ① 产品成分申报 ② SPF测试报告（ISO 24444） ③ 美白功效宣称需临床数据 ④ 英文/菲语标签 ⑤ 菲律宾代理人 |
| **成分限制** | 汞<1ppm, 氢醌禁用, 维A酸需处方 |
| **风险点** | 美白+防晒双重宣称，审批更严格 |
| **缓解方案** | ① 先以普通防晒注册，后续补充美白宣称 ② 使用已认证成分（熊果苷/VC） ③ 聘请当地FDA代理 |

| **费用** | — |
| **周期** | — |
| **建议** | 可自愿申请BPS认证增强信任 |

### #3 低糖胶原蛋白软糖 — 🔴 高风险

| 项目 | 详情 |
|------|------|
| **监管机构** | FDA |
| **认证名称** | LTO (License to Operate) + CPR |
| **费用** | USD 800-2,000 |
| **周期** | 4-8个月 |
| **要求** | ① 工厂GMP认证 ② 产品安全评估 ③ 营养成分标签 ④ 进口商LTO ⑤ 菲语标签 |
| **成分限制** | 胶原蛋白来源需声明（鱼/牛/猪），清真认证可选 |
| **风险点** | 食品补充剂审批严格，软糖剂型可能被归类为食品 |
| **缓解方案** | ① 先以普通食品注册 ② 避免功效宣称 ③ 选择已有LTO的进口商合作 |
