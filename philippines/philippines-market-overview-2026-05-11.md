# 菲律宾跨境电商宏观环境分析简报

**报告日期**：2026年5月11日  
**研究范围**：菲律宾跨境电商市场宏观环境  
**数据来源**：Google-Temasek-Bain (e-Conomy SEA 2025)、Cube.asia Tradewinds、Statista、DataReportal、Mordor Intelligence、Ken Research、CloudEcommerce、Anchanto、Portcalls、DHL、Traddal、CFGlobal、World Bank、IMF、PSA 等

---

## 执行摘要

菲律宾是**东南亚增长最快的数字经济体**。2024年电商GMV达 **$210亿**（CAGR ~23%），2025年预计达 **$220-360亿**（不同口径），2030年预计达 **$330-380亿**。1.1亿人口中互联网渗透率约 **84-89%**，日均上网时间近 **9小时**（全球最高之一），移动端占比超 **80%**。

然而，市场也面临隐忧：物流成本占GDP比重 **27.5%**（东盟最高），地理分散（7,641个岛屿）导致配送难度大，消费者价格敏感度高且COD占比仍达 **11%**。对于中国卖家而言，菲律宾市场意味着高增长机会，但需要做好本地化、物流和合规准备。

### 3.2 消费能力评估

- **海外汇款（OFW）**：约1000万海外劳工，年汇款超$300亿，是消费的重要支撑
- **消费信心**：总体积极，但对经济波动敏感

## 四、平台竞争格局

### 4.1 平台市场份额（FY2025，基于三大平台合计GMV）

| 平台 | GMV份额 | FY2025 GMV | YoY变化 | 核心定位 |
|------|---------|-----------|---------|---------|
| **Shopee** | **55%** | $120亿 | 稳定 | 综合电商领导者，高频低客单 |
| **TikTok Shop** | **29%** | $60亿 | **+53%** | 社交电商，美妆/时尚主导 |
| **Lazada** | **16%** | $30亿 | **-34%** | 品质/品牌电子，高客单 |
| **其他** | **~1%** | — | — | — |

> 三大平台合计占菲律宾平台电商GMV的 **98.8%+**（Momentum Works）

### 4.2 平台详细对比

#### Shopee — 市场领导者
- **GMV**：$120亿（FY2025），订单3.99亿单
- **客单价**：PHP 316（~$5.7）
- **用户年均订单**：11单（东南亚最高）
- **品类优势**：时尚(27%)、电子(24%)、美妆(14%)
- **核心优势**：物流网络最广、促销力度最大、用户习惯最强
- **Q4占比**：全年GMV的31%来自Q4
- **新动向**：2025年底推出 Instant Mart（4小时内送达）；2026年初要求卖家使用自有物流（J&T Express被移除）

#### TikTok Shop — 增长引擎
- **GMV**：$60亿（FY2025），+53% YoY
- **客单价**：PHP 198（~$3.6），仅为Shopee的63%
- **品类优势**：美妆(28%)、时尚(26%)、电子(12%)
- **核心优势**：直播带货转化率极高、Gen Z用户占比最高、内容发现→购买闭环
- **Q4占比**：全年GMV的35%来自Q4，季度份额已超Lazada
- **关键洞察**：美妆品类从Shopee向TikTok Shop迁移趋势明显

#### Lazada — 战略性收缩
- **GMV**：$30亿（FY2025），-34% YoY
- **客单价**：PHP 416（~$7.5），三平台最高
- **品类优势**：电子(26%)、品牌商品
- **核心优势**：LazMall正品保障、高客单价用户
- **挑战**：GMV持续下滑，Q4仅占三大平台合计GMV的10%
- **定位**：正转型为更专注的高端/品牌平台

### 4.3 平台费用对比（参考）

| 交易佣金 | 1-6% | 1-4% | 2-5% |

## 五、跨境进口政策与关税环境

### 5.1 关税制度

| 项目 | 详情 | 来源 |
|------|------|------|
| **关税制度** | ASEAN Harmonized Tariff Nomenclature (AHTN)，8位编码 | DHL / Traddal |
| **关税税率** | 0%-65%（视品类而定） | Traddal |
| **增值税（VAT）** | **12%**（适用于所有进口商品） | Traddal |
| **免税额度（De Minimis）** | **PHP 10,000（~$180）** FOB/FCA价值 | Traddal |
| **计税基础** | CIF价值（成本+保险+运费） | DHL |
| **消费税** | 酒精、烟草、石油等特定品类加征 | Traddal |

### 5.2 进口税计算方式

```
关税 = CIF价值 × 关税税率
VAT = (CIF价值 + 关税 + 消费税) × 12%
总到岸成本 = CIF价值 + 关税 + 消费税 + VAT + 报关费 + 处理费
```

**免税规则**：
- FOB/FCA价值 ≤ PHP 10,000：**免关税 + 免VAT**
- 注意：BOC对同一收件人同一天的多件低价值包裹实行**合并计算**，合计超PHP 10,000则全部征税

### 5.3 跨境电商新规（2025年关键变化）

**海关行政令 CAO No. 01-2025**（2025年1月28日签署）：

| 要求 | 详情 |
|------|------|
| **强制认证** | 所有电商参与者（平台、卖家、物流商、报关行等）必须向BOC申请认证 |
| **电商处理系统（EPS）** | 建立专门的电商清关系统，仅认证参与者可使用 |
| **数据共享** | BOC可与平台提供商共享数据，用于风险管理和统计分析 |
| **预付费账户** | 认证货代需在BOC开设预付费账户，保证税费即时缴纳 |
| **风险管理系统** | 建立动态风险管理系统，利用先进技术识别高风险货物 |
| **禁止品类** | 违禁品即使通过电商渠道也不得进口 |
| **排除范围** | B2B交易、C2C交易、无商业价值物品不适用电商清关程序 |

**互联网交易法（ITA, RA 11967, 2023年12月）**：
- 设立电商局（E-Commerce Bureau）
- 明确消费者、商家、平台三方权利义务
- 建立在线争议解决机制
- 要求平台对卖家进行资质审核

### 5.4 自由贸易协定（FTA）红利

菲律宾签署的主要FTA：

| FTA | 覆盖范围 | 关税优惠 |
|-----|---------|---------|
| **ATIGA**（东盟货物贸易协定） | 东盟成员国 | 绝大多数商品0-5%关税 |
| **ACFTA**（中国-东盟FTA） | 中国+东盟 | Normal Track商品已零关税 |
| **RCEP** | 15个亚太成员国 | 渐进式关税减免 |
| **PHL-KR FTA**（韩国） | 韩国 | 2024年12月31日生效 |
| **AANZFTA** | 澳新+东盟 | 90%+税目逐步零关税 |
| **AHKFTA** | 香港+东盟 | 珠宝、服装、玩具等减免 |

> **中国卖家注意**：通过ACFTA和RCEP，符合原产地规则的中国商品可享受优惠关税。需提供 **Form D（ATIGA）** 或 **Form RCEP** 等原产地证书。

### 5.5 合规要求清单

| 要求 | 适用品类 | 负责机构 |
|------|---------|---------|
| BOC认证 | 所有跨境电商参与者 | 海关总署 |
| FDA注册 | 食品、药品、化妆品、医疗器械 | FDA Philippines |
| ICC认证 | 电子产品、家电 | DTI-BPS |
| 进口许可证 | 特定品类 | DTI |
| 原产地证书 | FTA优惠关税 | 贸促会/海关 |

> **数据来源**：Portcalls / BOC CAO 01-2025；DHL Philippines；Traddal；Trade.gov

## 六、物流基础设施

### 6.1 物流市场概况

| 指标 | 数值 | 来源 |
|------|------|------|
| 物流市场总规模（2024） | **~$550-720亿** | Ken Research / Insight Pro Analytics |
| 物流市场CAGR | **6.3-8%** | 综合估算 |
| 物流成本占GDP比重 | **27.5%**（东盟最高） | CFGlobal |
| CEP市场规模（2025） | **$13.3亿** | Mordor Intelligence |
| CEP市场CAGR（2026-2031） | **7.11%** | Mordor Intelligence |

### 6.2 物流挑战

**地理挑战**：
- **7,641个岛屿**，跨岛运输依赖海运+空运+陆运多式联运
- 80%国内货运通过海运
- 港口拥堵（马尼拉港为核心瓶颈）

**成本挑战**：
- 物流成本占GDP **27.5%**，远高于东盟平均水平
- 最后一公里配送成本占订单价值的 **20-30%**（Blackbox Research）
- 地址系统不完善，GPS定位不准确

**时效挑战**：
- 大马尼拉区：1-2天
- 其他吕宋岛城市：2-4天
- 维萨亚斯/棉兰老：3-7天
- 偏远岛屿：7-14天

### 6.3 物流改善趋势

| 领域 | 进展 |
|------|------|
| **港口现代化** | 2024年81个港口项目；Batangas和Subic港分流马尼拉压力 |
| **铁路货运** | Subic-Clark-马尼拉铁路货运线规划中 |
| **同日达** | 2025年覆盖15个主要城市（2024年为8个） |
| **自动化仓储** | 履约中心容量增加40% |
| **区域仓** | 宿务、达沃、伊洛伊洛、卡加延德奥罗设立区域仓 |
| **海关清关** | NAIA和Clark机场跨境清关从5天缩短至2天 |
| **微型履约中心** | 城市内MFC缩短最后一公里 |

### 6.4 主要物流服务商

| 类型 | 主要玩家 | 特点 |
|------|---------|------|
| 传统快递 | LBC Express（2,800+网点）、2GO Express（95%覆盖） | 覆盖最广 |
| 电商物流 | Ninja Van、J&T Express、Shopee Express | 电商绑定 |
| 即时配送 | Lalamove、GoGoX | 大马尼拉/宿务/达沃同日达 |

| 支付方式 | 占比（2025-2026） | 趋势 | 来源 |

### 7.2 支付生态分析

**GCash（主导者）**：
- 菲律宾最大电子钱包，用户超7,600万
- 覆盖线上线下全场景
- 提供 GCredit（消费信贷）、GSave（储蓄）、GInvest（投资）
- 政府推动的 Higala（IIPS）即时支付系统

**Maya（原PayMaya）**：
- 第二大电子钱包
- 提供 Maya Credit（分期）、Maya Bank
- 线下商户覆盖广

**ShopeePay**：
- Shopee生态内闭环支付
- 提供 SPayLater（先买后付）
- 在Shopee交易中占比约40%

**COD（货到付款）**：
- 占比从2023年的~20%降至2025年的~11%
- 主要存在于农村地区和高龄人群
- **风险**：退货率高、现金流压力、欺诈风险

### 7.3 BNPL（先买后付）

| 平台 | BNPL产品 | 特点 |
|------|---------|------|
| Shopee | SPayLater | Shopee生态内，中期采用率达"mid-teens" |
| GCash | GCredit | 消费信贷 |
| Maya | Maya Credit | 消费信贷 |
| 独立平台 | Atome, BillEase | 跨平台使用 |

### 7.4 对跨境电商的支付启示

1. **必须接入 GCash + Maya**，覆盖80%+数字支付用户
2. **提供分期付款**选项（尤其电子、家居品类）
3. **COD可作为补充**，但不应作为主要方式（高退货率）
4. **ShopeePay**在Shopee平台内几乎是必选项

## 八、跨境电商增长驱动因素与障碍

### 8.1 增长驱动因素

| 驱动因素 | 描述 | 影响程度 |
|----------|------|---------|
| **人口红利** | 1.1亿人口，中位年龄26岁，数字化原住民 | ⭐⭐⭐⭐⭐ |

| **数字支付普及** | 电子钱包占比64%且持续增长 | ⭐⭐⭐⭐ |
| **中产阶级扩张** | 消费能力提升，品质需求增加 | ⭐⭐⭐⭐ |
| **政府支持** | ITA 2023法律框架、数字转型政策 | ⭐⭐⭐ |
| **海外汇款** | 年汇款$300亿+，支撑消费 | ⭐⭐⭐ |
| **物流改善** | 同日达覆盖扩大、履约中心增加 | ⭐⭐⭐ |
| **FTA红利** | ACFTA/RCEP为中国卖家提供关税优惠 | ⭐⭐⭐ |

### 8.2 增长障碍

| 食品/保健品 | ★★★☆☆ | ★★★★ | **P2**（FDA注册周期长） |

#### 进入策略建议

**阶段一：市场验证（0-6个月）**
- 通过 Shopee 跨境店试水，验证品类需求和定价
- 小规模 TikTok Shop 直播测试
- 接入 GCash + Maya + ShopeePay
- 与本地物流商合作（Ninja Van、J&T Express）

**阶段二：规模扩张（6-18个月）**
- 设立海外仓（大马尼拉区）
- 多平台布局（Shopee + TikTok Shop + Lazada）
- 组建本地运营团队或深度合作代理
- 启动 ICC/FDA 认证（如适用）

**阶段三：品牌深耕（18个月+）**
- 设立菲律宾本地公司
- 品牌建设和用户运营
- 拓展至宿务、达沃等二线城市
- 探索 O2O 模式

#### 关键成功因素

1. **移动优先**：所有listing、图片、checkout流程必须移动端优化
2. **直播带货能力**：TikTok Shop 直播是美妆/时尚的必争之地
3. **价格竞争力**：菲律宾消费者对价格极度敏感
4. **物流体验**：海外仓是提升时效和降低退货率的关键
5. **合规先行**：BOC认证、FDA注册、ICC认证等提前规划
6. **Q4聚焦**：Q4占全年GMV约31-35%，必须提前备货和营销

## 九、总结与市场吸引力评估

| 维度 | 评分（1-5） | 说明 |
|------|-----------|------|
| 市场规模 | ⭐⭐⭐⭐ | $210亿（2025），东南亚第四 |

| 竞争强度 | ⭐⭐⭐⭐ | 三强鼎立，TikTok快速崛起 |
| 进入门槛 | ⭐⭐⭐ | 监管趋严但de minimis较高（PHP 10,000） |
| 物流便利性 | ⭐⭐ | 地理分散，成本高，持续改善中 |

| 政策友好度 | ⭐⭐⭐ | FTA红利存在，但监管趋严 |

### 核心结论

1. **菲律宾是东南亚跨境电商的高增长市场**——最快增速、最年轻人口、最高互联网使用时长
2. **TikTok Shop 是最大变量**——53%的GOV增速正在重塑品类格局，美妆/时尚品牌必须入场
3. **物流是最大瓶颈**——27.5%的物流成本占比意味着海外仓和物流优化是核心竞争力
4. **中国卖家有结构性优势**——ACFTA/RCEP关税优惠 + 供应链优势，但需做好本地化和合规
5. **建议行动**：立即启动 Shopee + TikTok Shop 双平台运营，同步规划海外仓和合规认证
