# 🇵🇭 菲律宾跨境电商宏观环境分析

> **报告日期**：2026-05-29  
> **编制**：趋势研究员 (product-trend-researcher)  
> **数据覆盖**：2024-2026 年公开数据  
> **数据来源**：Momentum Works 2025、Cube.asia Tradewinds、DataReportal Digital 2025、Mordor Intelligence、e-Conomy SEA、Statista、BSP（菲律宾央行）

## 摘要

菲律宾是东南亚**增长最快的数字经济体**，2025 年数字经济增长 16% 至 $360 亿 GMV。三大电商平台（Shopee、Lazada、TikTok Shop）占据 98.8% 的市场份额，市场已进入高度集中的"三强格局"。Shopee 以 55% 的份额绝对领先，TikTok Shop 以 53% 的年增速快速追赶并已超越 Lazada 成为第二。菲律宾拥有东南亚**最高的社交媒体使用率**和**最强的直播电商接受度**，但也面临群岛物流成本高、COD 退货率大、人均消费力偏低的结构性挑战。

**核心判断**：菲律宾是值得进入的市场，但需要完全适应其"移动优先 + 直播驱动 + COD 主导"的独特消费生态。

**数据来源**：Cube.asia Tradewinds（FY2025，截至 2025 年 12 月）

### 2.2 平台画像对比

| 维度 | Shopee | TikTok Shop | Lazada |
|:----|:------|:-----------|:------|
| **份额趋势** | ↑ 55%（2024: 50%） | ↑ 29%（2024: 22%） | ↓ 16%（2024: 28%） |
| **客单价** | PHP 316（中） | PHP 198（低） | PHP 416（高） |
| **Top 品类** | 时尚 27%、电子 24%、美妆 14% | 美妆 28%、时尚 26%、电子 12% | 电子 26%、时尚 17% |
| **核心驱动** | 大促驱动（9.9/11.11/12.12） | 直播 + 算法推荐 | LazMall 品牌信任 |
| **物流** | Shopee Xpress + 第三方 | TikTok 物流（合作） | Lazada 物流 + 第三方 |
| **优势** | 高频习惯 + 最大流量池 | 直播转化率最高 | 高客单价 + 正品信任 |
| **劣势** | 品类流量正在流失给 TikTok | 客单价低，退换率高 | GMV 持续萎缩 |

### 2.3 关键趋势

- **Shopee 不可撼动但正在失守美妆阵地**：Shopee 仍占 55% 的市场份额，且用户习惯极强（年均 11 单），但美妆品类 GMV 份额从 2023 年的 17% 降至 2025 年的 14%，被 TikTok Shop 的直播带货分流。
- **TikTok Shop 增速惊人**：+53% YoY，是东南亚所有国家中 TikTok Shop 增长最快的市场之一。Q4 2025 已超越 Lazada 成为季度第二大平台（占 Q4 GMV 的 35%）。
- **Lazada 加速萎缩**：-34% YoY，份额从 28% 跌至 16%。正在退化为"高客单价品牌专用渠道"，但这一策略能否稳住基本盘尚不确定。
- **Temu 入场**：2024-2025 年进入菲律宾，但尚未形成实质性威胁。Temu 的低价策略与 TikTok Shop 的冲动消费模式有重叠。

## 4. 物流基础设施

### 4.1 物流概况

| 维度 | 现状 | 挑战 |
|:----|:-----|:-----|
| **群岛地理** | 7,641 个岛屿 | 岛屿间物流成本极高 |
| **国际物流** | 马尼拉港、NAIA 空运为主 | 清关效率中等 |
| **最后一公里** | J&T Express 市占率最高，Flash Express、LBC 紧随 | 外岛配送 5-10 天 |
| **电商自有物流** | Shopee Xpress（SPX Express）快速增长 | 覆盖范围仍在扩展 |
| **仓储设施** | 大马尼拉区仓库集中，外岛仓储匮乏 | 多地分仓成本高 |
| **冷链接力** | 极不成熟 | 生鲜/冷冻品类受限 |

### 4.2 主要物流商

| 物流商 | 特点 | 电商合作 |
|:------|:----|:---------|
| **J&T Express** | 覆盖最广，价格适中，但 Shopee 2026 年初已通知卖家不可再选择 J&T | 曾是 Shopee 主力，现被排除 |
| **Shopee Xpress（SPX）** | Shopee 自建物流，增长最快 | 仅限 Shopee 卖家 |
| **Flash Express** | 价格竞争型，覆盖中等 | Shopee、Lazada 均可 |
| **LBC Express** | 老牌物流，群岛覆盖最广 | 多平台 |
| **2GO / DHL / FedEx** | 高端/重货物流 | 跨境大件 |

### 4.3 物流挑战

- **群岛配送成本高**：从马尼拉到偏远岛屿（如棉兰老岛、巴拉望），最后一公里成本可达城市配送的 3-5 倍
- **COD 退货率高**：菲律宾 COD 比例约 30-40%，退货带来的逆向物流成本是重要负担
- **配送时效分层严重**：大马尼拉 1-3 天，主要城市 3-5 天，偏远地区 5-10 天
- **Shopee 2026 年物流整合**：排除 J&T 意味着卖家物流选择减少，佣金+物流费用可能上升

## 5. 支付生态

### 5.1 支付结构

| 支付方式 | 占比 | 趋势 |
|:--------|:---:|:----|
| **COD（货到付款）** | ~30-40% | 缓慢下降，但仍是主力 |
| **GCash** | ~25-30% | 快速增长，市占率最高 |
| **Maya（原 PayMaya）** | ~10-15% | 稳定增长 |
| **ShopeePay / SPayLater** | ~10-12% | Shopee 生态内快速渗透 |
| **银行卡（信用卡/借记卡）** | ~5-8% | 渗透率低 |
| **银行转账 / InstaPay** | ~5% | 数字化趋势 |

**数据来源**：BSP 2024 报告 + Cube.asia 估算

### 5.2 数字支付生态

| 指标 | 数值 |
|:----|:-----|
| 数字支付占总零售交易额比例（BSP 2024） | **57.4%（笔数）/ 59.0%（金额）** |
| 数字钱包市场份额（GCash + Maya） | >95% 的电子钱包市场 |
| GCash 用户数 | >9,400 万（2025 年） |
| Maya 用户数 | ~5,000 万 |
| Shopee 自有支付占比（2025） | ~40%（含 SPayLater 中双位数） |
| 信用卡渗透率 | <5%（东南亚最低之一） |

### 5.3 支付特征

- **COD 仍是最大痛点也是最大机会**：菲律宾是东南亚 COD 比例最高的市场之一。虽然数字支付增长迅速，但 COD 的退出速度远慢于印尼和泰国。
- **GCash 一枝独秀**：已从电子钱包演变为数字银行，提供理财、贷款、保险等综合金融服务。电商场景中 GCash 是 Shopee 和 TikTok Shop 的主要数字支付选项。
- **BNPL（先买后付）快速兴起**：SPayLater（Shopee）和 GGives（GCash）推动年轻用户从 COD 转向分期付款。
- **ShopeePay 的闭环优势**：Shopee 正在推动用户从 COD 转向 ShopeePay/SPayLater，以提高交易数据可见性和降低退货率。

## 6. 政策与法规环境

| 政策 | 内容 | 影响 |
|:----|:-----|:----|
| **EO 系列（电商行政令）** | 促进电商发展，简化注册流程 | 利好 |
| **CREATE MORE 法案** | 降低企业税负（从 30% → 25%） | 利好 |
| **数据隐私法（DPA）** | 严格个人信息保护要求 | 合规成本 |
| **关税门槛** | $0 以下免关税 → 2024 年起收紧低价值进口关税 | 对跨境低价模式有影响 |
| **数字服务税** | 正在讨论中 | 潜在影响 |

> 菲律宾的政策环境相比印尼要**友好得多**——没有 $100 最低限价、没有社交电商禁令、没有强制 Halal 认证。这是菲律宾对跨境卖家最大的吸引力。

**关键洞察**：

## 8. 跨境卖家的进入建议

### ✅ 菲律宾的优势

| 优势 | 说明 |
|:----|:-----|
| **增速最快** | 2025 年东南亚增速最高的电商市场 |
| **政策友好** | 无 $100 限价、无社交电商禁令、无强制 Halal |
| **英语普及率高** | 英语是官方语言之一，内容/客服本地化成本低 |
| **直播电商爆发** | 直播转化率东南亚最高 |
| **年轻人口红利** | 中位年龄 26 岁，数字消费意愿强 |

### ⚠️ 菲律宾的挑战

| 挑战 | 说明 |
|:----|:-----|
| **人均消费力低** | 客单价极低（$3-7），利润空间薄 |
| **COD 比例高** | 30-40% COD 退货率影响利润率 |
| **群岛物流成本高** | 外岛配送成本高、时效长 |
| **Shopee 物流排他** | J&T 被排除后物流选择减少 |
| **数字支付渗透仍在爬坡** | 虽有 GCash 但 COD 仍是主流 |

### 🎯 建议策略

1. **主攻 Shopee + TikTok Shop 双渠道**
   - Shopee：大促驱动，做 GMV 规模
   - TikTok Shop：直播驱动，做品牌曝光和品类突破

2. **聚焦中低客单价但可打包提价的品类**
   - 菲律宾不适合纯低价白牌（Temu/Shein 已打穿），也不适合纯高端（市场太小）
   - 最佳策略：$10-30 区间的品牌化产品

3. **优化 COD 退货策略**
   - 使用 ShopeePay/SPayLater 降低 COD 比例
   - 建立本地退货仓（大马尼拉区域）

4. **利用英语优势降低本地化成本**
   - 相比印尼（需要 Bahasa Indonesia），菲律宾的英语环境大幅降低内容成本
