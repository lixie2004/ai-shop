## 4. 搜索趋势洞察

### 4.1 上升趋势
1. **"tinted sunscreen"** — 全年搜索，夏季峰值，双功能需求
2. **"lip oil"** — 自2020年持续上升，Gen Z 驱动
3. **"malunggay capsule"** — 本土草药复兴趋势
4. **"aesthetic organizer"** — 社交媒体美学驱动
5. **"collagen gummy"** — 口服美容新兴趋势

### 4.2 季节性波动
1. **大促期（9.9/11.11/12.12）** — 全品类搜索量翻倍
2. **夏季（3-5月）** — 防晒、泳装、冷饮相关搜索激增
3. **圣诞季（10-12月）** — 礼品、装饰、食品搜索高峰
4. **开学季（6-8月）** — 文具、电子、服装需求上升

### 4.3 平台搜索差异
| 平台 | 搜索特征 | 热门品类 |
|------|---------|---------|
| **Shopee** | 价格敏感型搜索多，"cheap/affordable/discount" 高频 | 时尚、电子、家居 |
| **TikTok Shop** | 发现型搜索多，"trending/viral/review" 高频 | 美妆、健康、宠物 |
| **Lazada** | 品牌型搜索多，"original/authentic/official" 高频 | 电子、品牌服饰 |
