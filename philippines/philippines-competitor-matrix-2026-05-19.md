## 4. 竞品 SWOT 总结

### Shopee 平台卖家
| 维度 | 分析 |
|------|------|
| **优势** | 流量最大、物流体系完善、支付生态成熟 |
| **劣势** | 价格战激烈、利润空间压缩、同质化严重 |
| **机会** | Instant Mart 即时配送、BNPL 提升转化 |
| **威胁** | TikTok Shop 持续蚕食份额 |

### TikTok Shop 平台卖家
| 维度 | 分析 |
|------|------|
| **优势** | 增速最快、内容驱动转化、美妆品类强势 |
| **劣势** | 客单价低、退货率高、内容制作门槛 |
| **机会** | 直播带货红利、宠物/健康蓝海 |
| **威胁** | 平台政策变化、内容竞争加剧 |

### Lazada 平台卖家
| 维度 | 分析 |
|------|------|
| **优势** | 品牌信任度高、客单价高、正品保障 |
| **劣势** | GMV 持续下滑、流量减少 |
| **机会** | 品牌化转型、高端市场 |
| **威胁** | 被 TikTok Shop 超越、卖家流失 |

## 5. 竞争空白点识别

| 空白点 | 机会描述 | 建议品类 |
|--------|---------|---------|
| **有机/天然定位** | 大多数竞品缺乏有机认证 | 健康保健、美妆 |
| **宠物高端市场** | 有机宠物食品几乎空白 | 宠物用品 |
| **男士护肤** | 菲律宾男士护肤市场被忽视 | 美妆个护 |
| **可持续家居** | 环保家居产品供给不足 | 家居收纳 |
| **大码时尚** | 大码服装选择有限 | 时尚服饰 |
| **敏感肌专用** | 针对敏感肌的产品稀缺 | 美妆个护 |
