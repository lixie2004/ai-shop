## 计算公式

```
CIF价值 = 采购成本 + 头程物流
完税价格 = CIF价值 + 进口关税
总成本 = 完税价格 + VAT(12%) + IPF + 平台佣金 
       + 交易费 + 广告费 + 尾程配送 + 退货损耗 
       + 仓储费 + 汇率缓冲 + 包装耗材 + 认证分摊

综合毛利率 = (售价 - 总成本) / 售价 × 100%

净利率 = (售价 - 总成本 - 企业所得税) / 售价 × 100%
```

## 风险声明

1. **汇率风险**：PHP对USD 2026年贬值约4.6%（年初57.5→5月61.8），是所有SKU需持续关注的风险。模型中已预留1%缓冲。
2. **平台费率变动风险**：菲律宾Shopee目前尚未新增5%技术服务费（MY/VN/TH已实施），但预计可能在2026下半年跟进。建议预留2%弹性空间。
3. **FDA认证周期**：化妆品通知需2-4个月，宠物食品注册需3-6个月。认证期间无法销售。
4. **群岛物流成本**：菲律宾为群岛国家，尾程配送费高于马来西亚/泰国/越南约20-30%。低客单价SKU受冲击最大。
5. **本模型为理论测算**，实际毛利率受库存周转、季节波动、平台促销力度、退货率波动等多因素影响。建议每季度更新一次成本模型。
