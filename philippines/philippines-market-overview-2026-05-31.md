## 三、消费者画像与消费力

### 3.1 基础画像

| 指标 | 数值 |
|------|------|
| 人口 | ~1.15 亿（2025） |
| 中位年龄 | ~25 岁 |
| 互联网渗透率 | ~73-78% |
| 智能手机渗透率 | ~73.7%（2023），预计 2025 > 78% |
| GCash 活跃用户 | **8100 万**（2025 年初） |
| Maya (PayMaya) | 数百万用户，第二大电子钱包 |

### 3.2 消费行为特征

- **移动优先**: 绝大部分 Shopee 交易来自移动端
- **直播购物高采纳**: 菲律宾是东南亚直播购物渗透率最高的国家之一
- **活动驱动**: 9.9、11.11、12.12大促集中全年 GMV 的大部分
- **支付方式**: 
  - GCash（主导）+ Maya + SPayLater + COD（仍占显著份额）
  - Shopee 原生支付方式占交易量 ~40%
  - SPayLater（先买后付）渗透率达两位数
- **COD 仍重要**: 尽管数字钱包增长，COD 在跨境场景仍占相当比例

## 四、跨境电商政策环境

### 4.1 关键法规

| 政策 | 内容 |
|------|------|
| **《互联网交易法》** (RA 11967, 2023) | 规范跨境电商，保护消费者权益 |
| **海关行政令 CAO 01-2025** | 跨境电商进口标准清关流程 |
| **CMTA（海关现代化与关税法）** | 主要关税法规 |
| **WTO 绑定关税** | 67.6% 税目绑定，平均 24.9% |

### 4.2 进口关税与税费

| 项目 | 税率/规则 |
|------|----------|
| 进口关税 | 0-30% 不等（依品类） |
| VAT | **12%** |
| DTI/SEC 注册 | 境外卖家需在菲律宾注册 |

### 4.3 合规要点

- 跨境包裹 **USD 50 以下**通常免税额（需确认最新政策）
- 需提供 **进口许可证** 的商品清单定期更新
- **FDA 认证** 对美妆、食品、保健品强制要求
- **BOC（海关总署）** 加强跨境电商监管

## 六、市场机会总结

### 6.1 高潜力方向

| 机会 | 理由 |
|------|------|
| **TikTok Shop 美妆** | 28% GMV 份额，+53% 增速，直播驱动 |
| **Shopee 时尚** | 27% 份额，高频复购 |
| **健康保健** | 恢复增长（8%），后疫情需求 |
| **家用消费电子** | Shopee 24% + Lazada 26% 品牌正品需求 |
| **即时配送** | Shopee Instant Mart 带动高客单价 |

### 6.2 风险提示

| 风险 | 说明 |
|------|------|
| **Lazada 衰退** | 避免将资源过度集中在 Lazada |
| **COD 现金流压力** | 跨境场景 COD 回款慢 |
| **关税波动** | 电子产品关税较高 |
| **FDA 认证周期** | 美妆/食品类 12-16 周认证 |
| **Shopee J&T 退出** | 物流切换期可能影响配送时效 |
