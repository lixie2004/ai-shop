## Ⅲ. 跨品类修饰词总览 & 分层策略

| 层级 | 修饰词 | 适用品类 |
|------|--------|----------|
| **价格层** | cheap、affordable、murang、best price、sale、discount | 所有品类 |
| **信任层** | korean、authentic、original、BPOM（健康）、certified | 美妆、健康、厨房（品牌） |
| **功能层** | waterproof、portable、collapsible、non‑stick、high potency、probiotic | 美妆、厨房、家居收纳 |
| **场景层** | for summer、for home、for travel、for office、for skin type | 服饰、厨房、家居 |
| **平台层** | shopee、lazada、tokopedia、tiktok shop | 所有品类（平台泄漏） |
| **否定层** | free、how to、tutorial、recipe、DIY、second hand、ukay、rent、hiring、download | 所有品类（信息/低转化） |

> **策略**：
> 1. **价格层** → 交易系列词组匹配，低出价；
> 2. **信任层** → 高出价、文案突出 K‑Beauty、BPOM、认证。
> 3. **功能层** → 用于长尾细分，提升相关度。
> 4. **场景层** → 细分广告组（夏季、防晒、旅行）。
> 5. **平台层** → 账户级否定，阻止平台词泄漏。
> 6. **否定层** → 账户级统一否定，防止无效点击。

## Ⅶ. 关键结论
1. **交易意图占比近 50%**，意味着关键词层级（精准 + 词组）是核心驱动。
2. **价格敏感词（affordable、cheap、murang）** 在所有品类均表现出高转化，建议用词组匹配并设置分层出价。
3. **Taglish 低竞争高意图**（如 *pampaputi ng balat*、*murang storage box*）应提前布局内容页并配合再营销。
4. **Pay‑day 与 11.11 双高峰**，必须在每月 12‑15 日及 11 月 8‑15 日前提升出价。
5. **平台泄漏（Shopee/Lazada/TikTok）** 仍是主要浪费来源，务必在账户级部署平台词否定。
6. **K‑Beauty 与 Health Supplements** 为跨品类信任信号，广告文案需突出 “Korean”, “BPOM certified”, “high potency”。
