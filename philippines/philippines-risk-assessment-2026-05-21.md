### 2.2 关键成本压力全景

菲律宾市场与东南亚其他市场的**关键差异**在于：
1. **FORM E 不适用** → 所有 SKU 均按 MFN 关税（0-15%），不像印尼/马来西亚/越南可享受零关税
2. **COD 拒收率 5-15%** → 显著高于新加坡（<5%）
3. **物流岛屿化** → 大件商品尾程配送成本占比 20-29%
4. **服装关税 15%** → 直接导致 2 个服装 SKU 亏损或微利

## 七、方案可行性评估结论

### 7.1 核心结论

| 指标 | 数值 |
|:----|:----:|
| **方案可行性评分** | **~6.2/10** |
| Go 优先 SKU | **1 个**（#9 多功能一体锅） |
| Go 条件 SKU | **6 个**（#3/#4/#5/#6/#7/#10） |
| 暂缓 SKU | **1 个**（#8 家居收纳 8件套） |
| No-Go SKU | **2 个**（#1 防晒乳霜、#2 收纳箱 3件套） |
| 整体利润质量 | 🟡 中偏弱（平均净利率 ~5.6%，5 个 ≤ 5% 或亏损） |
| 合规负担 | 🟡 中（4 个需认证 SKU，3 个快速+2 个长周期） |
| 市场适配度 | 🟡 中（缺乏高利润零认证的爆品） |

### 7.2 方案核心问题

1. **5 个 SKU 净利率 ≤ 5% 或亏损（50%）**——一半 SKU 利润过低或亏损
2. **FORM E 不适用导致关税不可减免**——与印尼/马来西亚/越南相比，这是菲律宾市场的结构性劣势
3. **大件物流成本占比过高**——菲律宾岛屿地理导致大件商品物流成本是其他市场的 1.5-2 倍
4. **COD 拒收率 5-15%**——高 COD 占比+高拒收率是菲律宾特有的运营风险
5. **缺乏高利润零认证 SKU**——最佳 SKU #9 多功能锅净利率 15% 但需 BPS 认证 6-12 周

### 7.3 最终建议

> **方案可行性 ~6.2/10，低于新加坡（7.5）和马来西亚（6.8），与印尼（5.8）和越南（6.3）接近。**
>
> **菲律宾市场的核心挑战不是需求不足，而是成本结构恶劣**——FORM E 不适用 + 岛屿物流 + COD 高拒收率三重压力导致 50% 的 SKU 利润过低或亏损。
>
> **建议执行策略：**
>
> 1. **Phase 1 聚焦 2 个电器 SKU（#9/#5）**——净利率 13-15% 是全场唯一健康利润的来源，共用 BPS-PS License 降低认证成本
> 2. **Phase 2 用低成本 SKU 做品牌基础**——#4 T恤（零认证）和 #7 精华（2-5 月认证）作为品牌建设的基础品类
> 3. **Phase 3 保健品 SKU 是长期引擎**——#3 胶原蛋白和 #6 维生素 C 需提前 8-18 个月启动 FDA 认证
> 4. **必须做好 COD 管理**——开通 GCash/PayMaya/SPayLater 等电子支付，目标将 COD 占比从 40-60% 降至 25-30%
> 5. **优化大件物流**——初期仅覆盖 Luzon 岛（大马尼拉+周边），使用本地仓降低尾程配送成本
>
> **与东南亚其他市场对比：** 新加坡 7.5 > 马来西亚 6.8 > 菲律宾 6.2 ≈ 越南 6.3 > 印尼 5.8
