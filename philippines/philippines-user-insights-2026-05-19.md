## 1. 用户画像

### 1.1 核心用户画像

**画像 A：年轻都市女性（25-34 岁）— 最大消费群体**
- **占比**：约 35% 的线上消费者
- **收入**：PHP 15,000-35,000/月
- **居住**：大马尼拉/宿务/达沃等城市
- **行为**：社交媒体重度用户，K-Beauty 追随者，直播购物活跃
- **痛点**：选择过载、假货担忧、配送延迟
- **偏好**：TikTok Shop 发现 → Shopee 比价购买
- **关键词**：aesthetic, Korean, affordable, hiyang (适合)

**画像 B：新手妈妈（28-38 岁）— 高复购群体**
- **占比**：约 20% 的线上消费者
- **收入**：家庭收入 PHP 25,000-50,000/月
- **行为**：注重产品安全性，品牌忠诚度高，批量购买
- **痛点**：产品质量、安全认证、性价比
- **偏好**：Shopee 为主，重视评价和评分
- **关键词**：safe, organic, for baby, trusted brand

**画像 C：年轻男性（22-30 岁）— 增长最快群体**
- **占比**：约 18% 的线上消费者
- **行为**：电子/游戏/运动品类为主，价格敏感
- **痛点**：缺乏男士专属产品选择
- **偏好**：Shopee/Lazada，搜索型购买
- **关键词**：original, durable, gaming, sports

**画像 D：健康意识人群（30-50 岁）— 高客单群体**
- **占比**：约 15% 的线上消费者
- **收入**：PHP 30,000+/月
- **行为**：健康补充剂、有机食品、健身器材
- **痛点**：产品真实性、功效验证
- **偏好**：品牌官网/Lazada，信息型购买
- **关键词**：organic, natural, herbal, authentic

**画像 E：宠物主人（25-40 岁）— 新兴群体**
- **占比**：约 12% 的线上消费者
- **行为**：宠物拟人化消费，愿意为宠物花更多钱
- **痛点**：优质宠物产品选择少、价格高
- **偏好**：Shopee/TikTok Shop，内容驱动
- **关键词**：organic pet, durable, cute, safe

## 3. 好评关键词分析

### 3.1 各品类高频好评词

| 品类 | 好评关键词 | 出现频率 |
|------|-----------|---------|
| 美妆 | "effective", "gentle", "hiyang", "glowing", "affordable" | 极高 |
| 健康 | "boost energy", "feel better", "natural", "no side effects" | 高 |
| 时尚 | "trendy", "comfortable", "good quality", "fits well" | 高 |
| 电子 | "fast charging", "long battery", "original", "worth it" | 极高 |
| 家居 | "space-saving", "aesturdy", "easy to install", "organized" | 中高 |
| 宠物 | "my pet loves it", "durable", "good value", "cute" | 中 |

### 3.2 好评驱动因素
1. **超出预期**："比想象中好" — 最大的好评来源
2. **性价比**："这个价格买到这个质量很值"
3. **配送快**："第二天就到了"
4. **包装好**："包装很用心，没有损坏"
5. **客服好**："客服回复很快，问题解决了"

## 4. 差评痛点分析

### 4.1 各品类高频差评词

| 品类 | 差评关键词 | 出现频率 | 根本原因 |
|------|-----------|---------|---------|
| 美妆 | "fake", "allergic", "not effective", "different from photo" | 高 | 假货+过度宣传 |
| 健康 | "no effect", "bad taste", "expired", "fake" | 中 | 功效不达预期 |
| 时尚 | "wrong size", "color different", "poor quality", "cheap material" | 高 | 尺码不准+图片失真 |
| 电子 | "not original", "stopped working", "slow charging" | 高 | 假货+质量差 |
| 家居 | "flimsy", "hard to install", "smaller than expected" | 中 | 质量+描述不符 |
| 宠物 | "pet won't eat", "broke easily", "wrong size" | 中 | 适口性+质量 |

### 4.2 差评驱动因素
1. **假货/非正品**（28%）— 最大差评来源
2. **配送延迟**（22%）— 第二大差评来源
3. **商品与描述不符**（18%）— 图片/规格失真
4. **质量差**（15%）— 材料/做工问题
5. **尺寸不准**（10%）— 特别是服装
6. **客服差**（7%）— 响应慢/不解决

## 6. 购买决策路径

```
发现 → 搜索 → 比较 → 信任建立 → 购买 → 收货 → 评价
  │        │        │          │         │       │
  │        │        │          │         │       └── 好评→复购
  │        │        │          │         │
  │        │        │          │         └── COD/GCash
  │        │        │          │
  │        │        │          └── 评价+评分+正品认证
  │        │        │
  │        │        └── 价格+运费+促销
  │        │
  │        └── Shopee搜索/Google/TikTok
  │
  └── TikTok直播/Instagram/Facebook
```

### 关键转化节点
1. **发现→搜索**：TikTok 内容种草是关键入口
2. **搜索→比较**：价格+评价+运费是核心比较维度
3. **比较→信任**：正品认证+高评分+详细图片
4. **信任→购买**：促销+BNPL+免运费降低决策门槛
