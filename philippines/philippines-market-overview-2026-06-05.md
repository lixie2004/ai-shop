# 菲律宾跨境电商宏观环境分析报告

**报告日期**：2026年6月5日  
**目标市场**：Philippines（菲律宾）  
**分析维度**：1) 电商市场规模（GMV）及增速 2) 平台格局 3) 消费者画像 4) 跨境进口规模 5) 关键趋势  
**数据来源**：Google-Temasek-Bain e-Conomy SEA 2024-2025，Mordor Intelligence，Anchanto，Bangko Sentral ng Pilipinas，Ken Research，DataReportal

---

## 一、电商市场规模（GMV）及增速

| 指标 | 数据 | 备注 |
|------|------|------|
| **2025年电商GMV** | **$16.75-20.05亿美元** | 根据不同机构数据，估值区间 $16.75B (Global Information) - $20.05B (Mordor Intelligence) |
| **2026年预测GMV** | **$22-25亿美元** | CAGR 10-13%+，预计突破 $22B |
| **2030年预测GMV** | **$33.65-37.95亿美元** | CAGR 10-13.6%，成长稳健 |
| **CAGR（2025-2029/2031）** | **10.05% - 13.78%** | 东南亚最快增长电商市场之一 |
| **数字经济占GDP比重** | **9.4% (2022)** | 快速增长中，政府目标2025年占比15%+ |

**市场增长驱动力**：
- 互联网渗率从2012年的27%提升至2025年的83.8%
- 平均每日在线时长达8小时52分钟（全球第二高）
- 移动支付领先东南亚
- 8-25岁年轻人口占比高（中位年龄26岁）

## 二、平台格局（Shopee/Lazada/TikTok Shop）

| 平台 | 市场份额 | 主要特性 | 2025-2026趋势 |
|------|---------|---------|-----------------|
| **Shopee** | ~38% | 海淘+本地市场双支柱，350,000+本地卖家 | 继续领跑，推出Shopee Xpress本地物流 |
| **Lazada** | ~23% | Alibaba旗下，聚焦中高端市场 | 持续� losses 市场份额但保持利润率 |
| **TikTok Shop** | ~20% | 社交电商模式，直播带货核心 | **冠军种草期**，live selling 74%企业采用 |
| **Zalora** | ~5% | 时尚电商，品牌店铺为主 | 专业时尚领域份额稳定 |
| **Facebook Marketplace** | ~8% | C2C模式，乡村/偏远地区活跃 | 社交+小微经济协同 |
| **Amazon/阿里巴巴国际站** | ~6% | 跨境电商入口 | 跨境进口商品主要来源之一 |

**平台关键数据**：
- **TikTok Shop 4,000+订单**/单月（部分品类已超过Shopee+Lazada总和）
- Shopee+Lazada仍是消费者首选比较价格的两大阵地
- live selling（直播销售）占74%企业新客户获取策略
- marketplace-centric（以平台为中心）消费习惯仍然根深蒂固

## 三、消费者画像（年龄、收入、购物习惯）

### 3.1 人口统计

| 指标 | 数据 |
|------|------|
| **总人口** | **11600万**（2025） |
| **年中年龄** | **26岁**（全东南亚最年轻） |
| **互联网用户** | **9750万**（83.8%渗率） |
| **网上购物用户** | **5340万**（2025，约占全人口50%） |
| **移动钱包用户** | **8900万+**（90%+渗率，GCash 9400万注册） |

### 3.2 年龄结构

| 年龄段 | 占比 | 购物特征 |
|-------|------|----------|
| **18-25岁** | 28% | 社交电商（TikTok Shop）偏好最高 |
| **26-35岁** | 32% | 多平台购物，价格比较习惯强 |
| **36-45岁** | 22% | 以Shopee/Lazada为主，信任本地卖家 |
| **46-55岁** | 12% | 仍以COD+Facebook Marketplace为主 |
| **55+岁** | 6% | 购物渗率低，但增速最快 |

### 3.3 收入分布（购买能力）

| 收入层级 | 人均月收入 | 购物特征 |
|--------|-----------|----------|
| **低收入** (₱5,000-15,000 / $90-270) | 40% | 以价格为主，偏好团购+直播秒杀 |
| **中等收入** (₱15,000-35,000 / $270-630) | 38% | 多品类消费，逐利润比较 |
| **高收入** (₱35,000+ / $630+) | 22% | 品牌导向，偏好Zalora/Amazon |

### 3.4 购物习惯特征

| 习惯维度 | 数据 |
|---------|------|
| **价格比较行为** | 82%消费者跨平台比价 |
| **直播购物接受度** | 74%企业+65%消费者参与 |
| **支付方式首选** | GCash/Maya（55%） > COD（30%） > 银行转账（15%） |
| **平均客单价** | ₱850-1,200 ($15-22) |
| **购物频次** | 月均2.8次（城市）/ 1.5次（农村） |
| **跨境购物率** | 50%消费者曾购买海外商品（主要来自中国、美国、韩国） |

## 四、跨境进口规模

| 指标 | 数据 |
|------|------|
| **2025年跨境进口电商市场规模** | **$6.2亿美元** |
| **2026-2030年CAGR** | **15-20%** |
| **主要来源国** | China (60%) > 美国 (22%) > 韩国 (12%) > 日本 (6%) |
| **主品类** | 电子产品 > 美妆 > 服装 > 家居小件 > 健康保健 |
| **年增购量** | 跨境商品复购率35%（高于本地商品25%） |

**跨境进口核心驱动**：
- "海外链接"价格消费者信心
- 进口商品被认为品质更好
- 社交平台种草（TikTok Shop直播）推动跨境采购
- 2025年新政策（JAO 001-2025）推动跨境商品监管规范化

## 五、关键趋势分析

### 5.1 数字支付趋势

| 支付方式 | 渗率 | 特点 |
|---------|------|------|
| **GCash** | 9400万注册用户（55%市场份额） | 菲律宾数字钱包龙头，支持Shopee/Lazada/TikTok Shop |
| **Maya**（原PayMaya） | 3000万注册用户 | 银行+钱包混合模式 |
| **ShopeePay** | 2500万注册用户 | 平台内聚/外支付场景 |
| **COD (Cash on Delivery)** | 30%交易额 | 仍是农村/偏远地区主流 |

**政府政策**：
- Digital Payments Transformation Roadmap 2025
- 目标：50%零售交易数字化，70%成年人拥有正式金融账户
- 2024年已达成57.4%交易量/59%交易额数字化（领先目标）

### 5.2 物流趋势

| 挑战/趋势 | 数据/描述 |
|-----------|----------|
| **配送成本占比** | 20-30%订单价值（高于邻国） |
| **岛屿物流复杂性** | 7641个岛屿，170+殖民地，配送难度极大 |
| **主要配送时间** | 主要城市24-48小时，偏远地区1-2周 |
| **政府基础设施投资** | "Build Better More"计划升级高速公路/港口/机场 |
| **Shopee Xpress** | 本地物流网络建设，覆盖200+城市 |
| **最后一公里效率** | 仍是成本和时间的瓶颈 |

### 5.3 政策趋势

| 政策/法规 | 时间 | 内容 | 对跨境卖家影响 |
|----------|------|------|---------------|
| **Internet Transactions Act (ITA)** | 2023年12月 | 统一电商监管框架，成立E-Commerce Bureau | 入市需合法注册，提供消费者保护 |
| **JAO 001-2025** | 2025年2月生效 | 跨境电商清关规定，数字发票制度 | 跨境卖家需遵循清关流程，提供清单/价目表 |
| **VAT Destination Principle** | 持续执行 | 跨境商品销售征VAT，平台责任制 | 跨境商品售价需含税，平台+商家双责任 |
| **关户认证要求** | 2025年起 | 跨境卖家需与BoC (Bureau of Customs) 备案 | 入市门槛提高，需提前备案 |
| **2026年计划** | 强化监管 | 预计进一步收紧跨境包裹≤$100豁免政策调整 | 低价商品策略需调整 |

### 5.4 社交电商趋势

| 指标 | 数据 |
|------|------|
| **TikTok Shop月订单增长** | 冠军品类直播4,000+单/品牌 |
| **live selling市场占比** | 社交电商25%交易额 |
| **企业采取live selling策略** | 74%企业确认有效 |
| **消费者参与率** | 65%消费者观看并购买 |
| **KOL/直播主营销投入占比** | 60%营销预算（2025） |

### 5.5 消费趋势

| 趋势 | 描述 |
|------|------|
| **omnichannel（全渠道）** | Click & Collect（点击自提）被SM Markets等大型零售采用 |
| **时尚/美妆** | 2025年增速最快，疫情后复苏 |
| **社交购物** | 社交平台购物信任度高于品牌官网 |
| **价格敏感** | 仍然是首要考虑因素 |

## 六、跨境卖家机会与策略

### 6.1 进入机会评估

| 维度 | 评估 | 理由 |
|------|------|------|
| **市场容量** | ⭐⭐⭐⭐ | 年轻人口+快速增长GMV |
| **竞争激烈程度** | 🟡 中等 | Shopee/Lazada主场，TikTok Shop机会新 |
| **政策门槛** | 🟢 逐步规范 | 2025年政策趋于明朗，需提前备案 |
| **物流难度** | 🔴 较高 | 岛屿分布+配送成本高 |
| **消费偏好** | ⭐⭐⭐⭐⭐ | 社交电商+跨境商品接受度高 |
| **汇总评分** | **7.8/10** | 高增长+社交电商驱动，政策趋势正向 |

### 6.2 推荐策略

| 策略 | 具体执行 |
|------|----------|
| **平台布局** | TikTok Shop（社交电商） + Shopee（价格比较） + Lazada（品牌溢价） |
| **KOL策略** | KOC（5k-50k粉）矩阵 > 大V单点投放 |
| **物流方案** | 合作Shopee Xpress/LBC/DMCA物流，提前布局各大岛屿仓库 |
| **支付接入** | GCash/Maya API集成，支持COD选项 |
| **产品选择** | 电子产品 > 美妆 > 家居小件 > 健康保健 |
| **政策合规** | 提前向BoC备案，确保JAO 001-2025合规 |
