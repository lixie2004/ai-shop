## SKU #1：智能宠物喂食器

| 项目 | 详情 |
|------|------|
| **风险评级** | 🟡 中 |
| **NTC认证** | ✅ **需要** — WiFi (2.4GHz) 和摄像头模组属于无线通信设备 |
| **ICC认证** | ✅ **需要** — 220V电源适配器需符合菲律宾电气安全标准 |
| **FDA** | ❌ 不需要 — 非食品/非医疗 |
| **认证周期** | 8-12周（NTC 6-8周 + ICC 4-6周，可并行） |
| **认证费用** | $500-1,500（NTC $300-800 + ICC $200-700） |
| **标签要求** | 英文标签，含型号、制造商、进口商、电压规格（220V/60Hz）、NTC/ICC标志 |
| **关税（RCEP）** | 5%（HS Code 8479.89 — 家用电动器具） |
| **VAT** | 12% |
| **数据隐私合规** | 摄像头功能需遵守 Data Privacy Act of 2012，需在App中提供隐私政策 |
| **替代方案** | 如认证周期过长，可先以蓝牙版本（无需NTC）切入，后续升级WiFi版本 |

## SKU #2：草本补充剂（辣木胶囊）

| 项目 | 详情 |
|------|------|
| **风险评级** | 🟡 中 |
| **FDA注册** | ✅ **必须** — 菲律宾FDA将膳食补充剂纳入监管，需获得 LTO（License to Operate）+ CPR（Certificate of Product Registration） |
| **注册主体** | 需有菲律宾本地进口商/分销商作为注册主体（或使用第三方注册代理人） |
| **认证周期** | 12-24周（含产品检测） |
| **认证费用** | $1,000-3,000（含检测费+注册费+代理费） |
| **成分限制** | 辣木（Malunggay）为菲律宾本土植物，无限制；维生素和矿物质含量不得超过FDA上限 |
| **标签要求** | 英文+菲律宾语双语标签：成分表、每份含量、每日推荐摄入量、保质期、批号、制造商信息、FDA注册号 |
| **广告限制** | ❌ 不能宣称治疗/预防疾病（如"治疗糖尿病""预防癌症"） |
| **关税（RCEP）** | 5%（HS Code 2106.90 — 食品补充剂） |
| **VAT** | 12%（基本食品免税，但补充剂不享受免税） |
| **替代策略** | 辣木为菲律宾传统食材，可宣称"传统上使用"而非医疗功效 |

## SKU #3：LED美容面罩

| 项目 | 详情 |
|------|------|
| **风险评级** | 🔴 高 |
| **FDA注册** | ✅ **必须** — LED光疗设备在菲律宾属于 **Class II 医疗器械**，需 FDA 医疗器械注册 |
| **ICC认证** | ✅ **需要** — USB充电设备需符合电气安全标准 |
| **认证周期** | 16-36周（医疗器械注册周期最长） |
| **认证费用** | $2,000-5,000（含临床数据要求+检测费） |
| **电压/电气安全** | USB充电设计（5V）可规避大部分电气安全问题 |
| **波长宣称限制** | 红光（630nm）和近红外（850nm）为常见美容波长，风险较低；蓝光（415nm）宣称"祛痘"可能触发医疗器械认定 |
| **标签要求** | 英文标签，含FDA医疗器械注册号、使用说明、禁忌症、警告语 |
| **广告限制** | ❌ 不能宣称"治疗痤疮""消除皱纹"等医疗效果；只能说"辅助护肤""提亮肤色" |
| **关税（RCEP）** | 7%（HS Code 9018.90 — 其他医疗设备/美容仪器） |
| **VAT** | 12% |
| **替代方案** | 降低风险策略：宣称"温热按摩面罩"而非"光疗"，可能规避医疗器械认定（但需法律确认） |

| 项目 | 详情 |
|------|------|

| **标签要求** | 英文标签：SPF值、PA等级、成分表、使用方法、保质期、制造商、FDA注册号、净重 |
| **广告限制** | SPF宣称需有检测报告支持；不能宣称"100%防晒" |
| **关税（RCEP）** | 5%（HS Code 3304.99 — 其他美容/化妆品） |
| **VAT** | 12% |
| **特殊要求** | 防水宣称（80分钟）需有测试数据支持 |
