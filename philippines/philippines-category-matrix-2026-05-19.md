## 2. 四象限分析

| **家居收纳** | 城市化+小户型趋势，收纳需求爆发 | TikTok Shop 优先 |
| **宠物用品** | 菲律宾宠物拥有率快速上升，蓝海市场 | TikTok Shop 优先 |

| **母婴用品** | 年轻父母线上购买习惯已建立 | Shopee 优先 |

## 3. 平台品类机会差异

### TikTok Shop 优先品类
1. **美妆个护**（28% GMV 占比）— 直播带货转化率最高
2. **健康保健** — 短视频教育+冲动购买
3. **家居收纳** — 视觉展示效果好，易 viral
4. **宠物用品** — 内容驱动，宠物视频天然吸引力
5. **运动健身** — 健身达人带货效果好

### Shopee 优先品类
1. **时尚服饰**（27% GMV 占比）— 大促驱动，高频复购
2. **消费电子**（24% GMV 占比）— 搜索+比价购买
3. **厨房用品** — 家庭采购，客单稳定
4. **母婴用品** — 复购率高，品牌忠诚度强
5. **食品饮料** — 高频刚需

## 5. 关键发现

1. **TikTok Shop 是增量最大的平台**：53% 增速，美妆/健康/家居是核心机会
2. **Shopee 是基本盘**：55% 份额，时尚/电子/厨房是稳定现金流
3. **健康保健是最大蓝海**：后疫情需求+低竞争+高利润
4. **宠物用品被严重低估**：增速 25%+，竞争度仅 4/10
5. **美妆个护需要差异化**：虽然市场最大，但 TikTok Shop 正在蚕食 Shopee 的美妆份额
