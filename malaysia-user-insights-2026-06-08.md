## 5. 购买决策路径

### 典型决策流程

```
发现 → 兴趣 → 比较 → 信任 → 购买 → 收货 → 评价
  │       │       │       │       │       │       │
  ▼       ▼       ▼       ▼       ▼       ▼       ▼
社交    查看    比价    看评价   Voucher  物流    分享
推荐    详情    3-5家   4.5+分  +免运费  速度    体验
(60%)  图片视频  Shopee  好评>50  COD选项  3天内   UGC
```

### 关键转化节点
1. **发现→兴趣**: 主图质量决定点击率 (TikTok视频 > 静态图)
2. **兴趣→比较**: 详情页信息完整性决定停留时间
3. **比较→信任**: 评价数量(>100)和质量(4.5+)决定信任度
4. **信任→购买**: Voucher/免运费是最后推动力
5. **购买→复购**: 收货体验(包装+速度)决定复购率
