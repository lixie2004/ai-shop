## 市场背景概览

| 维度 | 数据 |
|------|------|
| 人口 | **1.16亿**，东南亚第二大人口国 |
| 互联网用户 | **9,580万**，渗透率82.6% |
| 社交媒体用户 | **9,010万**（Facebook 94.9%渗透率，全球最高之一） |
| 日均社媒使用 | **4小时48分钟**（全球第一） |
| 平台格局 | Facebook（94.9%）> TikTok（82.2%）> YouTube > Instagram |
| 电商GMV（2025） | ~**$28B**，CAGR 12-15% |
| 平台偏好 | Shopee ~50% / Lazada ~20% / TikTok Shop ~15% / Temu ~5% |
| 购买决策因素 | 价格(35%) > 口碑(25%) > 物流(15%) > 品牌(15%) > KOL(10%) |
| 支付方式 | **COD占比60-70%**（东南亚最高），GCash快速上升 |
| 社交电商占比 | 25-30%，增速+18% YoY |

**关键市场特征**：

- **"Taglish"文化**：菲律宾消费者在社媒上使用Tagalog+English混合（Taglish），内容必须Taglish才能获得高互动
- **COD占比60-70%**：菲律宾消费者对在线支付信任度低，货到付款是绝对主流，但退货率也高达15-25%
- **Facebook 94.9%渗透率全球第一**：Facebook仍是最大社交平台，但TikTok增速最快（82.2%）
- **家庭结构庞大**：平均家庭规模4.2人（东南亚最高），大家庭消费模式驱动多件/大包装需求
- **OFW（海外菲律宾劳工）经济**：1,200万OFW汇款支撑消费，礼物经济和代购文化盛行
- **TikTok直播转化率高**：菲律宾消费者对直播购物的接受度仅次于印尼，TikTok Shop直播转化率3-5%
