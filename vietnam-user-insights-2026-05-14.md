# 越南用户需求挖掘报告

**报告日期**：2026年5月14日  
**数据来源**：YouNet Media 700 人消费者调研、YouNet ECI 390 万店铺评论数据、Q&Me 消费者行为研究  
**分析框架**：好评关键词 + 差评痛点 → 用户画像 → 需求优先级

---

## 一、用户画像

### 1.1 核心用户群：Gen Z（18–25岁）

| 维度 | 特征 |
|------|------|
| **人口占比** | 约 20% 的越南人口（~2,000万） |
| **线上行为** | 日均社交媒体使用 3.5 小时，TikTok 是最常用 App |
| **购物频率** | 53.4% 每周至少网购 1 次 |
| **平均客单价** | $15–30/单，月均电商消费 $50–80 |
| **品类偏好** | 美妆个护 > 女装时尚 > 食品饮料 > 消费电子 |
| **决策路径** | 社交媒体发现 → 微网红评测 → 平台搜索比较 → 下单 |
| **支付方式** | 电子钱包（MoMo、ZaloPay）快速增长，但仍偏好 COD |
| **核心诉求** | 潮流、性价比、社交认同、即时满足 |

### 1.2 高价值用户群：Millennials（26–40岁）

| 维度 | 特征 |
|------|------|
| **人口占比** | 约 25% 的越南人口（~2,500万） |
| **线上行为** | 日均社交媒体使用 2 小时，Facebook 为主 |
| **购物频率** | 46.6% 每周至少网购 1–3 次 |
| **平均客单价** | $50–125/单，月均电商消费 $150–300 |
| **品类偏好** | 家居生活 > 消费电子 > 母婴用品 > 健康保健品 |
| **决策路径** | 需求驱动 → 平台搜索 → 比较评价 → 下单 |
| **支付方式** | 信用卡/电子钱包占比高，COD 占比下降 |
| **核心诉求** | 品质保障、服务体验、便利性、家庭需求 |

### 1.3 潜力用户群：Gen X（41–55岁）

| 维度 | 特征 |
|------|------|
| **人口占比** | 约 20% 的越南人口（~2,000万） |
| **线上行为** | 日均社交媒体使用 1 小时，Facebook + Zalo |
| **购物频率** | 约 30% 每周网购 1 次 |
| **平均客单价** | $30–60/单 |
| **品类偏好** | 食品生鲜 > 家居用品 > 保健品 |
| **决策路径** | 熟人推荐 → Facebook 搜索 → 平台下单 |
| **支付方式** | COD 偏好最高（>80%） |
| **核心诉求** | 信任、实用、价格、本地化服务 |

### 5.2 选品启示

基于用户需求分析，以下产品方向具有高潜力：

1. **男性护肤套装**：越南男性护肤市场几乎空白，Gen Z 男性美容意识觉醒
2. **大码女装**：越南肥胖率上升，大码服装供给严重不足
3. **宠物智能设备**：自动喂食器、智能猫砂盆等，解决上班族痛点
4. **有机儿童食品**：父母愿为安全付溢价，但供给不足
5. **越南气候适配运动装**：透气速干、防紫外线，适合热带气候
6. **家居智能照明**：LED 灯带、智能灯泡，TikTok 爆款潜力大
