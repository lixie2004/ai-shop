# 越南市场 Top 10 SKU 合规深度分析

**报告日期**: 2026-05-20
**目标市场**: 越南 (Socialist Republic of Vietnam)
**监管框架**: 越南标准与质量体系 (TCVN), CR 标志认证, MIC 认证, MoH 注册
**分析类型**: 进口合规 & 市场准入

## 越南进口合规总览

### 核心监管机构

| 机构 | 缩写 | 管辖范围 |
|------|------|---------|
| 越南标准与质量局 (Bộ Khoa học và Công nghệ) | MOST / STAMEQ | 产品认证 (CR 标志)、质量标准 |
| 信息与通信部 (Bộ Thông tin và Truyền thông) | MIC | 电信/无线电/ICT 设备认证 |
| 卫生部 (Bộ Y tế) | MoH | 食品补充剂、化妆品、药品、医疗器械 |
| 农业农村发展部 (Bộ Nông nghiệp và Phát triển nông thôn) | MARD | 农药、化肥、饲料、兽药 |
| 工商部 (Bộ Công thương) | MOIT | 进口许可、贸易政策 |
| 财政部 - 海关总署 (Tổng cục Hải quan) | MOF / GDVC | 清关、关税、VAT |
| 自然资源与环境部 (Bộ Tài nguyên và Môi trường) | MONRE | 化学品、有害物质 |

### 通用合规要求

#### 1. CR 标志 (Compulsory Registration / Chứng nhận hợp quy)

- **法律依据**: 产品质量法 (Law on Product and Goods Quality No. 05/2007/QH12) + Decree 74/2018/ND-CP
- **覆盖范围**: 对人身安全和环境有潜在风险的目录内产品必须获得 CR 标志
- **认证流程**: 产品测试（TCVN 标准）→ 工厂审核 → 颁发 CR 证书 → 加贴 CR 标志
- **认证机构**: QUACERT (越南质量认证中心)、Bureau Veritas Vietnam、TÜV Rheinland Vietnam、Intertek Vietnam 等指定机构
- **有效期**: 通常 1-3 年，需年度监督审核
- **清单**: 政府定期更新"必须进行合格认证的产品清单"

#### 2. MIC 认证 (Ministry of Information and Communications)

- **法律依据**: 电信法 (Law on Telecommunications) + Circular 10/2020/TT-BTTTT
- **覆盖范围**: 无线电设备、电信终端设备、ICT 设备（WiFi、蓝牙、4G/5G、手机等）
- **认证类型**: SDoC (供应商符合性声明) 或 Type Approval (型号核准)
- **标志**: MIC CR 标志或 DoC 编号

#### 3. 越南语标签 (Decree 43/2017/ND-CP)

进口商品必须附带越南语标签，内容应包括：
- 商品名称
- 负责任组织/个人的名称和地址
- 原产地
- 数量/重量
- 生产日期和有效期（如适用）
- 成分清单（如适用）
- 使用说明和警告
- 保存条件（如适用）

**可选方案**: 可使用附标 (auxiliary label) 覆盖在原始标签上，但越南语必须清晰可见。

#### 4. 进口许可证和海关

- 进口商需持有营业执照和相关经营许可
- 部分商品需取得 MOIT 进口许可证
- 海关清关需提交：商业发票、装箱单、提单、原产地证明(CO)、CR 证书、进口许可证（如需要）、产品测试报告
- 食品/化妆品进口需取得卫检证书 (Health Certificate / Certificate of Free Sale)

#### 5. 税务

- **进口关税 (Import Duty)**: 0–35%，取决 HS 编码和原产地
- **增值税 (VAT/GDP)**: 大部分商品 10% (GTGT)，食品 5%
- **优惠税率**: 东盟自贸区 ATIGA (原产地越南 = 0%), CPTPP, EVFTA 等自贸协定可享受关税减免

## SKU 1: 宠物智能饮水机 (Pet Smart Water Fountain)

| 合规维度 | 详细要求 |
|---------|---------|
| **HS 编码** | 8413.20 / 8413.70 (水泵), 或 8509.80 (家用电动器具) |
| **准入认证** | **CR 标志 (强制)** — 家用电器属于 CR 认证目录，需取得 CR 证书 |
| **产品安全标准** | TCVN 5699-1 (IEC 60335-1 家用电器安全) + TCVN 5699-2 (特定要求) — 电气安全、防水等级(IPX4+)、接地保护 |
| **成分/材料限制** | 食品接触材料需符合 TCVN 6514 (食品安全级塑料) — 水箱材质不得释放双酚A等有害物质 |
| **电磁兼容(EMC)** | 如含电子控制板，需符合 TCVN 7189 (CISPR 14-1) EMC 要求 |
| **标签要求** | 越南语标签: 名称、电压/功率、生产商、原产地、防水等级、安全警告（禁止干烧、定期清洗等） |
| **进口许可** | 无特殊许可证要求，需 CR 证书清关 |
| **能效标签** | 如含水泵，可能需要能效标签 (Decree 21/2011/ND-CP) |
| **风险评级** | 🔴 **中等** — CR 认证周期较长，但认证路径清晰 |
| **认证费用** | CR 认证: $2,000–$4,000 (测试+工厂检查+证书) |
| **认证周期** | 8–12 周 |
| **合规替代方案** | 跨境直邮 (BC/C2C): 个人自用无需 CR，但商业量大仍需 CR |

## SKU 3: 胶原蛋白果冻 (Collagen Jelly)

| 合规维度 | 详细要求 |
|---------|---------|
| **HS 编码** | 2106.90 (食品补充剂/保健食品) |
| **准入认证** | **MoH 保健食品注册 (强制)** — 需取得卫生部 (Vietnam Food Administration - VFA) 的保健食品准入许可 |
| **产品安全标准** | TCVN 要求: TCVN 7604-1/2 (食品添加剂)、TCVN 相关微生物标准。需满足国家技术法规 QCVN (如 QCVN 4-5:2011/BYT 对于功能性食品) |
| **成分/材料限制** | — 胶原蛋白来源需符合食品安全法规<br>— 禁止添加成分清单: 需核对越南卫生部发布的禁止食品添加成分目录<br>— 色素、防腐剂需符合越南食品添加剂白名单 |
| **标签要求** | **严格**: 越南语标签必须包括—产品名称、成分表(按重量降序)、净含量、生产日期/保质期、保存条件、保健功能声明、每日推荐用量、警告语("本品不是药品，不能替代药物治疗")、负责任企业的名称和地址 |
| **进口许可** | — 需取得 VFA (越南食品安全局) 的产品注册号<br>— 进口前需申请卫生检查 (Hygiene Clearance)<br>— 每批货物需取得卫生证书 (Certificate of Health) |
| **健康声明控制** | 保健功能声明需经 VFA 审批，禁止未经批准的疗效声明 |
| **风险评级** | 🔴 **高** — 保健食品注册周期长，要求严格 |
| **认证费用** | $3,000–$8,000 (产品注册、测试、文件翻译公证) |
| **认证周期** | 6–12 个月 |
| **合规替代方案** | **跨境直邮佳选** — BC 模式可作为一般食品(非保健食品)清关，减少注册成本 |
| **特别提示** | 产品名称不能使用"保健"或暗示治疗功能的用词，否则直接触发 VFA 审批 |

## SKU 5: 智能插座 - 越南标准 (Smart Plug - Vietnam Standard)

| 合规维度 | 详细要求 |
|---------|---------|
| **HS 编码** | 8536.69 (插头/插座) 或 8537.10 (电气控制装置) |
| **准入认证** | **CR 标志 (强制)** — 电气产品必须 CR 认证 |
| **产品安全标准** | — TCVN 5699-1 (IEC 60335-1 家用电器安全)<br>— TCVN 6620 (IEC 60884 插头插座安全标准)<br>— 必须适配越南插头标准 (SVT 型: 两扁圆插头; Type A/C/F/G 混合) |
| **EMC 要求** | **MIC 认证 (强制)** — 含 WiFi/BLE/Zigbee 等无线功能的智能插座必须取得 MIC 型号核准 |
| **射频要求** | — WiFi 2.4GHz/5GHz: 需符合 MIC Circular 47/2022/TT-BTTTT<br>— 频率范围须在越南允许范围内<br>— 发射功率须符合越南限制 |
| **能效标签** | 可能需要能效认证 (Decree 21/2011/ND-CP 节能标签) |
| **标签要求** | 越南语标签: 电压(220V 50Hz)、额定电流(6A/10A/16A)、功率、安全警告、CR 标志编号 |
| **进口许可** | 需 CR 证书 + MIC 认证清关 |
| **风险评级** | 🔴 **高** — 需要双重认证 (CR + MIC)，认证路径复杂 |
| **认证费用** | CR 认证: $2,000–$5,000<br>MIC 认证: $2,000–$6,000 (视无线功能复杂程度)<br>能效标签: $500–$1,000 |
| **认证周期** | CR: 8–12 周<br>MIC: 6–10 周<br>总计: 12–20 周 |
| **合规替代方案** | — 使用仅蓝牙 (BLE) 而非 WiFi 可简化 MIC 认证<br>— 或使用不含无线功能的机械定时插座<br>— 跨境直邮仍受 MIC 管辖 |

## SKU 7: 老年人关节保健片 (Joint Health Tablets for Elderly)

| 合规维度 | 详细要求 |
|---------|---------|
| **HS 编码** | 2106.90 (保健食品) 或 3004.90 (药品, 如有治疗声明) |
| **准入认证** | **MoH / VFA 保健食品注册 (强制)** |
| **产品安全标准** | — QCVN 20-1:2015/BYT (保健食品微生物标准)<br>— QCVN 4-5:2011/BYT (功能性食品)<br>— TCVN 相关重金属/农药残留限值 |
| **成分/材料限制** | — 氨糖(Glucosamine)、软骨素(Chondroitin)、MSM 等活性成分需在越南保健食品允许成分清单内<br>— 每日推荐剂量需符合越南规定<br>— 禁止含处方药成分<br>— 禁止疗效声明（药品专属） |
| **标签要求** | **严格**: 越南语标签必须包括：<br>— 产品名称<br>— 全部成分表 (INCI 格式)<br>— 每单位活性成分含量<br>— 每日推荐服用量<br>— 警告语: "本品不是药品"<br>— 保质期和批次号<br>— VFA 注册编号 |
| **进口许可** | — VFA 产品注册证书 (Product Registration Certificate)<br>— 进口前批次测试 (Lot Release Testing)<br>— 每批进口需卫检证明 |
| **风险评级** | 🔴 **高** — 注册周期最长，合规要求最严格 |
| **认证费用** | $5,000–$15,000 (完整注册、药物化学测试、微生物测试、稳定性测试) |
| **认证周期** | 12–18 个月 |
| **合规替代方案** | **强烈建议跨境直邮** — BC/DTC 模式可将产品以"个人自用食品补充剂"名义清关，避免 12-18 个月注册 |
| **特别提示** | — 避免任何疗效声明（如"减轻关节疼痛"改为"支持关节灵活"）<br>— 产品声称"治疗"则自动被归类为药品 (Drug)，需额外药品上市许可 |

## SKU 9: 儿童益生菌软糖 (Children's Probiotic Gummies)

| 合规维度 | 详细要求 |
|---------|---------|
| **HS 编码** | 2106.90 (保健食品) 或 1704.90 (糖果) |
| **准入认证** | **MoH / VFA 注册 (强制)** — 作为保健食品 |
| **产品安全标准** | — QCVN 20-1:2015/BYT (微生物标准)<br>— TCVN 微生物指标: 益生菌活性计数、沙门氏菌/大肠杆菌/b2 等致病菌不得检出<br>— QCVN 4-5:2011/BYT (功能性食品)<br>— TCVN 重金属限值 (铅 ≤ 0.5mg/kg, 砷 ≤ 1mg/kg) |

## 越南跨境直邮合规 (Cross-Border E-commerce / BC Model)

### BC 模式 (跨境电商零售进口)
- 适用平台: Shopee Vietnam、Lazada Vietnam、TikTok Shop Vietnam
- 单笔限额: 价值 ≤ 2,000,000 VND (~$85 USD) 可享受小额免税
- 超过限额部分需缴纳关税 + VAT (10%)
- **保健食品/化妆品 BC 模式**: 不需要 VFA 注册，但产品需符合进口国标准
- **注意**: 2025年起越南加强了对跨境直邮保健食品的监管

### 重点建议

1. **优先跨境直邮的产品**: SKU 3 (胶原蛋白果冻)、SKU 7 (关节保健片)、SKU 9 (儿童益生菌软糖)
   - 这些产品的 VFA 注册周期长，跨境直邮可作为市场测试方式
   - 但需注意: 越南海关可能将多批次同一产品行为视为商业进口，进而要求注册
   - 需在当地仓库备货 (保税仓) 或使用海外直邮 (≤15天送达)

2. **必须走一般贸易的产品**: SKU 1, 5, 8 (电气/智能产品)
   - CR 和 MIC 认证是强制性的，不符海关直接扣留
   - 建议先取得越南标准插头的 CR + MIC 认证，再批量进口

3. **建议越南本地合作伙伴**:
   - 对于认证要求复杂的产品 (SKU 7, 8, 9, 10)，建议与有资质的越南本地进口商合作
   - 本地合作伙伴可持有进口许可证并负责清关
