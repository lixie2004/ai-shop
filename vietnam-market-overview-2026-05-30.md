# 市场分析简报：越南跨境电商宏观环境（2025–2026）

## 摘要（Executive Summary）

越南是东南亚第三大电商市场（仅次于印尼和泰国），2025年四大平台总GMV达约165亿美元，同比增长34.75%。市场正处于**从野蛮增长向质量竞争过渡**的关键阶段：TikTok Shop以"娱乐购物"模式迅猛追赶Shopee，已占据超41%市场份额；2026年新电商法及多项配套法规生效，对跨境卖家的税务合规、产品认证和物流能力提出更高要求。对中国跨境卖家而言，越南既是近水楼台的增量市场，也是合规门槛快速抬升的"压力测试场"。

## 三、平台格局与市场份额

### 2025年全年市场份额（Metric.vn数据）

| 平台 | 2025年市场份额 | 2024年对比 | 增速趋势 |
|------|--------------|-----------|---------|
| **Shopee** | **56.04%** | 仍居首位 | Q3增速放缓至+4% YoY |
| **TikTok Shop** | **41.31%** | 从约30%快速上升 | H1 GMV +148% YoY；Q3 +69% |
| **Lazada** | ~**3%** | 稳定，略降 | Q3销售额-10% |
| **Tiki** | <1% | 持续萎缩 | Q3销售额-80% YoY |

### 平台竞争格局演变

| 时间节点 | Shopee | TikTok Shop | Lazada | Tiki |
|---------|--------|------------|--------|------|
| 2024全年 | ~67% | ~24% | ~7% | ~2% |
| Q1 2025 | ~60% | ~35% | ~3.5% | ~1.5% |
| Q3 2025 | 56% | 41% | 3% | <1% |
| 2025全年 | 56.04% | 41.31% | ~3% | 极小 |

### 关键平台动态

**Shopee（56%）**
- 仍居主导地位但增长放缓，面临TikTok Shop强劲竞争
- SLS物流分区调整，细化末端定价结构
- Mall店数量+11.81%，收入+63.54%
- SIP国际平台：超过35万越南卖家通过其向全球出口

**TikTok Shop（41.31%）**
- 2025年H1 GMV同比激增148%，增长最快的平台
- "Shoppertainment"模式在越南极度成功
- Mall店收入暴增99.14%
- 2026年4月起实施更严格卖家准入规则（打击假货、空包、欺诈）
- 中国美妆品牌在TikTok Shop跨境美妆销售中占比超50%

**Lazada（~3%）**
- 保持稳定但份额持续被蚕食
- 14周年庆强化跨境生态：整合7,000+国际商家（含天猫、Gmarket）
- 优化跨境购物体验（免押金、7天配送、30天退换）
- 差异化定位：跨境购物体验升级

**Tiki（<1%）**
- 销售额暴跌80%，市场份额从约2%进一步萎缩
- 面临生存危机，已基本退出主流竞争

**Temu / Shein**
- Temu曾于2024年被越南要求暂停运营（未完成电商牌照注册）
- 截至2026年5月，Temu和Shein在越南仍面临合规审查压力，未大规模恢复运营

## 四、政策环境

### 2026年核心法规变化

| 法规 | 生效时间 | 核心内容 | 对跨境卖家的影响 |
|------|---------|---------|----------------|
| **新电商法（E-commerce Law 2026）** | 2026年7月 | 建立全面的电商法律框架，扩大平台责任，加强消费者保护 | 跨境卖家面临更严格的合规要求，需注册备案 |
| **第70号令（Decree 70/2025/ND-CP）** | 2025年6月1日 | 强制外国供应商在电商平台开具电子发票 | 跨境卖家必须在平台上启用电子发票功能 |
| **第117号令（Decree 117/2025/ND-CP）** | 2025年7月1日 | 电商平台需代扣代缴个人卖家的增值税和个人所得税 | 平台负责税务扣缴，间接推动卖家正规化 |
| **电子海关集成令** | 2025年4月1日 | 海关系统与电商平台、物流实时数据对接 | 跨境订单需实时申报海关数据，清关更透明但也更严格 |
| **跨境电商发展总体规划（2026–2030）** | 2026年起 | 将跨境电商列为战略支柱，目标是建成可持续数字出口生态系统 | 长期利好，政府将投入基础设施和培训支持 |

### 关税政策

| 项目 | 内容 |
|------|------|
| 免税门槛 | VND 100万（约$40）以下免征进口税/增值税 |
| 超过门槛 | 增值税10% + 进口关税（品类不同） |
| 2025年变化 | 跨境包裹VND 100万以下不再享受免税，削弱低价商品价格优势 |
| 执法趋势 | 日益严格，海关查验力度加大 |

### 产品合规要求

| 品类 | 要求 |
|------|------|
| 化妆品 | 越南FDA注册 |
| 食品 | 进口许可证 + 安全认证 |
| 电子产品 | MIC安全认证（无线设备需型号核准） |
| 一般消费品 | 越南语标签、产地溯源 |

**趋势判断**：2025–2026年是越南电商监管的"合规元年"。政策方向明确：**收紧跨境电商监管、推动交易透明化、保护本土市场**。短期增加合规成本，中长期淘汰不合规卖家、利好正规化运营者。

## 五、跨境卖家机会与威胁

### 机会（Opportunities）

| 机会点 | 说明 | 时间窗口 | 影响程度 |
|--------|------|---------|---------|
| **市场增速领先区域** | 年增34%+，东南亚第三大市场，1.01亿人口红利 | 持续3–5年 | 🔴 高 |
| **社交媒体电商爆发** | TikTok Shop占41%份额，直播带货模式成熟，转化率高 | 当前–2028 | 🔴 高 |
| **中国供应链优势** | 中国占越南进口近40%，地缘近、物流成本比欧美低40–60% | 持续 | 🔴 高 |
| **品类空白机会** | 健康（+47%）、童装（+80%）、文具（+55%）等品类快速增长 | 6–12个月 | 🟡 中 |
| **政府支持跨境电商** | 2026–2030总体规划将跨境电商列为战略支柱 | 中长期 | 🟡 中 |
| **Mall化趋势** | 品牌旗舰店占比极小但贡献超35%销售额，品牌化卖家有先发优势 | 12–18个月 | 🟡 中 |
| **Lazada跨境差异化** | Lazada主动升级跨境体验（7天达、30天退换），为跨境卖家提供差异化渠道 | 6–12个月 | 🟢 低 |

### 威胁（Threats）

| 威胁点 | 说明 | 紧迫程度 |
|--------|------|---------|
| **合规成本快速上升** | 电子发票、代扣代缴、海关实时对接、产品注册等增加运营成本 | 🔴 当前 |
| **免税门槛取消** | VND 100万以下包裹不再免税，削弱中国低价商品竞争力 | 🔴 当前 |
| **平台规则趋严** | TikTok Shop 2026年4月起严查假货、空包；Shopee细化物流分区成本 | 🔴 当前 |
| **价格战白热化** | 中国卖家之间竞争激烈，利润率被压至5–8% | 🟡 6个月内 |
| **消费者信任问题** | 部分消费者对"中国制造"有低价低质刻板印象 | 🟡 中期 |
| **本土品牌崛起** | 越南本土品牌品质提升，价格竞争力增强 | 🟡 12–18个月 |
| **韩日品牌竞争** | 韩国美妆/日本消费品享有更高品质信任，在高端市场挤压中国卖家 | 🟡 持续 |
| **物流成本上涨** | 边境拥堵、集装箱价格涨20–30%，末端配送成本上升 | 🟡 当前 |
| **灰色清关风险加大** | 越南加强产地溯源和标签检查，灰色通道风险上升 | 🟡 当前 |

### 建议行动

| 时间线 | 行动 | 投入 | 预期收益 |
|--------|------|------|---------|
| **现在** | 完成越南电商平台合规注册（电子发票、税务登记、产品认证） | 1人×2周+认证费用 | 确保运营合法性，避免下架风险 |
| **1个月内** | 评估TikTok Shop直播带货策略，签约越南本地KOL | 预算视规模 | 抢占社交电商增长红利 |
| **3个月内** | 建立越南海外仓或与本地物流伙伴合作（目标2–5天达） | 3人×1月+仓储成本 | 提升配送速度、店铺评分和平台曝光 |
| **3个月内** | 产品组合优化：重点布局健康、童装、文具等高增品类 | 产品开发1–2月 | 避开红海品类价格战 |
| **6个月内** | 品牌化升级：越南语包装、本地化设计、品牌旗舰店入驻 | 品牌投入视规模 | Mall店35.7%销售额占比证明品牌化ROI显著 |
| **6个月内** | 多平台布局：Shopee+TikTok Shop为主，Lazada跨境店为补充 | 运营团队扩招 | 分散风险，覆盖不同消费群体 |
| **12个月内** | 评估从灰色清关转向正规进口渠道的可行性 | 法务+供应链评估 | 长期可持续运营，获得现代渠道（MT）准入 |
