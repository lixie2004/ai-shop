## 建模方法论与核心假设

### 成本模型公式

```
总成本 = 采购成本 + 头程物流 + 进口关税 + 增值税(VAT 12%) 
         + 平台佣金(含交易费) + 广告费 + 尾程配送费 
         + 退货损耗 + COD拒收分摊 + 汇率波动预留 
         + 认证/包装/其他

毛利额 = 零售价 - 总成本（不含广告费）
净利润 = 零售价 - 总成本（含广告费）

毛利率 = 毛利额 ÷ 零售价
净利率 = 净利润 ÷ 零售价
```

### 菲律宾跨境进口税制说明

菲律宾对进口商品实行两级税制：

```
计算公式：
1. 关税(Duty) = CIF × 关税税率（MFN或FTA优惠）
2. 增值税(VAT) = (CIF + Duty) × 12%
```

**关键点**：
- 菲律宾**无**类似印尼PPh 22的所得税预提
- 菲律宾VAT为12%（高于印尼11%和越南10%）
- ASEAN-CHINA FTA（ACFTA）提供0%关税，需凭FORM E
- 菲律宾海关要求发票金额如实申报，低报风险高

1. **利润之王**：多功能一体锅（⑨）净利率69.2%，空气炸锅（⑤）61.9%——高客单价小家电的固定成本占比极低，ROI最高。

5. **FORM E是菲律宾的制胜关键**：菲律宾MFN关税5-25%，ACFTA优惠0%。尤其对纺织品（MFN 15-25%）和化妆品（MFN 15%）节税效果显著。
