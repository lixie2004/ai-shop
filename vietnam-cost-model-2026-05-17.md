# Vietnam Top 10 SKU 全成本模型与毛利测算

**报告日期**: 2026-05-17 | **目标市场**: Vietnam  
**平台**: Shopee + TikTok Shop | **币种**: VND / USD  
**汇率基准**: 1 USD = 25,500 VND（2026年5月参考中间价）

---

## 核心假设与参数

### 平台费率（2026年Q2 有效）

| 费用项 | Shopee (Non-Mall) | TikTok Shop (Marketplace) |
|--------|-------------------|---------------------------|
| 基础佣金 | 1–6%（按品类） | 12.5%（统一，自2026年3月起） |
| 交易处理费 | 4.0%（含VAT） | 2.16% |
| 技术服务费 | 5.0%（2026年2月起新增） | — |
| 基础设施费 | — | 约1.5% |
| 订单处理费 | — | 视市场而定 |
| Free Shipping 计划 | ~3%（可选但影响搜索排名） | — |
| Coins Cashback 计划 | ~3%（可选） | — |
| **平台综合抽成（不含广告）** | **~13–18%** | **~16–20%** |

> 注：Shopee Vietnam 2026年5月8日起调整费率，部分品类第三方仓库佣金下调1%。本模型采用调整后费率。

### 税费参数

| 税项 | 税率 | 说明 |
|------|------|------|
| 进口关税（MFN） | 0–25%（按HS编码） | 消费电子类通常5–15%，纺织品20–25% |
| 增值税（VAT） | 8–10% | 临时减免至8%至2026年底；标准10% |
| 企业所得税（CIT） | 20% | 按净利润征收 |
| 预扣税（外国卖家） | 适用情况下 | 通过Shopee/TikTok Shop平台代扣 |

### 物流成本基准

| 运输方式 | 成本 | 适用场景 |
|----------|------|---------|
| 海运拼箱（LCL） | $3–8/CBM（约$0.3–0.8/kg） | 重货/大批量（>100kg） |
| 陆运（中越边境） | $0.5–1.2/kg | 中小批量，时效3–5天 |
| 空运 | $3–7.5/kg | 高价值/紧急补货 |
| 越南尾程配送（Shopee物流） | 15,000–40,000 VND/单 | 按重量和区域 |
| 越南尾程配送（TikTok Shop） | 15,000–35,000 VND/单 | 按重量和区域 |

### 通用成本假设

| 成本项 | 假设值 | 说明 |
|--------|--------|------|
| 退货损耗率 | 3–8% | 按品类；高退货品类（美妆）取高值 |
| 广告费占收入比 | 8–15% | Shopee广告+ TikTok 信息流 |
| 汇率波动缓冲 | 1–2% | VND对CNY/USD波动 |
| 仓储费（本地仓） | 约1–2% 货值/月 | 第三方仓储 |
| 认证/合规分摊 | 视品类而定 | CE/ASTM/化妆品备案 |
| 包装耗材 | 约0.5–1% | 含说明书越南语印刷 |

# 附录：成本模型方法论

## 参数来源

| 参数 | 数据来源 |
|------|---------|
| Shopee Vietnam 2026 Q2费率 | digitalinasia.com 平台费用对比报告 |
| TikTok Shop Vietnam 费率 | TikTok Shop Seller Centre + dataede.com 费率计算器 |

| 越南进口关税 | DHL Vietnam 进口税指南 + Forest Shipping 2026更新 |
| VAT税率 | 越南新VAT法（2025年7月生效，8%临时减征至2026年底） |
| 尾程配送费 | Shopee Vietnam物流费率卡 + GHN标准费率 |
| 采购成本 | 1688/Alibaba 批发价 + 选品报告BOM参考 |
| 汇率基准 | 2026年5月VND/USD中间价25,500 |

## 计算公式

```
总成本 = 采购成本 + 头程物流 + 进口关税 + VAT + 平台佣金 
       + 交易费 + 技术服务费 + 广告费 + 尾程配送 
       + 退货损耗 + 仓储费 + 汇率缓冲 + 包装耗材 
       + 认证分摊 + 促销分摊

综合毛利率 = (零售价 - 总成本) / 零售价 × 100%

净利率 = (零售价 - 总成本 - 企业所得税) / 零售价 × 100%
```

## 风险声明

1. **汇率风险**：VND对USD/CNY年波动3–5%，模型中已预留1%缓冲。建议通过远期锁汇或人民币结算对冲。
2. **平台费率变动风险**：Shopee 2026年新增5%技术服务费，TikTok Shop 2026年佣金从2-3%升至12.5%。预计费率仍有上升趋势。
3. **关税政策风险**：2026年越南调整增值税法，低价值进口商品免税额度取消，化妆品等品类监管趋严。
4. **竞争风险**：模型假设竞争度不变，实际入场后竞品跟风可能推高广告费，需预留15%以上的广告预算弹性。
5. **本模型为理论测算**，实际毛利率受库存周转、季节波动、平台促销力度、退货率波动等多因素影响。建议每季度更新一次成本模型。
