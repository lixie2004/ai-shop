## 一、成本模型框架

每个 SKU 全成本 = 采购成本 + 头程物流 + 平台佣金 + 广告费 + 尾程配送 + 退货损耗 + 支付手续费 + 税费 + 汇率波动预留

**汇率基准**: 1 USD = 56 PHP | 1 CNY = 7.8 PHP

| 物流方式 | 头程(中国→菲律宾) | 时效 | 适用 |

| 海外仓(马尼拉) | ¥25-40/kg(头程)+仓储 | 2-4天 | 大件/高周转 |

### 2.3 菲律宾税费结构

| 税种 | 率 | 说明 |
|-----|---|------|
| 进口关税 | 0-30% | 品类不同（电子较高，护肤较低） |
| VAT | 12% | 增值税 |
| 所得税 | 如有本地实体 | 暂不涉及跨境直发 |

> **免税额**: 单票 < USD 50 (~$2,800 PHP) 通常免征关税

## 六、风险提示

| 风险 | 影响 |
|------|------|
| PHP/CNY 汇率波动 | ±5% 会影响毛利 3-4pp |
| COD 拒收 | 增加退货运费，影响净利 5-10% |
| VAT 政策变化 | 如取消免税额，每件成本 +12% |
| 进口关税上调 | 电子产品影响最大（+15-30%） |
| Shopee 佣金上调 | 如从 4%→6%，净利 -2pp |
