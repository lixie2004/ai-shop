## 二、重点品类深度分析

### 🥇 P1 优先品类

#### 1. 宠物用品（综合得分 4.4）

| 维度 | 分析 |
|------|------|
| **市场背景** | 越南宠物经济爆发，城镇化+单身经济驱动 |
| **增速** | 年增速40-60%，电商渠道增速最快 |
| **竞争格局** | 本土品牌弱，国际品牌价格高，中间地带空白 |
| **跨境优势** | 中国供应链成本优势明显，物流友好（轻小件） |
| **利润率** | 40-60%（宠物零食、玩具、护理用品） |
| **机会SKU** | 宠物零食、猫砂、宠物玩具、洗护用品、宠物服饰 |
| **价格带** | VND 50,000-300,000 ($2-12) |
| **风险** | 食品类需检疫认证，玩具类竞争加剧 |

#### 2. 健康保健（综合得分 4.2）

| 维度 | 分析 |
|------|------|
| **市场背景** | 后疫情健康意识提升，越南中产阶级扩大 |
| **增速** | 年增速35-50%，维生素/补充剂领跑 |
| **竞争格局** | 国际品牌主导高端，中低端市场空白 |
| **跨境优势** | 中国保健品供应链成熟，可差异化定价 |
| **利润率** | 45-65%（膳食补充剂、草本产品） |
| **机会SKU** | 维生素C/D、胶原蛋白、益生菌、鱼油、草本茶 |
| **价格带** | VND 100,000-500,000 ($4-20) |
| **风险** | 需关注越南保健品注册法规，避免夸大宣传 |

#### 3. 美容个护（综合得分 3.8）

| 维度 | 分析 |
|------|------|
| **市场背景** | 2025年线上销售额$3亿，第一大品类 |
| **增速** | +30% YoY，TikTok Shop Q4领先 |
| **竞争格局** | 官方品牌店占64%，但白牌/跨境仍有空间 |
| **跨境优势** | 中国美妆供应链全球领先，KOL营销高效 |
| **利润率** | 35-50%（护肤、彩妆工具、身体护理） |
| **机会SKU** | 面膜、防晒、美妆工具、身体护理、护发 |
| **价格带** | VND 200,000-500,000（主力区间） |
| **风险** | 竞争激烈，需品牌化运营；越南语标签要求 |
