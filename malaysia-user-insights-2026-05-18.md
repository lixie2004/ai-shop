## 一、用户画像

| 维度 | 描述 |
|------|------|
| **年龄** | 25-40岁为主力消费群体 |

| **收入** | 月收入MYR 3,000-8,000（中产阶层） |
| **地域** | 西马（吉隆坡、雪兰莪、槟城）占70%，东马占30% |

## 二、各品类用户需求分析

### 2.1 🐾 宠物用品

**用户画像**: 25-35岁女性为主，城市白领，将宠物视为家庭成员

**需求优先级**:
1. **食品安全** (最重要): 成分透明、无添加剂、有认证
2. **便利性**: 自动喂食、易清洁
3. **耐用性**: 宠物玩具耐咬、耐抓
4. **美观性**: 宠物服饰设计感
5. **智能化**: GPS追踪、健康监测

**好评关键词**: "safe", "durable", "my pet loves it", "good value", "fast delivery"
**差评关键词**: "broke after 1 week", "wrong size", "smell bad", "not as described"

**未满足需求**:
- 宠物健康监测设备（智能项圈）
- 环保/可降解宠物用品
- 宠物保险/医疗服务信息

---

### 2.2 🏠 家居收纳

**用户画像**: 25-45岁，以小户型住户为主，追求空间利用最大化

**需求优先级**:
1. **空间效率** (最重要): 多功能、可折叠、模块化
2. **易安装**: 免工具、免打孔
3. **材质安全**: 无异味、防潮防霉（马来西亚气候）
4. **美观性**: 简约设计、多颜色选择
5. **承重能力**: 稳固耐用

**好评关键词**: "space saving", "easy to install", "sturdy", "looks good", "worth the price"
**差评关键词**: "too flimsy", "doesn't stick", "smaller than expected", "smell"

**未满足需求**:
- 模块化可定制收纳系统
- 智能收纳（带标签/APP管理）
- 环保材质收纳产品

---

### 2.3 💻 3C配件

**用户画像**: 18-35岁，男性略多，科技爱好者，价格敏感

**需求优先级**:
1. **价格** (最重要): 极致性价比
2. **兼容性**: 支持多种手机型号
3. **品质**: 耐用、不伤手机
4. **功能**: 快充、防水、防摔
5. **外观**: 设计感、个性化

**好评关键词**: "cheap but good", "works perfectly", "fast charging", "nice design"
**差评关键词**: "stopped working", "not compatible", "cheap material", "fake"

**未满足需求**:
- 环保材料手机壳
- MagSafe生态配件
- 品牌化/设计师联名款

---

### 2.4 🧴 Halal美妆

**用户画像**: 20-40岁女性，穆斯林为主但非穆斯林也购买，注重成分安全

**需求优先级**:
1. **Halal认证** (最重要): JAKIM认证、成分透明
2. **成分安全**: 无有害化学物质、天然成分
3. **效果**: 美白、保湿、防晒
4. **肤感**: 不油腻、易吸收
5. **价格**: 性价比

**好评关键词**: "halal certified", "gentle on skin", "effective", "not greasy", "good value"
**差评关键词**: "allergic reaction", "no effect", "fake product", "strong smell"

**未满足需求**:
- 中国Halal认证美妆品牌
- 男士Halal护肤
- 天然/有机+Halal双重认证

---

### 2.5 🏋️ 运动户外

**用户画像**: 22-35岁，健康意识强，城市白领

**需求优先级**:
1. **舒适度** (最重要): 透气、吸汗、贴合
2. **质量**: 耐用、不变形
3. **功能**: 速干、防臭、UPF防晒
4. **设计**: 时尚、多颜色
5. **价格**: 性价比

**好评关键词**: "comfortable", "good quality", "quick dry", "perfect for yoga"
**差评关键词**: "too tight", "smells after wash", "color faded", "not as pictured"

**未满足需求**:
- 穆斯林运动服饰系列
- 家用智能健身设备
- 环保运动装备

---

### 2.6 👗 穆斯林时尚

**用户画像**: 18-45岁马来女性，宗教文化驱动，追求时尚与端庄平衡

**需求优先级**:
1. **端庄合规** (最重要): 符合伊斯兰教义
2. **舒适度**: 透气、轻便（热带气候）
3. **时尚感**: 现代设计、多颜色
4. **便利性**: 易穿戴、不易滑落
5. **面料**: 高级面料、不起皱

**好评关键词**: "comfortable", "beautiful", "easy to wear", "good fabric", "modest"
**差评关键词**: "slips off", "too thin", "color faded", "not as pictured"

**未满足需求**:
- 现代穆斯林职场服饰
- 大码穆斯林时尚
- 穆斯林运动/泳装

## 四、购买决策因素

### 4.1 决策权重

| 因素 | 权重 | 说明 |
|------|------|------|
| **价格** | 25% | 最重要因素，马来西亚消费者极度价格敏感 |
| **评价/评分** | 20% | 高评分产品转化率显著更高 |
| **品牌信任** | 15% | 品牌认知度影响购买决策 |
| **物流速度** | 15% | 免费配送和快速送达是关键差异化 |
| **产品质量** | 15% | 质量投诉是最大痛点 |
| **售后服务** | 10% | 退换货政策和客服响应 |

### 4.2 决策路径

```
需求触发 → 平台搜索 → 价格比较(2-3个listing) → 查看评价(重点看差评)
→ 查看卖家评分 → 确认物流方式 → 下单(倾向COD或FPX)
→ 收货验货 → 评价(好则复购/推荐，差则投诉/退货)
```

## 五、用户留存与复购

### 5.1 复购驱动因素

| 因素 | 影响度 | 策略 |
|------|--------|------|
| 产品质量 | 高 | 严格QC，一致性 |
| 价格稳定 | 高 | 避免频繁调价 |
| 客服响应 | 中高 | 24小时内回复 |
| 物流体验 | 中 | 准时+包装完好 |
| 会员权益 | 中 | 积分、优惠券 |

### 5.2 流失原因

| 原因 | 占比 |
|------|------|
| 产品质量差 | 35% |
| 物流体验差 | 25% |
| 价格更高(竞品) | 20% |
| 售后不满意 | 15% |
| 其他 | 5% |
