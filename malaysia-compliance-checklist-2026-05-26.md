# 🇲🇾 马来西亚合规检查清单 — Top 10 SKU 准入分析

**评估日期**：2026-05-26  
**目标市场**：马来西亚（电商渠道 + 跨境准入）  
**核心法规背景**：

- **SIRIM QAS 认证**：强制性产品认证体系，电气/电子产品/玩具必须获 ST COA 证书
- **MCMC Type Approval**：无线/RF 设备（WiFi/蓝牙）强制马来西亚通信与多媒体委员会审批
- **JAKIM Halal 认证**：全球黄金标准，食品/化妆品/保健品/宠物食品均需认证
- **NPRA 产品注册**：保健品/药品需经马来西亚国家药品监管局（QUEST 3+ 系统）全注册
- **NPRA 化妆品通知**：化妆品走简化的通知程序（Notification），非完全注册
- **海关关税制度**：SST（销售服务税）10%（标准）/5%（IT 电子），进口关税按 HS 编码 0–30%
- **ASEAN Cosmetic Directive**：化妆品东盟统一法规
- **2025 海关关税令**（2025年11月生效）：税号重分类和税率调整

---

## 总体摘要

| 维度 | 评估 |
|------|------|
| 电商准入门槛 | 🟢 **中低** — 无需卖家许可证，但 Halal/NPRA 认证前置 |
| 高风险品类 | 保健品（NPRA 注册 9-12 个月）、Halal 认证产品（3-6 个月） |
| 低风险品类 | 竹餐具、环保包装（无强制认证） |

| 数据合规 | 个人数据保护法（PDPA 2010）已生效，但电商卖方要求低于越南 |
| 本地代表要求 | SIRIM/NPRA/JAKIM 均需马来西亚注册公司作为申请主体 |

## SKU 1: Halal 认证宠物食品

### 核心合规路径

| 维度 | 详情 |
|------|------|
| **认证名称** | **JAKIM Halal 认证** — 马来西亚伊斯兰发展局 |
| **主管机构** | JAKIM（BPPIH）— 通过 MyHDL 线上平台申请 |
| **产品分类** | 宠物食品（非人类食用）→ 仍可申请 Halal |
| **认证费用** | 申请费：RM 500–1,000（约 $110–$220 USD）<br>顾问/审核费用：RM 3,000–8,000<br>工厂审核差旅费：另计（境外工厂按实际）<br>**总计约 $800–$2,200 USD** |
| **认证周期** | **3–6 个月** |
| **验证方法** | MyHDL 门户 → 文件审核 → 现场检查（生产设施+原料仓库）→ 核发证书 |
| **续证** | 2 年有效，需年度监督审核 |
| **进口关税** | HS 2309.90（宠物食品）：**0–5%**（从 ASEAN/RCEP 国家可 0%） |
| **SST** | 10%（跨境平台代扣） |
| **标签要求** | 马来语 + 英语双语：产品名、成分表（Halal 原料来源）、净重、保质期、JAKIM Halal 标志及证书编号、进口商信息 |
| **其他要求** | 原料来源证明（Halal 屠宰/非猪肉/非酒精）<br>生产设备无交叉污染<br>产品不可含猪源性明胶/酶/甘油 |
| **海关文件** | 进口许可证（Veterinary Import Permit — DVS）/ 健康证书（Veterinary Health Certificate） |

### 🐾 风险评级：🟡 中

| 风险点 | 说明 |
|--------|------|
| Halal 认证周期 3-6 个月 | 需提前规划上市节奏 |
| 境外工厂审核 | JAKIM 可能要求现场审核中国工厂 — 需额外差旅费和时间 |
| 原料溯源要求严格 | 中国宠物食品的明胶/脂肪来源需 Halal 证明文件 |
| 兽医进口许可证 | 需要马来西亚兽医局（DVS）进口许可 — 额外 2-4 周 |

### 替代方案 / 加速建议

1. **先申请马来西亚本地 Halal 认证**（通过本地进口商作为申请人），再反向认证中国工厂
2. **选择已获 Halal 认证的中国代工厂** — 省去工厂首次审核
3. **走马来西亚本地 OEM 生产** — 完全规避进口许可复杂度，但成本提高约 20-30%
4. **初期只做猫粮（非肉源性/纯植物蛋白配方）** → 免除兽医进口证的肉类审核

## SKU 2: 智能家居套装（马来语版）
*含智能插座×2 + 智能灯泡×2 + 智能遥控器*

### 核心合规路径

| 维度 | 详情 |
|------|------|
| **认证名称** | **SIRIM ST COA**（Certificate of Approval）+ **MCMC Type Approval**（WiFi 射频） |
| **主管机构** | SIRIM QAS International / Suruhanjaya Tenaga（ST）/ MCMC |
| **适用标准** | MS IEC 60335（家用电器安全）、MS IEC 62368（AV/IT 设备）、MCMC TS 标准（射频） |
| **认证费用** | SIRIM ST COA（电气安全）：~$1,500–$3,000 USD<br>MCMC Type Approval：~$800–$1,500 USD<br>**合计 ~$2,300–$4,500 USD** |
| **认证周期** | **8–12 周**（SIRIM）+ **4–6 周**（MCMC）= **并行可压缩至 10 周** |
| **续证** | SIRIM 3 年有效（Scheme B Type Approval）、每年监督审核 |
| **进口关税** | HS 8536.69（插座）+ 8539.50（灯泡）：**15–20%** 基础税率 |
| **SST** | 5%（IT/电子类）— 比标准 10% 低 |
| **标签要求** | ST-SIRIM 标签必须直接贴在 **产品本体**（非包装）<br>马来语+英语：型号、额定电压/电流、SIRIM 编号、进口商 |
| **MEPS 能效要求** | 灯泡需满足 MEPS（最低能效标准）— SEDA 认证 |
| **马来语 APP** | 无强制认证要求，但建议适配马来语 UI 以提升市场接受度 |

### 风险评级：🟡 中

| 风险点 | 说明 |
|--------|------|
| 多 SKU 套装 = 多认证 | 插座+灯泡+遥控器 = 3 个产品分别测试 |
| 标签要求严格 | ST-SIRIM 标签必须贴在产品机身，2022 年起已不能只贴包装 |
| 能效测试 | 灯泡能效不达标则无法上市 |

### 替代方案 / 加速建议

1. **使用已有 SIRIM 认证的供应商成品** → 仅做进口商备案和标签
2. **灯泡和插座分开走批量认证（Scheme A Batch）** → 减少首次投资
3. **优先使用 CB 测试报告** → 被 SIRIM 认可，可节省重复测试约 30-50%

## SKU 3: Halal 认证美妆套装
*粉底液 + 口红 + 面霜*

### 核心合规路径

| 维度 | 详情 |
|------|------|
| **认证名称** | **NPRA 化妆品通知（Notification）** + **JAKIM Halal 认证** |
| **主管机构** | NPRA / DCA（药品管制局）— 化妆品通知<br>JAKIM — Halal 认证 |
| **法规依据** | Control of Drugs and Cosmetics Regulations 1984<br>ASEAN Cosmetic Directive（ACD） |
| **认证费用** | **NPRA 通知**：RM 50/单品（约 $11 USD）**极低**<br>PIF 编制：$500–$1,500/款<br>**JAKIM Halal** 认证：$800–$2,200/品牌 |
| **认证周期** | **NPRA 通知：3–4 周**（最快）<br>**JAKIM Halal：3–6 个月** |
| **NPRA 续证** | 通知有效期 2 年，到期需更新 |
| **成分限制** | 符合 ASEAN Cosmetic Directive 禁限用清单（2026年 1 月更新）<br>粉底液：禁用铅/汞/对羟基苯甲酸酯类<br>口红：禁用 CI 色素限定<br>面霜：美白成分需在许可范围 |
| **进口关税** | HS 3304.99（护肤品）：**20–25%**<br>HS 3304.10（口红）：**20–25%** |
| **SST** | 10% |
| **标签要求** | 马来语+英语：INCI 成分表、净含量、EO/PO、生产商/进口商信息、NPRA 通知编号、JAKIM Halal 标志 |
| **CNH 要求** | 需马来西亚本地 Cosmetic Notification Holder（本地公司） |
| **JAKIM 化妆品要求** | 产品不可含动物源性成分（除非 Halal 屠宰）<br>不可含酒精（乙醇/变性酒精）— 合成乙醇可能豁免，需个案确认 |

### 风险评级：🟡 中

| 风险点 | 说明 |
|--------|------|
| Halal 认证是最大瓶颈 | 3-6 个月，化妆品配方中的酒精/动物甘油是常见被拒原因 |
| 双认证协同复杂 | NPRA 通知快但 Halal 慢 → 整体时间由 Halal 决定 |
| 粉底液/口红色素合规 | ASEAN 2026 年更新了限用色素清单 — 需逐一比对 |

### 替代方案 / 加速建议

1. **先做 NPRA 通知快速上市（非 Halal 版）** → 同时申请 Halal 认证，3 个月后再切换 Halal 版
2. **选择植物基/合成原料配方** → 避免动物源性原料的 Halal 溯源问题
3. **与 SKU #7（Halal 保健品）共用 JAKIM 申请通道** — 同一申请主体可合并审核

### 核心合规路径

| 维度 | 详情 |
|------|------|
| **认证名称** | **SIRIM ST COA**（电气安全）+ **MCMC Type Approval**（WiFi/摄像头 RF） |
| **主管机构** | SIRIM / ST / MCMC |
| **适用标准** | MS IEC 62368（AV/IT安全）、MS IEC 60950（电源）、射频 EMC TS 标准 |
| **认证费用** | SIRIM：~$2,000–$3,500<br>MCMC：~$800–$1,500<br>**合计 ~$2,800–$5,000** |
| **认证周期** | **8–12 周** |
| **进口关税** | HS 8525.80（安防摄像头）：**0–5%**（IT 电子类，通常低关税） |
| **SST** | 5%（IT 电子类） |
| **数据合规** | 云端存储需符合马来西亚 **PDPA 2010**（个人数据保护法）<br>若提供本地服务器存储 → 加分项，但非强制 |
| **标签要求** | ST-SIRIM 标签贴机身、型号、电源参数、进口商信息 |

### 风险评级：🟢 低至中

| 风险点 | 说明 |
|--------|------|
| 摄像头隐私审查 | 马来西亚对监控设备无额外进口限制 — 比中国/越南宽松 |

| 云端数据存储 | 若收集用户面部/行为数据 → PDPA 要求 |

### 替代方案 / 加速建议

1. **使用 CB 测试报告缩短 SIRIM 周期**
2. **考虑本地化云端方案**（马来西亚 AWS/Azure 区域）→ 数据本地化加分

## SKU 5: 环保竹制餐具套装

### 核心合规路径

| 维度 | 详情 |
|------|------|
| **认证名称** | **SIRIM 食品接触材料合规**（一般贸易合规级） |
| **主管机构** | SIRIM QAS / MOH（食品安全部） |
| **适用标准** | MS EN 1186（迁移测试）、MS EN 1528（重金属） |
| **认证费用** | 迁移测试：~$500–$1,000<br>SIRIM 合规声明：~$300–$500<br>**总计 ~$800–$1,500** |
| **认证周期** | **4–6 周** |
| **进口关税** | HS 4419.00（竹餐具）：**20–25%** |
| **SST** | 10% |
| **标签要求** | 马来语+英语：材质（竹）、耐温范围、进口商信息、食品接触声明 |
| **竹材检疫** | 需 **植物检疫证书（Phytosanitary Certificate）**— 出口国签发 |
| **甲醛测试** | 竹材加工胶粘剂 → 需符合甲醛迁移限值 |

### 风险评级：🟢 低

| 风险点 | 说明 |
|--------|------|
| 甲醛迁移超标 | 竹餐具常用脲醛胶 → 甲醛迁移需 < 15mg/kg |
| 植物检疫 | 马来西亚对竹制品偶有检疫抽检 |

### 替代方案 / 加速建议

1. **选择纯机械加工（无胶粘剂）竹材** → 免甲醛测试
2. **考虑与越南 SKU 共享供应链** — 同一产品可同时覆盖两国

## SKU 7: Halal 认证保健品
*胶原蛋白/维生素C/鱼油*

### 🚨 马来西亚最严格品类

| 维度 | 详情 |
|------|------|
| **认证名称** | **NPRA 产品注册（全注册）** + **JAKIM Halal 认证** |
| **主管机构** | NPRA（QUEST 3+ 系统）— 产品全注册<br>JAKIM — Halal 认证 |
| **法规依据** | Control of Drugs and Cosmetics Regulations 1984<br>Drug Registration Guidance Document（DRGD）|
| **认证费用** | **NPRA 注册费**：RM 1,200–5,000/产品（约 $270–$1,100 USD）<br>**GMP 合规**：费用取决于工厂认证状态<br>**稳定性数据**：Climatic Zone IVB（30°C/75%RH）— 6-12 个月<br>**JAKIM Halal**：$800–$2,200<br>**总计约 $2,000–$5,000+ USD** |
| **认证周期** | **NPRA 注册：9–12 个月**（最长周期品类）<br>JAKIM Halal：3–6 个月<br>**合计约 12–18 个月** |
| **成分限制** | 维生素 C 上限 1000mg/日（与 NPRA DRGD 一致）<br>胶原蛋白需来源合规（Halal 屠宰认证的动物皮/鱼鳞）<br>鱼油需来源 Halal 认证鱼种（所有海鱼通常视为 Halal） |
| **稳定测试** | 必须在马来西亚 Climatic Zone IVB 条件下测试（30°C / 75%RH） |
| **进口关税** | HS 2106.90（保健品）：**0–5%**（从 ASEAN/RCEP 可 0%） |
| **SST** | 10% |
| **标签要求** | 马来语+英语：产品名、成分列表（按 INCI/NPRA 格式）、食用方法、警告语、**MAL 注册编号**、JAKIM Halal 标志 |
| **GMP 要求** | 工厂必须持有 WHO/PIC/S GMP 证书 — 中国工厂若在 PIC/S 国家列表内可接受 |
| **本地 PRH** | 必须在马来西亚指定 Product Registration Holder（本地公司） |

### 风险评级：🔴 高

| 风险点 | 说明 |
|--------|------|
| NPRA 注册 12-18 个月 | 市场进入时间最长，需提前规划 |
| GMP 工厂门槛 | 无 WHO/PIC/S GMP → 无法注册 |
| Climatic Zone IVB 稳定性数据 | 中国/欧美数据可能被拒，需在马来西亚或同等气候区域重新测试 |
| Halal + NPRA 双重审查 | 每一步都是单独审批流程，相互依赖 |
| 产品声明受限 | "美白""降脂"等功能性声明需临床试验证据 |
| 年度报告义务 | 注册后需提交不良反应监测报告 |

### 替代方案 / 加速建议

1. **先从维生素 C 单一产品入手** → 首次注册周期稳住后再扩展产品线
2. **使用马来西亚本地工厂 OEM**（已拥有 NPRA 注册 + Halal 认证）→ 周期缩短至 4-6 个月
3. **不申请 Halal 认证作为第一版** → 仅走 NPRA 注册（9 个月），后期升级 Halal
4. **利用 ASEAN 互认**：已在泰国/新加坡注册的保健品可部分复用测试数据

## SKU 8: 环保快递包装套装（B2B）

### 核心合规路径

| 维度 | 详情 |
|------|------|
| **认证名称** | **一般贸易合规** — 无强制产品认证 |
| **主管机构** | 马来西亚海关（RMCD） |
| **可选认证（加分）** | SIRIM 可降解测试（若标"可降解"）→ MS 标准测试 |
| **认证费用** | 基础合规：~$100–$300<br>可降解测试（可选）：~$500–$1,000 |
| **周期** | **1–2 周**（基础），可降解测试 4–6 周（选做） |
| **进口关税** | HS 4819.10（瓦楞纸箱）：**20–25%**<br>HS 3923.29（塑料快递袋）：**20–25%** |
| **SST** | 10%（B2B 可申请 SST 注册后可抵扣进项税） |
| **标签要求** | 马来语+英语：产品描述、规格、进口商信息 |
| **可降解声明** | "可生物降解"→ 需提供 **MS 测试报告** 或 ISO 14855 |

### 风险评级：🟢 低

| 风险点 | 说明 |
|--------|------|
| B2B 回款周期长 | 合规不是问题，现金流管理更重要 |
| 可降解声明合规 | 无测试报告背书即标"可降解"→ 消费者法风险 |

### 替代方案 / 加速建议

1. **分两个版本**：普通版（最快上市）+ 可降解版（测试完成后）

## SKU 10: 有机棉婴儿用品套装
*口水巾×5 + 婴儿毛巾×2 + 围兜×3*

### 核心合规路径

| 维度 | 详情 |
|------|------|
| **认证名称** | **SIRIM 一般消费品安全测试**（可选）<br>**婴儿产品安全标准** — 非强制但市场期望高 |
| **主管机构** | SIRIM QAS（检测机构） |
| **适用标准** | MS ISO 8124（玩具安全，类似 EN 71）<br>纺织品安全：MS 标准 |
| **认证费用** | 纺织品安全测试（甲醛/偶氮/色牢度）：~$300–$800<br>SIRIM 合规证书：~$200–$500<br>**总计 ~$500–$1,300** |
| **认证周期** | **3–5 周** |
| **进口关税** | HS 6302.60（毛巾类）：**20–25%** |
| **SST** | 10% |
| **有机棉声明** | "有机棉"需 **GOTS（全球有机纺织品标准）** 或 **OCS 证书** 背书 |
| **婴儿产品标签** | 马来语+英语：材质（100% 有机棉）、尺寸、洗涤说明、进口商信息<br>婴儿产品要求更严格的 **甲醛和偶氮染料限值** |
| **防火安全** | 婴儿纺织品可能需符合 **MS EN 14878**（儿童睡衣防火）— 口水巾/围兜一般豁免 |

### 风险评级：🟢 低

| 风险点 | 说明 |
|--------|------|
| 有机棉声称 | GOTS/OCS 缺失则不能使用"有机"字样 |
| 婴儿产品信任门槛 | 虽无强制认证，但消费者期望第三方检测背书 |
| 甲醛超标 | 婴儿纺织品甲醛限值更严格（≤20ppm vs 一般 75ppm） |
| 偶氮染料 | 禁用芳香胺偶氮染料 — 需测试证明 |

### 替代方案 / 加速建议

1. **先做国标/欧标（GB/EN）织物测试** → 马来西亚通常接受同等国际报告
2. **不做有机声明** = 100% 纯棉婴儿套装 → 免 GOTS 认证成本
3. **SGS/BV 等第三方检测** — SIRIM 认可的实验室覆盖广
