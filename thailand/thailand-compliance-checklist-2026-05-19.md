## 1. 泰国合规框架概览

### 1.1 核心监管机构

| 机构 | 职责 | 相关品类 |
|------|------|---------|
| **FDA泰国** | 食品/药品/化妆品/医疗器械注册 | 美妆、食品、保健 |
| **TISI** | 泰国工业标准认证 | 电子、电器 |
| **NBTC** | 无线电通信认证 | 无线设备 |
| **泰国海关** | 进口关税/VAT征收 | 所有进口商品 |
| **ETDA** | 数字平台监管 | 所有电商卖家 |
| **消费者保护委员会** | 消费者权益 | 所有商品 |

### 1.2 通用合规要求

| 要求 | 说明 | 适用 |
|------|------|------|
| **VAT注册** | 所有进口商品征收7% VAT | 全部 |
| **进口申报** | 需提供HS编码、原产地证明 | 全部 |
| **泰语标签** | 产品标签必须含泰语 | 全部 |
| **价格含税** | 商品页面必须显示含税价格 | 全部 |
| **7天退货** | 非食品类商品需提供7天无理由退货 | 全部 |
| **PDPA合规** | 用户数据保护 | 全部 |
