4.2 竞品关键发现

  1. 防晒霜: Mistine/Biore 主导但"油腻/含酒精"是高频差评 → 无酒精+不泛白+控油是明确差异化
  2. 收纳: IKEA 选择有限+白牌生锈 → 品牌化+防潮+公寓适配是空白
  3. 宠物食品: 中端天然无谷几乎空白 → SmartHeart含谷物 vs Orijen价高之间的最大价格空白
  4. 保健品: 低糖软糖剂型几乎空白 → BioLife含糖高+Swisse片剂大
  5. 智能家居: Xiaomi 米家APP泰语差 → 泰语APP+Google Home兼容是明确差异化
