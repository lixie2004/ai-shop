## 一、泰国消费者画像

### 1. 核心人群特征

| 维度 | 特征 |
|------|------|
| 年龄 | 25-40 岁为主力消费群体 |
| 性别 | 女性占比 ~60%（美妆/家居/母婴），男性 ~40%（电子/汽车/运动） |
| 地域 | 曼谷+城市圈占 ~65% 电商消费 |
| 收入 | 中产阶级（THB 15,000-50,000/月） |
| 购物习惯 | 移动端为主、COD 偏好、社交驱动 |

### 2. 消费决策路径

```
社交媒体种草 → 搜索比较 → 看评论/评分 → 比价 → 下单（COD）→ 分享
```
