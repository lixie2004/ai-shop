## 成本模型框架

每 SKU 全成本 = 采购成本 + 头程物流 + 平台佣金 + 广告费 + 尾程配送 + 退货损耗 + 仓储费 + 汇率波动 + 税费

| 费用项 | 比例/金额 |
|--------|----------|
| 广告费（ACOS） | 销售额的 15-25% |
| 退货率 | 3-8%（品类差异） |
| 退货成本 | 货值的 30-50%（含运费损失） |
| VAT | 7%（售价>1500฿时） |
