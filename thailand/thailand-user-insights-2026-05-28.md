## 3. 需求优先级清单

### 3.1 跨品类通用需求优先级

| 优先级 | 需求类别 | 具体需求 | 影响分数 |
|--------|---------|---------|---------|

| P0 | **正品保障** | 消费者极度担忧假货，必须提供认证 | 90 |
| P1 | **热带适配** | 产品必须适合泰国湿热气候 | 88 |
| P1 | **泰语服务和标签** | 产品标签、说明书、客服必须泰语 | 85 |

| P2 | **FDA/安全认证** | 美妆/健康品类FDA认证是入场门票 | 80 |
| P2 | **尺码准确性** | 服饰品类退货主因，尺码指南必须完善 | 78 |

| P3 | **环保可持续** | 新兴趋势，次要差异化 | 50 |

### 3.2 品类特定需求优先级

| 品类 | P0 需求 | P1 需求 | P2 需求 |
|------|---------|---------|---------|
| **美妆** | 正品保障（FDA编号） | 热带气候适配（清爽不油腻） | 成分安全（无酒精/无香精） |

## 4. 消费者驱动路径总结

```
触发点    发现        考虑        购买       使用       复购
  │        │           │          │          │          │
  ▼        ▼           ▼          ▼          ▼          ▼
社交      KOL/朋友    价格对比    Shopee/    开箱体验   PromptPay
推荐      推荐        正品验证   Lazada      评价      优惠券
(60%)    (40%)     Pantip测评  大促        分享      新品推送
                                  │          │          │
                              关键决策点   UGC传播   忠诚度计划
```

**核心洞察**：
- **60%的购买决策**由社交推荐触发（TikTok/Pantip/Facebook）
- **价格对比**是考虑阶段的核心行为（80%消费者会跨平台比价）
- **正品验证**是购买阶段的必经环节（在FDA编号、官方面板等验证后才下单）
- **UGC分享**在美妆/时尚品类特别活跃，好评分享是二次销售引擎

## 5. 跨境卖家行动建议

| 维度 | 具体行动 | 优先级 |
|------|---------|--------|
| **选型** | 优先选择: 防晒护肤(清爽型)、宠物用品、家居收纳、功能性零食 | P0 |
| **定价** | 美妆定价 ฿300-599 带竞争最强，服饰定价 ฿100-299 最适合冲击 | P0 |
| **认证** | 美妆/食品必须先获取FDA认证(\$500-1000/SKU, 4-8周) | P0 |
| **本地化** | 所有产品标签/说明书必须泰语，雇佣泰语本地化审核人员 | P1 |
| **物流** | Shopee FBA/海外仓解决3天配送门槛 | P1 |
| **内容** | 与泰国本土KOL/Facebook群组合作，持续产出UGC内容 | P1 |
| **客服** | 设置泰语客服，23:00前必须回复所有咨询（泰国高效时段） | P2 |
