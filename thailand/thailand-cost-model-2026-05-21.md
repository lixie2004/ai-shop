## 方法论与假设

### 成本模型框架

```
终端售价
  ├── 采购成本 (OEM/原材料)
  ├── 头程物流 (中国→泰国)
  ├── 进口关税 (泰国海关)
  ├── 增值税 VAT (7%)
  ├── 平台佣金 (按平台费率)
  ├── 支付手续费 (~1.5%)
  ├── 广告费 (ACOS 估算)
  ├── 尾程配送费
  ├── 退货损耗 (按品类)
  ├── 汇率波动预留 (3%)
  └── 认证费用分摊
  = 净利润
```

### 通用假设

| 参数 | 值 | 说明 |
|------|-----|------|
| **头程物流（海运）** | CNY 3-5/kg | 中国→泰国，15-20天 |
| **头程物流（空运）** | CNY 15-25/kg | 中国→泰国，3-5天 |
| **泰国 VAT** | 7% | 标准税率 |
| **支付手续费** | 1.5% | ShopeePay/LINE Pay/Credit Card |
| **汇率波动预留** | 3% | 覆盖 THB/CNY 波动 |
| **退货损耗** | 按品类 | 美妆3-5%，电子5-8%，食品1-2% |
| **认证费用分摊** | 按周期 | 将认证总费用分摊到首批12个月 |

### 品类关税参考

| 品类 | 泰国进口关税 | 说明 |
|------|-------------|------|
| 化妆品 | 5-10% | 防晒霜/护肤品 |
| 家居用品 | 5-15% | PP/ABS材质收纳 |
| 宠物食品 | 5-10% | 猫粮/宠物零食 |
| 膳食补充剂 | 5-10% | 软糖剂型 |
| 电子产品 | 0-5% | LED灯泡/TWS耳机（东盟优惠） |
| 纺织品 | 10-20% | 运动服饰 |
| 食品 | 10-30% | 零食类 |
