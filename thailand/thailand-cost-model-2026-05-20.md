## 建模方法论与假设

### 成本模型公式

```
总成本 = 采购成本 + 头程物流 + 进口关税 + VAT(7%) 
         + 平台佣金 + 广告费 + 尾程配送费 + 退货损耗 
         + 汇率波动预留 + 认证摊销 + 仓储费 + 支付手续费

毛利额 = 零售价 - 总成本（不含广告费）
净利润 = 零售价 - 总成本（含广告费）

毛利率 = 毛利额 ÷ 零售价
净利率 = 净利润 ÷ 零售价
```

### 关键假设（2026年5月）

| 假设项 | 取值 | 依据 |
|--------|------|------|
| **VAT税率** | **7%** | 泰国增值税标准税率，长期维持7% |
| **进口关税** | 0-30%（按HS Code） | 泰国MFN平均关税约10%，ASEAN-China FTA可降至0% |
| **低价值商品免税** | THB 1,500以下免关税+VAT | 2024年政策，电商小包裹优惠 |
| **ASEAN-China FTA** | 凭FORM E享0%关税 | 必须获取 |
| **海运中国→泰国** | CNY 1.5-4/kg（普货）；CNY 4-8/kg（敏感货） | 林查班港/曼谷港 |
| **THB/CNY汇率** | 1 THB ≈ 0.20 CNY | 2026年5月 |
| **汇率波动预留** | 3% | THB年均波动率3-5% |
| **Shopee TH佣金** | 3.0-6.0%（跨境） | Shopee Thailand跨境卖家费率 |
| **Lazada TH佣金** | 3.0-5.5%（跨境） | Lazada Thailand费率 |
| **TikTok Shop TH佣金** | 2.0-4.0% | TikTok Shop TH 2026费率 |
| **支付手续费** | 2.5%（加权） | 线上支付+COD混合费率 |
| **ACoS（广告支出占比）** | 12-22%（按品类） | 泰国广告成本中等 |
| **尾程配送费** | THB 30-80/单 | J&T Thailand / Flash Express / Kerry |

### 泰国市场关键特征

| 特征 | 说明 | 成本影响 |
|------|------|---------|
| **VAT 7%（全球最低之一）** | 进口时征收，可注册VAT后抵扣 | 税负成本低 |
| **COD占60%+** | 货到付款是主流支付方式 | COD拒收率3-8%需计入退货损耗 |
| **TikTok Shop爆发式增长** | 2024-2026年社交电商增速东南亚第一 | TikTok佣金低但内容投入高 |
| **ASEAN-China FTA** | 中国商品凭FORM E零关税 | 必须执行，否则关税10-30% |
| **低价值商品免税THB 1,500** | CIF≤THB 1,500免关税+VAT | 多数SKU在此阈值以上 |
| **社交媒体深度渗透** | LINE是国民通讯+购物入口 | LINE OA运营是必要成本 |
| **季节性消费明显** | 泼水节(4月)和年末双11/12.12是核心节点 | 库存需提前2个月备货 |
