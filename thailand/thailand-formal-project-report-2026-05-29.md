## 1. 执行摘要

### 核心发现

泰国是**东南亚增长最快**的电商市场，2025年 GMV 达 **$355亿**，同比增长 **51.8%**。本次选品调研扫描 **18个品类**，综合搜索热点、竞品格局、用户需求和成本模型，最终筛选出 **Top 10 产品**，其中 **7个立即启动**，**3个月梯队2个**，**调整梯队1个**。

### 关键数字

| 指标 | 数据 |
|------|------|
| 泰国电商增速 | **+51.8% YoY**（东南亚第一） |
| 网购渗透率 | >50% 人口定期网购 |
| 跨境购物占比 | **30%** |
| 移动端交易占比 | **>80%** |
| Top 3 平台市场份额 | **98.8%** |
| 消费者核心年龄 | 17-36岁（**62%**） |

### 选品结论

| 结论 | 详情 |
|------|------|
| **进入建议** | ✅ **强烈推荐进入** |
| **立即启动 SKU** | 7 个 |
| **3个月启动 SKU** | 2 个 |
| **调整后再启动** | 1 个 |
| **平均毛利率** | **34.2%** |
| **首年预期净利润** | 基准 ¥22,000/月 |

## 2. 市场概览

### 2.1 电商规模与增速

| 年份 | 电商 GMV | 增速 |
|------|---------|------|
| 2024 | $218亿 | — |
| 2025 | **$355亿**（平台电商）/ $234.5亿（协会数据） | **+51.8%** |
| 2029E | $540亿（B2C） | CAGR 10.9% |

### 2.2 平台格局

| 平台 | 市场份额 | 特点 |
|------|---------|------|
| **Shopee** | ~75% | 绝对霸主，移动优先 |
| **Lazada** | ~67% | 品牌+品质定位 |
| **TikTok Shop** | 快速上升 | 社交电商+直播，增长最快 |
| **Temu** | 新兴 | 中国跨境卖家涌入 |
| **LINE Shopping** | 稳定 | 5600万用户，社交购物 |

### 2.3 消费者画像

| 维度 | 数据 |
|------|------|
| 核心年龄 | 17-36岁（62%），千禧+Z世代 |
| 网购人数 | ~4500万 |
| 月均网购频率 | 71% 每月≥2次 |
| 支付方式 | QR码 PromptPay（61.5%）、数字钱包（23%）、银行卡（20%） |
| 跨境购物 | 43% 网购者进行过跨境交易 |
| 日均屏幕时间 | ~8 小时 |

### 2.4 政策环境

| 政策 | 内容 | 对跨境卖家的影响 |
|------|------|----------------|
| < 1500 ฿ 免税 | 进口价值 < 1500 泰铢免税 | ✅ 利好定价在 200-1000 ฿ |
| 数字平台法令 | 境外平台需指定本地联系人 | 需合规 |
| PDPA | 个人数据保护法 | 需用户数据合规 |
| IP 保护 | 商标先到先得 | 建议提前注册 |
