## 五、跨境卖家搜索优化建议

### 5.1 标题关键词策略

1. **泰语优先**：标题和描述必须使用泰语
2. **核心词前置**：将品类核心搜索词放在标题前50字符
3. **促销词**：包含"ส่งฟรี"（包邮）、"ลดราคา"（打折）
4. **认证词**：包含"ของแท้"（正品）

### 5.2 品类优先级

| 优先级 | 品类 | 核心关键词 |
|--------|------|-----------|
| 🔴 高 | 美妆护肤 | เครื่องสำอาง, ครีมกันแดด, ซีรัม |
| 🔴 高 | 家居生活 | ของตกแต่งบ้าน, ไฟ LED, อุปกรณ์ |
| 🔴 高 | 时尚服饰 | เสื้อผ้า, รองเท้า, กระเป๋า |
| 🟡 中 | 母婴用品 | ผ้าอ้อม, ของเล่นเด็ก |
| 🟡 中 | 数码配件 | อุปกรณ์มือถือ, หูฟัง |
| 🟢 低 | 宠物用品 | อาหารสัตว์, ของเล่นสัตว์ |
