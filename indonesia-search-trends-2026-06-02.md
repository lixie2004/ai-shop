## 六、核心洞察与行动建议

### 6.1 三大搜索行为特征

```
特征① "Halal" 是最大信任词
  → 在各品类搜索中，+halal 修饰词直接提升 25-35% 点击率
  → 清真认证不是加分项，是入场券

特征② "cicilan"（分期）是转化加速器
  → BNPL 相关搜索词转化率比非分期高 40-50%
  → 尤其对客单价 >IDR 300K 的产品

特征③ "gratis ongkir"（免邮）是购物车救星
  → 含"免邮"的搜索词转化率比不含高 60%+
  → 印尼消费者对运费极度敏感
```
